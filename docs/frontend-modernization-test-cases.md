# 前端覆盖式重构验收样本

## 用途

本文件定义 Vue 3 + Vite 覆盖式重构的验收样本。后续阶段每次覆盖学生端或管理端前，都应使用这些样本验证题目展示、编辑、保存、答题和交卷行为。

## Renderer 边界样本

这些样本来自当前 `scripts/test-markdown-renderer.js`，后续应迁入 `packages/question-renderer` 单元测试：

| 编号 | 内容形态 | 验收点 |
| --- | --- | --- |
| R01 | `<p>下面的程序用于判断$N$ 是否为偶数。</p>` | HTML 文本节点中的 `$N$` 渲染为 KaTeX |
| R02 | `下面的程序用于判断$N$ 是否为偶数。` | 纯 Markdown 行内公式渲染 |
| R03 | `<p>$$x^2+y^2=z^2$$</p>` | HTML 包裹块级公式渲染为 display math |
| R04 | `<pre><code>if ($N$ % 2 == 0)</code></pre>` | 代码块中的 `$N$` 保持字面量 |
| R05 | `<p><code>$N$</code> 是代码；$X$ 是公式。</p>` | 行内代码跳过公式渲染，外部公式正常 |
| R06 | `<p title="a > $N$">正文 $X$</p>` | 属性里的 `$N$` 不被注入 KaTeX，正文公式正常 |
| R07 | `<p><span>判断$N$ 是否为偶数</span></p>` | 嵌套 HTML 文本节点公式正常 |

## 安全样本

后续 `question-renderer` 和新富文本必须覆盖：

| 编号 | 输入 | 预期 |
| --- | --- | --- |
| X01 | `<script>alert(1)</script>` | 渲染输出中无 script |
| X02 | `<img src=x onerror=alert(1)>` | 事件属性被清理 |
| X03 | `<a href="javascript:alert(1)">x</a>` | 危险 URL 被清理或禁用 |
| X04 | HTML 属性中包含 `>` 和 `$N$` | 不破坏 HTML 结构，不注入公式 HTML |

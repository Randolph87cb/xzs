# 判断题

来源：
- 洛谷有题页：https://ti.luogu.com.cn/problemset/1123
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1569361525473312.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

计算机硬件主要包括运算器、控制器、存储器、输入设备和输出设备。

答案：正确

解析：本题判断为正确。常见的冯·诺依曼结构把计算机硬件分为运算器、控制器、存储器、输入设备和输出设备五大部分。运算器负责计算，控制器负责指挥各部件工作，存储器保存数据和程序，输入、输出设备负责和外部交互。

## 第2题

诞生于 1958 年的 103 机是中国第一台通用数字电子计算机，比 1946 年在美国诞生的第一台通用电子计算机 ENIAC 晚了十多年。

答案：正确

解析：本题判断为正确。103 机诞生于 1958 年，ENIAC 诞生于 1946 年，二者相差 12 年。12 年确实可以说是“十多年”，所以题干表述成立。

## 第3题

在 C++ 语言中，计算结果必须存储在变量中才能输出。

答案：错误

解析：本题判断为错误。C++ 可以直接输出常量或表达式的计算结果，不要求先存入变量。例如 `cout << 1 + 2;` 可以直接输出 `3`，`cout << "Hello";` 也可以直接输出字符串内容。

## 第4题

在 C++ 语言中，标识符的命名不能完全由数字组成，至少有一个字母就可以。

答案：错误

解析：本题判断为错误。C++ 标识符通常由字母、数字和下划线组成，但不能以数字开头。比如 `a1` 可以作为标识符，`1a` 虽然含有字母，却因为以数字开头而不合法，所以“至少有一个字母就可以”这个说法不准确。

## 第5题

10 是一个 int 类型常量。

答案：正确

解析：本题判断为正确。`10` 是没有小数点、没有后缀的整数字面量，在数值能用 `int` 表示时，它的类型就是 `int`。它不同于 `10.0` 这样的浮点常量，也不同于 `'1'` 这样的字符常量。

## 第6题

if 语句可以没有 else 子句。

答案：正确

解析：本题判断为正确。`if` 语句可以只写条件成立时要执行的语句，例如 `if (x > 0) cout << x;`。`else` 只是条件不成立时的分支，需要时才写，不是必须部分。

## 第7题

do ... while 语句的循环体至少会执行一次。

答案：正确

解析：本题判断为正确。`do ... while` 的执行顺序是先执行一次循环体，再判断 `while` 后面的条件。即使条件第一次判断就是假，循环体也已经执行过一次。

## 第8题

如果 a 和 b 为 int 类型的变量，则表达式 `a = b` 可以判断 a 和 b 是否相等。

答案：错误

解析：本题判断为错误。`a = b` 是赋值表达式，作用是把 `b` 的值赋给 `a`，不是比较两者是否相等。判断相等应使用 `a == b`。少写一个等号会改变变量的值，也会让判断含义完全不同。

## 第9题

如果 a 为 int 类型的变量，则表达式 `(a % 4 == 2)` 可以判断 a 的值是否为偶数。

答案：错误

解析：本题判断为错误。`a % 4 == 2` 只能判断 `a` 除以 4 的余数是不是 2，例如 2、6、10 满足；但 4、8、12 也是偶数，却不满足这个条件。判断偶数通常应写成 `a % 2 == 0`。

## 第10题

表达式 `(37 / 4)` 的计算结果为 9，且结果类型为 int。

答案：正确

解析：本题判断为正确。`37` 和 `4` 都是 `int` 类型常量，所以 `37 / 4` 做的是整数除法。整数除法只保留商的整数部分，余数 1 被舍去，因此结果是 `9`，结果类型仍为 `int`。

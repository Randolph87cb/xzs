## 第1题

计算机硬件主要包括运算器、控制器、存储器、输入设备和输出设备。

答案：正确

解析：本题判断为正确。计算机硬件可以分为运算器、控制器、存储器、输入设备和输出设备这五类基本组成部分。运算器负责算术和逻辑运算，控制器负责指挥各部件协调执行指令，存储器保存程序和数据，输入设备把外部信息送入计算机，输出设备把处理结果反馈给外部环境。

## 第2题

诞生于 1958 年的 103 机是中国第一台通用数字电子计算机，比 1946 年在美国诞生的第一台通用电子计算机 ENIAC 晚了十多年。

答案：正确

解析：本题判断为正确。题干给出的两个关键时间是 103 机诞生于 1958 年，ENIAC 诞生于 1946 年。两者相差 $1958 - 1946 = 12$ 年，12 年可以表述为“十多年”。因此题干关于时间先后和相差年数的说法都成立。

## 第3题

在 C++ 语言中，计算结果必须存储在变量中才能输出。

答案：错误

解析：本题判断为错误。C++ 的输出语句可以直接输出常量、变量，也可以直接输出表达式的结果，并不要求先把结果存入变量。例如 `cout << 1 + 2;` 会先计算表达式 `1 + 2`，再输出结果 `3`；`cout << "Hello";` 也可以直接输出字符串常量。

## 第4题

在 C++ 语言中，标识符的命名不能完全由数字组成，至少有一个字母就可以。

答案：错误

解析：本题判断为错误。C++ 标识符通常由字母、数字和下划线组成，并且不能以数字开头，也不能是关键字。题干前半句“不能完全由数字组成”是对的，但“至少有一个字母就可以”不充分。例如 `1a` 含有字母，却以数字开头，仍然不是合法标识符。

## 第5题

10 是一个 int 类型常量。

答案：正确

解析：本题判断为正确。`10` 是没有小数点、没有类型后缀的整数字面量，并且数值很小，可以用 `int` 表示，所以它是 `int` 类型常量。它不同于 `10.0` 这样的浮点常量，也不同于 `'1'` 这种用单引号括起来的字符常量。

## 第6题

if 语句可以没有 else 子句。

答案：正确

解析：本题判断为正确。`if` 语句的基本作用是在条件成立时执行一段语句，`else` 只是在需要处理“条件不成立”的情况时才添加。例如 `if (x > 0) cout << x;` 是合法写法，没有 `else` 也能单独构成条件语句。

## 第7题

do ... while 语句的循环体至少会执行一次。

答案：正确

解析：本题判断为正确。`do ... while` 和 `while` 的关键区别在于判断条件的位置：`do ... while` 先执行循环体，再判断条件是否继续循环。因此即使条件第一次判断就是假，循环体也已经执行过一次，所以它至少执行一次。

## 第8题

如果 a 和 b 为 int 类型的变量，则表达式 `a = b` 可以判断 a 和 b 是否相等。

答案：错误

解析：本题判断为错误。`a = b` 是赋值表达式，含义是把 `b` 的值存入 `a`，它会改变 `a` 的值，不是在比较两个变量是否相等。判断相等应写成 `a == b`。在条件语句中误写成 `a = b` 时，判断的是赋值表达式结果转换成的真假，语义与“比较相等”完全不同。

## 第9题

如果 a 为 int 类型的变量，则表达式 `(a % 4 == 2)` 可以判断 a 的值是否为偶数。

答案：错误

解析：本题判断为错误。`a % 4 == 2` 只能判断 `a` 除以 `4` 的余数是否为 `2`，例如 `2`、`6`、`10` 满足这个条件。偶数还包括 `4`、`8`、`12` 等，它们除以 `4` 的余数是 `0`，并不满足 `a % 4 == 2`。判断整数是否为偶数通常应使用 `a % 2 == 0`。

## 第10题

表达式 `(37 / 4)` 的计算结果为 9，且结果类型为 int。

答案：正确

解析：本题判断为正确。`37` 和 `4` 都是 `int` 类型常量，所以表达式 `37 / 4` 执行的是整数除法。整数除法只保留商的整数部分：`37 = 4 * 9 + 1`，余数 `1` 不会出现在结果中，因此计算结果为 `9`，表达式结果类型仍为 `int`。

## 第1题

以下不属于计算机输出设备的有（ ）。

A. 麦克风
B. 音箱
C. 打印机
D. 显示器

答案：A

解析：本题选 A。输出设备负责把计算机处理后的结果呈现出来。音箱输出声音，打印机输出纸质内容，显示器输出图像和文字；麦克风的作用是把声音输入到计算机中，属于输入设备，不是输出设备。

## 第2题

ChatGPT 是 OpenAI 研发的聊天机器人程序，它能通过理解和学习人类的语言来进行对话，还能根据聊天的上下文进行互动，完成很多工作。请你猜猜看，下面任务中，ChatGPT 不能完成的是（ ）。

A. 改邮件
B. 编剧本
C. 擦地板
D. 写代码

答案：C

解析：本题选 C。ChatGPT 是运行在计算机中的软件，可以根据文字要求改邮件、编剧本、写代码，这些都属于信息处理任务。擦地板需要真实的清洁工具和身体动作，聊天机器人本身不能直接完成。

## 第3题

常量 '3' 的数据类型是（ ）。

A. double
B. float
C. char
D. int

答案：C

解析：本题选 C。`'3'` 外面有一对单引号，表示字符常量，类型是 `char`。它和整数 `3` 不同，也不是带小数点的 `float` 或 `double` 常量。

## 第4题

下列关于 C++ 语言变量的叙述，正确的是（ ）。

A. 变量可以没有定义
B. 对一个没有定义的变量赋值，相当于定义了一个新变量
C. 执行赋值语句后，变量的类型可能会变化
D. 执行赋值语句后，变量的值可能不会变化

答案：D

解析：本题选 D。C++ 中变量使用前需要先定义，所以 A 错；给未定义变量赋值不会自动定义新变量，所以 B 错；变量的类型由定义决定，赋值后类型不会变，所以 C 错。D 是可能发生的，例如变量 `x` 原来就是 5，再执行 `x = 5;`，它的值就没有变化。

## 第5题

以下可以作为 C++ 标识符的是（ ）。

A. `number_of_Chinese_people_in_millions`
B. `360AntiVirus`
C. `Man&Woman`
D. `break`

答案：A

解析：本题选 A。C++ 标识符可以由字母、数字、下划线组成，且不能以数字开头，也不能是关键字。A 只含字母和下划线，合法；B 以数字开头；C 含有 `&`；D 的 `break` 是 C++ 关键字，不能作为普通标识符。

## 第6题

以下哪个不是 C++ 语言的关键字？（ ）

A. double
B. else
C. while
D. endl

答案：D

解析：本题选 D。`double` 是浮点类型关键字，`else` 是条件分支关键字，`while` 是循环关键字。`endl` 常用于 `cout << endl` 表示换行并刷新输出，它是标准库中的名字，不是 C++ 语言关键字。

## 第7题

如果 a、b 和 c 都是 int 类型的变量，下列哪个语句不符合 C++语法？（ ）

A. ` a = (b == c); `
B. ` b = 5.5; `
C. ` c = a + b + c; `
D. ` a + c = b + c; `

答案：D

解析：本题选 D。赋值号左边必须是可以被修改的变量，例如 `a`、`b`、`c`。`a + c` 是一个加法表达式的计算结果，不能放在赋值号左边。A 中比较结果可以赋给 `a`，B 中 `5.5` 可以转换后赋给 `int`，C 是普通的加法赋值语句，语法上都可以通过。

## 第8题

如果用一个 int 类型的变量 a 表达正方形的边长，则下列哪个表达式不能用来计算正方形的面积？（ ）

A. ` a * a `
B. ` 1 * a * a `
C. ` a ^ 2 `
D. ` a * 2 * a / 2 `

答案：C

解析：本题选 C。正方形面积应为边长乘边长，所以 `a * a` 可以计算面积，`1 * a * a` 结果相同，`a * 2 * a / 2` 按从左到右计算也等于 `a * a`。C++ 中 `^` 不是乘方运算符，而是按位异或运算符，因此 `a ^ 2` 不能表示 `a` 的平方。

## 第9题

表达式 `(4 * (11 + 12) / 4)` 的计算结果为（ ）。

A. ` 47 `
B. ` 20 `
C. ` 23 `
D. ` 56 `

答案：C

解析：本题选 C。先算括号内的 `11 + 12 = 23`，表达式变为 `4 * 23 / 4`。乘除同级，从左到右计算：`4 * 23 = 92`，`92 / 4 = 23`，所以结果是 23。

## 第10题

如果 a 为 int 类型的变量，且 a 的值为 6，则执行 `a %= 4;` 之后，a 的值会是（ ）。

A. ` 1 `
B. ` 2 `
C. ` 3 `
D. ` 4 `

答案：B

解析：本题选 B。`a %= 4;` 等价于 `a = a % 4;`，意思是把 `a` 除以 4 的余数再存回 `a`。原来 `a = 6`，`6 % 4 = 2`，执行后 `a` 的值就是 2。

## 第11题

如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断 “a 等于 0 且 b 等于 0”的是（ ）。

A. ` (a == b == 0) `
B. ` !(a || b) `
C. ` (a + b == 0) `
D. ` (a == 0) + (b == 0) `

答案：B

解析：本题选 B。`a || b` 在 `a` 或 `b` 至少有一个非 0 时为真，前面加 `!` 后，只有 `a` 和 `b` 都为 0 时才为真。A 的 `a == b == 0` 会先算 `a == b`，不是数学里的链式比较；C 中 `a + b == 0` 在 `a = 1, b = -1` 时也成立；D 的两个比较结果相加，不能准确表示“两个都等于 0”这个逻辑条件。

## 第12题

如果 a 和 b 为 int 类型的变量，且值分别为 7 和 2，则下列哪个表达式的计算结果不是 3.5？（ ）

A. ` 0.0 + a / b `
B. ` (a + 0.0) / b `
C. ` (0.0 + a) / b `
D. ` a / (0.0 + b) `

答案：A

解析：本题选 A。`a` 和 `b` 都是 `int`，所以 A 中先计算 `a / b`，即 `7 / 2 = 3`，再加 `0.0` 得到 `3.0`，不是 `3.5`。B、C、D 都在除法发生前引入了 `0.0`，使除法按实数计算，结果为 `3.5`。

## 第13题

在下列代码的横线处填写（ ），使得输出是 “20 10”。

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 10, b = 20;
    a = ________; // 在此处填入代码
    b = a + b;
    a = b - a;
    cout << a << " " << b << endl;
    return 0;
}

```

A. ` a + b `
B. ` b `
C. ` a - b `
D. ` b - a `

答案：C

解析：本题选 C。初始 `a = 10, b = 20`。若填 `a - b`，第一句后 `a = -10`；再执行 `b = a + b` 得到 `b = 10`；最后 `a = b - a` 得到 `a = 20`，输出就是 `20 10`。其他选项会让第二步得到的 `b` 不等于 10，无法输出题目要求的结果。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是 “147”。

```cpp
#include <iostream>
using namespace std;
int main() {
    for (int i = 1; i <= 8; i++)
        if (________) // 在此处填入代码
            cout << i;
    return 0;
}
```

A. ` i % 2 == 1 `
B. ` i % 3 == 1 `
C. ` i = i + 3 `
D. ` i + 3 `

答案：B

解析：本题选 B。循环中 `i` 从 1 到 8，满足 `i % 3 == 1` 的数是 1、4、7，所以会依次输出 `147`。A 会输出 1、3、5、7；C 是赋值表达式，会改变 `i` 的值；D 中 `i + 3` 在循环范围内一直非 0，会使每次判断都成立。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    int sum;
    for (int i = 1; i <= 20; i++)
        if (i % 3 == 0 || i % 5 == 0)
            sum += i;
    cout << sum << endl;
    return 0;
}

```

A. ` 63 `
B. ` 98 `
C. ` 113 `
D. 无法确定

答案：D

解析：本题选 D。程序只写了 `int sum;`，没有把 `sum` 初始化为 0。后面执行 `sum += i;` 时，会在一个不确定的初始值上累加，所以最终输出无法确定。若题目写成 `int sum = 0;`，才可以稳定计算 1 到 20 中 3 或 5 的倍数之和。

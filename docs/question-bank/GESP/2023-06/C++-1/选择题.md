## 第1题

以下不属于计算机输出设备的有（ ）。

A. 麦克风
B. 音箱
C. 打印机
D. 显示器

答案：A

解析：本题选 A。输出设备用于把计算机处理结果呈现给用户，麦克风 不属于输出结果的设备；其他选项可以输出声音、图像或纸质结果。

## 第2题

ChatGPT 是 OpenAI 研发的聊天机器人程序，它能通过理解和学习人类的语言来进行对话，还能根据聊天的上下文进行互动，完成很多工作。请你猜猜看，下面任务中，ChatGPT 不能完成的是（ ）。

A. 改邮件
B. 编剧本
C. 擦地板
D. 写代码

答案：C

解析：本题选 C。ChatGPT 是软件程序，能处理文本生成、修改和代码等信息任务，但不能直接完成擦地板这类物理动作。

## 第3题

常量 '3' 的数据类型是（ ）。

A. double
B. float
C. char
D. int

答案：C

解析：本题选 C。C++ 中带小数点的数字字面量默认是 double，单引号括起的是 char，不带小数点的整数字面量通常是 int；char 符合该规则。

## 第5题

以下可以作为 C++ 标识符的是（ ）。

A. `number_of_Chinese_people_in_millions`
B. `360AntiVirus`
C. `Man&Woman`
D. `break`

答案：A

解析：本题选 A。C++ 标识符只能由字母、数字和下划线组成，不能以数字开头，也不能使用关键字；`number_of_Chinese_people_in_millions` 符合题干要求，其他选项违反了这些规则之一。

## 第7题

如果 a、b 和 c 都是 int 类型的变量，下列哪个语句不符合 C++语法？（ ）

A. ` a = (b == c); `
B. ` b = 5.5; `
C. ` c = a + b + c; `
D. ` a + c = b + c; `

答案：D

解析：本题选 D。该选项违反 C++ 语法规则，例如赋值左侧必须是可修改的对象、运算符用法必须合法；其他选项可以按隐式转换或正常表达式规则通过。

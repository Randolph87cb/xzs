## 第1题

以下不属于计算机输出设备的有（ ）。

A. 麦克风
B. 音箱
C. 打印机
D. 显示器

答案：A

解析：本题选 A。输出设备的作用是把计算机处理后的信息呈现给人或外部环境。音箱输出声音，打印机输出纸质文字或图像，显示器输出屏幕画面，都属于输出设备。麦克风的方向相反，它把外部声音采集进计算机，属于输入设备，因此不属于输出设备。

## 第2题

ChatGPT 是 OpenAI 研发的聊天机器人程序，它能通过理解和学习人类的语言来进行对话，还能根据聊天的上下文进行互动，完成很多工作。请你猜猜看，下面任务中，ChatGPT 不能完成的是（ ）。

A. 改邮件
B. 编剧本
C. 擦地板
D. 写代码

答案：C

解析：本题选 C。ChatGPT 是聊天机器人程序，擅长处理和生成文字、代码等信息，所以改邮件、编剧本、写代码都属于它可以通过对话协助完成的任务。擦地板需要清洁工具、运动能力和对真实环境的直接操作，单独的软件聊天机器人不能直接完成这个物理动作。

## 第3题

常量 '3' 的数据类型是（ ）。

A. double
B. float
C. char
D. int

答案：C

解析：本题选 C。在 C++ 中，单引号括起来的一个字符是字符常量，`'3'` 的类型是 `char`，表示字符“三”。它和不加引号的整数常量 `3` 不同；`float`、`double` 通常用于带小数部分的浮点数，例如 `3.0` 或 `3.0f`。

## 第4题

下列关于 C++ 语言变量的叙述，正确的是（ ）。

A. 变量可以没有定义
B. 对一个没有定义的变量赋值，相当于定义了一个新变量
C. 执行赋值语句后，变量的类型可能会变化
D. 执行赋值语句后，变量的值可能不会变化

答案：D

解析：本题选 D。C++ 中变量必须先定义再使用，所以“变量可以没有定义”和“给未定义变量赋值就相当于定义新变量”都不符合语法规则。变量的类型由定义语句决定，例如 `int x;` 定义后，后续赋值不会把 `x` 变成别的类型。赋值后变量的值“可能不会变化”，例如 `x` 原来就是 `5`，再执行 `x = 5;`，存进去的值与原值相同。

## 第5题

以下可以作为 C++ 标识符的是（ ）。

A. `number_of_Chinese_people_in_millions`
B. `360AntiVirus`
C. `Man&Woman`
D. `break`

答案：A

解析：本题选 A。C++ 标识符可以由字母、数字和下划线组成，但不能以数字开头，也不能和关键字相同。`number_of_Chinese_people_in_millions` 虽然比较长，但只含字母和下划线，符合规则。`360AntiVirus` 以数字开头，`Man&Woman` 含有非法符号 `&`，`break` 是循环和分支控制相关的关键字，都不能作为普通标识符。

## 第6题

以下哪个不是 C++ 语言的关键字？（ ）

A. double
B. else
C. while
D. endl

答案：D

解析：本题选 D。关键字是 C++ 语言预先保留、具有固定语法含义的单词，不能再拿来当普通标识符。`double` 用来声明双精度浮点类型，`else` 用在条件分支中，`while` 用在循环中，都是关键字。`endl` 常见于 `cout << endl`，它来自标准库，表示换行并刷新输出流，不是语言关键字。

## 第7题

如果 a、b 和 c 都是 int 类型的变量，下列哪个语句不符合 C++语法？（ ）

A. ` a = (b == c); `
B. ` b = 5.5; `
C. ` c = a + b + c; `
D. ` a + c = b + c; `

答案：D

解析：本题选 D。赋值语句左边必须是可以被修改的变量，通常称为“可修改的左值”。A 中 `(b == c)` 的结果是布尔值，可以转换为 `0` 或 `1` 后赋给 `a`；B 中 `5.5` 可以经过类型转换赋给 `int`，虽然小数部分会丢失，但语法成立；C 是把右侧表达式结果赋给变量 `c`。D 的左边 `a + c` 只是一个加法表达式的临时计算结果，不是变量，不能被赋值。

## 第8题

如果用一个 int 类型的变量 a 表达正方形的边长，则下列哪个表达式不能用来计算正方形的面积？（ ）

A. ` a * a `
B. ` 1 * a * a `
C. ` a ^ 2 `
D. ` a * 2 * a / 2 `

答案：C

解析：本题选 C。正方形面积应为边长乘边长，也就是 $a \times a$。`a * a` 和 `1 * a * a` 都直接等于这个结果；`a * 2 * a / 2` 按乘除同级、从左到右计算，等价于先把 `a * a` 乘以 `2` 再除以 `2`，结果仍为面积。C++ 中 `^` 不是乘方运算符，而是按位异或运算符，所以 `a ^ 2` 不能表示 $a^2$。

## 第9题

表达式 `(4 * (11 + 12) / 4)` 的计算结果为（ ）。

A. ` 47 `
B. ` 20 `
C. ` 23 `
D. ` 56 `

答案：C

解析：本题选 C。表达式中最内层括号优先计算，`11 + 12 = 23`，原式变为 `4 * 23 / 4`。乘法和除法优先级相同，按从左到右计算：先得 `4 * 23 = 92`，再得 `92 / 4 = 23`。因此结果是 `23`，不是只算前一部分得到的 `47`，也不是把运算顺序随意改变得到的其他数。

## 第10题

如果 a 为 int 类型的变量，且 a 的值为 6，则执行 `a %= 4;` 之后，a 的值会是（ ）。

A. ` 1 `
B. ` 2 `
C. ` 3 `
D. ` 4 `

答案：B

解析：本题选 B。`%=` 是取模并赋值的复合赋值运算符，`a %= 4;` 等价于 `a = a % 4;`。原来 `a = 6`，`6` 除以 `4` 的商是 `1`、余数是 `2`，所以 `6 % 4 = 2`。执行语句后，余数 `2` 被重新存入 `a`。

## 第11题

如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断 “a 等于 0 且 b 等于 0”的是（ ）。

A. ` (a == b == 0) `
B. ` !(a || b) `
C. ` (a + b == 0) `
D. ` (a == 0) + (b == 0) `

答案：B

解析：本题选 B。在 C++ 的条件判断中，`0` 表示假，非 `0` 表示真。`a || b` 表示“`a` 非 `0` 或 `b` 非 `0`”，再取反后，只有 `a` 和 `b` 都为 `0` 时 `!(a || b)` 才为真。A 不是数学中的链式比较，会先算 `a == b`，再把结果和 `0` 比较；C 在 `a = 1, b = -1` 时也为真；D 得到的是两个比较结果的和，不是一个严格表示“二者都为 0”的逻辑表达式。

## 第12题

如果 a 和 b 为 int 类型的变量，且值分别为 7 和 2，则下列哪个表达式的计算结果不是 3.5？（ ）

A. ` 0.0 + a / b `
B. ` (a + 0.0) / b `
C. ` (0.0 + a) / b `
D. ` a / (0.0 + b) `

答案：A

解析：本题选 A。`a` 和 `b` 都是 `int`，A 中除法 `a / b` 会先发生，因此按整数除法计算，`7 / 2 = 3`，再加上 `0.0` 得到 `3.0`，不是 `3.5`。B、C、D 都在除法之前让其中一个运算数变成实数：`a + 0.0`、`0.0 + a` 或 `0.0 + b` 都会触发实数除法，所以结果是 `3.5`。

## 第13题

在下列代码的横线处填写（ ），使得输出是 “20 10”。

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 10, b = 20;
    a = ________; // 在此处填入代码
    b = a + b;
    a = b - a;
    cout << a << " " << b << endl;
    return 0;
}

```

A. ` a + b `
B. ` b `
C. ` a - b `
D. ` b - a `

答案：C

解析：本题选 C。初始时 `a = 10, b = 20`。若横线处填 `a - b`，第一句后 `a = 10 - 20 = -10`；接着 `b = a + b = -10 + 20 = 10`；最后 `a = b - a = 10 - (-10) = 20`，所以输出 `20 10`。如果填 `a + b`、`b` 或 `b - a`，第一句后的 `a` 分别为 `30`、`20`、`10`，后续计算都得不到题目要求的 `a = 20, b = 10`。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是 “147”。

```cpp
#include <iostream>
using namespace std;
int main() {
    for (int i = 1; i <= 8; i++)
        if (________) // 在此处填入代码
            cout << i;
    return 0;
}
```

A. ` i % 2 == 1 `
B. ` i % 3 == 1 `
C. ` i = i + 3 `
D. ` i + 3 `

答案：B

解析：本题选 B。循环变量 `i` 依次取 `1` 到 `8`，要输出 `147`，条件应只在 `i = 1, 4, 7` 时成立。这三个数除以 `3` 的余数都为 `1`，所以 `i % 3 == 1` 正好匹配。A 会输出所有奇数 `1357`；C 是赋值表达式 `i = i + 3`，会改变循环变量且表达式值通常非 `0`；D 的 `i + 3` 在这里始终非 `0`，几乎等价于条件一直为真。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    int sum;
    for (int i = 1; i <= 20; i++)
        if (i % 3 == 0 || i % 5 == 0)
            sum += i;
    cout << sum << endl;
    return 0;
}

```

A. ` 63 `
B. ` 98 `
C. ` 113 `
D. 无法确定

答案：D

解析：本题选 D。程序中的 `sum` 是局部变量，只写了 `int sum;`，没有初始化。第一次执行 `sum += i;` 时，等价于在 `sum` 原有的不确定值上加 `i`，因此后续即使累加的 `i` 是固定的，最终输出也无法确定。若改成 `int sum = 0;`，才会稳定求出 `1` 到 `20` 中能被 `3` 或 `5` 整除的数之和；但原程序没有这一步初始化。

## 第1题

高级语言编写的程序需要经过以下（ ）操作，可以生成在计算机上运行的可执行代码。

A. 编辑
B. 保存
C. 调试
D. 编译

答案：D

解析：本题选 D。高级语言源程序不能直接作为机器可执行代码运行，通常需要经过编译，把源代码翻译成目标代码并进一步生成可执行文件。编辑、保存只是编写和保存源文件，调试是查找错误的过程，都不能直接生成可执行代码。

## 第2题

排序算法是稳定的（Stable Sorting），就是指排序算法可以保证，在待排序数据中有两个相等记录的关键字 R 和 S（R 出现在 S 之前），在排序后的列表中 R 也一定在 S 前。下面关于排序稳定性的描述，正确的是（ ）。

A. 冒泡排序是不稳定的。
B. 插入排序是不稳定的。
C. 选择排序是不稳定的。
D. 以上都不正确。

答案：C

解析：本题选 C。稳定排序要求相等关键字在排序后的相对先后顺序不变。冒泡排序只交换相邻逆序元素，相等元素不会被交换；插入排序把当前元素插入到前面有序区间时，不需要越过相等元素，因此二者可以保持稳定。选择排序每轮把未排序区间的最小元素换到前面，这次交换可能让相等关键字互相越过，所以说选择排序不稳定是正确的。

## 第3题

下列关于 C++ 语言中指针的叙述，不正确的是（ ）。

A. 指针变量中存储的是内存地址。
B. 定义指针变量时必须指定其指向的类型。
C. 指针变量只能指向基本类型变量，不能指向指针变量。
D. 指针变量指向的内存地址不一定能够合法访问。

答案：C

解析：本题选 C。指针变量中保存的是地址，定义时要写明它指向的数据类型，例如 `int *p` 指向 `int`。指针也可以指向另一个指针，例如 `int **q` 可以指向 `int *` 类型的变量，所以 C 的说法错误。某个地址是否能合法访问还取决于它是否指向有效对象，因此 D 是正确描述。

## 第4题

下列关于 C++ 语言中数组的叙述，不正确的是（ ）。

A. 一维数组在内存中一定是连续存放的。
B. 二维数组是一维数组的一维数组。
C. 二维数组中的每一个一维数组在内存中都是连续存放的。
D. 二维数组在内存中可以不是连续存放的。

答案：D

解析：本题选 D。C++ 中数组元素在内存中连续存放，一维数组如此，二维数组也按“数组的数组”连续存放。对 `int a[3][4]` 来说，每一行是一个连续的一维数组，三行整体也连续排列。因此说二维数组在内存中可以不连续是错误的。

## 第5题

下列关于 C++ 语言中函数的叙述，正确的是（ ）。

A. 函数必须有名字。
B. 函数必须有参数。
C. 函数必须有返回值。
D. 函数定义必须写在函数调用前。

答案：A

解析：本题选 A。普通函数定义需要函数名，例如 `int f()` 中的 `f`。函数可以没有参数，如 `void g()`；也可以没有返回值，此时返回类型写 `void`。函数调用前不一定要出现完整函数定义，只要前面已有函数声明，函数定义可以写在后面。

## 第6题

下列关于 C++ 语言中变量的叙述，正确的是（ ）。

A. 变量定义后可以一直使用。
B. 两个变量的变量名不能是相同的。
C. 两个变量的变量名可以相同，但它们的类型必须是不同的。
D. 两个变量的变量名可以相同，但它们的作用域必须是不同的。

答案：D

解析：本题选 D。变量只能在自己的作用域内使用，离开作用域后就不能再访问，所以 A 错。同一作用域内不能定义两个同名变量，哪怕类型不同也不行，所以 B、C 的说法都不准确。不同作用域中可以出现同名变量，例如函数内的局部变量和另一个函数内的局部变量同名。

## 第7题

一个二维数组定义为 `double array[3][10];`，则这个二维数组占用内存的大小为（ ）。

A. 30
B. 60
C. 120
D. 240

答案：D

解析：本题选 D。`double array[3][10]` 一共有 $3 \times 10 = 30$ 个 `double` 元素。通常一个 `double` 占 8 字节，因此总大小是 $30 \times 8 = 240$ 字节。

## 第8题

一个变量定义为 `int *p = nullptr;`，则下列说法正确的是（ ）。

A. 该指针变量的类型为 int。
B. 该指针变量指向的类型为 int。
C. 该指针变量指向的内存地址是随机的。
D. 访问该指针变量指向的内存会出现编译错误。

答案：B

解析：本题选 B。`int *p` 表示变量 `p` 本身是指针类型，它指向的对象类型是 `int`，所以 A 错、B 对。`nullptr` 表示空指针，不是随机地址。对空指针解引用属于错误的运行时行为，通常不是“编译错误”，所以 D 也不对。

## 第9题

一个二维数组定义为 `int array[5][3];`，则 array[1][2] 和 array[2][1] 在内存中的位置相差多少字节？（ ）

A. 2 字节。
B. 4 字节。
C. 8 字节。
D. 无法确定。

答案：C

解析：本题选 C。二维数组 `int array[5][3]` 按行连续存放，`array[i][j]` 前面有 $i \times 3 + j$ 个 `int`。`array[1][2]` 的下标位置是 $1 \times 3 + 2 = 5$，`array[2][1]` 的下标位置是 $2 \times 3 + 1 = 7$，相差 2 个 `int`，即 $2 \times 4 = 8$ 字节。

## 第10题

如果 a 为 int 类型的变量，且 a 的值为 $6$，则执行 `a &= 3;` 之后，a 的值会是（ ）。

A. 3
B. 9
C. 2
D. 7

答案：C

解析：本题选 C。`&=` 是按位与后赋值，`a &= 3` 等价于 `a = a & 3`。$6$ 的二进制可看作 `110`，$3$ 的二进制是 `011`，按位与得到 `010`，也就是十进制 $2$。

## 第11题

一个数组定义为 `int a[5] = {1, 2, 3, 4, 5};`，一个指针定义为 `int *p = &a[2];`，则执行 `a[1] = *p;` 后，数组 a 中的值会变为（ ）。

A. {1, 3, 3, 4, 5}
B. {2, 2, 3, 4, 5}
C. {1, 2, 2, 4, 5}
D. {1, 2, 3, 4, 5}

答案：A

解析：本题选 A。数组下标从 0 开始，`&a[2]` 是第三个元素的地址，因此 `*p` 的值是 `a[2]`，也就是 3。执行 `a[1] = *p;` 后，第二个元素由 2 改成 3，数组变为 `{1, 3, 3, 4, 5}`。

## 第12题

以下哪个函数声明在调用时可以传递二维数组的名字作为参数？（ ）

A. `void BubbleSort(int a[][4]);`
B. `void BubbleSort(int a[3][]);`
C. `void BubbleSort(int a[][]);`
D. `void BubbleSort(int ** a);`

答案：A

解析：本题选 A。二维数组作为函数参数时，第一维长度可以省略，但第二维及之后的长度必须给出，这样编译器才能计算 `a[i][j]` 的位置。因此 `int a[][4]` 合法；`int a[3][]` 和 `int a[][]` 缺少第二维长度，不合法。`int **a` 是指向指针的指针，不能直接接收普通二维数组名作为同类型参数。

## 第13题

在下列代码的横线处填写（ ），可以使得输出是“20 10”。

```cpp
#include <iostream>
using namespace std;
void xchg(________) { // 在此处填入代码
    int t = *x;
    *x = *y;
    *y = t;
}
int main() {
    int a = 10, b = 20;
    xchg(&a, &b);
    cout << a << " " << b << endl;
    return 0;
}

```

A. `int x, int y`
B. `int * x, int * y`
C. `int a, int b`
D. `int & a, int & b`

答案：B

解析：本题选 B。调用语句是 `xchg(&a, &b);`，传入的是 `a` 和 `b` 的地址，所以形参应当是两个 `int *` 指针。函数体中使用 `*x`、`*y` 访问并交换这两个地址上的整数，才能把主函数中的 `a` 和 `b` 改为 `20 10`。A、C 是普通整数参数，不能接收地址并被 `*x` 解引用；D 是引用参数，但调用时传入的不是 `int` 变量本身而是地址。

## 第14题

执行以下 C++ 语言程序后，输出结果是（ ）。  

```cpp
#include <iostream>
using namespace std;
int main() {
    int array[3][3];
    for (int i = 0; i < 3; i++)
        for (int j = 0; j < 3; j++)
            array[i][j] = i * 10 + j;
    int sum;
    for (int i = 0; i < 3; i++)
        sum += array[i][i];
    cout << sum << endl;
    return 0;
}
```

A. 3
B. 30
C. 33
D. 无法确定。

答案：D

解析：本题选 D。数组元素会被赋成 `array[i][j] = i * 10 + j`，主对角线理论上是 0、11、22，和为 33。但变量 `sum` 定义后没有初始化，第一次执行 `sum += array[i][i]` 时会先读取一个不确定的旧值，所以最终输出无法确定。若写成 `int sum = 0;`，才会稳定输出 33。

```cpp
                min = j;
        temp = array[min];

```

A. `array[min] > array[j]`
B. `array[min] > array[i]`

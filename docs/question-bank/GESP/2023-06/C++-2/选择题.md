## 第1题

高级语言编写的程序需要经过以下（ ）操作，可以生成在计算机上运行的可执行代码。

A. 编辑
B. 保存
C. 调试
D. 编译

答案：D

解析：本题选 D。高级语言程序不能直接被计算机当作可执行机器代码运行，通常需要经过编译，把源程序翻译成目标代码或可执行程序。编辑和保存只是写入、保留源文件，调试是查找并修正错误，都不是生成可执行代码的主要翻译步骤。

## 第2题

能够实现下面流程图功能的伪代码是（ ）。

![](https://cdn.luogu.com.cn/upload/image_hosting/04ikzj0t.png)

A. if 条件判断 then 语句块
B. if 条件判断 then 什么也不做 else 语句块
C. while 条件判断 do 语句块
D. while not 条件判断 do 语句块

答案：D

解析：本题选 D。流程图先进行条件判断，条件不满足时执行语句块，然后回到判断处继续检查；条件满足时才离开循环。这等价于“当条件不成立时反复执行语句块”，也就是 `while not 条件判断 do 语句块`。A、B 是一次性的 `if` 分支，C 则是在条件成立时循环，方向相反。

## 第3题

下列关于 C++ 语言的叙述，正确的是（ ）。

A. char 类型变量不能赋值给 int 类型的变量。
B. 两个 int 类型变量相乘，计算结果还是 int 类型。
C. 计算两个 int 类型变量相乘时，如果乘积超出了 int 类型的取值范围，程序会报错崩溃。
D. 计算两个 double 类型变量相除时，如果除数的值为 0.0，程序会报错崩溃。

答案：B

解析：本题选 B。两个 `int` 变量相乘时，表达式结果仍按 `int` 类型计算。A 错在 `char` 可以转换并赋给 `int`；C 错在整数溢出并不会按语法规则保证“报错崩溃”；D 错在浮点数除以 0.0 通常得到无穷大或非数等浮点结果，也不是按题干说法必然报错崩溃。

## 第4题

下列关于 C++ 语言的叙述，不正确的是（ ）。

A. if 语句中的判断条件必须用小括号 ‘(’ 和 ‘)’ 括起来。
B. for 语句中两个 ‘;’ 之间的循环条件可以省略，表示循环继续执行的条件一直满足。
C. 循环体包含多条语句时，可以用缩进消除二义性。
D. 除了“先乘除、后加减”，还有很多运算符优先级。

答案：C

解析：本题选 C。C++ 判断语句和循环语句的控制范围由语法决定，不由缩进决定；循环体有多条语句时应使用 `{}` 组成语句块，单靠缩进不能消除二义性。A 中 `if` 条件需要小括号，B 中 `for` 的循环条件省略表示条件恒为真，D 中 C++ 确实有很多运算符优先级，都是正确说法。

## 第5题

以下哪个是 C++ 语言的关键字？（ ）

A. main
B. max
C. double
D. sqrt

答案：C

解析：本题选 C。`double` 是 C++ 用来表示双精度浮点类型的关键字，不能作为普通标识符使用。`main` 是程序入口函数常用名称但不是关键字，`max` 常作为函数名或变量名，`sqrt` 是数学库函数名，它们都不是 C++ 关键字。

## 第6题

以下哪个不是 C++ 语言的运算符？（  ）

A. >=
B. /=
C. ||
D. <>

答案：D

解析：本题选 D。`>=` 是大于等于运算符，`/=` 是除后赋值的复合赋值运算符，`||` 是逻辑或运算符。`<>` 不是 C++ 中表示“不等于”的运算符，C++ 的不等于是 `!=`，所以 D 符合“不是运算符”的要求。

## 第7题

如果 a 为 int 类型的变量，b 为 char 类型的变量，则下列哪个语句不符合 C++语法？（ ）

A. ` a = a + 1.0; `
B. ` a = (int)(b - '0'); `
C. ` b = (char)(a + '0'); `
D. ` (int)b = a; `

答案：D

解析：本题选 D。赋值号左边必须是可以被修改的变量或对象，`(int)b` 是把 `b` 转成 `int` 后得到的临时结果，不能放在赋值号左侧。A 中 `a + 1.0` 可再转换赋给 `int`，B、C 使用强制类型转换后赋值，语法上都可以成立。

## 第8题

如果用两个 int 类型的变量 a 和 b 分别表达平行四边形的两条边长，用 int 类型的变量 h 表达 a 边对应的高，则下列哪个表达式不能用来计算 b 边对应的高？（ ）

A. ` a / b * (0.0 + h) `
B. ` (0.0 + a * h) / b `
C. ` a * h / (b + 0.0) `
D. `(a + 0.0) * h / b`

答案：A

解析：本题选 A。平行四边形面积可写成 `a * h`，也等于 `b * b边对应的高`，所以目标表达式应等价于 `a * h / b`。A 先计算 `a / b`，因为 `a`、`b` 都是 `int`，会先做整数除法，可能把小数部分截掉，之后再乘 `h` 已经无法恢复正确结果。B、C、D 都在除法前引入了 `0.0`，能按实数运算保留小数。

## 第9题

以下哪个循环语句会无限次执行？（ ）

A. ` for (int a = 0; a; a++); `
B. ` for (bool b = false; b <= true; b++); `
C. ` for (char c = 'A'; c < 'z'; c++); `
D. ` for (double d = 0.0; d < 10.0; d += 0.001); `

答案：B

解析：本题选 B。按原题/早期 C++ 教学语境理解，`false` 可看作 0，`true` 可看作 1，`b` 初值为 `false`，满足 `false <= true`；执行 `b++` 后会变成 `true`，之后 `true <= true` 一直成立，因此形成无限循环。不过在较新的 C++ 标准中，对 `bool` 使用 `++` 已不合法/已移除，实际写程序不应这样写；这里按原题考查语境判断。A 初值 `a = 0` 时条件为假，不会进入循环；C 的字符会从 `'A'` 增加到不小于 `'z'` 后结束；D 中 `d` 不断增加，最终会达到或超过 10.0 后结束。

## 第10题

如果 a 为 char 类型的变量，且 a 的值为 'C'（已知 'C' 的 ASCII 码为 67），则执行 `cout << (a + 2);` 会输出（ ）。

A. E
B. C+2
C. C2
D. 69

答案：D

解析：本题选 D。`a` 是字符 `'C'`，参与 `a + 2` 运算时会先转换为对应的整数编码值。已知 `'C'` 的 ASCII 码为 67，所以 `a + 2` 的值是 69；表达式类型是整数，`cout` 输出整数 `69`，而不是字符 `E`。

## 第11题

如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断 “a 等于 1 且 b 等于 1” 的是（ ）。

A. ` (a == b) && (b == 1) `
B. ` (a && b) `
C. ` (a == b == 1) `
D. ` (a * b == 1) `

答案：A

解析：本题选 A。A 要求 `a == b` 且 `b == 1`，两者同时成立时可以推出 `a` 和 `b` 都等于 1。B 只能判断二者是否都非 0；C 会先算 `a == b` 得到布尔值，再和 1 比较，不是数学中的连续等式；D 在 `a = -1, b = -1` 时也满足乘积为 1，不能准确表达题意。

## 第12题

如果 a 为 char 类型的变量，下列哪个表达式可以正确判断 “a 是数字”？（ ）

A. ` '0' <= a && a <= '9' `
B. ` '1' <= a && a <= '0' `
C. ` '0' <= a <= '9' `
D. ` '1' <= a <= '0' `

答案：A

解析：本题选 A。数字字符在编码中从 `'0'` 到 `'9'` 连续排列，判断 `a` 是数字应同时满足 `a` 不小于 `'0'` 且不大于 `'9'`。B 的区间左右端点反了；C、D 写成了连续比较，C++ 会先计算左边的比较得到布尔值，再继续和字符比较，不能表达“夹在两个字符之间”。

## 第13题

在下列代码的横线处填写（ ），使得输出是 9。

```cpp
#include <iostream>
using namespace std;
int main() {
    char a = '3', b = '6';
    cout << ________; // 在此处填入代码
    return 0;
}
```

A. ` (a + b) `
B. ` (a + b - '0') `
C. ` (char)(a + b) `
D. ` (char)(a + b - '0') `

答案：D

解析：本题选 D。字符 `'3'` 和 `'6'` 的编码值相加不是直接得到数字 9；在 ASCII 中可看作 51 和 54，相加为 105。`a + b - '0'` 等于 57，正好是字符 `'9'` 的编码；再强制转换为 `char` 后，`cout` 按字符输出 `9`。A 会输出整数 105，B 会输出整数 57，C 会把 105 当字符输出。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是 42。

```cpp
#include <iostream>
using namespace std;
int main() {
    int sum = 0;
    for (int i = 1; i <= 20; i++)
        if (________) // 在此处填入代码
            sum += i;
    cout << sum << endl;
    return 0;
}
```

A. ` i % 3 == 0 `
B. ` 20 % i == 0 `
C. ` i <= 8 `
D. ` i >= 18 `

答案：B

解析：本题选 B。程序把满足条件的 `i` 累加到 `sum` 中。若条件是 `20 % i == 0`，被累加的是 20 在 1 到 20 之间的因数：1、2、4、5、10、20，它们的和正好是 42。A 会累加 3 的倍数，C 会累加 1 到 8，D 会累加 18 到 20，结果都不是 42。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    for (char x = 'A'; x <= 'D'; x++)
        if ((x != 'A') + (x == 'C') + (x == 'D') + (x != 'D') == 3)
            cout << x;
    return 0;
}
```


A. A
B. B
C. C
D. D

答案：C

解析：本题选 C。条件中每个比较结果会按 `true` 为 1、`false` 为 0 参与求和。`x = 'A'` 时和为 1，`x = 'B'` 时和为 2，`x = 'C'` 时四项分别为 1、1、0、1，和为 3，会输出 `C`；`x = 'D'` 时和为 2。因此最终只输出 C。

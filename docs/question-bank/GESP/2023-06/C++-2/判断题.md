# 判断题

来源：
- 洛谷有题页：https://ti.luogu.com.cn/problemset/1124
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1569361588387872.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

诞生于 1986 年的中华学习机 CEC-I 入选了 2021 年的 CCF 计算机历史记忆（一类），它的内存只有 64KB。当时的汉字编码字符集 GB2312 中共有 6763 个汉字，假如每个汉字用 2 个字节编码，将整个 GB2312 汉字字符集都放入 CEC-I 的内存，也只占用了不超过 1/5 的内存空间。

答案：错误

解析：本题判断为错误。64KB 内存按 $64 \times 1024$ 字节计算，共 65536 字节；其中的 $1/5$ 约为 13107 字节。GB2312 中 6763 个汉字若每个占 2 字节，需要 $6763 \times 2 = 13526$ 字节，已经超过内存的 $1/5$，所以题干中“不超过 1/5”的说法不成立。

## 第2题

域名是由一串用点分隔的名字来标识互联网上一个计算机或计算机组的名称，CCF 编程能力等级认证官方网站的域名是 gesp.ccf.org.cn，其中顶级域名是 gesp。

答案：错误

解析：本题判断为错误。域名层级从右向左看，`gesp.ccf.org.cn` 最右侧的 `cn` 是顶级域名，`org`、`ccf`、`gesp` 依次是更低层级的域名部分。因此把 `gesp` 说成顶级域名是错误的。

## 第3题

在使用 C++ 语言编写程序时，不能使用 sqrt、abs 等数学函数，包含 <cmath> 或 <math.h> 头文件后就能够使用了。

答案：正确

解析：本题判断为正确。`sqrt`、`abs` 这类数学函数需要先有函数声明，程序才能正确识别调用；在 C++ 中包含 `<cmath>` 或兼容 C 风格的 `<math.h>` 后，就可以使用这些常见数学函数。题干表达的是“包含相应头文件后能够使用”，这一点是正确的。

## 第4题

在 C++ 语言中，标识符中可以有下划线 `_`。同时，`_` 也是 C++ 语言的运算符。

答案：错误

解析：本题判断为错误。C++ 标识符中确实可以出现下划线，例如 `sum_1` 是合法写法；但单独的 `_` 并不是 C++ 运算符。题干把“可用于标识符的字符”和“运算符”混在一起，因此整体说法错误。

## 第5题

如果 a 是 double 类型的变量，而且值为 3.5，则表达式 `a * 10` 的计算结果为 35，且结果类型为 int。

答案：错误

解析：本题判断为错误。`a` 是 `double` 类型，整数常量 `10` 参与 `a * 10` 运算时会转换成 `double` 再计算，结果数值是 35.0，结果类型仍是 `double`，不会因为看起来是整数 35 就变成 `int`。

## 第6题

在 if 语句中，如果条件成立时需要执行多条语句，可以使用大括号 { 和 } 将这些语句括起来。

答案：正确

解析：本题判断为正确。`if` 后如果不加大括号，通常只把紧跟着的一条语句作为条件成立时执行的内容；如果需要执行两条或更多语句，就要用 `{}` 把它们组成一个语句块，避免只有第一条受 `if` 控制。

## 第7题

循环语句的循环体有可能无限制地执行下去。

答案：正确

解析：本题判断为正确。例如 `while (true) {}` 的条件始终为真，循环体中也没有 `break` 或改变条件的语句，就会一直执行下去。实际程序中如果循环变量没有正确更新，也可能形成这样的无限循环。

## 第8题

++ 和 == 都是 C++ 语言的运算符，但 += 不是。

答案：错误

解析：本题判断为错误。`++` 是自增运算符，`==` 是相等比较运算符，`+=` 也是合法的复合赋值运算符，含义是“在原值基础上加上右侧表达式并赋回去”。所以说 `+=` 不是 C++ 运算符是错误的。

## 第9题

如果 a 为 char 类型的变量，且取值为大写字母 'F'，则执行语句 `a = a+1;` 后，a 的值会变为大写字母 'G'。

答案：正确

解析：本题判断为正确。题目考查字符参与整数运算时使用编码值计算这一规则；大写英文字母的编码按字母顺序连续，因此 `'F' + 1` 得到的编码对应下一个大写字母 `'G'`。表达式 `a + 1` 先用字符编码值做加法，再把结果赋回 `char` 变量 `a`，所以 `a` 会变为 `'G'`。

## 第10题

表达式 `sqrt(9.0)` 的计算结果为 3，且结果类型为 int。

答案：错误

解析：本题判断为错误。`sqrt(9.0)` 的数值结果是 3.0，但 `sqrt` 处理浮点参数时返回的是浮点类型结果，通常为 `double`，不是 `int`。题干把数值结果和结果类型都说成整数类型，因此错误。

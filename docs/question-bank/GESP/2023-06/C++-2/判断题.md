# 判断题

来源：

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

诞生于 1986 年的中华学习机 CEC-I 入选了 2021 年的 CCF 计算机历史记忆（一类），它的内存只有 64KB。当时的汉字编码字符集 GB2312 中共有 6763 个汉字，假如每个汉字用 2 个字节编码，将整个 GB2312 汉字字符集都放入 CEC-I 的内存，也只占用了不超过 1/5 的内存空间。

答案：错误

解析：本题判断为错误。英文字母编码连续，减去小写字母基准再加上大写字母基准，可把对应小写字母转换成大写字母。

## 第2题

域名是由一串用点分隔的名字来标识互联网上一个计算机或计算机组的名称，CCF 编程能力等级认证官方网站的域名是 gesp.ccf.org.cn，其中顶级域名是 gesp。

答案：错误

解析：本题判断为错误。IPv4 点分十进制每段范围包含 0 到 255；域名从右向左看层级，最右侧才是顶级域名。

## 第3题

在使用 C++ 语言编写程序时，不能使用 sqrt、abs 等数学函数，包含 <cmath> 或 <math.h> 头文件后就能够使用了。

答案：正确

解析：本题判断为正确。赋值号 `=` 与相等比较 `==` 含义不同，C++ 运算符也必须写成规定形式；题干说法需与这些语法规则一致。

## 第4题

在 C++ 语言中，标识符中可以有下划线 `_`。同时，`_` 也是 C++ 语言的运算符。

答案：错误

解析：本题判断为错误。C++ 标识符可以包含字母、数字和下划线，不能以数字开头；下划线可以出现在标识符中，也可以作为开头。

## 第5题

如果 a 是 double 类型的变量，而且值为 3.5，则表达式 `a * 10` 的计算结果为 35，且结果类型为 int。

答案：错误

解析：本题判断为错误。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第6题

在 if 语句中，如果条件成立时需要执行多条语句，可以使用大括号 { 和 } 将这些语句括起来。

答案：正确

解析：本题判断为正确。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

## 第7题

循环语句的循环体有可能无限制地执行下去。

答案：正确

解析：本题判断为正确。如果循环条件一直为真且循环体不能改变退出条件，循环体就可能一直执行下去。

## 第8题

++ 和 == 都是 C++ 语言的运算符，但 += 不是。

答案：错误

解析：本题判断为错误。赋值号 `=` 与相等比较 `==` 含义不同，C++ 运算符也必须写成规定形式；题干说法需与这些语法规则一致。

## 第9题

如果 a 为 char 类型的变量，且取值为大写字母 'F'，则执行语句 `a = a+1;` 后，a 的值会变为大写字母 'G'。

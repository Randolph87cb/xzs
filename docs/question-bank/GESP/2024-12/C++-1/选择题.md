## 第1题

2024 年 10 月 8 日，诺贝尔物理学奖“意外地”颁给了两位计算机科学家约翰·霍普菲尔德（John J. Hopfield）和杰弗里·辛顿（Geoffrey E. Hinton）。这两位科学家的主要研究方向是（ ）。

A. 天体物理
B. 流体力学
C. 人工智能
D. 量子理论

答案：C

解析：约翰·霍普菲尔德和杰弗里·辛顿都与人工智能的发展密切相关，尤其是神经网络和机器学习方向。天体物理、流体力学、量子理论虽然也是物理或自然科学中的重要方向，但不是题干中这两位科学家的主要研究领域。因此选 C。

## 第2题

下列软件中是操作系统的是（ ）。

A. 高德地图
B. 腾讯会议
C. 纯血鸿蒙
D. 金山永中

答案：C

解析：操作系统负责管理硬件资源，并为应用软件提供运行环境。“纯血鸿蒙”属于操作系统；高德地图、腾讯会议是应用软件，金山永中通常指办公软件，也不是操作系统。因此选 C。

## 第3题

有关下列 C++ 代码的说法，正确的是（ ）。

```cpp
printf("Hello,GESP!");
```

A. 配对双引号内，不可以有汉字
B. 配对双引号可以相应改变为英文单引号而输出效果不变
C. 配对双引号可以相应改变为三个连续英文单引号而输出效果不变
D. 配对双引号可以相应改变为三个连续英文双引号而输出效果不变

答案：D

解析：`printf` 中的 `"Hello,GESP!"` 是字符串字面量。C++ 中相邻的字符串字面量会自动拼接，若把两端各写成三个连续英文双引号，等价于空字符串、原字符串、空字符串相邻拼接，输出内容仍是 `Hello,GESP!`。双引号内可以出现汉字，所以 A 错；单引号表示字符字面量，不能直接替代字符串，所以 B 错；三个连续英文单引号也不能表示这个字符串，所以 C 错。因此选 D。

## 第4题

C++ 表达式 `12 - 3 * 2 && 2` 的值是（ ）。

A. 0
B. 1
C. 6
D. 9

答案：B

解析：先按优先级计算乘法，`3 * 2 = 6`，再算减法，`12 - 6 = 6`。逻辑与 `&&` 的两边分别是 `6` 和 `2`，在 C++ 中非零值都当作真，真并且真结果为真；作为整数输出或作为表达式值时，真对应 `1`。因此选 B。

## 第5题

在 C++ 中，假设 $N$ 为正整数 $2$，则 `cout << (N / 3 + N % 3)` 将输出（ ）。

A. 0
B. 2
C. 3
D. 4

答案：B

解析：`N` 是正整数 `2`，表达式中使用整数除法和取余。`2 / 3` 的商是 `0`，`2 % 3` 的余数是 `2`，两者相加为 `0 + 2 = 2`。因此输出 `2`，选 B。

## 第6题

C++ 语句 `cout << 7%3 << ' ' << "7%3" << ' ' << "7%3={7%3}"` 执行后的输出是（ ）。

A. `1 1 1=1`
B. `1 7%3 1=1`
C. `1 7%3 7%3=1`
D. `1 7%3 7%3={7%3}`

答案：D

解析：`7%3` 没有引号包住，是表达式，结果为 `1`。后面的 `"7%3"` 和 `"7%3={7%3}"` 都被双引号包住，是字符串，会按原样输出，里面的 `%`、`{}` 不会再被计算。三个输出项之间各输出了一个空格字符，所以最终结果是 `1 7%3 7%3={7%3}`。因此选 D。

## 第7题

下面 C++ 代码执行后，求出几天后星期几。如果星期日则输出“星期天”否则输出形如“星期1”。横线上应填入的代码是（ ）。

```cpp
int N, nowDay, afterDays;
cout << "今天星期几？" << endl;
cin >> nowDay;
cout << "求几天后星期几？" << endl;
cin >> afterDays;
N = nowDay + afterDays;
if(_____________)
    printf("星期天");
else
    printf("星期%d", N%7);
```

A. `N % 7 != 0`
B. `N % 7 == 0`
C. `N == 0`
D. `N % 7`

答案：B

解析：`N = nowDay + afterDays` 表示从星期编号继续往后数，星期按 7 天循环。当 `N % 7 == 0` 时，说明落在一周循环中的第 7 天，也就是星期日，应输出“星期天”。若余数不是 `0`，就输出 `N%7` 得到“星期1”到“星期6”。因此横线处应判断 `N % 7 == 0`，选 B。

## 第8题

下面 C++ 代码执行后输出是（ ）。

```cpp
int N=0,i;
for (i = 1; i < 10; i++)
    N += 1;
cout << (N + i);
```

A. 54
B. 20
C. 19
D. 18

答案：C

解析：循环从 `i = 1` 开始，只要 `i < 10` 就执行一次 `N += 1`，所以 `i = 1,2,...,9` 共执行 9 次，循环结束时 `N = 9`。最后一次自增后 `i` 变为 `10`，不再满足循环条件。输出的是 `N + i = 9 + 10 = 19`。因此选 C。

## 第9题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int tnt = 0;
for (i = 0; i < 100; i++)
    tnt += i % 10;
cout << tnt;
```

A. 4950
B. 5050
C. 450
D. 100

答案：C

解析：`i` 从 `0` 到 `99`，`i % 10` 依次得到个位数字。每连续 10 个数的余数和是 `0 + 1 + ... + 9 = 45`，从 `0` 到 `99` 一共有 10 组这样的循环，所以总和是 `45 * 10 = 450`。因此选 C。

## 第10题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int N=0,i;
int tnt = 0;
for (i = 5; i < 100; i+=5){
    if (i % 2 == 0)
        continue;
    tnt += 1;
    if (i >= 50)
        break;
}
cout << tnt;
```

A. 10
B. 9
C. 6
D. 5

答案：C

解析：循环变量依次取 `5,10,15,20,...`。遇到偶数时 `continue`，这次循环后面的 `tnt += 1` 和 `break` 判断都不执行，所以只有奇数倍的 5 会计数：`5,15,25,35,45,55`。当 `i = 55` 时先执行 `tnt += 1`，此时 `tnt = 6`，再因为 `i >= 50` 执行 `break` 退出循环。最终输出 `6`，选 C。

## 第11题

下面的程序用于判断输入的整数 $N$ 是否为能被 $3$ 整除的偶数，横线处应填写代码是（ ）。

```cpp
int N;
cin >> N;

if(_____________)
    cout << "能被3整除的偶数" << endl;
else
    cout << "其他情形" << endl;

cout << endl;
```

A. `N%2 && N%3`
B. `N%2==0 && N%3`
C. `N%2 && N%3==0`
D. `N%2==0 && N%3==0`

答案：D

解析：题目要求同时满足两个条件：是偶数，并且能被 `3` 整除。偶数应写成 `N % 2 == 0`，能被 `3` 整除应写成 `N % 3 == 0`，两个条件要同时成立，用逻辑与 `&&` 连接。A 没有判断余数是否为 0；B 只保证是偶数，却要求 `N % 3` 为真；C 只保证能被 3 整除，却要求 `N % 2` 为真。因此选 D。

## 第12题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int cnt;

cnt = 0;
for(int i = 1; i < 10; i++)
    cnt += i++;

cout << cnt;
cout << endl;
```

A. 54
B. 45
C. 25
D. 10

答案：C

解析：循环体是 `cnt += i++`，这里先把当前 `i` 加到 `cnt`，再让 `i` 自增 1；随后 `for` 语句末尾的 `i++` 又会再自增 1。因此每轮实际让 `i` 增加 2。参与累加的 `i` 依次是 `1,3,5,7,9`，总和为 `25`。因此选 C。

## 第13题

`int` 类型变量 `a` 的值是一个正方形的边长，如下图中的正方形的四条边长都为 4，则下列哪个语句执行后能够使得正方形的周长（四条边长的和）增加 4？（ ）。

```
+ + + + +
+       +
+       +
+       +
+ + + + +
```

A. `a*4;`
B. `a+4;`
C. `a+1;`
D. `++a;`

答案：D

解析：正方形周长是边长的 4 倍。若要周长增加 `4`，边长只需要增加 `1`。`++a;` 会真正把变量 `a` 的值增加 1，因此周长随之增加 `4`。`a*4;`、`a+4;`、`a+1;` 都只是计算表达式的值，没有把结果赋回 `a`，不会改变边长。因此选 D。

## 第14题

C++ 表达式 `(6 > 2) * 2` 的值是（ ）。

A. 1
B. 2
C. true
D. truetrue

答案：B

解析：关系表达式 `6 > 2` 的结果为真。在 C++ 中，真参与算术运算时可按 `1` 处理，所以 `(6 > 2) * 2` 等价于 `1 * 2`，结果是 `2`。因此选 B。

## 第15题

下面 C++ 代码用于判断输入的整数是否为位增数，即从首位到个位逐渐增大，是则输出 1。如 123 是一个位增数。下面横线处应填入的是（ ）。

```cpp
int N;
int n1,n2;

cin >> N;

_______________________;

while(N){
    n1 = N % 10;

    if(n1 >= n2){
        cout << 0;
        return 1;
    }
    n2 = n1, N /=10;
}

cout << 1;
cout << endl;
return 0;
```

A. `n2 = N%10`
B. `N /= 10`
C. `n2 = N/10, N %= 10`
D. `n2 = N%10, N /= 10`

答案：D

解析：程序从个位向高位检查。要比较相邻两位，进入 `while` 前应先把原数的个位保存到 `n2`，再通过 `N /= 10` 去掉这一位；循环中取新的个位到 `n1`，判断更高位 `n1` 是否小于右边一位 `n2`。以 `123` 为例，先保存 `3`，再比较 `2 < 3`、`1 < 2`，都满足才输出 `1`。A 只保存个位但没有去掉个位，第一次会拿同一位和自己比较；B 没有初始化 `n2`；C 把十位以上保存到 `n2`、只留下个位，方向和含义都不对。因此选 D。

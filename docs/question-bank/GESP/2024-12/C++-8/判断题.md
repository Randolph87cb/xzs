# 判断题

来源：

- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1650239836520480.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

表达式 `'3' + '5'` 的结果为 `'8'`，类型为 `char`。

答案：错误

解析：`'3'` 和 `'5'` 是字符字面量，但参与 `+` 运算时会发生整型提升，表达式按两个字符编码值相加，结果类型是 `int`，不是 `char`。在常见 ASCII 编码中，`'3'` 为 51、`'5'` 为 53，相加为 104，并不是字符 `'8'`。

## 第2题

在 C++ 语言中，可以在函数内定义结构体，但该结构体类型只能在该函数内使用。

答案：正确

解析：在 C++ 中，结构体本质上也是类类型，可以定义在函数的块作用域内。这样的局部结构体类型名只在该函数作用域内可见，函数外不能直接用这个类型名声明变量或参数，所以题干说法按常规作用域规则是正确的。

## 第3题

对 $n$ 个元素的数组进行排序，快速排序和归并排序的平均时间复杂度都为 $O(n \log n)$。但快速排序存在退化情况，使得时间复杂度升高至 $O(n^2)$；归并排序需要额外的空间开销。

答案：正确

解析：快速排序在平均情况下每层处理 $O(n)$ 个元素，递归层数平均为 $O(\log n)$，所以平均复杂度为 $O(n \log n)$；但若每次划分都极不平衡，会退化为 $O(n^2)$。归并排序每次把数组分半并在线性时间合并，时间复杂度为 $O(n \log n)$，通常需要额外数组保存合并结果，因此有额外空间开销。

## 第4题

二维数组的最后一维在内存中一定是连续的，但第一维在内存中可能不连续。

答案：错误

解析：真正的 C++ 二维数组如 `int a[R][C]` 在内存中是连续存放的，先存第 0 行的 $C$ 个元素，再存第 1 行的 $C$ 个元素，行与行之间也是连续的。它传参时可能退化为指向一行数组的指针，但这不改变原数组对象本身连续存储的事实。因此“第一维在内存中可能不连续”用于二维数组是不正确的。

## 第5题

使用 `math.h` 或 `cmath` 头文件中的函数，表达式 $\log(1000)$ 的结果类型为 `double`、值约为 3。

答案：错误

解析：`math.h` 或 `cmath` 中的 `log` 表示自然对数，不是以 10 为底的对数。`log(1000)` 的返回类型按重载和转换为 `double`，但值约为 $\ln 1000 \approx 6.907$；值约为 3 的是 `log10(1000)`。题干把函数含义混淆了，所以为错误。

## 第6题

你有三种硬币，分别面值 2 元、5 元和 7 元，每种硬币都有足够多。买一本书需要 27 元，则有 8 种硬币组合（组合与顺序无关，“1 个 2 元+1 个 5 元+1 个 2 元”与“1 个 5 元+2 个 2 元”认为是同样的组合）可以正好付清，且不需要对方找钱。

答案：正确

解析：设 2 元、5 元、7 元硬币个数分别为 $a,b,c$，需要非负整数解 $2a+5b+7c=27$。枚举 $c$：$c=0$ 时 $b=1,3,5$ 共 3 种；$c=1$ 时 $b=0,2,4$ 共 3 种；$c=2$ 时只有 $b=1$ 共 1 种；$c=3$ 时只有 $b=0$ 共 1 种。合计 $3+3+1+1=8$ 种，且组合不区分顺序，所以题干正确。

## 第7题

使用哈希函数 $f(x) = x \bmod p$ 建立键值为 `int` 类型的哈希表，只要 $p$ 取小于等于哈希表大小的素数，可保证不发生碰撞。

答案：错误

解析：取素数作为模数只能在一定程度上改善散列分布，不能保证没有碰撞。只要存在两个键值相差 $p$，例如 $x$ 和 $x+p$，就有 $x \bmod p = (x+p) \bmod p$，它们会落到同一个位置。因此题干中的“可保证不发生碰撞”错误。

## 第8题

杨辉三角中的第 $n$ 行、第 $m$ 项，即为将二项式 $(a + b)^n$ 展开后 $a^{n-m}b^m$ 项的系数。

答案：错误

解析：按通常教材和考试中的 1 基编号，杨辉三角第 1 行是 `1`，第 $n$ 行第 $m$ 项对应 $\binom{n-1}{m-1}$，也就是 $(a+b)^{n-1}$ 中 $a^{n-m}b^{m-1}$ 项的系数。题干把它说成 $(a+b)^n$ 中 $a^{n-m}b^m$ 的系数，相当于使用 0 基行号和项号；这与“第 $n$ 行、第 $m$ 项”的通常含义不一致，所以判为错误。

## 第9题

判断图是否连通，可以通过广度优先搜索实现。

答案：正确

解析：对无向图，从任意一个顶点开始做广度优先搜索，若搜索结束后所有顶点都被访问到，则图连通；若存在未访问顶点，则不连通。因此 BFS 可以用来判断图是否连通。若讨论有向图，还需要额外说明是强连通还是弱连通，但本题按常规“图连通”理解为无向连通性。

## 第10题

要求解一元二次方程 $x^2 + ax + b = 0$，需要先判断表达式 `a ^ 2 - b * 4 >= 0` 是否为真。

答案：错误

解析：一元二次方程 $x^2+ax+b=0$ 是否有实根，应判断判别式 $a^2-4b \ge 0$。在 C++ 中 `^` 不是乘方运算，而是按位异或；并且该表达式还会按 C++ 运算符优先级解析，不等价于数学上的 $a^2-4b \ge 0$。正确写法应使用 `a * a - 4 * b >= 0`。

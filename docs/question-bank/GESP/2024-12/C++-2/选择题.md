## 第1题

2024 年 10 月 8 日，诺贝尔物理学奖“意外地”颁给了两位计算机科学家约翰·霍普菲尔德（John J. Hopfield）和杰弗里·辛顿（Geoffrey E. Hinton）。这两位科学家的主要研究方向是（ ）。

A. 天体物理
B. 流体力学
C. 人工智能
D. 量子理论

答案：C

解析：约翰·霍普菲尔德和杰弗里·辛顿的代表性工作都与神经网络、机器学习等人工智能方向有关。天体物理、流体力学、量子理论虽然也是物理或数学相关领域，但不是题干中这两位科学家的主要研究方向，因此选 C。

## 第2题

计算机系统中存储的基本单位用 B 来表示，它代表的是（ ），比如某个照片大小为 3MB。

A. Byte
B. Block
C. Bulk
D. Bit

答案：A

解析：计算机存储容量中的大写 `B` 通常表示 `Byte`，即字节，`3MB` 表示约 $3$ 兆字节。`Bit` 是比特，常用小写 `b` 表示；`Block` 和 `Bulk` 不是存储容量基本单位的标准写法，因此选 A。

## 第3题

C++ 语句 `cout << (3 + 3 % 3 * 2 - 1)` 执行后输出的值是（ ）。

A. -1
B. 4
C. 56
D. 2

答案：D

解析：表达式中 `%` 和 `*` 的优先级高于 `+`、`-`，并且同级从左到右计算。先算 `3 % 3 = 0`，再算 `0 * 2 = 0`，所以括号内为 `3 + 0 - 1 = 2`，输出为 `2`，对应 D。

## 第4题

下面 C++ 代码执行后其输出是（ ）。

```cpp
for (int i=0; i<10; i++)
    printf("%d",i);
```

A. 123456789
B. 0123456789
C. 12345678910
D. 012345678910

答案：B

解析：循环从 `i = 0` 开始，每次输出当前的 `i`，然后 `i++`。当 `i` 依次为 `0, 1, ..., 9` 时满足 `i < 10`，`i = 10` 时循环结束。`printf("%d", i)` 不会自动输出空格或换行，因此结果连续为 `0123456789`，选 B。

## 第5题

下面 C++ 代码的相关说法中，正确的是（ ）。

```cpp
int tnt;
for (int i=0; i<10; i++)
    tnt += i;
cout << tnt;
```

A. 上述代码执行后其输出相当于求 1-10 的和（包含 10）
B. 上述代码执行后其输出相当于求 1-10 的和（不包含 10）
C. 上述代码执行后其输出相当于求 0-10 的和（不包含 10）
D. 上述代码执行后将输出不确定的值

答案：D

解析：变量 `tnt` 定义后没有初始化，第一次执行 `tnt += i` 时会先读取 `tnt` 原来的值，而这个值是不确定的。因此这段代码不能稳定表示某个求和过程，输出也不是确定的 $0+1+\cdots+9$ 或 $1+\cdots+9$，应选 D。

## 第6题

下面 C++ 代码执行后输出是（ ）。

```cpp
int i;
for (i=1; i<10; i++)
    if (i % 2)
        continue;
    else
        break;
cout << i;
```

A. 1
B. 2
C. 9
D. 10

答案：B

解析：初始 `i = 1`，`1 % 2` 为真，执行 `continue`，本轮结束后 `i++` 变为 `2`。下一轮 `2 % 2` 为 `0`，条件为假，进入 `else` 执行 `break`，循环立即结束，此时 `i` 仍为 `2`，所以输出 `2`，选 B。

## 第7题

下面 C++ 代码执行后的输出是（ ）。

```cpp
for (i=0; i<10; i++){
    if (i % 3)
        continue;
    printf("0#");
}
if(i>=10)
    printf("1#");
```

A. 0#0#0#0#0#0#0#1#
B. 0#0#0#0#0#0#1#
C. 0#0#0#0#1#
D. 0#0#0#0#

答案：C

解析：循环中只有当 `i % 3 == 0` 时才不会执行 `continue`，因此 `i = 0, 3, 6, 9` 各输出一次 `0#`，共四次。循环正常结束后，`i` 已经自增到 `10`，满足 `i >= 10`，再输出一次 `1#`。最终结果是 `0#0#0#0#1#`，对应 C；D 少了最后的 `1#`，A、B 的 `0#` 次数过多。

## 第8题

下面 C++ 代码用于输出 0-100 之前（包含 100）能被 7 整除但不能被 3 整除的数，横线处**不能**填入的代码是（ ）。

```cpp
for (i=0; i<100; i++)
    if(______________)
        cout << i << endl;
```

A. `i % 7 == 0 && i % 3 != 0`
B. `!(i % 7) && i % 3 != 0`
C. `i % 7 && i % 3`
D. `i % 7 == 0 && !(i % 3 == 0)`

答案：C

解析：要输出的数必须满足“能被 $7$ 整除”且“不能被 $3$ 整除”。A 直接写成 `i % 7 == 0 && i % 3 != 0`；B 中 `!(i % 7)` 等价于 `i % 7 == 0`；D 中 `!(i % 3 == 0)` 等价于不能被 $3$ 整除。C 的 `i % 7 && i % 3` 要求两个余数都非零，反而表示既不能被 $7$ 整除又不能被 $3$ 整除，所以不能填入，选 C。

## 第9题

下面 C++ 代码用于求正整数各位数字之和，横线处**不应**填入代码是（ ）。

```cpp
int tnt, N;
printf("请输入正整数：");
cin >> N;
tnt = 0;
while (N != 0){
    ________________
    N /= 10;
}
cout << tnt;
```

A. `tnt = tnt + N % 10`
B. `tnt += N % 10`
C. `tnt = N % 10 + tnt`
D. `tnt = N % 10`

答案：D

解析：每轮循环应把当前个位 `N % 10` 加到累计和 `tnt` 中，然后通过 `N /= 10` 去掉个位。A、B、C 都是在原有 `tnt` 基础上加当前个位，只是写法不同。D 会把 `tnt` 直接改成当前个位，覆盖之前已经累加的结果，例如 `123` 会依次得到 `3, 2, 1`，最后不是 $1+2+3$，所以选 D。

## 第10题

下图的 C++ 程序执行后的输出是（ ）。

```cpp
for (i=0; i<5; i++)
    for (j=0; j<i; j++)
        cout << j;
```

A. 0010120123
B. 01012012301234
C. 001012012301234
D. 01012012301234012345

答案：A

解析：外层 `i` 取 `0` 到 `4`。当 `i = 0` 时内层 `j < 0` 不成立，不输出；`i = 1` 输出 `0`；`i = 2` 输出 `01`；`i = 3` 输出 `012`；`i = 4` 输出 `0123`。把这些结果连续拼接为 `0`、`01`、`012`、`0123`，得到 `0010120123`，选 A。

## 第11题

下面 C++ 代码用于实现图示的九九乘法表。相关说法错误的是（ ）。

```cpp
/*
1*1=1
1*2=2 2*2=4
1*3=3 2*3=6 3*3=9
1*4=4 2*4=8 3*4=12 4*4=16
1*5=5 2*5=10 3*5=15 4*5=20 5*5=25
1*6=6 2*6=12 3*6=18 4*6=24 5*6=30 6*6=36
1*7=7 2*7=14 3*7=21 4*7=28 5*7=35 6*7=42 7*7=49
1*8=8 2*8=16 3*8=24 4*8=32 5*8=40 6*8=48 7*8=56 8*8=64
1*9=9 2*9=18 3*9=27 4*9=36 5*9=45 6*9=54 7*9=63 8*9=72 9*9=81
*/
```

```cpp
for (int Hang=1; Hang<10; Hang++){
    for (int Lie=1; Lie<Hang+1; Lie++){
        if (Lie * Hang > 9)
            printf("%d*%d=%d ", Lie, Hang, Lie*Hang);
        else
            printf("%d*%d=%d ", Lie, Hang, Lie*Hang);
        // L2
    }
    printf("\n"); // L1
}

```

A. 将 L1 注释的 `printf("\n")` 移到 L2 注释所在行，效果相同
B. 将 L1 注释的 `printf("\n")` 修改为 `print("%c", '\n')` 效果相同
C. 将 `Lie * Hang > 9` 修改为 `Lie * Hang >= 10` 效果相同
D. 将 `Lie * Hang > 9` 修改为 `Hang * Lie > 9` 效果相同

答案：A

解析：本题问“说法错误”的一项。`printf("\n")` 放在 L1 时，内层循环会先输出当前行的所有乘法式，等这一行结束后才换行；如果把它移到 L2，每输出一个乘法式就立即换行，原来同一行的内容会被拆成多行，输出格式明显不同，所以 A 是错误说法。C 中整数乘积满足 `> 9` 等价于 `>= 10`；D 中乘法满足交换律，`Lie * Hang` 和 `Hang * Lie` 的结果相同。B 写成 `print("%c", '\n')` 时，按给定文本看，`print` 不是标准 C/C++ 输出函数，也不能直接说它与 `printf` 等效；若意图是使用正确的输出函数输出换行字符 `'\n'`，换行效果才与原语句接近，不会像 A 那样改变每行输出几个乘法式。

## 第12题

在数学中 $N!$ 表示 $N$ 的阶乘，即 $1$ 到 $N$ 的乘积，如 $3! = 1*2*3$。下面的 C++ 用于求 $1$-$N$ 的阶乘之和，如 $N$ 为 $3$，则是 $1! + 2! + 3!$。下面代码段补充选项后用于实现上述功能，其中不能实现阶乘和的选项是（ ）。

```cpp
int N;
cin >> N;

int tnt = 0, nowNum = 1; // tnt 保存求和之值，当前 N 的阶乘

for (int i = 1; i < N + 1; i++) {
    ____________ // 基于上一个计算出当前数的阶乘
    ____________ // 从 1 到 i 每个数阶乘之和
}
cout << tnt;
```

A.

```cpp
nowNum *= i;
tnt += nowNum;
```

B.

```cpp
nowNum = nowNum * i;
tnt = tnt + nowNum;
```

C.

```cpp
nowNum *= i;
tnt = nowNum + tnt;
```

D.

```cpp
nowNum = nowNum + i;
tnt *= nowNum;
```

答案：D

解析：`nowNum` 应保存当前的阶乘。第 `i` 轮要在上一轮阶乘基础上乘以 `i`，即得到 `i!`，再把它加到 `tnt` 中。A、B、C 都是先令 `nowNum` 乘以 `i`，再把 `nowNum` 加入总和。D 使用 `nowNum = nowNum + i`，不是阶乘递推；又用 `tnt *= nowNum`，而 `tnt` 初始为 `0`，乘法也不会得到阶乘和，因此选 D。

## 第13题

下面 C++ 代码用于输出 $N$ 和 $M$ 之间（可以包括 $N$ 和 $M$）的孪生素数。孪生素数是指间隔为 $2$ 的两个数均为素数，如 $11$ 和 $13$ 分别是素数，且间隔为 $2$。`isPrime(N)` 用于判断 $N$ 是否为素数的函数。为完成上述功能，横线处应填上的代码是（ ）。

```cpp
int N, M;

// 本题假设 N 小于 M
cin >> N >> M;

for (int i = N; i < __________; i++)
    if (isPrime(i) && isPrime(i + 2))
        printf("%d %d\n", i, i + 2);
```

A. `M - 2`
B. `M - 1`
C. `M`
D. `M + 1`

答案：B

解析：循环变量 `i` 表示孪生素数中较小的那个数，另一项是 `i + 2`。为了保证可以包含右端点 $M$，最大的合法 `i` 应为 $M - 2$。`for` 条件写作 `i < 上界`，若上界为 `M - 1`，最后一次可取到 `M - 2`。A 会漏掉 `(M - 2, M)` 这一对，C、D 则可能让 `i + 2` 超过 $M$，因此选 B。

## 第14题

下面 C++ 代码实现输出如下图形。横线应填入的代码是（ ）。

```cpp
/*
高度:5
    *
   ***
  *****
 *******
*********
*/
int height;
cout << "高度：";
// 获取用户输入的高度
cin >> height;
for (i = 0; i < height; i++) {
    // 打印每行前面的空格
    for (j = 0; j < __________; j++)
        cout << " ";
    // 打印每行的星号
    for (k = 0; k < __________; k++)
        cout << "*";
    // 输出一行后，换行
    cout << endl;
}
```

A.

```cpp
height - i
2 * i
```

B.

```cpp
height
2 * i
```

C.

```cpp
height - i
2 * i + 1
```

D.

```cpp
height - i - 1
2 * i + 1
```

答案：D

解析：行号 `i` 从 `0` 开始。第 `i` 行前面的空格数应逐行减少，分别是 `height - 1, height - 2, ..., 0`，所以空格循环次数为 `height - i - 1`。星号数应为奇数序列 `1, 3, 5, ...`，对应 `2 * i + 1`。只有 D 同时满足这两个规律。

## 第15题

下面 C++ 代码执行后的输出是 $30$，则横线处不能填入（ ）。

```cpp
int a = 10, b = 20, c = 30;

cout << __________ << endl;

cout << endl;
```

A. `max(max(a, b), c)`
B. `min(a + b, c)`
C. `sqrt(a + b + c)`
D. `(a + b + c) / 2`

答案：C

解析：`a + b + c = 60`。A 的 `max(max(a, b), c)` 为 `30`；B 的 `min(a + b, c)` 是 `min(30, 30)`，也为 `30`；D 的 `(a + b + c) / 2` 是 `60 / 2 = 30`。C 的 `sqrt(a + b + c)` 是 $\sqrt{60}$，约为 $7.75$，不能输出 `30`，所以选 C。

```cpp
1*4=4 2*4=8 3*4=12 4*4=16
```

```cpp
cin >> N;

for (int i = 1; i < N + 1; i++) {
    ____________ // 基于上一个计算出当前数的阶乘
    ____________ // 从 1 到 i 每个数阶乘之和
}
cout << tnt;
```

A.

```cpp
nowNum *= i;
tnt += nowNum;
```

B.

```cpp
nowNum = nowNum * i;
tnt = tnt + nowNum;
```

C.

```cpp

tnt = nowNum + tnt;
```

```cpp

```

B. B
C. C
D. D

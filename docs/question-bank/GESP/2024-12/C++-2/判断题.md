## 第1题

在 Windows 的资源管理器中为已有文件 A 建立副本的操作是 `Ctrl+C`，然后 `Ctrl+V`。（ ）

答案：正确

解析：在 Windows 资源管理器中，`Ctrl+C` 是复制选中的文件，`Ctrl+V` 是把剪贴板中的文件粘贴到当前位置。对已有文件 A 执行这两个操作，会生成该文件的副本，因此题干说法正确。

## 第2题

在 C++ 代码中，假设 $N$ 为正整数，则 `cout << (N - N / 10 * 10)` 将获得 $N$ 的个位数。（ ）

答案：正确

解析：`N / 10` 是整数除法，会去掉 $N$ 的个位；再乘以 `10`，得到不含个位的部分。例如 $N = 123$ 时，`N / 10 * 10 = 12 * 10 = 120`，`N - 120 = 3`。这与正整数 $N$ 取个位的结果相同，所以题干说法正确。

## 第3题

在 C++ 语句 `cout << (10 <= N <= 12)` 中，假设 N 为 12，则其输出为 1。（ ）

答案：正确

解析：C++ 不会把 `10 <= N <= 12` 当作数学中的连续不等式，而是从左到右计算。若 `N = 12`，先算 `10 <= N`，结果为真，可按数值 `1` 参与下一步比较；再算 `1 <= 12`，仍为真，输出为 `1`。因此题干给定 `N` 为 `12` 时说法正确。

## 第4题

如果 C++ 表达式 `int(sqrt(N))*int(sqrt(N)) == N` 的值为 True，则说明 $N$ 为完全平方数，如 4、9、25 等。（ ）

答案：正确

解析：`sqrt(N)` 求 $N$ 的平方根，`int(sqrt(N))` 会取其整数部分。若 `int(sqrt(N)) * int(sqrt(N)) == N` 成立，说明存在一个整数 $k = int(sqrt(N))$，使得 $k^2 = N$，这正是完全平方数的定义。对于题目讨论的 $4, 9, 25$ 这类非负整数，判断成立，因此题干说法正确。

## 第5题

下面 C++ 代码执行后将输出 `2*3=6`。（ ）

```cpp
int a = 2, b = 3;
printf("%%a*%%b=%d", a*b);
```

答案：错误

解析：`printf` 格式串中 `%%` 表示输出一个普通的百分号字符，不会去读取变量 `a` 或 `b`。因此 `printf("%%a*%%b=%d", a*b);` 的前半部分会输出 `%a*%b=`，最后 `%d` 才输出 `a*b` 的值 `6`。实际输出是 `%a*%b=6`，不是 `2*3=6`，所以题干说法错误。

## 第6题

以下 C++ 代码因为循环变量为 `_` 将导致错误，即 `_` 不能作为变量名称，不符合 C++ 变量命名规范。（ ）

```cpp
for (int _ = 0; _ < 10; _++)
    continue;
```

答案：错误

解析：C++ 标识符可以由字母、数字和下划线组成，且不能以数字开头。单独的 `_` 在这段局部 `for` 循环中可以作为变量名使用，循环变量名本身不会导致语法错误。代码会正常循环并执行 `continue`，所以题干说法错误。

## 第7题

下面 C++ 代码执行后因为有 `break`，将输出 0。（ ）

```cpp
int i;
for (i = 0; i < 10; i++){
    continue;
    break;
}
cout << i;
```

答案：错误

解析：循环体中先执行 `continue`，它会直接结束当前轮循环并进入下一轮的更新步骤，因此后面的 `break` 永远没有机会执行。`i` 从 `0` 增加到 `10` 后不再满足 `i < 10`，循环结束，随后输出的是 `10`，不是 `0`，所以题干说法错误。

## 第8题

下面的 C++ 代码执行后将输出 18 行 “OK”。（ ）

```cpp
int i,j;
for (i = 8; i > 2; i-=2)
    for (j =0; j < i; j++)
        printf("OK\n");
```

答案：正确

解析：外层循环中 `i` 依次取 `8, 6, 4`，当减到 `2` 时不满足 `i > 2` 而停止。内层循环每次输出 `i` 行 `OK`，所以总行数为 $8 + 6 + 4 = 18$。因此题干说法正确。

## 第9题

将下面 C++ 代码中的 `i = 1` 调整为 `i = 0` 的输出结果相同。（ ）

```cpp
int i;
int cnt = 0;
for (i = 1; i < 5; i++)
    if(i%2) cnt += 1;
cout << cnt;
```

答案：正确

解析：原代码中 `i` 取 `1, 2, 3, 4`，满足 `i % 2` 的是 `1` 和 `3`，`cnt` 最终为 `2`。若改成 `i = 0`，`i` 取 `0, 1, 2, 3, 4`，其中奇数仍然只有 `1` 和 `3`，`cnt` 还是 `2`。输出结果相同，所以题干说法正确。

## 第10题

下面两段 C++ 代码都是用于求 1-10 的和，其运行结果相同。通常说来，for 循环都可以用 while 循环实现。（ ）

```cpp
int tnt;
int i;

tnt = 0;
for (i = 1; i < 10 + 1; i++)
    tnt += i;
cout << tnt << endl;
```

```cpp
int tnt;
int i;

tnt = 0;
i = 1;
while (i <= 10){
    tnt += i;
    i += 1;
}
cout << tnt << endl;
```

答案：正确

解析：第一段 `for` 循环让 `i` 从 `1` 到 `10`，每次把 `i` 加到 `tnt`，得到 $1+2+\cdots+10=55$。第二段 `while` 循环先设 `i = 1`，每轮加上 `i` 后再 `i += 1`，直到 `i` 变成 `11` 停止，同样累加 $1$ 到 $10$。两段程序结果相同，而且一般来说 `for` 的初始化、条件和更新步骤都可以改写成 `while` 的对应语句，因此题干说法正确。

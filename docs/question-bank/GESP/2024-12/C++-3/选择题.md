## 第1题

下列二进制表示的十进制数值分别是（ ）

```cpp
[10000011]原 = ( )
[10000011]补 = ( )
```

【知识点：进制转换与进制字面量】

A. $-125$，$-3$
B. $-3$，$-125$
C. $-3$，$-3$
D. $-125$，$-125$

答案：B

解析：原码的最高位是符号位，`10000011` 的符号位为 $1$，数值位 `0000011` 表示 $3$，所以 $[10000011]原=-3$。补码 `10000011` 的最高位也表示负数，对其求真值时先按位取反得 `01111100`，再加 $1$ 得 `01111101`，即 $125$，所以 $[10000011]补=-125$。两个结果依次是 $-3$、$-125$，选 B。

## 第2题

关于计算机中的编码，下列说法中错误的是（ ）

【知识点：原码反码补码与整数表示】

A. 对于无符号数，原码就是真值
B. 正数的反码是其本身
C. 负数的反码和补码是不一样的
D. 负数的反码，在其原码的基础上，各个位取反

答案：D

解析：题目要求找错误说法。无符号数没有符号位，二进制编码直接表示真值，A 正确；正数的原码、反码、补码相同，B 正确；负数的补码是在反码基础上加 $1$，通常与反码不同，C 正确。负数由原码求反码时，符号位保持 $1$ 不变，只把数值位取反，而不是“各个位”都取反，所以 D 错误。

## 第3题

$16$ 进制数 $B2025$ 转换成 $8$ 进制数是（ ）

【知识点：进制转换与进制字面量】

A. $2620045$
B. $2004526$
C. $729125$
D. $2420045$

答案：A

解析：把每一位十六进制数写成 $4$ 位二进制：$B2025_{16}=1011\ 0010\ 0000\ 0010\ 0101_2$。再从右向左每 $3$ 位分组，左侧补 $0$ 后为 `010 110 010 000 000 100 101`，对应八进制数 $2620045$。因此选 A。

## 第4题

$625.625$ 变成二进制是（ ）。

【知识点：进制转换与进制字面量】

A. $1001110001.101$
B. $101.101$
C. $101.1001110001$
D. $1001110001.1101$

答案：A

解析：整数部分 $625=512+64+32+16+1$，对应二进制为 $1001110001$。小数部分 $0.625=0.5+0.125=2^{-1}+2^{-3}$，所以二进制小数为 $.101$。合起来是 $1001110001.101$，选 A。

## 第5题

下面逻辑运算中，正确的是（ ）。

【知识点：位运算】

A. `5 && 3 == 0`
B. `5 | 3 == 8`
C. `5 || 3 == 11`
D. `5 & 3 == 0001`

答案：D

解析：$5$ 的二进制是 `0101`，$3$ 的二进制是 `0011`。按位与时只有同一位都为 $1$ 才得到 $1$，所以 `0101 & 0011 = 0001`，D 正确。`5 | 3` 应为 `0111`，不是 $8$；`&&` 和 `||` 是逻辑运算，非零值只按“真”处理，不会得到题中 A、C 写出的结果。

## 第6题

补码 $1111\ 1101$ 进行运算 $1111\ 1101 >> 1$ 以后得到的结果是（ ）

【知识点：原码反码补码与整数表示】

A. $1111\ 1100$
B. $-2$
C. $1111\ 1101$
D. $1111\ 1010$

答案：B

解析：补码 `1111 1101` 表示负数，取反加 $1$ 得 `0000 0011`，真值为 $-3$。按补码的算术右移理解，右移一位时最高位补 $1$，得到 `1111 1110`，它的真值是 $-2$。因此结果对应选项 B。

## 第7题

下列代码输出的是

```cpp
string s = "1234@@chenadai";
string str = "12345";
s.replace(1, 5, str);
cout << s << endl;
```

【知识点：字符串处理】

A. 12345
B. 2345@
C. 112345chenadai
D. 12345chenadai

答案：C

解析：`replace(1, 5, str)` 从下标 $1$ 开始替换连续 $5$ 个字符。原串 `1234@@chenadai` 的下标 $1$ 到 $5$ 分别是 `2`、`3`、`4`、`@`、`@`，被替换成 `12345`；下标 $0$ 的 `1` 保留，后面的 `chenadai` 也保留。因此输出为 `112345chenadai`，选 C。

## 第8题

`a | 10`（$a$ 与 $10$ 都是 $10$ 进制，且二进制表示最高位为 $1$）运算的结果是（ ）。

【知识点：位运算】

A. 使 $a$ 的二进制表示从右往左的第二位为 $1$
B. 使 $a$ 的二进制表示从右往左的第一位为 $0$
C. 使 $a$ 的二进制表示从右往左第二位为 $0$
D. 使 $a$ 的二进制表示最高位为 $0$

答案：A

解析：$10$ 的二进制是 `1010`，其中从右往左第二位为 $1$。按位或 `|` 的规则是只要有一个操作数在某一位为 $1$，结果该位就是 $1$，所以 `a | 10` 一定会把 $a$ 从右往左第二位变成 $1$。按位或不会把某一位清成 $0$，因此 B、C、D 都不符合，选 A。

## 第9题

下列程序输出的是（ ）。

```cpp
string ch = "hello";
if (ch[5] == NULL)
{
    cout << "right" << endl;
}
else if (ch[5] == '\0')
{
    cout << "wrong" << endl;
}
else
{
    cout << "hello" << endl;
}
```

【知识点：字符串处理】

A. `right`
B. `wrong`
C. `hello`
D. 不能正确执行

答案：A

解析：`"hello"` 的有效字符下标是 $0$ 到 $4$，题目按字符串末尾的空字符判断，`ch[5]` 的值为 `'\0'`。`NULL` 通常也是值 $0$，所以第一个条件 `ch[5] == NULL` 成立，程序直接输出 `right`，不会继续判断后面的 `else if`。因此选 A。

## 第10题

下列程序中，假设一个字符占用的内存空间是 $1$，下列程序中，`ch` 占用的内存空间是（ ）

```cpp
char ch[] = "hello world";
size_t ret = strlen(ch);
cout << ret << endl;
```

【知识点：数组与字符数组】

A. $11$
B. $10$
C. $13$
D. $12$

答案：D

解析：字符串字面量 `"hello world"` 有 $11$ 个可见字符，但字符数组初始化时还会在末尾保存一个字符串结束符 `'\0'`。题目问的是数组 `ch` 占用的内存空间，而不是 `strlen(ch)` 的返回值，所以需要 $11+1=12$ 个字符空间。答案选 D。

## 第11题

下列程序最后输出的是（ ）

```cpp
int a = 65;
cout << tolower(a) << endl;
```

【知识点：字符串处理】

A. $65$
B. `A`
C. `a`
D. $97$

答案：D

解析：ASCII 中 $65$ 对应字符 `A`。`tolower(a)` 会把大写字母对应的编码转换为小写字母编码，`a` 的编码是 $97$；该函数返回的是整数值，直接用 `cout` 输出时显示 $97$，不是字符 `a`。因此选 D。

## 第12题

想要计算从数字 $n$ 到数字 $m$ 之间（包含 $n$ 和 $m$ ）有多少个数字 $d$ 出现，下列程序哪个能够实现（ ）。

【知识点：控制结构与程序阅读】

A.

```cpp
int n, m, res, d;
cin >> n >> m >> d;
for (int i = n + 1; i <= m; i++) {
    int temp = i;
    while (temp) {
        if (temp % 10 == d) res++;
        temp /= 10;
    }
}
cout << res << endl;
return 0;
```

B.

```cpp
int n, m, res, d;
cin >> n >> m >> d;
for (int i = n; i <= m; i++) {
    int temp = i;
    while (temp) {
        if (temp % 10 = d) res++;
        temp /= 10;
    }
}
cout << res << endl;
return 0;
```

C.

```cpp
int n,m,res=0,d;
cin >> n >> m>>d;
for(int i = n; i <= m; i++){
    int temp = i;
    while(temp){
        if(temp % 10 == d) res++;
        temp /= 10;
    }
}
cout << res << endl;
return 0;
```

D.

```cpp
int n,m,res=0,d;
cin >> n >> m>>d;
for(int i = n; i <= m; i++){
    while(temp){
        if(temp % 10 == d) res++;
        temp /= 10;
    }
}
cout << res << endl;
return 0;
```

答案：C

解析：要统计区间 $[n,m]$ 中数字 $d$ 出现的次数，需要让循环从 $n$ 到 $m$ 都参与统计，并且每次把当前数复制到 `temp`，通过 `temp % 10` 逐位检查。C 中 `res` 初始化为 $0$，循环范围包含两端，每个 `i` 都有 `int temp = i;`，判断也使用 `==`，逻辑完整。A 从 `n + 1` 开始会漏掉 $n$，且 `res` 没有初始化；B 把比较写成赋值 `=`；D 没有为每个数定义并初始化 `temp`。因此选 C。

## 第13题

兔子五元一只，鸡三元一只，小鸭子一元三只，现在你有一百元要买一百只，兔子、鸡、鸭子，问兔子、鸡、鸭子各买多少只，每种必须至少一只，下列哪个程序能实现。（ ）。

【知识点：控制结构与程序阅读】

A.
```cpp
for(int i = 0; i <= 100; i++){
    for(int j = 0; j <= 100; j++){
        for(int k = 0; k <= 100; k++){
            if(i + j + k == 100 && 25 * i + 10 * j + k == 300)
                cout << i << " " << j << " " << k << endl;
        }
    }
}
```

B.
```cpp
for(int i = 0; i <= 100; i++){
    for(int j = 0; j <= 100; j++){
        for(int k = 0; k <= 100; k++){
            if(i + j + k == 100 && 20 * i + 10 * j + k == 300)
                cout << i << " " << j << " " << k << endl;
        }
    }
}
```

C.
```cpp
for(int i = 0; i <= 20; i++){
    for(int j = 0; j <= 34; j++){
        for(int k = 0; k <= 20; k++){
            if(i + j + k == 100 && 15 * i + 9 * j + k == 300)
                cout << i << " " << j << " " << k << endl;
        }
    }
}
```

D.
```cpp
for(int i = 0; i <= 100; i++){
    for(int j = 0; j <= 100; j++){
        for(int k = 0; k <= 100; k++){
            if(i + j + k == 100 && 15 * i + 9 * j + k == 300)
                cout << i << " " << j << " " << k << endl;
        }
    }
}
```

答案：D

解析：设兔子、鸡、小鸭子的数量分别为 $i$、$j$、$k$，数量条件是 $i+j+k=100$。为了避免小鸭子“一元三只”的分数价格，可以把总价都乘以 $3$，得到兔子单价 $15$、鸡单价 $9$、小鸭子单价 $1$，总价为 $300$，所以价格条件是 $15i+9j+k=300$。D 同时使用这两个正确条件，并枚举到 $100$ 的完整范围；A、B 的价格系数不对，C 把 $k$ 限制到 $20$，会漏掉可能的小鸭子数量。按官方选项应选 D。需要注意的是，题干写有“每种必须至少一只”，而 D 选项循环从 $0$ 开始，严格按题干应让循环从 $1$ 开始，或在判断条件中增加 `i > 0 && j > 0 && k > 0` 过滤；否则程序可能输出 `(0,25,75)` 这类不满足每种至少一只的解。

## 第14题

求小于等于 $N$ 的素数的方法中，有一种方法是将所有从 $2$ 到它本身减 $1$ 的数都除一遍，如果不能整除，就是素数。下列哪个程序，体现了这种方法（ ）？

【知识点：枚举与基础算法】

A.
```cpp
if(N >= 3){
    cout<<2<<endl;
    for(int i = 2; i <= N; i++){
        for(int j = 2; j < i; j++){
            if(i % j != 0){
                flag = 1;
            }
        }
        if(flag == 0){
            cout<<i<<endl;
        }
        flag = 0;
    }
}
```

B.
```cpp
if(N >= 2){
    for(int i = 3; i <= N; i++){
        for(int j = 2; j < i; j++){
            if(i % j == 0){
                flag = 1;
            }
        }
        if(flag == 0){
            cout<<i<<endl;
        }
        flag = 0;
    }
}
```

C.
```cpp
if(N >= 2){
    cout<<2<<endl;
    for(int i = 3; i <= N; i++){
        for(int j = 2; j < i; j++){
            if(i % j == 0){
                flag = 1;
            }
        }
        if(flag == 0){
            cout<<i<<endl;
        }
        flag = 0;
    }
}
```

D.

```cpp
if(N >= 2){
    cout<<2<<endl;
    for(int i = 3; i <= N; i++){
        for(int j = 2; j < i; j++){
            if(i % j == 0){
                flag = 1;
            }
        }
        if(flag == 0){
            cout<<i<<endl;
        }
    }
}
```

答案：C

解析：题目描述的方法是：对候选数 $i$，枚举 $2$ 到 $i-1$，只要找到能整除的数就说明 $i$ 不是素数；一轮检查结束后如果没有找到因子，就输出 $i$。C 先在 $N \ge 2$ 时输出最小素数 $2$，再从 $3$ 枚举到 $N$，用 `flag` 记录是否存在因子，并在每个 $i$ 检查后把 `flag` 重置为 $0$，符合这种枚举方法。A 把“不能整除”当成标记条件，会把素数误判掉；B 没有输出 $2$；D 没有在每轮结束后重置 `flag`，遇到合数后会影响后续判断。因此选 C。

## 第15题

工人工作一天，会得到一个金环作为工资报酬，某个工作需要 $15$ 天完成，一条 $15$ 个环的金环项链，为了严格执行每天工作结束时，完成对工人工资的结算，最少需要将金环项链剪裁成几段，每段几个金环（ ）

【知识点：枚举与基础算法】

A. $4$ 段，分别是 $1$、$2$、$4$、$8$
B. $15$ 段，每段 $1$ 个
C. $6$ 段，分别是 $3$、$3$、$3$、$3$、$2$、$1$
D. $9$ 段，分别是 $2$、$2$、$2$、$2$、$2$、$2$、$1$、$1$、$1$

答案：A

解析：每天结算后，工人手中需要能表示从 $1$ 到 $15$ 的任意累计工资。若剪成 $3$ 段，最多只有 $2^3-1=7$ 种非空组合，无法覆盖 $15$ 个不同的累计金额，所以至少需要 $4$ 段。把 $15$ 个环分成 $1$、$2$、$4$、$8$ 后，任意 $1$ 到 $15$ 都能用二进制组合表示，例如 $13=1+4+8$、$14=2+4+8$、$15=1+2+4+8$，因此 $4$ 段已经足够且最少。B、C、D 的段数都不是最少，选 A。

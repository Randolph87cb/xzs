## 第1题

为了简化计算机基本运算电路，使加减法都只需要通过加法电路实现，也就是让减去一个正数或加上一个负数这样的运算可以用加上一个正数来代替。于是改变负数存储的形式，存储成一种可以直接当成正数来相加的形式，这种形式就是补码。

【知识点：原码反码补码与整数表示】

答案：正确

解析：补码设计的目的之一，就是把减法转化为加法处理。例如计算 $a-b$ 可以看成 $a+(-b)$，只要负数的机器表示满足加法电路的运算规则，就不需要为减法单独设计一套完全不同的基本电路。题干对补码用途的描述符合计算机整数表示的基本思想，因此正确。

## 第2题

使用原码进行的计算， $2 + (-1)$ 的结果是 $-3$

【知识点：原码反码补码与整数表示】

答案：正确

解析：如果直接把原码当作普通二进制数参与加法，$2$ 可写成 `0010`，$-1$ 的原码可写成 `1001`，相加得到 `1011`。按原码解释，`1011` 的符号位是 $1$，数值位是 $3$，结果就是 $-3$。这也说明原码不适合直接做加减运算；题干说“使用原码进行的计算”会得到 $-3$，判断为正确。

## 第3题

反码计算加减法：加法与减法结果都是正确的，只是解决不了 $-0$ 的问题

【知识点：原码反码补码与整数表示】

答案：正确

解析：反码比原码更适合加减法，负数通过数值位取反参与运算，超出最高位的进位还需要回卷到最低位继续相加。它能处理很多加减法结果，但仍然存在 `+0` 和 `-0` 两种零的表示，零不唯一。题干指出反码解决不了 $-0$ 的问题，表述正确。

## 第4题

10 进制数 63，在转换成二进制的计算过程中，产生了如下的式子：

```text
1  63/2=31 余 1
2  31/2=15 余 1
3  15/2=7  余 1
4  7/2=3   余 1
5  3/2=1   余 1
6  1/2=0   余 1
```

按照从前往后的顺序，获得 63 的二进制值是 111111

【知识点：进制转换与进制字面量】

答案：错误

解析：十进制整数转二进制时，连续除以 $2$ 后应把余数从后往前读，而不是从前往后读。$63$ 的各次余数恰好全是 $1$，所以按错误顺序也会碰巧写出 `111111`，但题干说“按照从前往后的顺序”获得二进制值，这个转换规则本身不正确，因此判断为错误。

## 第5题

下列程序输出的是 A

```cpp
char x=65;
x=x&00001111;
cout<<x<<endl;
```

【知识点：位运算】

答案：正确

解析：`char x=65;` 中 $65$ 对应 ASCII 字符 `A`。代码里的 `00001111` 在 C++ 中不是二进制字面量，而是前导 $0$ 的八进制整数，值为 $1111_8=585$；$65$ 与 $585$ 按位与时，$65$ 的两个置位仍被保留，结果还是 $65$。赋回 `char` 后输出字符 `A`，所以题干正确。

## 第6题

下列可执行程序段中，最后 pos 的值是 4

```cpp
string str="chenADai";
int pos = str.find('D');
--pos&11;
```

【知识点：位运算】

答案：正确

解析：字符串 `chenADai` 中字符 `D` 的下标是 $5$，所以 `find('D')` 得到 `pos=5`。表达式 `--pos&11;` 先执行前置自减，把 `pos` 改为 $4$，再把 $4$ 与 $11$ 做按位与；这个按位与结果没有赋值给 `pos`。因此语句执行完后 `pos` 的值仍是 $4$，题干正确。

## 第7题

```cpp
string ch="chen";
cout<<ch[4]<<endl;
```

该段程序将不能正确执行

【知识点：字符串处理】

答案：错误

解析：`string ch="chen"` 的有效字符下标是 $0$ 到 $3$，题目按字符串末尾的空字符理解，`ch[4]` 对应结束位置的 `'\0'`。输出这个字符通常看不到可见内容，但程序本身可以执行，并不是“不能正确执行”。因此题干说法错误。

## 第8题

```cpp
char a='A';
a=a+32;
cout<<(int)a<<endl;
```

将输出 97

【知识点：ASCII/字符编码】

答案：正确

解析：字符 `A` 的 ASCII 编码是 $65$，执行 `a = a + 32` 后编码变为 $97$，对应小写字母 `a`。输出时写了 `(int)a`，所以输出的是字符编码数值 $97$，而不是字符本身。题干正确。

## 第9题

自然界中，最小的素数是 2。

【知识点：枚举与基础算法】

答案：正确

解析：素数是大于 $1$ 且只有 $1$ 和自身两个正因数的自然数。$1$ 不是素数，$2$ 的正因数只有 $1$ 和 $2$，并且是最小的满足条件的自然数。因此题干正确。

## 第10题

CCF（十六进制） = 12363（七进制）

【知识点：计算机基础常识】

答案：正确

解析：$CCF_{16}=12\times16^2+12\times16+15=3279$。再计算 $12363_7=1\times7^4+2\times7^3+3\times7^2+6\times7+3=2401+686+147+42+3=3279$。两边转换成十进制后相等，所以题干正确。

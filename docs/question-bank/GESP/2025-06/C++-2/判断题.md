## 第10题

为实现如下效果，即 $N$ 行 $N$ 列字符。当输入是奇数时，中间列为 `*`，其他是 `-`；当输入是偶数时，则中间两列为 `*`，其他是 `-`。字符串列后的代码能实现其效果。（ ）

```cpp
/*
5
--*--
--*--
--*--
--*--
--*--
*/
```

```cpp
int N;
cin >> N;

int i, j;
for (i = 0; i < N; i++){
    for (j = 0; j < N; j++)
        if ((j == N / 2) || (j == (N-1) / 2))
            cout << "*";
        else
            cout << "-";
    cout << endl;
}
```

答案：正确

解析：题干表述成立。输入输出题要注意格式控制符、转义字符、整数除法和输出语句中字面量与变量的区别。 将题干条件代入相关规则后不会产生矛盾，因此判断为正确。

## 第1题

2025 年 4 月 19 日在北京举行了一场颇为瞩目的人形机器人半程马拉松赛。比赛期间，跑动着的机器人会利用身上安装的多个传感器所反馈的数据来调整姿态、保持平衡等，那么这类传感器类似于计算机的（ ）。

A. 处理器
B. 存储器
C. 输入设备
D. 输出设备

答案：C

解析：传感器的作用是把机器人身体姿态、速度、周围环境等信息收集进计算机系统，让程序据此调整动作。这和键盘、鼠标一样，都是把外部信息送入计算机，所以属于输入设备。处理器负责计算，存储器负责保存数据，输出设备负责把结果显示或发出，都不符合“反馈数据”的作用。



## 第2题

小明购置的计算机使用一年后觉得内存不够用了，想购置一个容量更大的内存条，这时他需要的内存条是（ ）。

A. RAM
B. ROM
C. CACHE
D. EPROM

答案：A

解析：平时说“加内存条”，指的是给计算机增加主存容量，也就是 RAM。RAM 可以被程序频繁读写，断电后数据通常会丢失。ROM、EPROM 偏向只读存储，CACHE 是速度很快但容量较小的高速缓存，都不是常说的内存条。



## 第3题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int a=3;
float b = 3.5;
cout << (a *= b);
```

A. 3
B. 3.5
C. 10
D. 11

答案：C

解析：`a *= b` 等价于 `a = a * b`。先计算右边，`3 * 3.5 = 10.5`；但 `a` 是 `int` 类型，赋值回 `a` 时会舍去小数部分，得到 `10`。最后输出的就是新的 `a`，所以结果为 `10`。



## 第4题

下面 C++ 代码用于获得正整数的第 3 位数，如 $1234$ 则输出 $2$。如果是一位数或两位数，则输出 $0$。横线处应填入的代码是（ ）。

```cpp
int N, remainder;
cout << "请输入正整数:";
cin >> N;
cout << _______________;
```

A. `N % 1000 / 100`
B. `N / 1000 % 100`
C. `N / 1000 / 100`
D. `N % 100 / 100`

答案：A

解析：题目中的“第 3 位数”按例子 $1234$ 输出 $2$，也就是百位。`N % 1000` 先保留最后三位，$1234$ 变成 $234$；再做整数除法 `/ 100`，$234 / 100 = 2$。如果 `N` 是一位数或两位数，`N / 100` 的整数结果就是 `0`。



## 第5题

下面 C++ 代码执行，其输出是（ ）。

```cpp
int a, b = (6, 28);
b == a;
a = b;
cout << a << ' ' << b;
```

A. 6 28
B. 6 6
C. 28 6
D. 28 28

答案：D

解析：`(6, 28)` 是逗号表达式，它会依次计算 `6` 和 `28`，整个表达式的值取最后一个，也就是 `28`，所以 `b` 被初始化为 `28`。`b == a;` 只是做一次相等比较，不是赋值语句，不会把 `b` 的值放进 `a`。后面的 `a = b;` 才是真正赋值，因此最后 `a` 和 `b` 都是 `28`。实际写程序时，比较前应先初始化 `a`，本题重点是区分 `==` 和 `=`。



## 第6题

今天星期六，其后第 $N$ 天星期几？如果是星期一到星期六输出形如：星期 1、星期 2 等，星期天则输出星期天。下面的 C++ 代码用于完成上述要求，横线处应填上的代码是（ ）。

```cpp
int N, remainder;
cin >> N;
remainder = ___________;
if(remainder == 0)
    printf("星期六后第%d天是星期天\n", N);
else
    printf("星期六后第%d天是星期%d\n", N, remainder);
```

A. `(N + 6) / 7`
B. `(N + 6) // 7`
C. `N % 7`
D. `(N + 6) % 7`

答案：D

解析：代码约定余数为 `0` 时输出“星期天”，余数为 `1` 到 `6` 时输出“星期1”到“星期6”。今天是星期六，过 $1$ 天是星期天，应该得到 `0`；过 $2$ 天是星期一，应该得到 `1`。`(N + 6) % 7` 正好满足这个对应关系，例如 $N=1$ 时为 $0$，$N=2$ 时为 $1$。



## 第7题

下面的 C++ 代码执行后其输出是（ ）。

```cpp
int i, Sum = 0;
for (i = 1; i < 10; i++) {
    Sum += i;
    if(i % 2) continue;
    if(i % 7) break;
}
cout << Sum;
```

A. 45
B. 28
C. 3
D. 0

答案：C

解析：循环从 `i = 1` 开始，先执行 `Sum += i`，所以 `Sum` 先变成 `1`。因为 `1 % 2` 为真，执行 `continue`，进入下一轮。`i = 2` 时先累加，`Sum = 1 + 2 = 3`；此时 `2 % 2` 为假，不执行 `continue`，再判断 `2 % 7` 为真，于是 `break` 结束循环。输出的就是 `3`。



## 第8题

下面 C++ 代码执行后其输出是（ ）。

```cpp
int i, j;
for(i = 1; i < 12; i++)
    for(j = 1; j < i; j++)
        if(i * j % 2 == 1)
            break;
cout << i * j;
```

A. 110
B. 22
C. 12
D. 3

答案：C

解析：外层循环没有花括号，内层 `for` 是它的循环体。`break` 只会跳出内层循环，不会直接跳出外层循环。对奇数 `i`，当 `j = 1` 时 `i * j` 是奇数，内层立刻 `break`；对偶数 `i`，内层会一直跑到 `j == i` 才结束。外层最后一次有效取值是 `i = 11`，这时内层在 `j = 1` 处退出；随后外层 `i++` 变成 `12`，条件 `i < 12` 不成立。此时 `i = 12`、`j = 1`，所以输出 `12`。



## 第9题

下面 C++ 代码执行后输出是（ ）。

```cpp
int i, cnt = 0;
for(i = -99; i < 100; i += 2)
    cnt = 1 + cnt;
cout << cnt;
```

A. 101
B. 100
C. 99
D. 98

答案：B

解析：`i` 从 `-99` 开始，每次加 `2`，依次是 `-99, -97, ..., 97, 99`，都是小于 `100` 的奇数。首项到末项相差 $99 - (-99) = 198$，步长是 $2$，一共有 $198 / 2 + 1 = 100$ 个数。循环每经过一个数就让 `cnt` 加 `1`，因此输出 `100`。



## 第10题

下面 C++ 代码执行后输出是（ ）。

```cpp
int i;
for(i = 1; i < 10; i++) {
    if(i % 3 != 0) {
        printf("A#");
        continue;
    }
    else
        break;
    printf("0#");
}
if(i == 10) cout << "1";
```

A. A#A#
B. A#0#A#0
C. A#A#1
D. A#0#A#0#1

答案：A

解析：`i = 1` 时，`1 % 3 != 0` 成立，输出 `A#`，然后 `continue` 跳到下一轮，下面的 `printf("0#")` 不会执行。`i = 2` 时同样输出一次 `A#`。`i = 3` 时条件不成立，进入 `else` 执行 `break`，循环结束。此时 `i` 不是 `10`，最后的 `cout << "1"` 也不会执行，所以总输出是 `A#A#`。



## 第11题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int i, j;
for(i = 0; i < 3; i++)
    for(j = 0; j < i; j++)
        printf("%d#%d-", i, j);
printf("END");
```

A. 0#0-1#0-2#0-2#1-END
B. 0#0-1#0-1#1-2#0-2#1-2#2-3#0-3#1-3#2-END
C. 0#0-1#0-1#1-2#0-2#1-2#2-END
D. 1#0-2#0-2#1-END

答案：D

解析：外层 `i = 0` 时，内层条件 `j < i` 也就是 `0 < 0` 不成立，不输出。`i = 1` 时，`j` 只取 `0`，输出 `1#0-`。`i = 2` 时，`j` 取 `0` 和 `1`，输出 `2#0-2#1-`。外层循环结束后再输出 `END`，合起来是 `1#0-2#0-2#1-END`。



## 第12题

下面 C++ 代码执行后，将输出不能被 3 整除且除以 5 余数为 2 的数。下列选项不能实现的是（ ）。

```cpp
int i, j;
for(i = 0; i < 100; i++)
    if(__________________)
        cout << i << endl;
```

A. `(i % 3 != 0) && (i % 5 == 2)`
B. `(i % 3) && (i % 5 == 2)`
C. `(i % 3) && !(i % 5 != 2)`
D. `!(i % 3) && (i % 5 == 2)`

答案：D

解析：题目要求输出“不能被 `3` 整除”并且“除以 `5` 余 `2`”的数。`i % 3 != 0` 表示不能被 `3` 整除；在 C++ 中，`i % 3` 作为条件时，只要余数不是 `0` 就为真，所以 B 项也能表示不能被 `3` 整除；`!(i % 5 != 2)` 等价于 `i % 5 == 2`。D 项中的 `!(i % 3)` 等价于 `i % 3 == 0`，表示能被 `3` 整除，和要求相反，因此不能实现。



## 第13题

下面 C++ 代码用于判断一个大于 $0$ 的正整数是几位数，横线处应填入代码先后是（ ）。

```cpp
int N, cnt;
cout << "请输入大于0的正整数：";
cin >> N;

cnt = 0;
while (______){
    cnt += 1;
    ________;
}
cout << cnt;
```

A.
```cpp
N > 1
N = N / 10
```

B.
```cpp
N > 1
N /= 10
```

C.
```cpp
N == 0
N /= 10
```

D.
```cpp
N > 0
N /= 10
```

答案：D

解析：统计位数时，每次去掉当前数字的个位，并把计数器加 `1`。例如 `N = 123`：第一次计数后 `N` 变成 `12`，第二次变成 `1`，第三次变成 `0`，共计 `3` 位。所以循环条件应为 `N > 0`，循环内应写 `N /= 10`。如果用 `N > 1`，输入 `1` 时会得到 `0` 位；如果用 `N == 0`，正整数一开始不会进入循环。



## 第14题

判断一个数是否为自守数。自守数的定义是如果一个数的平方其尾数与该数相同，则为自守数，如 $25$ 的平方是 $625$，其尾数是 $25$，所以 $25$ 是自守数。相关说法错误的是（ ）。

```cpp
int N, N1, M1;
cout << "输入一个正整数：";
cin >> N;
N1 = N, M1 = N * N;

bool Flag = true;

while (N1 > 0) {
    if (N1 % 10 != M1 % 10) {
        Flag = false;
        break;
    }
    else {
        N1 = N1 / 10, M1 = M1 / 10;
    }
}

if (Flag == true)
    printf("%d的平方是%d,是自守数", N, N * N);
else
    printf("%d的平方是%d,不是自守数", N, N * N);
```

A. 如果 Flag 在循环中不被改为 false，则说明该数是自守数
B. 代码 `if (N1 % 10 != M1 % 10)` 用于判断其个位数是否相等，如果不等，则表明不是自守数
C. 代码 `N1 = N1 / 10, M1 = M1 / 10` 将个位数去掉
D. 将 `N1 > 0` 改为 `N > 0` 效果相同

答案：D

解析：这段程序用 `N1` 保存原数的副本，用 `M1` 保存平方，然后从个位开始一位一位比较。每比较完一位，`N1 = N1 / 10, M1 = M1 / 10` 会同时去掉两边的个位；当 `N1` 变成 `0`，说明原数的所有位都比较完了。如果把循环条件改成 `N > 0`，`N` 在循环中从来没有改变，对于正整数会一直为真，循环不能按“原数位数比较完”正常结束，效果当然不同。



## 第15题

下面 C++ 代码实现输出如下图形，相关说法错误的是（ ）。

```
0
12
345
6789
01234
567890
1234567
89012345
678901234
5678901234
```

```cpp
int line_number, now_number;
int i, row;

cout << "请输入行数：";
cin >> line_number;
now_number = 0;

for (row = 1; row < line_number + 1; row++) { // L1
    for (i = 0; i < row; i++) { // L2
        cout << now_number;
        now_number += 1;
        if (now_number == 10) now_number = 0; // L3
    }
    cout << endl;
}
```

A. 代码 `now_number = 0` 移动到 L1 和 L2 标记的两行代码之间，效果维持不变
B. 代码 `now_number += 1` 修改为 `now_number = 1 + now_number` 效果维持不变
C. 将代码 `now_number == 10` 调整为 `now_number > 9` 效果维持不变
D. 将最后一行的 `cout << endl` 修改为 `cout << "\n"`，效果维持不变

答案：A

解析：原程序的 `now_number` 只在所有循环开始前设为 `0`，所以数字会跨行连续增加：第一行输出 `0`，第二行接着输出 `12`，第三行接着输出 `345`。如果把 `now_number = 0` 放到 L1 和 L2 之间，它会在每一行开始前都重置为 `0`，每行都会从 `0` 重新输出，图形就变成 `0`、`01`、`012` 这类效果。B 项只是把加一写成等价形式，C 项对本程序中加到 `10` 后归零的判断等价，D 项中 `endl` 和 `"\n"` 都能换行。

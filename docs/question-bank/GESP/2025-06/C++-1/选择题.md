## 第1题

2025 年 4 月 19 日在北京举行了一场颇为瞩目的人形机器人半程马拉松赛。比赛期间，跑动着的机器人会利用身上安装的多个传感器所反馈的数据来调整姿态、保持平衡等，那么这类传感器类似于计算机的（ ）。

A. 处理器
B. 存储器
C. 输入设备
D. 输出设备

答案：C

解析：传感器的作用是把外界信息交给机器人，比如姿态、速度、受力、周围环境等数据。计算机收到这些数据后才能继续处理，所以传感器更像键盘、鼠标、摄像头这类“把信息送进去”的输入设备。处理器负责计算，存储器负责保存数据，输出设备负责把结果显示或发出，因此选 C。



## 第2题

在某集成开发环境中调试下面代码段时尝试设置断点和检查局部变量，下面哪个说法是错误的（ ）。

```cpp
int i,N = 0; // L1
cin >> N; // L2
for (i = 1; i < 9; i++)
    if (N % i == 0) break;  // L3
if (i < 9)
    printf("N不能大于9\n"); // L4
```

A. 断点不可以设在 L1 标记的代码行
B. 执行暂停在 L2 标记的代码行时，可以检测 i 的值
C. 执行暂停在 L3 标记的代码行时，可以修改 i 的值
D. 执行有可能暂停在 L4 标记的代码行

答案：A

解析：断点通常可以设在会执行的语句行上，L1 是变量定义并初始化 `N = 0` 的语句，可以作为暂停位置，所以 A 的说法错误。程序暂停在 L2 时，`i` 已经定义在同一作用域中，调试器可以查看它；暂停在 L3 时，也可以在调试器里修改 `i` 的值；如果循环中发现某个 `i` 能整除 `N` 并执行 `break`，后面的 L4 也可能被执行到。因此应选 A。



## 第3题

对下列 C++ 的代码，描述准确的是（ ）。

```cpp
int first = 10;
printf("{%d}\n", First)
```

A. 执行后输出 $10$
B. 执行后输出 {First}
C. 执行后输出 "{First}"
D. 编译报错，因为 First 应该是 first

答案：D

解析：C++ 区分大小写，`first` 和 `First` 是两个不同的名字。代码中只定义了 `int first = 10;`，没有定义名为 `First` 的变量，`printf` 里使用 `First` 时编译器找不到这个标识符，会报错。题目要找准确描述，所以选 D。



## 第4题

在 C++ 中，下列可以做变量名的是（ ）。

A. X.cpp
B. X-cpp
C. X#cpp
D. X_cpp

答案：D

解析：C++ 变量名可以由字母、数字和下划线组成，并且不能以数字开头。`X_cpp` 只包含字母和下划线，可以做变量名。`X.cpp` 含有点号，`X-cpp` 含有减号，`X#cpp` 含有井号，这些字符都不能出现在普通变量名中，所以选 D。



## 第5题

C++ 表达式 `14 - 3 * 3 % 2` 的值是（ ）。

A. $0$
B. $11$
C. $13$
D. $-67$

答案：C

解析：乘法 `*` 和取余 `%` 的优先级高于减法 `-`，并且同一优先级从左到右计算。先算 `3 * 3 = 9`，再算 `9 % 2 = 1`，最后算 `14 - 1 = 13`。所以表达式的值是 $13$，选 C。



## 第6题

下面的 C++ 代码执行后，其输出是（ ）。

```cpp
int x = 10, y = 20;
x = x + y;
y = x - y;
x = x - y;
cout << x << ' ' << y;
```

A. $10$ $20$
B. $20$ $10$
C. $10$ $10$
D. $20$ $20$

答案：B

解析：开始时 `x = 10`，`y = 20`。执行 `x = x + y` 后，`x = 30`，`y` 仍是 $20$；执行 `y = x - y` 后，`y = 30 - 20 = 10`；执行 `x = x - y` 后，`x = 30 - 10 = 20`。最后输出 `x` 和 `y`，结果是 `20 10`，选 B。



## 第7题

定义整型变量 `int a = 16`，则执行 `++a += 3` 之后，a 的值会是（ ）。

A. $3$
B. $17$
C. $19$
D. $20$

答案：D

解析：`++a` 是前置自增，先把 `a` 从 $16$ 加到 $17$，并且在 C++ 中这个结果还可以继续作为被赋值的对象。`++a += 3` 可以理解为先得到自增后的 `a`，再给它加上 $3$，所以 `a` 变成 $17 + 3 = 20$。答案是 D。



## 第8题

C++ 的 `int` 类型变量 X 的值为 8，如果执行 `cout << (++X)++;`，则输出和执行后 X 的值分别是（ ）。

A. $8$ $9$
B. $9$ $9$
C. $9$ $10$
D. 编译错误，无法执行

答案：C

解析：初始 `X = 8`。表达式里的 `++X` 是前置自增，先把 `X` 变成 $9$；外层的后置 `++` 会先使用当前值再自增，所以 `cout` 输出的是 $9$，随后 `X` 再变成 $10$。因此输出和执行后 `X` 的值分别是 $9$、$10$，选 C。



## 第9题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int a, b;
a = 3;
b = 4;
printf("a+b=%02d#a+b={a+b}", a + b, a + b);
```

A. a+b=07#a+b={a+b}
B. a+b= 7#a+b=7
C. a+b=7#a+b={a+b}
D. a+b=7#a+b=7

答案：A

解析：`a + b = 3 + 4 = 7`。格式 `"%02d"` 表示按至少 $2$ 位输出整数，不足两位时前面补 `0`，所以第一部分输出 `a+b=07`。后面的 `#a+b={a+b}` 在格式字符串里没有对应的 `%d`，花括号中的 `a+b` 只是普通文字，会原样输出；第二个 `a + b` 实参没有被使用。最终是 `a+b=07#a+b={a+b}`，选 A。



## 第10题

下面的 C++ 代码用于求 $M$ 天后是星期几，两处横线处分别应填入的代码是（ ）。

```cpp
int N, M, D;
cout << "当前星期几？"; // 星期日是0
cin >> N;
cout << "求多少天后？"; // 输入正整数
cin >> M;
D = ____________;
if (___________)
    printf("%d天后是星期日", M);
else
    printf("%d天后是星期%d", M, D);
```

A.
```cpp
(N + M) / 7
D == 0
```

B.
```cpp
(N + M) % 7
D == 0
```

C.
```cpp
(N + M) / 7
D <= 0
```

D.
```cpp
(N + M) % 7
D = 0
```

答案：B

解析：星期编号每 $7$ 天循环一次，所以 $M$ 天后的星期应为 `(N + M) % 7`，不能用除法 `/`。如果算出的 `D` 等于 $0$，说明是星期日，需要执行第一条输出语句，因此判断条件应写成 `D == 0`。`D = 0` 是赋值，不是比较，所以正确填法是 B。



## 第11题

下面 C++ 代码执行后输出是（ ）。

```cpp
int i;
for (i = 1; i < 11; i += 3) {
    continue;
    if (i % 2 == 0)
        break;
    printf("%d#", i);
}
if (i >= 11)
    printf("END");
```

A. END
B. 1#
C. 1#4#END
D. 1#4#7#10#END

答案：A

解析：循环中第一句就是 `continue;`，它会立刻跳到本轮循环的结尾，后面的 `if`、`break` 和 `printf("%d#", i)` 都不会执行。因此循环过程中不会输出 `1#`、`4#` 等数字。`i` 依次为 $1,4,7,10$，再加 $3$ 后变成 $13$，循环结束，此时 `i >= 11` 成立，所以只输出 `END`，选 A。



## 第12题

下面的 C++ 代码用于求 $N$ 的所有因数（即能将 $N$ 整除的数），如输入 $12$ 则将输出 $12,6,4,3,2,1$。（ ）。

```cpp
int i, N;
cin >> N;
i = N;
while (______) {
    if (N % i == 0)
        printf("%d,", i);
    i -= 1;
}
printf("1");
```

A. i -= 1
B. i == 1
C. i > 1
D. i >= 1

答案：C

解析：程序把 `i` 从 `N` 开始往下试，如果 `N % i == 0` 就输出这个因数。循环里已经会输出从 `N` 到 $2$ 之间的因数，循环结束后又单独 `printf("1")` 输出因数 $1$，所以循环条件应该让 `i` 大于 $1$ 时继续，即 `i > 1`。如果写 `i >= 1`，$1$ 会在循环里输出一次，最后又输出一次；如果写 `i == 1`，一开始通常进不了循环。答案是 C。



## 第13题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int Sum = 0;
for (int i = 0; i < 10; i++) {
    if (i % 2 == 0)
        continue;
    if (i % 5 == 0)
        break;
    Sum += i;
}
cout << Sum;
```

A. $55$
B. $15$
C. $9$
D. $4$

答案：D

解析：循环从 `i = 0` 到 `i = 9`。偶数遇到 `i % 2 == 0` 会执行 `continue`，不累加；`i = 1` 时累加，`Sum = 1`；`i = 3` 时累加，`Sum = 4`；`i = 5` 是奇数，但满足 `i % 5 == 0`，先执行 `break` 退出循环，不会把 $5$ 加进去。最后输出 $4$，选 D。



## 第14题

试图编译并执行下面 C++ 代码，下面描述正确的是（ ）。

```cpp
float x;
x = 101;
x++;
cout << ++x;
```

A. 输出 $101$
B. 输出 $102$
C. 输出 $103$
D. 编译将报错，无法执行

答案：C

解析：`float` 类型变量可以参与自增运算。先执行 `x = 101`，然后 `x++` 使 `x` 变成 $102$；输出语句里的 `++x` 是前置自增，先把 `x` 变成 $103$，再输出这个值。因此程序可以编译执行，输出 $103$，选 C。



## 第15题

以下 C++ 代码可以找出百位、十位和个位满足特定条件的三位数，横线处应该填入的是（ ）。

```cpp
int count = 0;
for (int i = 100; i <= 999; i++) {
    int a = i / 100;
    _______________________
    int c = i % 10;
    if (a * a + b * b == c * c) {
        count++;
    }
}
```

A. `int b = (i / 10) / 10;`
B. `int b = (i / 10) % 10;`
C. `int b = (i % 10) / 10;`
D. `int b = (i % 10) % 10;`

答案：B

解析：三位数 `i` 的百位已经用 `i / 100` 得到，个位用 `i % 10` 得到。要取十位，可以先用 `i / 10` 去掉个位，再对 $10$ 取余得到新的个位，也就是原数的十位。例如 `i = 345` 时，`i / 10 = 34`，`34 % 10 = 4`。所以应填 `int b = (i / 10) % 10;`，选 B。

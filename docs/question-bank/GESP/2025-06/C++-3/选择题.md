## 第11题

以下程序使用枚举法（穷举法）求解满足条件的三位数，横线处应该填入的是（ ）

```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 0;
    for (int i = 100; i <= 999; i++) {
        int a = i / 100;
        // 横线处
        int c = i % 10;
        if (a * a + b * b == c * c) {
            count++;
        }
    }
    cout << count << endl;
    return 0;
}
```

【知识点：枚举与基础算法】

A. `int b = (i / 10) / 10;`
B. `int b = (i / 10) % 10;`
C. `int b = (i % 10) / 10;`
D. `int b = (i % 10) % 10;`

答案：B

## 第12题

```cpp
#include <iostream>
using namespace std;

int main() {
    int height = 10;
    int distance = 0;
    for (int i = 1; i <= 5; i++) { // 模拟5次落地
        ________________
        height /= 2;
        distance += height;
    }
    cout << distance << endl;
    return 0;
}
```

【知识点：控制结构与程序阅读】

A. `distance += height/2;`
B. `distance += height;`
C. `distance += height*2;`
D. `distance += height+1;`

答案：B

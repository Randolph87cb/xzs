## 第1题

8 位二进制原码能表示的最小整数是：（ ）

【知识点：原码反码补码与整数表示】

A. -127
B. -128
C. -255
D. -256

答案：A

解析：8 位原码中最高位是符号位，后面 $7$ 位表示数值大小。数值位最大是 $1111111_2=127$，所以能表示的负整数最小为 $-127$。原码还有 $+0$ 和 $-0$ 两种零的形式，因此不能再多表示一个 $-128$；$-128$ 是 8 位补码能表示的最小值。



## 第2题

反码表示中，零的表示形式有：

【知识点：原码反码补码与整数表示】

A. 1 种
B. 2 种
C. 8 种
D. 16 种

答案：B

解析：在反码中，正数和 $+0$ 与原码相同，所以 $+0$ 写作 $0000\ 0000$。负数反码是符号位为 $1$、数值位取反，因此 $-0$ 写作 $1111\ 1111$。同一个数值零有 $+0$ 和 $-0$ 两种编码，所以答案是 $2$ 种。



## 第3题

补码 $1011\ 1011$ 对应的真值是（ ）

【知识点：原码反码补码与整数表示】

A. -69
B. -59
C. -68
D. -58

答案：A

解析：补码 $1011\ 1011$ 的最高位是 $1$，表示这是负数。求它的绝对值可以先按位取反得到 $0100\ 0100$，再加 $1$ 得到 $0100\ 0101$，也就是十进制 $64+4+1=69$。因此这个补码对应的真值是 $-69$。



## 第4题

若 $X$ 的 $8$ 位补码为 $0000\ 1010$，则 $X/2$ 的补码是（ ）。

【知识点：原码反码补码与整数表示】

A. $0000\ 0101$
B. $1000\ 0101$
C. $0000\ 0101$ 或 $1000\ 0101$
D. 算术右移后结果取决于符号位

答案：A

解析：补码 $0000\ 1010$ 的最高位是 $0$，说明 $X$ 是正数，数值就是十进制 $10$。$X/2=5$，正数的补码和普通二进制写法相同，$5$ 的 8 位表示为 $0000\ 0101$。选项 B 的最高位为 $1$，表示负数，不可能是 $5$ 的补码。



## 第5题

二进制数 $1101.101$ 对应的十进制数是（ ）

【知识点：进制转换与进制字面量】

A. 13.625
B. 12.75
C. 11.875
D. 14.5

答案：A

解析：按二进制位权展开：整数部分 $1101_2=1\times8+1\times4+0\times2+1\times1=13$；小数部分 $.101_2=1\times\frac12+0\times\frac14+1\times\frac18=0.625$。合起来是 $13+0.625=13.625$。



## 第6题

补码加法中，若符号位无进位而次高位有进位，则说明（ ）

【知识点：原码反码补码与整数表示】

A. 结果正确
B. 发生上溢
C. 发生下溢
D. 结果符号位错误

答案：B

解析：补码加法判断溢出时，要比较“进入符号位的进位”和“从符号位产生的进位”。次高位有进位，说明有进位进入符号位；符号位无进位，说明没有进位从符号位送出。两者不同，表示发生溢出，并且这种情况对应正数相加超过可表示的最大正数，即上溢。



## 第7题

八进制数 $35.6$ 对应的十进制数是（ ）

【知识点：进制转换与进制字面量】

A. 29.75
B. 28.5
C. 27.625
D. 30.25

答案：A

解析：八进制每一位的权值是 $8$ 的幂。$35.6_8=3\times8^1+5\times8^0+6\times8^{-1}=24+5+\frac68$。其中 $\frac68=0.75$，所以结果是 $29.75$。



## 第8题

二进制数 $1010 \mid 1100$ 的结果是（ ）

【知识点：进制转换与进制字面量】

A. 1000
B. 1110
C. 1010
D. 1100

答案：B

解析：按位或 `|` 的规则是：同一位上只要有一个 $1$，结果就是 $1$；两个都是 $0$ 时结果才是 $0$。逐位计算：$1|1=1$，$0|1=1$，$1|0=1$，$0|0=0$，所以 $1010\ |\ 1100=1110$。



## 第9题

以下哪个位运算可以交换两个变量的值（无需临时变量）（ ）

【知识点：位运算】

A. `a = a ^ b; b = a ^ b; a = a ^ b;`
B. `a = a & b; b = a | b; a = a & b;`
C. `a = a | b; b = a ^ b; a = a ^ b;`
D. `a = ~a; b = ~b; a = ~a;`

答案：A

解析：异或有两个重要性质：$x\oplus x=0$，$x\oplus0=x$。设原来的值是 $a_0$ 和 $b_0$，第一步后 `a` 变成 $a_0\oplus b_0$；第二步 `b = a ^ b`，得到 $(a_0\oplus b_0)\oplus b_0=a_0$；第三步 `a = a ^ b`，得到 $(a_0\oplus b_0)\oplus a_0=b_0$。这样 `a` 和 `b` 的值就交换了。



## 第10题

如何正确定义一个长度为 5 的整型数组（ ）

【知识点：数组与字符数组】

A. `int array = new int[5];`
B. `array int[5];`
C. `int[] array = {1,2,3,4,5};`
D. `int array[5];`

答案：D

解析：C++ 中定义普通数组的格式是“元素类型 数组名[长度]”，所以长度为 $5$ 的整型数组应写成 `int array[5];`。A 项把 `new int[5]` 得到的指针赋给普通 `int`，类型不匹配；B 项顺序错误；C 项是其他语言常见写法，不是 C++ 普通数组的声明格式。



## 第11题

以下程序使用枚举法（穷举法）求解满足条件的三位数，横线处应该填入的是（ ）

```cpp
#include <iostream>
using namespace std;

int main() {
    int count = 0;
    for (int i = 100; i <= 999; i++) {
        int a = i / 100;
        // 横线处
        int c = i % 10;
        if (a * a + b * b == c * c) {
            count++;
        }
    }
    cout << count << endl;
    return 0;
}
```

【知识点：枚举与基础算法】

A. `int b = (i / 10) / 10;`
B. `int b = (i / 10) % 10;`
C. `int b = (i % 10) / 10;`
D. `int b = (i % 10) % 10;`

答案：B

解析：三位数 `i` 的百位已经用 `i / 100` 得到，个位用 `i % 10` 得到。十位要先去掉个位，即 `i / 10`，再对 $10$ 取余，保留新的个位，所以应写 `int b = (i / 10) % 10;`。例如 $i=345$ 时，$345/10=34$，$34\bmod10=4$，正好是十位。



## 第12题

以下程序模拟了一个简单的小球反弹过程，横线处应该填入的是（ ）

```cpp
#include <iostream>
using namespace std;

int main() {
    int height = 10;
    int distance = 0;
    for (int i = 1; i <= 5; i++) { // 模拟5次落地
        ________________
        height /= 2;
        distance += height;
    }
    cout << distance << endl;
    return 0;
}
```

【知识点：控制结构与程序阅读】

A. `distance += height/2;`
B. `distance += height;`
C. `distance += height*2;`
D. `distance += height+1;`

答案：B

解析：循环每次表示小球从当前高度落到地面一次，所以总路程要先加上这段下落距离 `height`。随后 `height /= 2;` 表示反弹到原来高度的一半，再用 `distance += height;` 加上这段上升距离。横线处如果写 `height/2`，就会把下落距离少算一半；写 `height*2` 又会把下落和上升混在一起多算。



## 第13题

C++ 代码 `string s = "GESP考试";`，s 占据的字节数是（ ）

【知识点：字符串处理】

A. 10
B. 8
C. 8 或 10
D. 取决于计算机采用什么编码

答案：D

解析：`string` 保存的是字节序列，不会自动按“汉字个数”来计算。`G`、`E`、`S`、`P` 这 $4$ 个英文字符通常各占 $1$ 字节；“考”“试”在不同编码中占用的字节数可能不同，例如 UTF-8 中通常每个汉字占 $3$ 字节，GBK 中通常每个汉字占 $2$ 字节。因此 `s` 占据的字节数取决于采用的编码。



## 第14题

C++ 语句 `string s="Gesp Test";` 执行 `s.rfind("e")` 以后，输出的是（ ）

【知识点：字符串处理】

A. 1
B. 2
C. 6
D. 3

答案：C

解析：`rfind("e")` 会从字符串右侧开始找最后一次出现的 `"e"`，返回它的下标。`"Gesp Test"` 的下标依次是：`G` 为 $0$，第一个 `e` 为 $1$，`s` 为 $2$，`p` 为 $3$，空格为 $4$，`T` 为 $5$，最后一个 `e` 为 $6$。所以返回值是 $6$。



## 第15题

字符串 "Gesp考试"，字符数是（ ）

【知识点：字符串处理】

A. 10
B. 8
C. 6
D. 字符数多少取决于编码

答案：C

解析：题目问的是字符数，不是 `string` 占用的字节数。`"Gesp考试"` 可以按字符数成 `G`、`e`、`s`、`p`、`考`、`试`，一共 $6$ 个字符。不同编码会影响汉字占几个字节，但不会把“考”“试”这两个字符变成不同数量的字符。

# 判断题

来源：
- 洛谷有题页：https://ti.luogu.com.cn/problemset/1188
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1687196088139808.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

C++ 语言中，表达式 `9 | 12` 的结果类型为 int、值为 $13$。

答案：正确

## 第2题

C++ 语言中，访问数据发生下标越界时，总是会产生运行时错误，从而使程序异常退出。

答案：错误

## 第3题

对 $n$ 个元素的数组进行归并排序，最差情况的时间复杂度为 $O(n \log n)$。

答案：正确

## 第4题

$5$ 个相同的红球和 $4$ 个相同的蓝球排成一排，要求每个蓝球的两侧都必须至少有一个红球，则一共有 $15$ 种排列方案。

答案：错误

## 第5题

使用 `math.h` 或 `cmath` 头文件中的函数，表达式 `log(8)` 的结果类型为 double、值约为 $3$。

答案：错误

## 第6题

C++ 是一种面向对象编程语言，C 则不是。继承是面向对象三大特性之一，因此，使用 C 语言无法实现继承。

答案：错误

## 第7题

$n$ 个顶点的无向完全图，有 $n^{n-2}$ 棵生成树。

答案：正确

## 第8题

已知三个 double 类型的变量 `a`、`b` 和 `theta` 分别表示一个三角形的两条边长及二者的夹角（弧度），则三角形的周长可以通过表达式 `sqrt(a * a + b * b - 2 * a * b * cos(theta))` 求得。

答案：错误

## 第9题

有 $V$ 个顶点、$E$ 条边的图的深度优先搜索遍历时间复杂度为 $O(V + E)$。

答案：正确

## 第10题

从 $32$ 名学生中选出 $4$ 人分别担任班长、副班长、学习委员和组织委员，老师要求班级综合成绩排名最后的 $4$ 名学生不得参选班长或学习委员（仍可以参选副班长和组织委员），则共有 $P(30, 4)$ 种不同的选法。

答案：正确

## 第1题

若函数声明为 `int f(int &x){ x+=3; return x; }`，则对声明的变量 `int a=3`，下面哪个调用能够改变 $a$ 的值（ ）。

A. `f(&a);`
B. `f(*a);`
C. `f(a);`
D. `f(a-3);`

答案：C

## 第8题

下面 C++ 函数中采用的算法是（ ）。

```cpp
int fib(int n)
{
    int i, f[n] = {0, 1};

    for (int i = 2; i <= n; i++)
        f[i] = f[i - 1] + f[i - 2];

    return f[n];
}
```

A. 递推
B. 递归
C. 迭代
D. 循环

答案：A

## 第10题


B. $X = Y$
C. $X < Y$
D. 不确定

## 第11题

第11题 如果文件 `1.txt` 中的内容如下，则执行下面 C++ 代码时，注释了 `####` 那行代码所输出的 $x$ 的值为（ ）。

```
50 2024 3.16 I
love
GESP!
```

```cpp
int main()
{
    ifstream fin;
    string line;
    int x;
    fin.open("1.txt", ios::in);

    for (int i = 0; i < 2; i++) {
        fin >> line;
        cout << line << endl;
    }
    fin >> x;
    cout << x << endl; //####
    
    cout << endl;
    return 0;
}
```

A. $5$
B. $2024$
C. $3$
D. $0$

答案：C

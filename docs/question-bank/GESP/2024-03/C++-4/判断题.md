## 第1题

对 `int a[] = {2, 0, 2, 4, 3, 1, 6}`，执行第一趟选择排序处理后 `a` 中数据变为 `{0, 2, 2, 4, 3, 1, 6}`。（ ）

答案：正确

解析：选择排序的第一趟会在整个未排序区间中找到最小元素，并把它放到第一个位置。数组 `{2, 0, 2, 4, 3, 1, 6}` 的最小值是 $0$，位于原数组第 $1$ 个下标处；第一趟把它与首元素 $2$ 交换后，数组变为 `{0, 2, 2, 4, 3, 1, 6}`。题干给出的变化符合选择排序第一趟处理结果，所以正确。

## 第2题

如果待排序数据不能都装进内存，需要使用外排序算法。（ ）

答案：正确

解析：内排序要求待排序数据能主要放在内存中完成比较和移动；当数据规模太大，不能一次性全部装入内存时，需要借助磁盘等外部存储分批处理、归并，这类方法称为外排序。题干描述的正是外排序的适用场景，所以正确。

## 第3题

定义变量 `int a = 5`，则 `cout << &++a` 会输出 $6$。（ ）

答案：错误

解析：`++a` 会先把 `a` 从 $5$ 增加到 $6$，并返回可取地址的左值；但表达式前面还有取地址运算符 `&`，所以 `&++a` 的结果是变量 `a` 的地址，而不是数值 $6$。`cout` 输出这个地址时通常显示为类似 `0x...` 的地址形式，题干说会输出 $6$，因此错误。

## 第4题

两个函数之间可以使用全局变量来传递数据。（ ）

答案：正确

解析：全局变量定义在函数外，在其作用域内的多个函数都可以访问同一个变量。因此一个函数修改全局变量后，另一个函数再读取这个全局变量，就能得到被修改后的数据。从程序设计习惯看，过度依赖全局变量会降低可维护性，但“可以使用全局变量来传递数据”这一说法本身成立。

## 第5题

定义数组 `int a[2024][3][16] = {2, 0, 2, 4, 3, 1, 6}`，则 `cout << a[2023][2][15]` 的结果不确定。（ ）

答案：错误

解析：数组使用列表 `{2, 0, 2, 4, 3, 1, 6}` 进行聚合初始化时，前 $7$ 个元素按顺序得到这些值，其余没有显式给出的元素会被零初始化。`a[2023][2][15]` 没有越界，它是整个三维数组中的最后一个元素之一，属于未显式赋值的部分，因此值确定为 $0$。题干说结果“不确定”，所以错误。

## 第6题

在 C++ 语言中，函数的参数为指针时，可以在函数内部修改该参数的值。（ ）

答案：错误

解析：题干说法错误。普通指针形参也是按值传递，函数得到的是实参指针值的一份副本。函数内如果执行 `p = other;`，改变的只是形参副本，调用方原来的指针变量不会随之改变；若要修改实参指针本身，需要使用指针引用（如 `int *&p`）或传入指针的指针（如 `int **p`）。需要区分的是，若 `p` 指向一个有效且非 `const` 的对象，函数内可以通过 `*p = ...` 修改“指向的对象”，但这不是修改指针参数本身。

## 第7题

在 C++ 语言中 `try` 子句里抛出的结构体等类型的异常无法被 `catch` 捕获。（ ）

答案：错误

解析：C++ 的异常可以抛出很多类型的对象，包括结构体对象。只要 `catch` 子句中的类型与抛出的异常类型匹配，或能按异常匹配规则接住，例如 `throw S{};` 后使用 `catch (S e)` 或 `catch (const S &e)`，就可以捕获该异常。题干说结构体等类型“无法被 `catch` 捕获”过于绝对，与 C++ 异常机制不符，所以错误。

## 第8题

C++ 语言中 `cout << 9^2 << endl;` 会输出 $81$。（ ）

答案：错误

解析：C++ 中 `^` 不是乘方运算符，而是按位异或运算符；如果单独计算 `9 ^ 2`，二进制 `1001` 与 `0010` 异或得到 `1011`，即十进制 $11$，不是 $81$。此外，题干写法没有给 `9 ^ 2` 加括号时，还会受到 `<<` 与 `^` 的优先级影响，并不能按“输出 $9$ 的平方”理解。无论按哪种教材口径，都不会输出 $81$，所以错误。

## 第9题

小杨今年春节回奶奶家了，奶奶家的数字电视要设置 IP 地址并接入到 WIFI 盒子才能收看节目，那这个 WIFI 盒子具有路由器的功能。（ ）

答案：正确

解析：数字电视需要设置 IP 地址并通过 WIFI 盒子接入网络，说明这个 WIFI 盒子承担了把终端设备接入局域网并与外部网络通信的功能。家用 WIFI 盒子通常集成无线接入、地址分配、网络转发等能力，这些都是路由器常见功能。因此题干说它具有路由器的功能，判断为正确。

## 第10题

任何一个 `for` 循环都可以转化为等价的 `while` 循环（ ）。

答案：正确

解析：`for (初始化; 条件; 更新) { 循环体 }` 可以改写为先执行初始化，再用 `while (条件) { 循环体; 更新; }` 表达同样的重复过程；若循环中存在 `continue` 等控制语句，也可以通过在等价位置补上更新语句或重构控制流保持语义一致。因此从语言表达能力看，`for` 循环可以转化为等价的 `while` 循环，题干正确。

## 第1题

C++ 表达式 `(3 - 2) * 3 + 5` 的值是（ ）。

A. $-13$
B. $8$
C. $2$
D. $0$

答案：B

## 第2题

C++ 语句 `cout << "5%2=" << 5 % 2` 执行后的输出是（ ）。

A. `2 2`
B. `1 1`
C. `5%2=2`
D. `5%2=1`

答案：D

B. 变量 a 将被赋值为字符串，字符串内容为 5+2

## 第6题

下面 C++ 代码执行时输入 21 后，有关描述正确的是（ ）。

```cpp
int N;
cin >> N;
if(N% 3 == 0)
    cout << "能被3整除";
else if (N % 7 == 0)
    cout << "能被7整除";
else
    cout << "不能被3和7整除";
cout << endl;
```

A. 代码第 4 行被执行
B. 第 4 和第 7 行代码都被执行
C. 仅有代码第 7 行被执行
D. 第 8 行代码将被执行，因为 input() 输入为字符串

答案：A

## 第10题

下面 C++ 代码用于判断键盘输入的整数是否为质数。质数是只能被 1 和它本身整除的数。在横线处应填入代码是（ ）。

```cpp
int N;
cin >> N;
int cnt = 0; // 记录N被整除的次数
for(int i=1; i<N+1; i++)
    if(_____________)
        cnt +=1;
if(cnt == 2)
    cout << N << "是质数";
else
    cout << N << "不是质数";
```

A. `N % i`
B. `N % i == 0`
C. `N / i == 0`
D. `N / i`

答案：B

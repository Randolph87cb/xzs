## 第1题

C++ 表达式 `(3 - 2) * 3 + 5` 的值是（ ）。

A. $-13$
B. $8$
C. $2$
D. $0$

答案：B

解析：括号优先计算，`3 - 2 = 1`，再算乘法得到 `1 * 3 = 3`，最后加上 `5`，结果是 `8`，所以选 B。A 把结果算成负数，C 只接近括号内或部分运算结果，D 则像是把表达式错误抵消后得到的结果，都没有按运算优先级完整求值。

## 第2题

C++ 语句 `cout << "5%2=" << 5 % 2` 执行后的输出是（ ）。

A. `2 2`
B. `1 1`
C. `5%2=2`
D. `5%2=1`

答案：D

解析：`cout` 会先原样输出字符串 `"5%2="`，然后输出表达式 `5 % 2` 的值。`%` 是取余运算，`5` 除以 `2` 的余数为 `1`，整体输出为 `5%2=1`，所以选 D。A、B 把字符串部分漏掉了，C 把取余结果误算成了 `2`。

## 第3题

执行 C++ 语句 `cin >> a` 时如果输入 `5+2`，下述说法正确的是（ ）。

A. 变量 a 将被赋值为整数 7
B. 变量 a 将被赋值为字符串，字符串内容为 5+2
C. 语句执行将报错，不能输入表达式
D. 依赖于变量 a 的类型。如果没有定义，会有编译错误

答案：D

解析：`cin >> a` 的读入规则取决于 `a` 的声明类型；如果 `a` 没有提前定义，代码本身会编译错误，所以 D 的说法完整。若 `a` 是整型，输入 `5+2` 通常只会先读入整数 `5`，不会计算成 `7`，A 错；若 `a` 是字符串，才可能读入 `"5+2"`，但题目没有说明类型，B 不能作为普遍结论；能否读入表达式也不等于语句必然报错，C 过于绝对。

## 第4题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int a = 1;
cout << "a+1= " << a+1 << endl;
```

A. `a+1= 2`
B. `a+1=2`
C. `2=2`
D. `2= 2`

答案：A

解析：`a` 的值是 `1`，表达式 `a+1` 的值为 `2`。输出语句中的字符串是 `"a+1= "`，等号后面已经带有一个空格，因此后接数字后得到 `a+1= 2`，所以选 A。B 少了这个空格；C、D 把字符串里的 `a+1` 误当成表达式求值，实际字符串内容会原样输出。

## 第5题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int a = 1;
printf("a+1=%d\n", a+1);
```

A. `a+1= 2`
B. `a+1=2`
C. `2=2`
D. `2= 2`

答案：B

解析：`printf` 的格式串是 `"a+1=%d\n"`，其中 `a+1=` 会原样输出，`%d` 被后面的整数表达式 `a+1` 替换。`a` 为 `1`，所以替换值是 `2`，且格式串中等号后没有空格，输出为 `a+1=2`，选 B。A 多了空格；C、D 都把文字 `a+1` 错误地当成要计算的表达式。

## 第6题

下面 C++ 代码执行时输入 21 后，有关描述正确的是（ ）。

```cpp
int N;
cin >> N;
if(N% 3 == 0)
    cout << "能被3整除";
else if (N % 7 == 0)
    cout << "能被7整除";
else
    cout << "不能被3和7整除";
cout << endl;
```

A. 代码第 4 行被执行
B. 第 4 和第 7 行代码都被执行
C. 仅有代码第 7 行被执行
D. 第 8 行代码将被执行，因为 input() 输入为字符串

答案：A

解析：输入 `21` 后，先判断 `N % 3 == 0`，因为 `21` 能被 `3` 整除，条件成立，于是执行第 4 行输出“能被3整除”。在 `if ... else if ... else` 结构中，前一个分支成立后不会再执行后面的 `else if` 或 `else`，所以第 7 行不会执行，B、C 错。`cin >> N` 读入整数，不是 Python 的 `input()` 字符串输入，D 错。

## 第7题

下面 C++ 代码第 2 行，总共被执行次数是（ ）。

```cpp
for(int i=-10; i<10; i++)
    cout << i << " ";
```

A. $10$
B. $19$
C. $20$
D. $21$

答案：C

解析：循环变量从 `-10` 开始，每次加 `1`，只要 `i < 10` 就执行第 2 行。因此实际取值为 `-10, -9, ..., 8, 9`，一共有 `9 - (-10) + 1 = 20` 个整数，所以选 C。A 只数了半段，B 少算了一个端点，D 则错误地把 `i = 10` 也算进去了。

## 第8题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int tnt = 0;

for(int i=0; i<10; i++)
    if(i%3 && i%7)
        tnt += i;
cout << tnt << endl;
```

A. $0$
B. $7$
C. $18$
D. $20$

答案：D

解析：`if(i%3 && i%7)` 中，非零值按真处理，所以条件成立表示 `i` 既不能被 `3` 整除，也不能被 `7` 整除。`i` 从 `0` 到 `9`，满足条件的是 `1, 2, 4, 5, 8`，它们的和为 `20`，所以选 D。`0, 3, 6, 9` 被 `3` 整除，`7` 被 `7` 整除，不能累加；A、B、C 都不是这些有效项的总和。

## 第9题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int N = 10;
while(N){
    N -= 1;
    if(N%3 == 0)
        cout << N << "#";
}
```

A. `9#6#3#`
B. `9#6#3#0#`
C. `8#7#5#4#2#1#`
D. `10#8#7#5#4#2#1#`

答案：B

解析：`while(N)` 在每轮开始时检查当前 `N` 是否非零，进入循环后先执行 `N -= 1`，再判断是否能被 `3` 整除。`N` 依次变成 `9, 8, 7, 6, 5, 4, 3, 2, 1, 0`，其中 `9, 6, 3, 0` 满足 `N % 3 == 0`，因此输出 `9#6#3#0#`，选 B。A 漏掉了最后一次进入循环后得到的 `0`；C 输出的是不能被 `3` 整除的数；D 还错误包含了初始值 `10`。

## 第10题

下面 C++ 代码用于判断键盘输入的整数是否为质数。质数是只能被 1 和它本身整除的数。在横线处应填入代码是（ ）。

```cpp
int N;
cin >> N;
int cnt = 0; // 记录N被整除的次数
for(int i=1; i<N+1; i++)
    if(_____________)
        cnt +=1;
if(cnt == 2)
    cout << N << "是质数";
else
    cout << N << "不是质数";
```

A. `N % i`
B. `N % i == 0`
C. `N / i == 0`
D. `N / i`

答案：B

解析：变量 `cnt` 要记录 `1` 到 `N` 中有多少个数能整除 `N`，因此横线处应判断余数是否为 `0`，也就是 `N % i == 0`，选 B。A 的 `N % i` 在余数非零时为真，反而会统计“不能整除”的情况；C 判断商是否为 `0`，在本循环 `i <= N` 时对正整数 `N` 基本不符合因数含义；D 只判断商是否非零，也不能表示整除。

## 第11题

下面关于整型变量 `int x` 的赋值语句不正确的是（ ）。

A. `x=(3.16);`
B. `x=3.16;`
C. `x=int(3.16);`
D. `x=3.16 int;`

答案：D

解析：题目问“不正确”的赋值语句。A、B 都是把浮点数 `3.16` 赋给整型变量 `x`，会发生到 `int` 的转换；C 显式写成 `int(3.16)` 后再赋值，也符合语法。D 写作 `x=3.16 int;`，类型名 `int` 不能这样放在表达式后面，语法错误，所以选 D。

## 第12题

C++ 语言中下面可以完成数据输入的语句是（ ）。

A. printf 语句
B. scanf 语句
C. default 语句
D. cout 语句

答案：B

解析：`scanf` 是从标准输入读取数据的函数，可以完成数据输入，所以选 B。`printf` 和 `cout` 主要用于输出，不能完成从键盘读入数据的任务；`default` 是 `switch` 语句中的分支标记，不是输入语句。

## 第13题

在 Dev C++ 中对一个写好的 C++ 源文件要生成一个可执行程序需要执行下面哪个处理步骤？（ ）

A. 创建
B. 编辑
C. 编译
D. 调试

答案：C

解析：源文件要变成可执行程序，需要经过编译，把 C++ 源代码转换并链接成机器可以运行的程序，因此选 C。创建和编辑只是产生、修改源文件；调试是在程序可以编译运行后定位问题的过程，不是生成可执行程序的必要处理步骤名称。

## 第14题

小杨的父母最近刚刚给他买了一块华为手表，他说手表上跑的是鸿蒙，这个鸿蒙是？（ ）

A. 小程序
B. 计时器
C. 操作系统
D. 神话人物

答案：C

解析：鸿蒙指华为的 HarmonyOS，是用于管理设备硬件资源、运行应用程序的操作系统，所以选 C。小程序是运行在特定平台上的应用形态，计时器只是具体功能或工具，神话人物与题干中的手表软件环境无关。

## 第15题

中国计算机学会（CCF）在 2024 年 1 月 27 日的颁奖典礼上颁布了王选奖，王选先生的重大贡献是（ ）。

A. 制造自动驾驶汽车
B. 创立培训学校
C. 发明汉字激光照排系统
D. 成立方正公司

答案：C

解析：王选先生的重要贡献是主持研制汉字激光照排系统，推动中文印刷出版从铅字排版走向计算机照排，所以选 C。自动驾驶汽车、创立培训学校都不是王选先生的代表性贡献；方正公司与相关成果产业化有关，但题目问“重大贡献”，不是公司成立这一事项。

## 第1题

在 C++ 代码中变量 n 被赋值为 27，则 `cout << n%10` 执行后输出的是 7。（ ）

答案：正确

解析：题干说法正确。`%` 是取余运算，`27 % 10` 表示 `27` 除以 `10` 的余数，余数为 `7`；`cout << n%10` 会输出这个表达式的值，因此输出结果是 `7`。

## 第2题

C++ 语句 `printf("%d#%d&",2,3)` 执行后输出的是 2#3&。（ ）

答案：正确

解析：题干说法正确。格式串 `"%d#%d&"` 中两个 `%d` 依次由后面的参数 `2` 和 `3` 替换，`#` 与 `&` 是普通字符，会原样输出，所以最终输出 `2#3&`。

## 第3题

C++ 函数 `scanf()` 必须含有参数，且其参数为字符串型字面量，其功能是提示输入。（ ）

答案：错误

解析：题干说法错误。`scanf` 的主要功能是按格式从标准输入读取数据，不是“提示输入”；如果需要提示，通常要另外用 `printf` 或 `cout` 输出提示文字。常见用法如 `scanf("%d", &x)`，第一个参数是格式字符串，后续参数还要提供变量地址，因此题干把功能说错了。

## 第4题

C++ 表达式 `"10"*2` 执行时将报错，因为 `"10"` 是字符串类型而 2 是整数类型，它们数据类型不同，不能在一起运算。（ ）

答案：正确

解析：题干说法正确。`"10"` 是字符串字面量，在表达式中通常会退化为指向字符数组首元素的指针；C++ 没有定义“字符串字面量乘以整数”的 `*` 运算，因此 `"10" * 2` 不能通过编译。严格说，报错的关键不是“只要类型不同就不能运算”，而是这两个操作数之间没有合法的乘法运算规则。

## 第5题

在 C++ 中，while 可能是死循环，而 for 循环不可能是死循环。（ ）

答案：错误

解析：题干说法错误。`while` 可以写成条件一直为真的死循环，例如 `while(true){}`；`for` 也可以写成死循环，例如 `for(;;){}` 或循环条件一直成立的形式。因此“for 循环不可能是死循环”不成立。

## 第6题

在 C++，break 语句用于提前终止当前层次循环，适用于 while 循环，但不适用于 for 循环。（ ）

答案：错误

解析：题干说法错误。`break` 用于提前跳出当前所在的一层循环或 `switch`，既可以用于 `while` 循环，也可以用于 `for` 循环。例如在 `for` 中满足某个条件后执行 `break;`，循环会立即结束。

## 第7题

C++ 语言中 3.0 和 3 的值相等，所以它们占用的存储空间也相同。（ ）

答案：错误

解析：题干说法错误。`3` 是整型字面量，`3.0` 是浮点型字面量，它们在数值比较时可以相等，但数据类型不同，通常占用的存储空间也不同，例如常见环境下 `int` 和 `double` 的字节数就不一样。数值相等不能推出存储空间相同。

## 第8题

在 C++ 的程序中，cin 是一个合法的变量名。（ ）

答案：正确

解析：题干按变量命名规则判为正确。`cin` 由字母组成，不是 C++ 关键字，因此从标识符命名规则看可以作为变量名。需要注意的是，标准输入对象通常写作 `std::cin`；如果在同一作用域中引入或使用了同名对象，可能产生名字冲突或遮蔽，但这不改变 `cin` 本身不是关键字这一点。

## 第9题

小杨今年春节回奶奶家了，奶奶家的数字电视可以通过遥控器输入电视剧名称来找到想播放的电视剧，所以可以推知里面有交互式程序在运行。（ ）

答案：正确

解析：题干说法正确。数字电视能根据用户通过遥控器输入的电视剧名称进行查找，说明系统会接收用户输入、处理输入并给出对应结果，这正是交互式程序的基本特征。它不是单纯播放固定内容，而是根据操作产生不同反馈。

## 第10题

任何一个 for 循环都可以转化为等价的 while 循环（ ）。

答案：正确

解析：题干说法正确。一般的 `for(初始化; 条件; 更新)` 可以改写为先执行初始化，再用 `while(条件)` 包住循环体，并在每轮末尾执行更新语句，因此能表达同样的循环过程。严格保持等价时要注意 `continue` 等语句对“更新语句执行时机”的影响，不能只把更新语句机械放在循环体最后而忽略这些控制流细节。

## 第8题

```cpp
int s,t,ans;
s = 2, t = 10;
ans = 0;
while (s != t){
    if (t % 2 == 0 && t / 2 >= s)
        t /= 2;
    else
        t -= 1;
    ans += 1;
}
cout << ans;
```

A. 2
B. 3
C. 4
D. 5

答案：B

## 第9题

下面 C++ 代码执行后的输出是？（ ）

```cpp
int n, masks, days, cur;
n = 17, masks = 10, days = 0;
cur = 2;
while (masks != n){
    if (cur == 0 || cur == 1)
        masks += 7;
    masks -= 1;
    days += 1;
    cur = (cur + 1) % 7;
}
cout << days;
```

A. 5
B. 6
C. 7
D. 8

答案：C

## 第12题

一个数的所有数字倒序排列后这个数的大小保持不变，这个数就是回文数，比如 $101$ 与 $6886$ 都是回文数，而 $100$ 不是回文数。以下程序代码用于判断一个数是否为回文数，横线处应填写？（ ）

```cpp
int n,a,k;
cin >> n;
a = 0;
k = n;
while (n > 0){
    a = _________; // 在此处填写代码
    n /= 10;
}
if (a == k)
    cout << "是回文数";
else
    cout << "不是回文数";
```

A. `10 * a + n % 10`
B. `a + n % 10`
C. `10 * a + n / 10`
D. `a + n / 10`

答案：A

## 第13题

给定两个整数 $n$ 与 $k$，打印出一个栅栏图形，这个栅栏应该分成 $n$ 段，段与段之间的间隔为 `+`，段内的填充为 $k$ 个 `-`。形如 $n=5$，$k=6$ 时，图形如下：

```
+------+------+------+------+------+
```

以下程序代码用于绘制该图形，横线处应填写？（ ）

```cpp
int n, k, i, j;
n = 5, k = 6;
for (i = 0; i < n; i++){
    _________ // 在此处填写代码
    for (j = 1; j < k; j++)
        cout << '-';
}
cout << '+';
```

A. `cout << '+' << endl;`
B. `cout << '+' << ' ' << endl;`
C. `cout << '+';`
D. `cout << '+' << ' ';`

答案：C

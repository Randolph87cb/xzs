## 第1题

下列流程图的输出结果是？（  ）

![](https://cdn.luogu.com.cn/upload/image_hosting/vvwkr8pr.png)

A. 优秀
B. 良好
C. 不及格
D. 没有输出

答案：B

解析：流程图先令 `cj = 75`。判断 `cj >= 60` 时，$75 \ge 60$ 成立，所以不会走到“不及格”；继续判断 `cj >= 90` 时，$75 \ge 90$ 不成立，因此走 NO 分支输出“良好”。所以选 B。A 需要成绩至少 $90$，C 需要第一步小于 $60$，D 与流程图中已经到达输出框不符。

## 第2题

以下选项中不符合 C++ 变量命名规则的是？（ ）

A. `student`
B. `2_from`
C. `_to`
D. `Text`

答案：B

解析：C++ 标识符可以由字母、数字和下划线组成，但不能以数字开头，也不能是保留字。`2_from` 的第一个字符是数字 `2`，不符合变量命名规则，所以选 B。`student`、`_to`、`Text` 都由合法字符组成，且没有以数字开头。

## 第3题

以下选项中，不能用于表示分支结构的 C++ 保留字是？（ ）

A. `switch`
B. `return`
C. `else`
D. `if`

答案：B

解析：`if`、`else`、`switch` 都直接用于组织分支结构：`if/else` 表示条件分支，`switch` 表示多分支选择。`return` 的作用是从函数返回，必要时带回返回值，它不是用来表示分支结构的保留字，所以选 B。

## 第4题

下列说法错误的是？（ ）

A. while 循环满足循环条件时不断地运行，直到指定的条件不满足为止
B. if 语句通常用于执行条件判断
C. 在 C++ 中可以使用 foreach 循环
D. break 和 continue 语句都可以用在 for 循环和 while 循环中

答案：C

解析：题目要求选错误说法。A 正确，`while` 会在条件为真时反复执行，条件变假后停止；B 正确，`if` 常用于条件判断；D 正确，`break` 和 `continue` 都可以用于 `for`、`while` 等循环中。C 的问题在于 C++ 没有名为 `foreach` 的标准关键字或语句；C++11 以后有范围 `for`，功能类似遍历容器，但写法不是 `foreach`，因此选 C。

## 第5题

下列 $4$ 个表达式中，答案不是整数 $8$ 的是？（ ）

A. `abs(-8)`
B. `min(max(8, 9), 10)`
C. `int(8.88)`
D. `sqrt(64)`

答案：B

解析：逐项计算：`abs(-8)` 的结果是 $8$；`max(8, 9)` 先得到 $9$，再算 `min(9, 10)` 得到 $9$，不是 $8$；`int(8.88)` 会截去小数部分得到 $8$；`sqrt(64)` 的数学结果是 $8$。按官方题目口径这里比较的是表达式的数值结果，因此选 B。

## 第6题

下面 C++ 代码执行后的输出是？（ ）

```cpp
int n,a,m,i;
n=3, a = 5;
m = (a - 1) * 2;
for (i=0; i<n-1; i++)
    m = (m - 1) * 2;
cout << m;
```

A. 8
B. 14
C. 26
D. 50

答案：C

解析：先赋值 `n = 3, a = 5`，所以 `m = (a - 1) * 2 = 8`。循环条件是 `i < n - 1`，即 `i < 2`，从 `i = 0` 开始共执行两次。第 1 次后 `m = (8 - 1) * 2 = 14`，第 2 次后 `m = (14 - 1) * 2 = 26`，随后 `i = 2` 不再满足条件，输出 $26$，选 C。A 是只看到了初始的 `m`，B 是只执行了一次循环，D 多执行了一次循环。

## 第7题

下面 C++ 代码执行后的输出是？（ ）

```cpp
int n,i,result;
n = 81;
i = 1, result = 1;
while (i * i <= n){
    if (n % (i * i) == 0)
        result = i * i;
    i += 1;
}
cout << result;
```

A. 16
B. 36
C. 49
D. 81

答案：D

解析：循环从 `i = 1` 开始，只要 `i * i <= 81` 就继续，因此会检查 $1^2$ 到 $9^2$。当 `81 % (i * i) == 0` 时，把 `result` 更新为这个平方数。能整除 $81$ 的平方数依次有 $1$、$9$、$81$，最后一次更新发生在 `i = 9`，`result = 81`。所以输出 $81$，选 D。$16$、$36$、$49$ 虽然是平方数，但都不能整除 $81$。

## 第8题

下面 C++ 代码执行后的输出是？（ ）

```cpp
int s,t,ans;
s = 2, t = 10;
ans = 0;
while (s != t){
    if (t % 2 == 0 && t / 2 >= s)
        t /= 2;
    else
        t -= 1;
    ans += 1;
}
cout << ans;
```

A. 2
B. 3
C. 4
D. 5

答案：B

解析：初始 `s = 2, t = 10, ans = 0`。第 1 轮，`t` 为偶数且 `t / 2 = 5 >= 2`，所以 `t` 变为 $5$，`ans` 变为 $1$；第 2 轮，`t = 5` 是奇数，执行 `t -= 1` 得到 $4$，`ans = 2`；第 3 轮，`t = 4` 为偶数且 `t / 2 = 2 >= 2`，执行除以 $2$，`t = 2`，`ans = 3`。此时 `s == t`，循环结束，输出 $3$，选 B。

## 第9题

下面 C++ 代码执行后的输出是？（ ）

```cpp
int n, masks, days, cur;
n = 17, masks = 10, days = 0;
cur = 2;
while (masks != n){
    if (cur == 0 || cur == 1)
        masks += 7;
    masks -= 1;
    days += 1;
    cur = (cur + 1) % 7;
}
cout << days;
```

A. 5
B. 6
C. 7
D. 8

答案：C

解析：初始 `masks = 10, days = 0, cur = 2`。每轮若 `cur` 是 $0$ 或 $1$，先增加 $7$ 个口罩，然后每天消耗 $1$ 个，`cur` 按星期对 $7$ 取模前进。前 5 天 `cur` 依次为 `2, 3, 4, 5, 6`，每天只减 $1$，口罩数从 $10$ 变到 $5$；第 6 天 `cur = 0`，先加 $7$ 再减 $1$，变为 $11$；第 7 天 `cur = 1`，再先加 $7$ 后减 $1$，变为 $17$。此时 `masks == n`，循环结束，输出 `days = 7`，选 C。

## 第10题

以下 C++ 代码判断一个正整数 $N$ 的各个数位是否都是偶数。如果都是，则输出“是”，否则输出“否”。例如 $N=2024$ 时输出“是”。则横线处应填入（ ）。

A. `break`
B. `continue`
C. `N = N / 10`
D. `N = N % 10`

答案：A

解析：循环中每次查看当前最低位。若 `N % 2` 为真，说明当前数位是奇数，已经可以确定“不是所有数位都是偶数”，程序把 `Flag` 置为 `false` 后应立刻停止继续检查，所以横线处填 `break` 最合适。`continue` 会跳过本轮后面的 `N = N / 10`，导致下一轮仍检查同一个奇数位，可能无法结束。`N = N % 10` 只保留个位，遇到奇数个位时也不能推进到更高位。`N = N / 10` 虽然能继续删去当前位，但放在这里会在已经发现反例后继续循环，不如 `break` 直接表达“发现一个奇数位就结束判断”的逻辑。

## 第11题

有句俗话叫“三天打渔，两天晒网”。如果小杨前三天打渔，后两天晒网，一直重复这个过程，以下程序代码用于判断，第 $n$ 天小杨是在打鱼还是晒网，横线处应填写？（ ）

A. `i == 0`
B. `i == 4`
C. `i == 0 && i == 4`
D. `i == 0 || i == 4`

答案：D

解析：程序中 `i = n % 5`，表示第 $n$ 天落在 $5$ 天周期中的位置。前三天打鱼，对应余数 $1,2,3$；后两天晒网，对应第 $4$ 天和第 $5$ 天，而第 $5$ 天取模后余数为 $0$。因此判断晒网应写成 `i == 0 || i == 4`，选 D。A 漏掉余数 $4$，B 漏掉余数 $0$，C 要求 `i` 同时等于 $0$ 和 $4$，永远不成立。

## 第12题

一个数的所有数字倒序排列后这个数的大小保持不变，这个数就是回文数，比如 $101$ 与 $6886$ 都是回文数，而 $100$ 不是回文数。以下程序代码用于判断一个数是否为回文数，横线处应填写？（ ）

```cpp
int n,a,k;
cin >> n;
a = 0;
k = n;
while (n > 0){
    a = _________; // 在此处填写代码
    n /= 10;
}
if (a == k)
    cout << "是回文数";
else
    cout << "不是回文数";
```

A. `10 * a + n % 10`
B. `a + n % 10`
C. `10 * a + n / 10`
D. `a + n / 10`

答案：A

解析：变量 `a` 用来逐步保存反转后的数字，`n % 10` 取出当前最低位，`n /= 10` 删除当前最低位。每读入一位，就要把原来的 `a` 左移一位十进制位，即乘以 $10$，再加上新取出的最低位，所以应填 `10 * a + n % 10`。例如 `n = 6886` 时，`a` 依次变为 $6,68,688,6886$，最后能与原数 `k` 比较。B、D 没有把旧结果乘以 $10$，C 加的是去掉个位后的 `n / 10`，都不能正确构造倒序数。

## 第13题

给定两个整数 $n$ 与 $k$，打印出一个栅栏图形，这个栅栏应该分成 $n$ 段，段与段之间的间隔为 `+`，段内的填充为 $k$ 个 `-`。形如 $n=5$，$k=6$ 时，图形如下：

```
+------+------+------+------+------+
```

以下程序代码用于绘制该图形，横线处应填写？（ ）

```cpp
int n, k, i, j;
n = 5, k = 6;
for (i = 0; i < n; i++){
    _________ // 在此处填写代码
    for (j = 1; j < k; j++)
        cout << '-';
}
cout << '+';
```

A. `cout << '+' << endl;`
B. `cout << '+' << ' ' << endl;`
C. `cout << '+';`
D. `cout << '+' << ' ';`

答案：C

解析：横线位于外层循环每一段的开头，作用是先输出该段左侧的栅栏分隔符 `+`；随后内层循环输出这一段里的若干个 `-`；外层循环全部结束后，最后一句 `cout << '+';` 再补上最右侧的分隔符。这样整体才能形成 `+---+---+...+` 这种由 `+` 分隔各段的图形，所以应填写 `cout << '+';`，选 C。A、B 都会输出换行，使栅栏被拆成多行；D 会在分隔符后多出空格，图形格式不符合题意。

## 第14题

小杨的父母最近刚刚给他买了一块华为手表，他说手表上跑的是鸿蒙，这个鸿蒙是。（ ）

A. 小程序
B. 计时器
C. 操作系统
D. 神话人物

答案：C

解析：鸿蒙指华为的 HarmonyOS，是运行在手表、手机等设备上的操作系统，负责管理硬件资源并为应用提供运行环境，所以选 C。小程序是运行在某个平台中的应用形态，计时器只是具体功能，神话人物不是这里“手表上跑的是鸿蒙”的计算机含义。

## 第15题

中国计算机学会（CCF）在 2024 年 1 月 27 日的颁奖典礼上颁布了王选奖，王选先生的重大贡献是（ ）。

A. 制造自动驾驶汽车
B. 创立培训学校
C. 发明汉字激光照排系统
D. 成立方正公司

答案：C

解析：王选先生最重要的贡献是主持研制并推动应用汉字激光照排系统，使中文出版印刷从铅字排版走向计算机照排，所以选 C。A 与其贡献领域无关；B 不是王选先生的重大科技贡献；D“成立方正公司”与相关产业化背景有关，但题目问的是王选先生的重大贡献，核心应是汉字激光照排系统。

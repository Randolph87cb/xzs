## 第1题

如果有以下 C++ 代码：

```cpp
double s;
int t;
s = 18.5;
t = int(s) + 10;
```

那么 `cout << t` 的结果为 $28.5$。

答案：错误

解析：`s` 被赋值为 `18.5`，但 `int(s)` 会把小数部分截去，得到整数 $18$。于是 `t = int(s) + 10 = 28`，并且 `t` 的类型是 `int`，`cout << t` 输出的是 `28`，不是 `28.5`。所以题干说法错误。

## 第2题

`Xyz`，`xYz`，`xyZ` 是三个不同的变量。

答案：正确

解析：C++ 标识符区分大小写，大小写不同就是不同的名字。`Xyz`、`xYz`、`xyZ` 虽然字母相同，但每个名字中大写字母的位置不同，编译器会把它们当作三个不同变量。因此题干说法正确。

## 第3题

`cout << (8 < 9 < 10)` 的输出结果为 `true`。

答案：错误

解析：关系运算从左到右求值，`8 < 9 < 10` 会先算 `8 < 9`，结果为 `true`，在继续参与比较时可转换为整数 $1$；再算 `1 < 10`，结果仍为 `true`。但是 `cout` 默认输出布尔值时不输出单词 `true`，而是输出 `1`；只有设置 `boolalpha` 后才会输出 `true`。所以题干说输出结果为 `true` 是错误的。

## 第4题

`for (i = 0; i < 100; i += 2);` 语句中变量 $i$ 的取值范围是 $0$ 到 $99$。

答案：错误

解析：语句末尾的分号表示这个 `for` 循环的循环体是空语句。循环判断过程中，`i` 从 $0$ 开始每次加 $2$，满足条件时会出现 $0,2,4,\ldots,98$，并不会取到所有 $0$ 到 $99$ 的整数；最后一次自增后 `i` 变为 $100$，条件 `i < 100` 才失败。题干说取值范围是 $0$ 到 $99$，既漏掉步长为 $2$ 的事实，也没有说明最终停止时的 $100$，因此错误。

## 第5题

C++ 中 `cout << float(2022)` 与 `cout << float('2022')` 运行后的输出结果均为 $2022$。

答案：错误

解析：`float(2022)` 是把整数 $2022$ 转成浮点数，默认输出通常显示为 `2022`。但 `'2022'` 在 C++ 中不是字符串，也不是数字 $2022$，而是多字符字符字面量，其类型和值具有实现相关特征；再转成 `float` 后不应按十进制数 $2022$ 理解。因此两者运行结果不能都判定为 $2022$，题干说法错误。

## 第6题

已知 A 的 ASCII 码值为 $65$，表达式 `int('C') + abs(-5.8)` 的值为 $72.8$。

答案：正确

解析：已知 `'A'` 的 ASCII 码是 $65$，则 `'C'` 比 `'A'` 大 $2$，所以 `int('C') = 67`。在常见 C++ 数学函数用法中，`abs(-5.8)` 会按浮点数求绝对值，结果是 $5.8$，两者相加得到 $67 + 5.8 = 72.8$。因此题干说法正确。

## 第7题

`bool()` 函数用于将给定参数或表达式转换为布尔类型。语句 `bool(-1)` 返回的是 `false` 值。（ ）

答案：错误

解析：把数值转换为 `bool` 时，$0$ 转换为 `false`，非 $0$ 值转换为 `true`。`-1` 虽然是负数，但它不是 $0$，所以 `bool(-1)` 的结果是 `true`，不是 `false`。因此题干说法错误。

## 第8题

如果变量 $a$ 的值使得 C++ 表达式 `sqrt(a) == abs(a)`，则 $a$ 的值为 $0$。（ ）

答案：错误

解析：表达式 `sqrt(a)` 要在实数范围内有意义，通常要求 $a \ge 0$。当 $a \ge 0$ 时，`abs(a) = a`，条件变为 $\sqrt a = a$，解得 $a = 0$ 或 $a = 1$。例如 $a = 1$ 时，`sqrt(1) == abs(1)` 也成立。因此题干把 $a$ 的值限定为 $0$ 是错误的。

## 第9题

小杨今年春节回奶奶家了，奶奶家的数字电视要设置 ip 地址并接入到 WIFI 盒子才能收看节目，那这个 WIFI 盒子具有路由器的功能。（ ）

答案：正确

解析：题干说数字电视需要设置 IP 地址并接入 WIFI 盒子才能收看节目，说明这个 WIFI 盒子承担了让设备加入网络、进行网络连接的功能。能让局域网设备通过 IP 接入并转发网络通信，是路由器的基本功能之一。因此说它具有路由器的功能是正确的。

## 第10题

任何一个 `for` 循环都可以转化为等价的 `while` 循环（ ）。

答案：正确

解析：`for` 循环的一般结构是“初始化、条件判断、循环体、迭代表达式”。把初始化语句放到循环前，把条件放进 `while`，再在循环体末尾执行迭代表达式，就可以表达同样的循环过程；如果循环体中有 `continue`，转换时也要保证先执行相当于迭代表达式的步骤再进入下一轮判断。只要按这些控制流程细节处理，`for` 循环可以转化为等价的 `while` 循环，所以题干说法正确。

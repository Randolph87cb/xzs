## 第1题

任意整数 $a$ 的二进制反码与补码都有 $1$ 位不同。（ ）

【知识点：原码反码补码与整数表示】

答案：错误

解析：题干说法错误。正整数的反码和补码通常都与原码相同，例如 $+5$ 的反码和补码没有不同位；负整数也不一定只差 $1$ 位，因为“反码加 $1$”可能产生进位，导致多个低位变化。例如 8 位表示下，$-2$ 的反码可写成 `11111101`，补码是 `11111110`，末两位都发生变化。因此“任意整数”“都有 $1$ 位不同”过于绝对。

## 第2题

对整型变量 `int a = 3`，执行 C++ 代码 `a<<2` 将把 $2$ 输出到 $a$ 中。（ ）

【知识点：位运算】

答案：错误

解析：题干说法错误。`a << 2` 是一个表达式，表示把 `a` 的二进制位左移 $2$ 位；当 `a=3` 时，该表达式的值是 $3 \times 2^2 = 12$。但题目中没有赋值语句 `a = a << 2`，也没有 `cout` 输出语句，所以它既不会把 $2$ 输出到 `a` 中，也不会自动改变 `a` 的值。

## 第3题

下面 C++ 代码可以计算 $1$ 到 $100$ 的累加和，采用的是穷举法。

```cpp
int main()
{
    int i, sum = 0;
    for (int i = 1; i <= 100; i++)
        sum += i;
    cout << sum << endl;

    cout << endl;
    return 0;
}
```

【知识点：枚举与基础算法】

答案：错误

解析：题干说法错误。这段代码确实通过循环把 $1$ 到 $100$ 依次加到 `sum` 中，最后输出累加和 $5050$。但按本题官方口径，它采用的是普通循环累加，不是“穷举法”；穷举法通常指枚举所有可能方案或候选答案再判断是否满足条件。这里没有枚举候选解并筛选，只是在固定范围内做求和。

## 第4题

一个 `int` 类型变量 $a$，执行操作 `(a<<2>>2)` 后的值一定是 $a$。

【知识点：位运算】

答案：错误

解析：题干说法错误。左移两位后，高位可能被移出并丢失，再右移两位无法恢复这些位，所以结果不一定等于原来的 `a`。例如在固定位宽整数中，如果 `a` 的高位本来是 $1$，`a << 2` 可能丢掉这些高位信息。并且对有符号 `int` 来说，移位还可能受到溢出、符号位和标准规则限制，更不能说“一定是 `a`”。

## 第5题

在C++语言中，`(010<<1)` 执行结果是 $100$。（ ）

【知识点：位运算】

答案：错误

解析：题干说法错误。在 C++ 中，整数常量 `010` 以 `0` 开头，表示八进制数，实际十进制值是 $8$。左移 $1$ 位相当于乘以 $2$，所以 `(010 << 1)` 的值是十进制 $16$，不是 $100$。即使写成二进制形式，$16$ 也是 `10000`，也不是题干中的 $100$。

## 第6题

执行下面 C++ 代码后将输出 $2$。（ ）

```cpp
int main()
{
    string str = "gEsP is Interesting";
    int x = str.find("s");

    cout << x << endl;

    cout << endl;
    return 0;
}
```

【知识点：字符串处理】

答案：正确

解析：题干说法正确。字符串 `gEsP is Interesting` 的下标从 $0$ 开始：`g` 在下标 $0$，`E` 在下标 $1$，第一个小写 `s` 在下标 $2$。`str.find("s")` 返回第一次找到子串 `"s"` 的起始位置，因此 `x` 的值为 $2$，程序输出 $2$。

## 第7题

在 C++ 语言中，字符数组被定义时，它的大小可以调整。（ ）

【知识点：数组与字符数组】

答案：错误

解析：题干说法错误。C++ 中普通字符数组一旦定义，其数组大小就是固定的，例如 `char s[20]` 的容量就是 $20$ 个 `char` 位置，后续不能把这个数组本身调整成更大或更小。可以修改数组中的字符内容，也可以用字符串结束符改变“当前字符串长度”，但这不等于改变数组大小。

## 第8题

对定义的数组 `int a[7] = {2, 0, 2, 4, 3, 1, 6}`，可以用简单循环就找到其中最小的整数。（ ）

【知识点：数组与字符数组】

答案：正确

解析：题干说法正确。可以先令当前最小值为第一个元素 $2$，再用循环依次比较后面的 $0,2,4,3,1,6$。比较到 $0$ 时更新最小值，之后的元素都不小于 $0$，最终得到最小整数 $0$。这种线性扫描只需要一个简单循环即可完成。

## 第9题

小杨今年春节回奶奶家了，奶奶家的数字电视要设置 ip 地址并接入到 WIFI 盒子才能收看节目，那这个 WIFI 盒子具有路由器的功能。（ ）

【知识点：计算机基础常识】

答案：正确

解析：题干说法正确。数字电视需要设置 IP 地址并通过 WIFI 盒子接入网络，说明这个盒子在家庭网络中为终端提供无线接入和网络转发能力。按本题计算机基础常识口径，这类设备具有路由器的功能，因此判断为正确。

## 第10题

任何一个 `for` 循环都可以转化为等价的 `while` 循环（ ）。

【知识点：控制结构与程序阅读】

答案：正确

解析：题干说法正确。`for (初始化; 条件; 更新)` 的基本执行顺序可以改写为：先执行初始化，再用 `while (条件)` 循环，在循环体末尾执行更新。若 `for` 省略条件，可按条件恒真处理；若循环体中有 `continue`，改写时需要保证 `continue` 前也能执行原来的更新表达式。经过这些等价处理后，`for` 循环可以转化为等价的 `while` 循环。

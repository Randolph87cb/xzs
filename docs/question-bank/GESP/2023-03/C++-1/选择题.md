## 第1题

以下不属于计算机输入设备的有（ ）。

A. 键盘
B. 音箱
C. 鼠标
D. 传感器

答案：B

解析：本题选 B。输入设备用于把数据或指令送入计算机，音箱 不承担输入功能；其他选项能采集或输入信息。

## 第2题

计算机系统中存储的基本单位用 **B** 来表示，它代表的是（ ）。

A. Byte
B. Block
C. Bulk
D. Bit

答案：A

解析：本题选 A。B 表示 Byte，即字节；bit 是位，Block、Bulk 不是这里的基本存储单位。

## 第3题

常量 $7.0$ 的数据类型是（ ）。

A. double
B. float
C. void
D. int

答案：A

解析：本题选 A。C++ 中带小数点的数字字面量默认是 double，单引号括起的是 char，不带小数点的整数字面量通常是 int；double 符合该规则。

## 第5题

以下不可以作为 C++ 标识符的是（ ）。

A. x321
B. 0x321
C. x321_
D. _x321

答案：B

解析：本题选 B。C++ 标识符只能由字母、数字和下划线组成，不能以数字开头，也不能使用关键字；0x321 符合题干要求，其他选项违反了这些规则之一。

## 第7题

如果 a、b 和 c 都是 int 类型的变量，下列哪个语句不符合 C++ 语法？（  ）

A. c = a + b;
B. c += a + b;
C. c = a = b;
D. c = a ++ b;

答案：D

解析：本题选 D。该选项违反 C++ 语法规则，例如赋值左侧必须是可修改的对象、运算符用法必须合法；其他选项可以按隐式转换或正常表达式规则通过。

A. a + b * 2
B. 2 * a + 2 * b

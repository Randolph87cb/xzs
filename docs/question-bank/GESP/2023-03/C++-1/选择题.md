## 第1题

以下不属于计算机输入设备的有（ ）。

A. 键盘
B. 音箱
C. 鼠标
D. 传感器

答案：B

解析：本题选 B。输入设备的作用是把数据、命令或外部信息送入计算机。键盘可以输入文字和命令，鼠标可以输入点击、移动等操作，传感器可以采集温度、光线等信息；音箱是把计算机中的声音播放出来，属于输出设备，不是输入设备。

## 第2题

计算机系统中存储的基本单位用 **B** 来表示，它代表的是（ ）。

A. Byte
B. Block
C. Bulk
D. Bit

答案：A

解析：本题选 A。大写字母 `B` 通常表示 `Byte`，中文叫“字节”，是计算机存储容量常用单位，例如 `KB`、`MB` 中的 `B` 都是字节。`Bit` 是“位”，通常用小写 `b` 表示；`Block` 和 `Bulk` 不是题目中 `B` 所代表的存储基本单位。

## 第3题

常量 $7.0$ 的数据类型是（ ）。

A. double
B. float
C. void
D. int

答案：A

解析：本题选 A。在 C++ 中，像 `7.0` 这样带小数点、又没有加 `f` 或 `F` 后缀的浮点常量，默认类型是 `double`。`float` 常量通常要写成 `7.0f`，`int` 是整数类型，`void` 表示“无类型/无返回值”，都不符合 `7.0` 的写法。

## 第4题

下列关于 C++ 语言的叙述，不正确的是（ ）。

A. 变量定义时可以不初始化
B. 变量被赋值之后的类型不变
C. 变量没有定义也能够使用
D. 变量名必须是合法的标识符

答案：C

解析：本题选 C。C++ 中变量必须先定义，编译器知道它的名字和类型后才能使用；如果直接使用没有定义过的变量，会产生编译错误。A 中 `int x;` 这样的定义可以不初始化，只是此时值不确定；B 中变量一旦定义为某种类型，后续赋值不会改变它的类型；D 中变量名必须是合法标识符，这些说法都是正确的。

## 第5题

以下不可以作为 C++ 标识符的是（ ）。

A. x321
B. 0x321
C. x321_
D. _x321

答案：B

解析：本题选 B。C++ 标识符可以由字母、数字和下划线组成，但不能以数字开头。`x321`、`x321_`、`_x321` 都符合这个规则；`0x321` 以数字 `0` 开头，在 C++ 中还很像十六进制整数常量的写法，不能作为标识符。

## 第6题

以下哪个不是 C++ 语言的关键字？

A. int
B. for
C. do
D. cout

答案：D

解析：本题选 D。`int`、`for`、`do` 都是 C++ 语言保留的关键字，分别用于整数类型、循环语句等语法结构，不能随便当普通变量名使用。`cout` 不是关键字，它是标准库中用于输出的对象名，使用时通常还需要包含 `<iostream>` 并配合命名空间。

## 第7题

如果 a、b 和 c 都是 int 类型的变量，下列哪个语句不符合 C++ 语法？（  ）

A. c = a + b;
B. c += a + b;
C. c = a = b;
D. c = a ++ b;

答案：D

解析：本题选 D。A 是普通赋值，表示把 `a + b` 的结果赋给 `c`；B 中 `c += a + b` 等价于 `c = c + (a + b)`，语法正确；C 中赋值运算可以连续使用，`c = a = b` 会先把 `b` 赋给 `a`，再把这个结果赋给 `c`。D 写成 `a ++ b`，`++` 是自增运算符，后面不能这样再直接接一个 `b`，两个变量之间缺少合法运算关系，所以不符合 C++ 语法。

## 第8题

如果用两个 int 类型的变量 a 和 b 分别表达长方形的长和宽，则下列哪个表达式不能用来计算长方形的周长？（  ）

A. a + b * 2
B. 2 * a + 2 * b
C. a + b + a + b
D. b + a * 2 + b

答案：A

解析：本题选 A。长方形周长应为 `2 * a + 2 * b`，也可以写成 `a + b + a + b`。B 和 C 正好是这个公式；D 中 `b + a * 2 + b` 按乘法优先计算，也就是 `2 * a + 2 * b`。A 中 `a + b * 2` 只等于 `a + 2 * b`，少加了一个 `a`，一般不能表示周长。

## 第9题

表达式 `(3 == 0) + 'A' + 1 + 3.0` 的结果类型为（ ）。

A. double
B. int
C. char
D. bool

答案：A

解析：本题选 A。`3 == 0` 的结果是 `false`，参与加法时可按 0 处理；字符 `'A'` 参与算术运算时会转换为整数编码；表达式中还有 `3.0`，它是 `double` 类型。整数、字符转换后的值和 `double` 相加时，最终会提升为 `double`，所以整个表达式的结果类型是 `double`。

## 第10题

如果 a 为 int 类型的变量，且 a 的值为 6，则执行 `a *= 3;` 之后，a 的值会是（）。

A. 3
B. 6
C. 9
D. 18

答案：D

解析：本题选 D。`a *= 3` 是复合赋值语句，含义等价于 `a = a * 3`。原来 `a` 的值是 6，执行后变成 `6 * 3 = 18`。A、B、C 分别忽略了原值或把乘法误当成其他运算，都不是执行后的结果。

## 第11题

如果 a 和 b 均为 int 类型的变量，下列表达式不能正确判断 “a 等于 0 且 b 等于 0” 的是（  ）

A. `(a == 0) && (b == 0)`
B. `(a == b == 0)`
C. `(!a) && (!b)`
D. `(a == 0) + (b == 0) == 2`

答案：B

解析：本题选 B。A 直接判断 `a == 0` 和 `b == 0` 同时成立，是正确写法；C 中整数 0 在逻辑中表示假，`!a` 在 `a` 为 0 时为真，所以也能判断两个数都为 0；D 中两个比较结果都为真时可按 1 相加，和为 2，也能表示两个条件都成立。B 的 `a == b == 0` 不会像数学里那样连着比较，而是先算 `a == b`，再把这个布尔结果拿去和 0 比较，因此不能正确表达“a 等于 0 且 b 等于 0”。

## 第12题

如果 a 为 int 类型的变量，下列哪个表达式可以正确求出满足“大于等于
a 且是 4 的倍数”的整数中最小的？（  ）

A. `a * 4`
B. `a / 4 * 4`
C. `(a + 3) / 4 * 4`
D. `a - a % 4 + 4`

答案：C

解析：本题选 C。题干只写 `a` 是 `int`，没有单独说明取值范围；在 GESP 1 级这类基础表达式题中，通常默认按非负整数或自然数输入来理解。在这个语境下，`(a + 3) / 4` 相当于把 `a` 除以 4 后向上取整，再乘回 4，所以能得到不小于 `a` 的最小 4 的倍数。例如 `a = 9` 时结果为 12，`a = 8` 时结果仍为 8。A 的 `a * 4` 通常会比所求结果大很多；B 的 `a / 4 * 4` 会舍去不足 4 的部分，`a` 不是 4 的倍数时会小于 `a`；D 在 `a` 本来就是 4 的倍数时仍会多加 4。需要注意，如果允许 `a` 是任意负整数，C++ 整数除法是向 0 截断，题面就存在范围不够严谨的问题，选项 C 并不对所有 `int` 都成立，例如 `a = -4` 时会得到 0，而不是 -4。

## 第13题

在下列代码的横线处填写（），可以使得输出是 “20 10”。

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 10, b = 20;
    a = ________; // 在此处填入代码
    b = a / 100;
    a = a % 100;
    cout << a << " " << b << endl;
    return 0;
}
```

A. `a + b`
B. `(a + b) * 100`
C. `b * 100 + a`
D. `a * 100 + b`

答案：D

解析：本题选 D。横线处执行前 `a = 10, b = 20`。若填 `a * 100 + b`，则 `a` 先变成 `1020`；接着 `b = a / 100` 得到 10，`a = a % 100` 得到 20，最后输出 `20 10`。若填 C，会得到 `2010`，最后输出 `10 20`；A 会得到 `30`，B 会得到 `3000`，都不能得到题目要求的两个数。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是“1248”。

```cpp
#include <iostream>
using namespace std;
int main() {
    for (int i = 1; i <= 8; ______) // 在此处填入代码
        cout << i;
    return 0;
}

```

A. `i++`
B. `i *= 2`
C. `i += 2`
D. `i * 2`

答案：B

解析：本题选 B。循环体每次先输出当前的 `i`，再执行横线处的更新语句。要输出 `1248`，`i` 应该依次变为 1、2、4、8，所以每次更新都要乘 2，`i *= 2` 正好满足。`i++` 会输出 1 到 8，`i += 2` 会输出 1、3、5、7，`i * 2` 只是计算表达式但没有把结果赋回 `i`，不能正确更新循环变量。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    int sum = 0;
    for (int i = 1; i <= 20; i++)
        if (i % 3 == 0 || i % 5 == 0)
            sum += i;
    cout << sum << endl;
    return 0;
}

```

A. 210
B. 113
C. 98
D. 15

答案：C

解析：本题选 C。程序把 1 到 20 中能被 3 或 5 整除的数加入 `sum`，注意 `||` 表示“或者”，同一个数只会在一次 `if` 中加一次。符合条件的数是 3、5、6、9、10、12、15、18、20，它们的和为 `3 + 5 + 6 + 9 + 10 + 12 + 15 + 18 + 20 = 98`，所以输出 98。

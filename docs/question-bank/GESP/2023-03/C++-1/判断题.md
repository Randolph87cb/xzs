# 判断题

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

在 Windows 系统中通过键盘完成对选定文本移动的按键组合是先 Ctrl+X，移动到目标位置后按 Ctrl+V

答案：正确

解析：本题判断为正确。Ctrl+X 会把选定文本剪切到剪贴板，原位置的文本被移走；到目标位置后按 Ctrl+V 粘贴，就完成了移动。

## 第2题

程序员用 C、C++、Python、Scratch 等编写的程序能在 CPU 上直接执行。

答案：错误

解析：本题判断为错误。CPU 能直接执行的是机器指令。C、C++、Python、Scratch 等源程序需要先经过编译、解释或运行环境处理，不能作为源代码直接在 CPU 上执行。

## 第3题

在 C++ 语言中，注释不宜写得过多，否则会使得程序运行速度变慢。

答案：错误

解析：本题判断为错误。注释是给人阅读的说明，编译时不会变成可执行语句。注释多可能影响人阅读代码，但不会让程序运行速度变慢。

## 第4题

在 C++ 语言中，标识符中可以有数字，但不能以数字开头。

答案：正确

解析：本题判断为正确。C++ 标识符可以由字母、数字和下划线组成，但第一个字符不能是数字。例如 `a1` 合法，`1a` 不合法。

## 第5题

'3' 是一个 int 类型常量。

答案：错误

解析：本题判断为错误。`'3'` 用单引号括起来，是字符常量，表示字符 3，不是整数常量 `3`。在 C++ 中它的类型是 `char`。

## 第6题

if 语句中的条件表达式的结果必须为 bool 类型。

答案：错误

解析：本题判断为错误。C++ 的 `if` 条件不只接受 `bool`。数值表达式也可以作为条件，值为 0 时按假处理，非 0 时按真处理。

## 第7题

for 语句的循环体至少会执行一次。

答案：错误

解析：本题判断为错误。`for` 语句会先判断循环条件。如果一开始条件就不成立，循环体一次也不会执行。

## 第8题

如果 a 为 int 类型的变量，则赋值语句 `a = a + 3;` 是错误的，因为这条语句会导致 a 无意义。

答案：错误

解析：本题判断为错误。`a = a + 3;` 的意思是先用当前的 `a` 加 3，再把结果重新赋给 `a`。只要 `a` 已经有确定的值，这条赋值语句就是有意义的。

## 第9题

如果 a 为 int 类型的变量，则表达式 `(a / 4 == 2)` 和表达式 `(a >= 8 && a <= 11)` 的结果总是相同的。

答案：正确

解析：本题判断为正确。`a` 是 `int` 时，`a / 4` 是整数除法，结果为 2 正好对应 `a` 取 8、9、10、11。因此 `(a / 4 == 2)` 与 `(a >= 8 && a <= 11)` 的真假总是相同。

## 第10题

表达式 `(3.5 * 2)` 的计算结果为 7.0，且结果类型为 double。

答案：正确

解析：本题判断为正确。`3.5` 是 `double` 类型常量，和整数 `2` 相乘时按浮点数计算，结果是 `7.0`，表达式结果类型仍为 `double`。

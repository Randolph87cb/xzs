# 判断题

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

在 Windows 系统中通过键盘完成对选定文本移动的按键组合是先 Ctrl+X，移动到目标位置后按 Ctrl+V

答案：正确

解析：本题判断为正确。Ctrl+X 是剪切，Ctrl+V 是粘贴，先剪切再到目标位置粘贴可以完成文本移动。

## 第2题

程序员用 C、C++、Python、Scratch 等编写的程序能在 CPU 上直接执行。

答案：错误

解析：本题判断为错误。CPU 直接执行的是机器指令，高级语言程序通常要经过编译或解释，不能把源代码直接交给 CPU 执行。

## 第3题

在 C++ 语言中，注释不宜写得过多，否则会使得程序运行速度变慢。

答案：错误

解析：本题判断为错误。注释在编译阶段会被忽略，不会生成可执行语句，因此不会因为写得多而直接拖慢程序运行。

## 第4题

在 C++ 语言中，标识符中可以有数字，但不能以数字开头。

答案：正确

解析：本题判断为正确。C++ 标识符可以包含字母、数字和下划线，不能以数字开头；下划线可以出现在标识符中，也可以作为开头。

## 第5题

'3' 是一个 int 类型常量。

答案：错误

解析：本题判断为错误。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第6题

## 第7题

for 语句的循环体至少会执行一次。

答案：错误

解析：本题判断为错误。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

## 第1题

明明和笑笑在“小庙会”上分别抽到一个 4GB 和 4096MB 的 U 盘，容量大的盘是笑笑的（ ）。

答案：错误

解析：1GB = 1024MB，所以 4GB = 4 × 1024MB = 4096MB。明明的 4GB U 盘和笑笑的 4096MB U 盘容量相同，并不是笑笑的更大，因此题干说法错误。

## 第2题

IPv4 的地址通常用“点分十进制”的表示形式，形如（a.b.c.d），其中 a、b、c、d 都是 1~255 之间的十进制整数（  ）。

答案：错误

解析：IPv4 地址由 32 位二进制数表示，点分十进制会把它分成 4 段，每段对应 8 位二进制数。8 位能表示的十进制范围是 0~255，题干写成 1~255，少了可以出现的 0，因此说法错误。

## 第3题

在 C++语言中，一个程序不能有多个 main 函数。

答案：正确

解析：`main` 函数是 C++ 程序的入口。一个完整程序只能有一个入口函数，如果定义多个 `main`，编译或链接时会出现重复定义的问题，因此题干说法正确。

## 第4题

在 C++语言中，标识符中可以有下划线_，但不能以下划线_开头。

答案：错误

解析：C++ 标识符可以由字母、数字和下划线组成，但不能以数字开头。下划线可以出现在标识符中，也可以作为开头，例如 `_count` 在语法形式上是标识符；只是某些以下划线开头的名字在标准库或实现中有保留规则，初学阶段判断合法标识符时关键是“不能以数字开头”。因此题干说“不能以下划线开头”是错误的。

## 第5题

如果 a 是 int 类型的变量，而且值为 1，则表达式'a'的值为'1'。

答案：错误

解析：表达式 `'a'` 是字符常量，表示字母 a 这个字符，和变量 `a` 的当前值没有关系。即使变量 `a` 是 `int` 类型且值为 1，`'a'` 也不会变成字符 `'1'`，因此题干说法错误。

## 第6题

在 if ... else 语句中，else 子句可以嵌套 if ... else 语句，但 if 子句不可以，因为会造成二义性。

答案：错误

解析：`if` 子句和 `else` 子句后面都可以跟一条语句，这条语句本身也可以是新的 `if ... else`。如果嵌套后担心 `else` 和哪个 `if` 配对不清楚，可以用 `{}` 明确语句范围；C++ 中 `else` 默认和最近的未匹配 `if` 配对，并不是禁止在 `if` 子句中嵌套。因此题干说法错误。

## 第7题

while 语句的循环体至少会执行一次。

答案：错误

解析：`while` 会先判断循环条件，条件为真才执行循环体。如果一开始条件就是假，循环体一次也不会执行。至少执行一次的是 `do ... while` 结构，不是普通 `while`，因此题干说法错误。

## 第8题

C++ 语言中 >= 是运算符，但 => 不是。

答案：正确

解析：`>=` 是 C++ 中的关系运算符，表示“大于等于”。`=>` 不是 C++ 的运算符，不能把两个字符顺序写反；例如判断 `x` 大于等于 `5` 应写 `x >= 5`，不能写 `x => 5`。因此题干说法正确。

## 第9题

如果 a 为 char 类型的变量，且取值为小写字母，则执行语句 `a = a - 'a' + 'A';` 后，a 的值会变为与原值对应的大写字母。

答案：正确

解析：在常见 ASCII 编码语境下，小写字母 `'a'` 到 `'z'` 连续排列，大写字母 `'A'` 到 `'Z'` 也连续排列。表达式 `a - 'a'` 得到当前小写字母相对 `'a'` 的偏移量，例如 `'c' - 'a'` 为 2；再加上 `'A'` 就得到对应的大写字母 `'C'`。因此题干说法正确。

## 第10题

表达式 `(10.0 / 2)` 的计算结果为 5.0，且结果类型为 double。

答案：正确

解析：`10.0` 是 `double` 类型的浮点字面量，和整数 `2` 做除法时，`2` 会按浮点运算参与计算。结果数值是 `5.0`，表达式结果类型也是 `double`，因此题干说法正确。

## 第1题

明明和笑笑在“小庙会”上分别抽到一个 4GB 和 4096MB 的 U 盘，容量大的盘是笑笑的（ ）。

答案：错误

解析：本题判断为错误。按容量单位和字节数换算即可验证，题干给出的大小关系或比例与实际计算结果不一致或一致。

## 第2题

IPv4 的地址通常用“点分十进制”的表示形式，形如（a.b.c.d），其中 a、b、c、d 都是 1~255 之间的十进制整数（  ）。

答案：错误

解析：本题判断为错误。IPv4 点分十进制每段范围包含 0 到 255；域名从右向左看层级，最右侧才是顶级域名。

## 第3题

在 C++语言中，一个程序不能有多个 main 函数。

答案：正确

解析：本题判断为正确。赋值号 `=` 与相等比较 `==` 含义不同，C++ 运算符也必须写成规定形式；题干说法需与这些语法规则一致。

## 第4题

在 C++语言中，标识符中可以有下划线_，但不能以下划线_开头。

答案：错误

解析：本题判断为错误。C++ 标识符可以包含字母、数字和下划线，不能以数字开头；下划线可以出现在标识符中，也可以作为开头。

## 第5题

如果 a 是 int 类型的变量，而且值为 1，则表达式'a'的值为'1'。

答案：错误

解析：本题判断为错误。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第6题

在 if ... else 语句中，else 子句可以嵌套 if ... else 语句，但 if 子句不可以，因为会造成二义性。

答案：错误

解析：本题判断为错误。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

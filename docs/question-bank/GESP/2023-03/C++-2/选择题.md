## 第1题

以下存储器中的数据不会受到附近强磁场干扰的是（ ）。

A. 硬盘
B. U盘
C. 内存
D. 光盘

答案：D

解析：光盘依靠激光读写盘面上的信息，属于光存储介质，不是靠磁性记录数据，所以附近强磁场通常不会改变光盘上的数据。硬盘使用磁性介质，强磁场可能影响数据；U 盘和内存虽然不是磁盘，但其中的电子电路也可能受到强电磁环境影响。题目问“不会受到附近强磁场干扰”的存储器，最符合的是光盘，选 D。

## 第2题

下列流程图，属于计算机的哪种程序结构？（ ）。

![](https://cdn.luogu.com.cn/upload/image_hosting/rbcxs9t6.png)

A. 顺序结构
B. 循环结构
C. 分支结构
D. 数据结构

答案：C

解析：题图中的流程会根据一个条件走向不同的处理路径，也就是“条件成立”和“条件不成立”时执行不同步骤，这正是分支结构的特征。顺序结构是按固定顺序一直执行，循环结构会重复执行一段流程，数据结构不是程序控制结构，所以选 C。

## 第3题

下列关于 C++ 语言的叙述，不正确的是（ ）。

A. double 类型的变量占用内存的大小是浮动的
B. bool 类型的变量占用1字节内存
C. int 类型变量的取值范围不是无限的
D. char 类型的变量有 256 种取值

答案：A

解析：题目要求选“不正确”的叙述。`double` 类型变量在同一编译环境下占用空间是确定的，常见环境中为 8 字节，并不是“浮动的”，所以 A 错。`bool` 在 GESP 常见教学语境中按 1 字节理解；`int` 的取值范围有限；`char` 通常占 1 字节，有 256 种不同取值模式，这些说法都不是本题要找的错误项。因此选 A。

## 第4题

下列关于 C++ 语言的叙述，不正确的是（ ）。

A. 变量定义后，可以使用赋值语句改变它的值
B. 变量定义时，必须指定类型
C. 变量名必须为合法标识符
D. 合法标识符可以以数字开始

答案：D

解析：题目要求选“不正确”的叙述。变量定义后可以通过赋值语句修改变量值，定义变量时需要写出类型，变量名也必须是合法标识符，这三项都正确。C++ 标识符可以包含数字，但不能以数字开头，例如 `a1` 可以，`1a` 不可以，所以 D 的说法错误，选 D。

## 第5题

以下哪个不是 C++语言的关键字？

A. return
B. max
C. else
D. case

答案：B

解析：`return` 用于从函数返回，`else` 用于分支语句，`case` 用于 `switch` 语句，它们都是 C++ 关键字。`max` 常见于库函数或普通标识符名称，本身不是 C++ 关键字，所以“不属于关键字”的是 B。

## 第6题

以下哪个不是 C++ 语言的运算符？

A. ` \= `
B. ` /= `
C. ` -= `
D. ` != `

答案：A

解析：C++ 运算符必须写成规定形式。`/=` 是除后赋值运算符，`-=` 是减后赋值运算符，`!=` 是不等比较运算符，都是合法运算符。选项 A 写成 `\=`，反斜杠加等号不是 C++ 运算符，因此选 A。

## 第7题

如果 a 和 b 都是 char 类型的变量，下列哪个语句不符合 C++ 语法？

A. ` b = a + 1; `
B. ` b = a + '1'; `
C. ` b = 'a'++; `
D. ` b = a++; `

答案：C

解析：`'a'` 是字符常量，不是变量，不能被 `++` 修改，所以 `b = 'a'++;` 不符合 C++ 语法。A、B 中右侧表达式会先按字符编码参与整数运算，再赋值给 `char` 变量；D 中 `a++` 是对变量 `a` 自增，表达式值可以再赋给 `b`。因此不合法的是 C。

## 第8题

如果 a、b、c 和 d 都是 int 类型的变量，则下列哪个表达式能够正确计算它们的平均值？

A. ` (a + b + c + d) / 4 `
B. ` (a + b + c + d) % 4 `
C. ` (a + b + c + d) / 4.0 `
D. ` (a + b + c + d) % 4.0 `

答案：C

解析：平均值应为四个数的和除以 4，并且可能不是整数。A 中除数是整数 `4`，若总和不能被 4 整除，会发生整数除法截断；B 求的是除以 4 的余数，不是平均值；D 中 `%` 只能用于整数，不能和 `4.0` 这样的浮点数一起使用。C 用 `4.0` 让表达式按浮点运算得到平均值，选 C。

## 第9题

如果 a 为 char 类型的变量，且 a 的值为 '2'，则下列哪条语句执行后，a 的值不会变为 '3'？

A. ` a = a + 1; `
B. ` a + 1; `
C. ` a = 1 + a; `
D. ` ++a; `

答案：B

解析：字符 `'2'` 的下一个字符编码是 `'3'`。A 的 `a = a + 1;` 和 C 的 `a = 1 + a;` 都会把加 1 后的结果重新赋给 `a`，D 的 `++a;` 会直接让 `a` 自增，所以它们都会使 `a` 变为 `'3'`。B 只是计算表达式 `a + 1`，没有赋值给任何变量，计算结果被丢弃，`a` 仍然是 `'2'`，所以选 B。

## 第10题

如果 a 为 int 类型的变量，且 a 的值为 9，则执行 `a -= 3;` 之后，a 的值会是（）。

A. 3
B. 6
C. 9
D. 12

答案：B

解析：复合赋值 `a -= 3;` 等价于 `a = a - 3;`。原来 `a` 的值是 9，执行后变成 `9 - 3 = 6`，所以选 B。A 的 3 是把 9 除以 3 才可能得到的结果；C 表示值没有变化；D 表示加 3，都不符合这条语句的含义。

## 第11题

如果 a 和 b 均为 int 类型的变量，下列表达式能正确判断“a 等于 0 或 b 等于 0”的是（ ）

A. ` (!a) || (!b) `
B. ` (a == b == 0) `
C. ` (a == 0) && (b == 0) `
D. ` (a == 0) - (b == 0) == 0 `

答案：A

解析：在 C++ 中，整数 0 转成逻辑值是假，非 0 为真，所以 `!a` 当且仅当 `a == 0` 时为真，`(!a) || (!b)` 表示“a 等于 0 或 b 等于 0”。B 不是数学里的链式比较，它会先算 `a == b`，再把结果和 0 比较；C 使用 `&&`，表示两个变量都等于 0；D 判断两个布尔结果是否相同，会在二者都不等于 0 时也为真。因此选 A。

## 第12题

如果 a 为 char 类型的变量，下列哪个表达式可以正确判断“a 是小写字母”？

A. ` a <= a <= z `
B. ` a - 'a' <= 'z' - 'a' `
C. ` 'a' <= a <= 'z' `
D. ` a >= 'a' && a <= 'z' `

答案：D

解析：判断小写字母需要同时满足 `a >= 'a'` 和 `a <= 'z'`，所以 D 正确。A 中 `a <= a` 总为真，后面再和 `z` 比较，既不是正确范围判断，且 `z` 若不是变量还会有语法问题；B 等价于只判断 `a <= 'z'`，没有排除 `'a'` 之前的字符；C 也是错误的链式比较，`'a' <= a` 会先得到 0 或 1，再拿这个结果和 `'z'` 比较，不能表达区间条件。因此选 D。

## 第13题

在下列代码的横线处填写（），使得输出是 `50 10`。

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 10, b = 50;
    ________; // 在此处填入代码
    b -= a;
    a += b;
    cout << a << " " << b << endl;
    return 0;
}

```

A. ` a -= b `
B. ` a += b `
C. ` a = b - a `
D. ` a = b `

答案：C

解析：初始时 `a = 10, b = 50`。若填 C：先执行 `a = b - a`，得到 `a = 40, b = 50`；再执行 `b -= a`，得到 `b = 10`；最后执行 `a += b`，得到 `a = 50`，输出正好是 `50 10`。A 会先让 `a` 变成 -40，后续输出不对；B 会让 `a` 变成 60；D 会让 `a` 直接等于 50，也都不能得到目标输出。因此选 C。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是 `5`。

```cpp
#include <iostream>
using namespace std;
int main() {
    int cnt = 0;
    for (char ch = '1'; ch <= '9'; ch++)
        if (________) // 在此处填入代码
            cnt++;
    cout << cnt << endl;
    return 0;
}

```

A. ` ch < '5' `
B. ` ch >= 5 `
C. ` ch >= '4' `
D. ` ch % 2 == 1 `

答案：D

解析：程序让 `ch` 从字符 `'1'` 依次变化到 `'9'`，并统计满足横线条件的字符个数。字符参与 `%` 运算时会使用对应的 ASCII 码，`'1'`、`'3'`、`'5'`、`'7'`、`'9'` 的 ASCII 码为奇数，一共有 5 个，所以填 `ch % 2 == 1` 会输出 5。`ch < '5'` 只会选 `'1'` 到 `'4'` 共 4 个；`ch >= 5` 中的 5 是整数值，不是字符 `'5'`，数字字符的 ASCII 码都大于 5，会统计 9 个；`ch >= '4'` 会选 `'4'` 到 `'9'` 共 6 个，因此选 D。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    int n = 17;
    bool isprime = true;
    for (int i = 2; i <= n; i++)
        if (n % i == 0)
            isprime = false;
    cout << isprime << endl;
    return 0;
}

```

A. false
B. true
C. 0
D. 1

答案：C

解析：程序令 `n = 17`，循环条件是 `i <= n`，所以最后会检查到 `i = 17`。当 `i` 等于 17 时，`17 % 17 == 0` 成立，`isprime` 被改成 `false`。普通 `cout` 输出 `bool` 值时，如果没有使用 `boolalpha`，会输出 `false` 对应的数字 `0`、`true` 对应的数字 `1`，因此最终输出是 `0`，选 C。

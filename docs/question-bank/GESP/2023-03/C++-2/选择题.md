## 第12题

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
#include <iostream>
using namespace std;
int main() {
    int n = 17;
    bool isprime = true;
    for (int i = 2; i <= n; i++)
        if (n % i == 0)
            isprime = false;
    cout << isprime << endl;
    return 0;
}

```

A. false
B. true
C. 0
D. 1

答案：C

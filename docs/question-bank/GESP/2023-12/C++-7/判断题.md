## 第1题

小杨这学期准备参加 GESP 的 7 级考试，其中有关于三角函数的内容，他能够通过下面的代码找到结束循环的角度值。（ ）

```cpp
int main()
{
    double x;

    do {
        cin >> x;
        x = x / 180 * 3.14;
    } while (int(sin(x) * sin(x) + cos(x) * cos(x)) == 1);
    cout << "//" << sin(x) << " " << cos(x);

    cout << endl;
    return 0;
}

答案：正确

解析：本题判断为正确。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第7题

某个哈希表键值 $x$ 为整数，为其定义哈希函数 $H(x) = x \% p$，则 $p$ 选择素数时不会产生冲突。（ ）

答案：错误

解析：本题判断为错误。哈希函数取模后不同键仍可能得到相同值，表长或模数为素数只能降低某些冲突概率，不能保证无冲突。

## 第8题

动态规划只要推导出状态转移方程，就可以写出递归程序来求出最优解。（ ）

答案：错误

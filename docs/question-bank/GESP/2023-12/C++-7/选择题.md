## 第1题

定义变量 `double x`，如果下面代码输入为 $100$，输出最接近（ ）。

```cpp
#include <iostream>
#include <string>
#include <cmath>
#include <vector>
using namespace std;

int main()
{
    double x;

    cin >> x;
    cout << log10(x) - log2(x) << endl;

    cout << endl;
    return 0;
}
```

A. 0
B. -5
C. -8
D. 8

答案：B

解析：本题选 B。程序输入 $x=100$ 后计算的是 $\log_{10}(100)-\log_2(100)$。其中 $\log_{10}(100)=2$，而 $\log_2(100)\approx 6.64$，差值约为 $-4.64$，最接近选项 B 的 $-5$。A 的 $0$ 表示两个对数相等，不成立；C 的 $-8$ 低估了 $\log_2(100)$ 与 $\log_{10}(100)$ 的差；D 的 $8$ 连符号也相反。

## 第2题

对于下面动态规划方法实现的函数，以下选项中最适合表达其状态转移函数的为（ ）。

```cpp
int s[MAX_N], f[MAX_N][MAX_N];
int stone_merge(int n, int a[]) {
    for (int i = 1; i <= n; i++)
        s[i] = s[i - 1] + a[i];
    for (int i = 1; i <= n; i++)
        for (int j = 1; j <= n; j++)
            if (i == j)
                f[i][j] = 0;
            else
                f[i][j] = MAX_F;
    for (int l = 1; l < n; l++)
        for (int i = 1; i <= n - l; i++) {
            int j = i + l;
            for (int k = i; k < j; k++)
                f[i][j] = min(f[i][j], f[i][k] + f[k + 1][j] + s[j] - s[i - 1]);
        }
    return f[1][n];
}
```

A. $f(i,j) = \min_{i \le k < j} (f(i,j), f(i,k) + f(k+1,j) + s(j) - s(i-1))$
B. $f(i,j) = \min_{i \le k < j} (f(i,j), f(i,k) + f(k+1,j) + \sum_{k=i}^{j} a(k))$
C. $f(i,j) = \min_{i \le k \le j} (f(i,k) + f(k+1,j) + \sum_{k=i}^{j+1} a(k))$
D. $f(i,j) = \min_{i \le k < j} (f(i,k) + f(k+1,j)) + \sum_{k=i}^{j} a(k)$

答案：D

解析：本题选 D。代码中 $f[i][j]$ 表示合并区间 $[i,j]$ 的最小代价，枚举最后一次合并的分界点 $k$，先分别合并 $[i,k]$ 和 $[k+1,j]$，再付出整段石子重量 $s[j]-s[i-1]=\sum_{t=i}^{j}a_t$。因此递推式为 $f(i,j)=\min_{i\le k<j}\{f(i,k)+f(k+1,j)\}+\sum_{t=i}^{j}a_t$。A、B 把当前的 $f(i,j)$ 也写进 $\min$ 中，更像程序更新语句而不是状态转移函数；C 允许 $k=j$，会出现空区间 $f(j+1,j)$，并且求和范围也不对。

## 第3题

下面代码可以用来求最长上升子序列（LIS）的长度，如果输入是：$5\ 1\ 7\ 3\ 5\ 9$，则输出是（ ）。

```cpp
int a[2023], f[2023];
int main()
{
    int n, i, j, ans = -1;

    cin >> n;
    for (i = 1; i <= n; i++) {
        cin >> a[i];
        f[i] = 1;
    }

    for (i = 1; i <= n; i++)
        for (j = 1; j < i; j++)
            if (a[j] < a[i])
                f[i] = max(f[i], f[j] + 1);
    for (i = 1; i <= n; i++) {
        ans = max(ans, f[i]);
        cout << f[i] << " ";
    }

    cout << ans << endl;
    return 0;
}
```

A. 9 7 5 1 1 9
B. 1 2 2 3 4 4
C. 1 3 5 7 9 9
D. 1 1 1 1 1 1

答案：B

解析：本题选 B。输入的 $n=5$，序列为 $1,7,3,5,9$。程序令 $f[i]$ 表示以 $a[i]$ 结尾的最长上升子序列长度：$a_1=1$ 时 $f_1=1$；$7$ 可接在 $1$ 后，$f_2=2$；$3$ 也只能接在 $1$ 后，$f_3=2$；$5$ 可接在 $1,3$ 后，$f_4=3$；$9$ 可接在 $1,3,5$ 后，$f_5=4$。随后程序依次输出 $f[i]$，最后输出最大值 $ans=4$，所以整体输出为 `1 2 2 3 4 4`。A、C 把数值本身当成了输出，D 忽略了状态递推。

## 第4题

C++ 语言中，下列关于关键字 `static` 的描述不正确的是（ ）。

A. 可以修饰类的成员函数。
B. 常量静态成员可以在类外进行初始化。
C. 若 a 是类 A 常量静态成员，则 a 的地址都可以访问且唯一。
D. 静态全局对象一定在 main 函数调用前完成初始化，执行完 main 函数后被析构。

答案：C

解析：本题选 C。`static` 可以修饰类的成员函数，A 正确；静态常量成员在需要类外定义或初始化时可以在类外给出定义，B 正确；静态全局对象具有静态存储期，通常在进入 `main` 前完成初始化，在 `main` 结束后按规则析构，D 正确。C 的问题在于“地址都可以访问且唯一”说得过绝对：类内声明的常量静态成员如果没有满足取地址所需的定义，取地址属于 odr-use，可能无法通过链接；因此不能说其地址总是可以直接访问。

## 第5题

$G$ 是一个非连通无向图，共有 $28$ 条边，则该图至少有（ ）个顶点。

A. 6
B. 7
C. 8
D. 9

答案：D

解析：本题选 D。要让一个非连通简单无向图在顶点数尽量少时拥有尽量多的边，应把尽可能多的顶点放在一个完全图中，另留至少一个孤立顶点保证非连通。若共有 $v$ 个顶点，最多可有 $\binom{v-1}{2}$ 条边。$v=8$ 时最多 $\binom{7}{2}=21$ 条，不够 $28$ 条；$v=9$ 时最多 $\binom{8}{2}=28$ 条，刚好可行，所以至少需要 $9$ 个顶点。

## 第6题

哈希表长 $31$，按照下面的程序依次输入 $4\ 17\ 28\ 30\ 4$，则最后的 $4$ 存入哪个位置？（ ）

```cpp
#include <iostream>
#include <string>
#include <cmath>
#include <vector>
using namespace std;

const int N = 31;
int htab[N], flag[N];
int main()
{
    int n, x, i, j, k;

    cin >> n;
    for (i = 0; i < n; i++) {
        cin >> x;
        k = x % 13;
        while (flag[k]) k = (k + 1) % 13;
        htab[k] = x;
        flag[k] = 1;
    }

    for (i = 0; i < N; i++)
        cout << htab[i] << " ";

    cout << endl;
    return 0;
}
```

A. 3
B. 4
C. 5
D. 6

答案：D

解析：本题选 D。代码先读入 $n$，所以输入 `4 17 28 30 4` 表示要插入 $4$ 个数：$17,28,30,4$。散列位置按 `x % 13` 计算，并用线性探测 `(k + 1) % 13` 解决冲突：$17\bmod 13=4$，占用位置 $4$；$28\bmod 13=2$，占用位置 $2$；$30\bmod 13=4$，位置 $4$ 已占用，探测到位置 $5$；最后 $4\bmod 13=4$，位置 $4$、$5$ 都已占用，继续探测到位置 $6$。因此最后的 $4$ 存入位置 $6$。

## 第7题

某二叉树 $T$ 的先序遍历序列为：$\{A\ B\ D\ F\ C\ E\ G\ H\}$，中序遍历序列为：$\{B\ F\ D\ A\ G\ E\ H\ C\}$，则下列说法中正确的是（ ）。

A. $T$ 的度为 $1$
B. $T$ 的高为 $4$
C. $T$ 有 $4$ 个叶节点
D. 以上说法都不对

答案：B

解析：本题选 B。先序序列首元素为根 $A$，在中序序列中 $A$ 左侧为 $\{B,F,D\}$、右侧为 $\{G,E,H,C\}$。左子树先序为 $B,D,F$，可得 $B$ 的右孩子为 $D$，$D$ 的左孩子为 $F$；右子树根为 $C$，其左子树根为 $E$，$E$ 的左右孩子分别为 $G,H$。最长根到叶路径如 $A\to B\to D\to F$，按结点数计高度为 $4$，所以 B 正确。该树存在结点有两个孩子，度不是 $1$；叶结点为 $F,G,H$，共 $3$ 个，不是 $4$ 个，因此 A、C 错，D 也错。

## 第8题

下面代码段可以求两个字符串 $s1$ 和 $s2$ 的最长公共子串（LCS），下列相关描述不正确的是（ ）。

```cpp
while (cin >> s1 >> s2)
{
    memset(dp, 0, sizeof(dp));
    int n1 = strlen(s1), n2 = strlen(s2);
    for (int i = 1; i <= n1; ++i)
        for (int j = 1; j <= n2; ++j)
            if (s1[i - 1] == s2[j - 1])
                dp[i][j] = dp[i - 1][j - 1] + 1;
            else
                dp[i][j] = max(dp[i - 1][j], dp[i][j - 1]);
    cout << dp[n1][n2] << endl;
}
```

A. 代码的时间复杂度为 $O(n^2)$
B. 代码的空间复杂度为 $O(n^2)$
C. 空间复杂度已经最优
D. 采用了动态规划求解

答案：C

解析：本题选 C。观察代码的转移关系：当 `s1[i - 1] == s2[j - 1]` 时由 `dp[i - 1][j - 1] + 1` 转移；不相等时取 `dp[i - 1][j]` 和 `dp[i][j - 1]` 的较大值。这类状态表示的是两个前缀的最长公共子序列长度，而不是要求连续匹配的一段字符串。无论按两个字符串长度分别记为 $n_1,n_2$，还是都记为 $n$，二维数组 `dp` 的时间和空间规模都是平方级。B 对二维数组空间复杂度的描述成立，但 C 说空间复杂度是最优并不正确，因为当前行只依赖上一行和当前行左侧信息，可以用滚动数组把空间降到与较短字符串长度成正比。A、B、D 都符合这段动态规划代码的性质，所以不正确项为 C。

## 第9题

图的广度优先搜索中既要维护一个标志数组标志已访问的图的结点，还需哪种结构存放结点以实现遍历？（ ）

A. 双向栈
B. 队列
C. 哈希表
D. 堆

答案：B

解析：本题选 B。广度优先搜索按“先发现的结点先扩展”的顺序逐层遍历，需要用先进先出的队列保存已经发现但尚未扩展的结点。栈会变成后进先出，更接近深度优先搜索；哈希表适合做查找或去重，不负责维护访问顺序；堆按优先级取元素，也不符合 BFS 的层次顺序。

## 第10题

对关键字序列 $\{44,\ 36,\ 23,\ 35,\ 52,\ 73,\ 90,\ 58\}$ 建立哈希表，哈希函数为 $h(k) = k \% 7$，执行下面的 `Insert` 函数，则等概率情况下的平均成功查找长度（即查找成功时的关键字比较次数的均值）为（ ）。

```cpp
#include <iostream>
#include <string>
#include <cmath>
#include <vector>
using namespace std;

typedef struct Node {
    int data;
    struct Node *next;
} Node;
Node* hTab[7];
int key[] = {44, 36, 23, 35, 52, 73, 90, 58, 0};
void Insert()
{
    int i, j;
    Node *x;

    for (i = 0; key[i]; i++) {
        j = key[i] % 7;
        x = new Node;
        x->data = key[i];
        x->next = hTab[j];
        hTab[j] = x;
    }

    return;
}
```

A. $7/8$
B. $1$
C. $1.5$
D. $2$

答案：C

解析：本题选 C。该哈希表采用链地址法，并且每次新结点插到链表表头。各关键字的桶号为：$35\bmod 7=0$，$36\bmod 7=1$，$44,23,58$ 都在桶 $2$，$52,73$ 都在桶 $3$，$90$ 在桶 $6$。按插入表头后的链表顺序，成功查找比较次数分别为：桶 $0$ 的 $35$ 为 $1$ 次，桶 $1$ 的 $36$ 为 $1$ 次，桶 $2$ 中 $58,23,44$ 分别为 $1,2,3$ 次，桶 $3$ 中 $73,52$ 分别为 $1,2$ 次，桶 $6$ 的 $90$ 为 $1$ 次。总比较次数为 $12$，等概率平均为 $12/8=1.5$。

## 第11题

学生在读期间所上的某些课程中需要先上其他的课程，所有课程和课程间的先修关系构成一个有向图 $G$，有边 $\langle U, V \rangle$ 表示课程 $U$ 是课程 $V$ 的先修课，则要找到某门课程 $C$ 的全部先修课下面哪种方法不可行？（ ）

A. BFS 搜索
B. DFS 搜索
C. DFS+BFS
D. 动态规划

答案：D

解析：本题选 D。边 $\langle U,V\rangle$ 表示 $U$ 是 $V$ 的先修课，要找课程 $C$ 的全部先修课，本质上是找所有能够沿有向边到达 $C$ 的课程，或者在反向图中从 $C$ 出发能到达的课程。用 BFS 或 DFS 都可以完成图搜索，先构造反向邻接关系后也可以混合使用 DFS、BFS。动态规划通常要求明确的阶段、状态和无环依赖关系；题目只给出一般有向图，可能存在环，不能保证用动态规划直接求出全部先修课，所以不可行的是 D。

## 第12题

B. $1013$
C. $1012$

## 第13题

用下面的邻接表结构保存一个有向图 $G$，`InfoType` 和 `VertexType` 是定义好的类。设 $G$ 有 $n$ 个顶点、$e$ 条弧，则求图 $G$ 中某个顶点 $u$（其顶点序号为 $k$）的度的算法复杂度是（ ）。

```cpp
typedef struct ArcNode {
    int adjvex;           // 该弧所指向的顶点的位置
    struct ArcNode *nextarc; // 指向下一条弧的指针
    InfoType *info;       // 该弧相关信息的指针

typedef struct VNode {
    VertexType data;      // 顶点信息
    ArcNode *firstarc;    // 指向第一条依附该顶点的弧

typedef struct {
    AdjList vertices;
    int vexnum, arcnum;
    int kind;             // 图的种类标志

```

A. $O(n)$
B. $O(e)$
C. $O(n + e)$
D. $O(n + 2 \times e)$

答案：B

解析：本题选 B。邻接表中每个 `ArcNode` 对应一条弧。求有向图中顶点 $u$ 的度时，出度可以扫描 $u$ 自己的出边链表；入度需要找出哪些弧指向顶点序号 $k$，通常要扫描弧结点。主导代价来自检查弧结点，弧的数量为 $e$，所以复杂度取 $O(e)$。A 只考虑顶点数，无法反映扫描弧的工作量；C 的 $O(n+e)$ 会把遍历顶点表头也算入，但在这些选项中更直接对应主导扫描代价的是 $O(e)$；D 的 $O(2e)$ 只是多了常数因子，大 $O$ 记号中不保留常数。

## 第14题

给定一个简单有向图 $G$，判断其中是否存在环路的下列说法哪个最准确？（ ）

A. BFS 更快
B. DFS 更快
C. BFS 和 DFS 一样快
D. 不确定

答案：D

解析：本题选 D。判断有向图是否存在环，可以用 DFS 维护访问状态和递归栈，也可以用 BFS 思路的拓扑排序：不断取出入度为 $0$ 的点，最后看是否仍有顶点未被删除。两类方法在邻接表表示下通常都能做到 $O(n+e)$，但实际运行中谁更快会受图的结构、存储方式、遍历顺序和实现细节影响。A、B 直接断言某一种更快依据不足；C 说“一样快”也过于绝对，因为渐进复杂度相同不代表实际耗时完全相同。因此最稳妥的判断是“不确定”，选 D。

## 第15题

从顶点 $v_1$ 开始遍历下图 $G$ 得到顶点访问序列，在下面所给的 4 个序列中符合广度优先的序列有几个？（ ）

$\{v_1\ v_2\ v_3\ v_4\ v_5\}$，$\{v_1\ v_2\ v_4\ v_3\ v_5\}$，$\{v_1\ v_4\ v_2\ v_3\ v_5\}$，$\{v_1\ v_2\ v_4\ v_5\ v_3\}$


![](https://cdn.luogu.com.cn/upload/image_hosting/vuh0s68z.png)

A. $4$
B. $3$
C. $2$
D. $1$

答案：B

解析：本题选 B。图中 $v_1$ 的相邻点是 $v_2$ 和 $v_4$，它们都属于 BFS 的第 $1$ 层；$v_3$、$v_5$ 需要经过 $v_2$ 或 $v_4$ 才能到达，属于后续层。因此合法 BFS 序列必须先访问完 $v_2$、$v_4$ 这两个第 $1$ 层结点，再访问 $v_3$、$v_5$。第 1 个序列 `{v1 v2 v3 v4 v5}` 把第 $2$ 层的 $v_3$ 放在第 $1$ 层的 $v_4$ 前，不符合 BFS。其余三个序列都可以通过不同的邻接点扫描顺序得到，所以符合广度优先的序列有 $3$ 个。

## 第1题

下面 C++ 数组的定义中，会丢失数据的是

【知识点：数组与字符数组】

A. `char dict_key[] = {'p','t','o'};`
B. `int dict_value[] = {33,22,11};`
C. `char dict_name[]={'chen','wang','zhou'};`
D. `float dict_value[]={3,2,1};`

答案：C

解析：`char` 数组的每个元素只能保存一个字符。C 选项中的 `'chen'`、`'wang'`、`'zhou'` 不是字符串，而是多字符字符常量，其类型通常按整数处理，再放入 `char` 元素时会发生截断，数据会丢失。A 是 3 个单字符，B 是整型数组，D 中 `3、2、1` 可作为浮点数保存，都不属于这里的丢失数据情形。

## 第2题

在下列编码中，不能够和二进制"1101 1101"相等的是()。

【知识点：进制转换与进制字面量】

A. $(221)_{10}$ 进制
B. $(335)_8$ 进制
C. $(dd)_{16}$ 进制
D. $(5d)_{16}$ 进制

答案：D

解析：二进制 `1101 1101` 按十进制计算为 `128+64+16+8+4+1=221`。八进制 `335` 等于 `3*64+3*8+5=221`，十六进制 `dd` 等于 `13*16+13=221`。十六进制 `5d` 等于 `5*16+13=93`，不能与原二进制数相等，所以选 D。

## 第3题

下面 C++ 代码执行后不能输出"GESP"的是()。

【知识点：字符串处理】

A. `string str("GESP"); cout<<str<<endl;`
B. `string str="GESP"; cout<<str<<endl;`
C. `string str("GESP"); cout<<str[1]<<str[2]<<str[3]<<str[4]<<endl;`
D. `string str{"GESP"}; cout<<str<<endl;`

答案：C

解析：A、B、D 都是合法地把字符串初始化为 `"GESP"`，再整体输出，所以会输出 `GESP`。C 使用下标逐个输出，但字符串下标从 `0` 开始：`str[0]` 是 `G`，`str[1]` 是 `E`，`str[2]` 是 `S`，`str[3]` 是 `P`；它没有输出 `str[0]`，却从 `str[1]` 开始输出，`str[4]` 也不是字符串中的有效字符位置，因此不能输出 `"GESP"`。

## 第4题

执行下面C++代码输出是()。
```cpp
int temp=0;
for(int i=1;i<7;i++)
{
    for(int j=1;j<5;j++)
    {
        if(i/j==2)
        {
            temp++;
        }
    }
}
cout<<temp<<endl;
```

【知识点：控制结构与程序阅读】

A. 10
B. 8
C. 4
D. 3

答案：C

解析：这里的 `/` 是整数除法，结果会向零截断。逐个数满足 `i/j==2` 的情况：`i=2,j=1`；`i=4,j=2`；`i=5,j=2`；`i=6,j=3`，共 4 次。其他 `i、j` 组合的整数商不是 2，所以 `temp` 最终为 `4`，选 C。

## 第5题

执行下面 C++ 代码后，输出是()。
```cpp
string str=("chen");
int x=str.length();
int temp=0;
for(int i=0;i<=x;i++)
{
    temp++;
}
cout<<temp<<endl;
```

【知识点：字符串处理】

A. $4$
B. $2$
C. $5$
D. $3$

答案：C

解析：字符串 `"chen"` 的长度是 `4`，所以 `x=4`。循环条件是 `i<=x`，不是 `i<x`，因此 `i` 会取 `0、1、2、3、4` 共 5 个值，每次 `temp++`，最终输出 `5`，选 C。

string str=("chen");

## 第7题

执行下面 C++ 代码后输出的是()。
```cpp
string str=("chen");
cout<<str[5]<<endl;
```

【知识点：字符串处理】

A. 输出未知的数
B. 输出'n'
C. 输出'\0'
D. 输出空格

答案：A

解析：`"chen"` 的有效下标是 `0、1、2、3`，分别对应 `c、h、e、n`。程序访问 `str[5]` 已经越过字符串范围，结果不确定，不能保证输出 `n`、空格或 `'\0'`。因此只能判断为输出未知内容，选 A。

## 第8题

下面 C++ 代码执行后的输出是()。
```cpp
char ch[10]={'1'};
cout<<ch[2]<<endl;
```

【知识点：数组与字符数组】

A. $0$
B. $1$
C. 输出空格
D. 什么也不输出

答案：D

解析：`char ch[10]={'1'};` 只把 `ch[0]` 初始化为字符 `'1'`，其余元素会被初始化为字符 `'\0'`。`ch[2]` 是空字符，使用 `cout` 输出时没有可见字符，只会执行后面的换行；它不是字符 `'0'`，也不是空格，所以选 D。

## 第9题

下面 C++ 代码用于统计每种字符出现的次数，当输出为 $3$ 时，横线上不能填入的代码是()。
```cpp
string str="GESP is a good programming test!";
int x=0;
for(int i=0;i<str.length();i++)
{
    if(_________)
    {
        x++;
    }
}
cout<<x<<endl;
```

【知识点：字符串处理】

A. `str[i]=='o'`
B. `str[i]=='a'+14`
C. `str[i]==115`
D. `str[i]==111`

答案：C

解析：字符串中小写字母 `o` 出现 3 次：`good` 中 2 次，`programming` 中 1 次。A 判断 `'o'`，D 判断 ASCII 码 `111`，都能统计到这 3 次；B 中 `'a'+14` 也是字符 `'o'`，同样得到 3。C 中 `115` 是小写 `'s'` 的 ASCII 码，字符串里的小写 `s` 只在 `is` 和 `test` 中各出现 1 次，共 2 次，不能使输出为 3，所以选 C。

## 第10题

32 位计算机中，C++ 的整型变量 int 能够表示的数据范围是()。

【知识点：原码反码补码与整数表示】

A. $2^{31} \sim (2^{31})-1$
B. $2^{32}$
C. $-2^{31} \sim +(2^{31})-1$
D. $-(2^{31})+1 \sim 2^{31}$

答案：C

解析：32 位有符号 `int` 通常用补码表示，共有 `2^32` 个不同取值，其中一半表示负数，一半表示非负数，范围是 `-2^31` 到 `2^31-1`。正数上界少 1 是因为还要表示 `0`。A 没有给出负数范围，B 只是数量不是范围，D 的两端都不符合补码范围，所以选 C。

## 第11题

下面 C++ 程序执行的结果是()。

```cpp
int cnt=0;
for(int i=0;i<=20;i++)
{
    if(i%3==0&&i%5==0)
    {
        cnt++;
    }
}
cout<<cnt;
```

【知识点：控制结构与程序阅读】

A. $2$
B. $3$
C. $5$
D. $4$

答案：A

解析：循环检查 `0` 到 `20` 中同时能被 `3` 和 `5` 整除的数，也就是 `15` 的倍数。这个范围内满足条件的是 `0` 和 `15`，共 2 个；注意循环从 `0` 开始且条件是 `i<=20`，所以 `0` 会被计入，答案选 A。

## 第12题

C++ 的数据类型转换让人很难琢磨透，下列代码输出的值是()。

```cpp
int a=3;
int b=2;
cout<<a/b*1.0<<endl;
```

【知识点：运算符、表达式与类型转换】

A. $1.5$
B. $1$
C. $2$
D. $1.50$

答案：B

解析：表达式按从左到右计算同优先级的 `/` 和 `*`。`a`、`b` 都是 `int`，所以 `a/b` 先进行整数除法，`3/2` 的结果是 `1`，小数部分已经被舍去；之后再乘以 `1.0` 只会得到浮点值 `1.0`，默认 `cout` 输出为 `1`，不会变回 `1.5`，选 B。

## 第13题

C++ 代码用于抽取字符串中的电话号码。约定：电话号码全部是数字，数字之间没有其他符号如连字符或空格等。代码中变量 strSrc 仅仅是示例，可以包含更多字符。下面有关代码说法，正确的说法是()。
```cpp
string strSrc="红十字：01084025890火警电话：119急救电话：120紧急求助：110";
string tel="";
for(int i=0;i<=strSrc.length();i++)
{
    if(strSrc[i]>='0'&&strSrc[i]<='9')
    {
        tel=tel+strSrc[i];
    }
    else if(tel!="")
    {
        cout<<tel<<endl;
        tel="";
    }
}
```

【知识点：ASCII/字符编码】

A. 代码将换行输出各个含有数字的电话号码。
B. 代码将不换行输出各个含有数字的电话号码，号码中间没有分隔。
C. 代码将不换行输出各个含有数字的电话号码，号码中间有分隔。
D. 不能够输出数字电话号码。

答案：A

解析：程序逐个检查 `strSrc` 中的字符，遇到 `'0'` 到 `'9'` 之间的数字就追加到 `tel`；一旦遇到非数字且 `tel` 不为空，就用 `cout << tel << endl` 输出当前号码并清空。`endl` 会换行，所以每段连续数字会单独占一行。循环写成 `i<=strSrc.length()`，还能在字符串最后一段数字后触发一次输出，因此示例中的 `110` 也能输出，选 A。

某公司新出了一款无人驾驶的小汽车，通过声控智能驾驶系统，乘客只要告诉汽车目的地，车子就能自动选择一条优化路线，告诉乘客后驶达那里。请问下面哪项不是驾驶系统完成选路所必须的。()

【知识点：计算机基础常识】

A. 麦克风

## 第15题

现代计算机是指电子计算机，它所基于的是()体系结构。

【知识点：计算机基础常识】

A. 艾伦·图灵
B. 冯·诺依曼
C. 阿塔纳索夫
D. 埃克特-莫克利

答案：B

解析：现代电子计算机通常基于冯·诺依曼体系结构，其核心思想包括存储程序、程序和数据统一存放、由运算器、控制器、存储器、输入设备、输出设备等部分组成。图灵更多对应理论计算模型，阿塔纳索夫和埃克特-莫克利与早期电子计算机发展有关，但题干所问的体系结构是冯·诺依曼，选 B。

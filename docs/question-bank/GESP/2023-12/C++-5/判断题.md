## 第1题

归并排序的时间复杂度是 $O(N \log N)$。

答案：正确

解析：本题考查归并排序的分治复杂度。归并排序每一层递归都要把当前区间中的元素归并一次，总工作量是 $O(N)$；区间每次大致二分，递归层数是 $O(\log N)$。两者相乘得到总时间复杂度 $O(N \log N)$，所以该说法正确。

## 第2题

小杨在生日聚会时拿一块 $H\times W$ 的巧克力招待来的 $K$ 个小朋友，保证每位小朋友至少能获得一块相同大小的巧克力。那么小杨想分出来最大边长的巧克力可以使用二分法。

答案：错误

解析：题干只说每个小朋友得到“一块相同大小”的巧克力，并没有明确要求切成正方形，也没有把问题限定成只由一个边长决定的模型。二分答案需要“边长越小越容易满足、边长越大越难满足”这样的单调判定；若巧克力块的形状和两条边都可变化，单独二分一个“最大边长”不能完整描述切分方案。因此按题干表述，该说法判为错误。

## 第3题

以下 C++ 代码能以递归方式实现斐波那契数列，该数列第 1、2项为 1，以后各项均是前两项之和。（ ）
```cpp
int Fibo(int N)
{
    if (N == 1 || N == 2)
        return 1;
    else
    {
        int m = fiboA(N - 1);
        int n = fiboB(N - 2);
        return m + n;
    }
}
```

答案：错误

解析：函数名是 `Fibo`，但递归调用写成了 `fiboA(N - 1)` 和 `fiboB(N - 2)`。如果没有另外定义这两个函数，代码无法通过编译；即使外部恰好存在同名函数，这段代码本身也不是用 `Fibo` 递归实现斐波那契。因此该说法错误。

## 第4题

贪心算法可以达到局部最优，但可能不是全局最优解。

答案：正确

解析：贪心算法每一步按当前看起来最优的规则做选择，所以能保证的是局部选择最优。只有当问题满足贪心选择性质和最优子结构时，这些局部选择才能推出全局最优；例如某些背包变形中局部最优选择可能导致整体更差。因此“可能不是全局最优解”的说法正确。

## 第5题

小杨设计了一个拆数程序，它能够将任意的非质数自然数 $N$ 转换成若干个质数的乘积，这个程序是可以设计出来的。

答案：正确

解析：本题考查质因数分解。在本题语境中，“非质数自然数”按大于 $1$ 的合数理解；合数一定能找到非 $1$、非自身的因数，继续分解后最终会得到若干个质数的乘积，这就是整数的质因数分解。程序可以通过枚举因数、试除到 $\sqrt N$ 等方法实现，所以该说法正确。严格数学表述下，$1$ 既不是质数也不是合数，不能分解成质数乘积；本题不把 $N=1$ 纳入该说法的讨论范围。

## 第6题

插入排序有时比快速排序时间复杂度更低。

答案：正确

解析：插入排序的最坏时间复杂度是 $O(N^2)$，但当序列已经有序或接近有序时，只需要很少移动，最好可达 $O(N)$。快速排序平均是 $O(N \log N)$，但如果划分极不均衡也可能退化到 $O(N^2)$。题干说“有时”，存在插入排序更低的场景，所以正确。

## 第7题

下面的 C++ 代码能实现十进制正整数 $N$ 转换为八进制并输出。
```cpp
char s[10];
int main()
{
    int N;
    cin >> N;
    string rst = "";
    while (N != 0)
    {
        s[0] = N % 8 + '0';
        rst += string(s);
        N /= 8;
    }
    cout << rst << endl;
    return 0;
}
```

答案：错误

解析：这段代码不能可靠输出八进制表示。首先，每次只给 `s[0]` 赋值，没有写入字符串结束符 `'\0'`，`string(s)` 会继续读取未初始化内容，结果不确定。其次，即使忽略这个问题，余数是从低位到高位依次得到的，直接追加到 `rst` 会把八进制数字反向输出。因此该说法错误。

## 第8题

对数组 `int arr[] = {2, 6, 3, 5, 4, 8, 1, 0, 9, 10}` 执行 `sort(arr, arr+10)`。则执行后 `arr` 中的数据调整为 `{0, 1, 2, 3, 4, 5, 6, 8, 9, 10}`。

答案：正确

解析：`sort(arr, arr+10)` 对半开区间 `[arr, arr+10)` 内的 10 个整数按默认升序排序。原数组元素是 `2, 6, 3, 5, 4, 8, 1, 0, 9, 10`，升序排列后正好是 `0, 1, 2, 3, 4, 5, 6, 8, 9, 10`，所以该说法正确。

## 第9题

小杨想写一个程序来算出正整数 $N$ 有多少个因数，经过思考他写出了一个重复没有超过 $N/2$ 次的循环就能够算出来了。

答案：正确

解析：除 $N$ 本身以外，正整数 $N$ 的其他正因数都不超过 $N/2$（当 $N>1$ 时）。因此可以循环检查 $1$ 到 $N/2$ 是否整除 $N$，最后再把 $N$ 本身计入因数个数；$N=1$ 时也可以单独处理。这样的循环次数不超过 $N/2$，所以该说法正确。

## 第10题

同样的整数序列分别保存在单链表和双向链中，这两种链表上的简单冒泡排序的复杂度相同。

答案：正确

解析：简单冒泡排序的核心是反复比较相邻元素并交换，比较轮数和相邻比较次数都与元素个数有关，数量级为 $O(N^2)$。单链表和双向链表在相邻节点访问方向上有差异，但不改变冒泡排序需要进行的相邻比较总量，所以两者的渐进时间复杂度相同。

## 第3题

以下 C++ 代码能以递归方式实现斐波那契数列，该数列第 1、2项为 1，以后各项均是前两项之和。（ ）
```cpp
int Fibo(int N)
{
    if (N == 1 || N == 2)
        return 1;
    else
    {
        int m = fiboA(N - 1);
        int n = fiboB(N - 2);
        return m + n;
    }
}
```

答案：错误

解析：本题判断为错误。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

## 第7题

下面的 C++ 代码能实现十进制正整数 $N$ 转换为八进制并输出。
```cpp
char s[10];
int main()
{
    int N;
    cin >> N;
    string rst = "";
    while (N != 0)
    {
        s[0] = N % 8 + '0';
        rst += string(s);
        N /= 8;
    }
    cout << rst << endl;
    return 0;
}
```

答案：错误

解析：本题判断为错误。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第1题

C++ 表达式 `-7/2` 的值为整数 $-3$。( )

答案：正确

解析：本题判断为正确。需要按 C++ 字面量和算术转换规则判断：字符常量、整型常量和浮点运算的结果类型不同，表达式不会随描述随意改变类型。

## 第4题

在 C++ 代码中，执行 `srand(0)` 后连续两次执行 `rand()` 的结果相等。( )

答案：错误

解析：本题判断为错误。赋值号 `=` 与相等比较 `==` 含义不同，C++ 运算符也必须写成规定形式；题干说法需与这些语法规则一致。

## 第5题

C++ 代码中 `while(1){...}` 的判断条件不是逻辑值，将导致语法错误。( )

答案：错误

解析：本题判断为错误。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

## 第1题

下面有关函数参数的说法，正确的是（ ）。

A. 函数参数传递时，主函数当中采用值传递方式将参数传递给子函数时，若子函数将参数值改变，主函数当中的参数值不变。
B. 函数参数传递时，主函数当中采用值传递方式将参数传递给子函数时，若子函数将参数值改变，主函数当中的参数值将随子函数一样改变而改变。
C. 函数参数传递时，主函数如果将参数的地址传递给子函数，若子函数将参数值改变，主函数当中的参数值将不改变。
D. 函数参数传递可以不满足子函数的参数个数要求。

答案：A

解析：值传递会把实参的值复制一份交给形参，子函数修改的是这份副本，主函数里的原变量不会随之改变，所以 A 正确。B 把值传递说成会影响原变量，错误。C 说传地址后子函数改变参数值也不影响主函数变量，和指针间接访问可修改原对象的规则相反。D 也错误，函数调用时实参数量通常必须与函数形参列表匹配，不能随意少传或多传。

## 第2题

下面 C++ 代码执行后，输出的是（ ）。
```cpp
int arr[10]={1};
string strArr="chen a dai";
cout<<strArr[arr[1]]<<endl;
```

A. chen
B. c
C. chen a dai
D. dai

答案：B

解析：`int arr[10]={1};` 只把 `arr[0]` 初始化为 `1`，其余元素会被补成 `0`，所以 `arr[1]` 的值是 `0`。字符串下标从 `0` 开始，`strArr[0]` 是字符串 `"chen a dai"` 的第一个字符 `c`，输出时只输出这一个字符，不会输出整个单词或整串字符串。

## 第3题

下面 C++ 代码最后执行后输出是（ ）。

```cpp
int fun1(int *n)
{
    return *n**n;
}
int main()
{
    int arr[10]={2};
    arr[1]=fun1(arr);
    cout<<arr[1]<<endl;
}
```

A. 1
B. 2
C. 3
D. 4

答案：D

解析：调用 `fun1(arr)` 时，数组名 `arr` 会退化为指向首元素的指针，也就是把 `&arr[0]` 传给形参 `n`。函数里的 `*n**n` 按运算符解析为 `(*n) * (*n)`，而 `arr[0]` 初始为 `2`，所以返回 `2*2=4`。返回值赋给 `arr[1]` 后，`cout<<arr[1]` 输出 `4`。

## 第5题

下面 C++ 代码执行后输出是（ ）。
```cpp
int arr[3]={1,2,3};
int *p=NULL;
p=arr;
p++;
cout<<*p<<endl;
```

A. 1,2,3
B. 1
C. 2
D. 3

答案：C

解析：`p=arr` 后，指针 `p` 指向 `arr[0]`，也就是值为 `1` 的位置。`p++` 会让指针移动到下一个 `int` 元素的位置，即 `arr[1]`。此时 `*p` 取到的是 `arr[1]` 的值 `2`，所以输出 `2`，不是整个数组，也不是首元素或末元素。

## 第6题

如果变量 x 的地址是 0x6ffe14,下面 C++ 代码执行以后输出的是（ ）。
```cpp
int *p=NULL;
int x=2;
p=&x;
p++;
cout<<p<<endl;
```

A. 0x6ffe11
B. 0x6ffe14
C. 0x6ffe18
D. 0x6ffe15

答案：C

解析：`p=&x` 后，`p` 保存的是 `x` 的地址 `0x6ffe14`。题目选项体现这里按 `sizeof(int)=4` 计算，`p++` 对 `int*` 来说不是地址数值加 1，而是移动到下一个 `int` 的位置，也就是地址增加 4 个字节。`0x6ffe14 + 0x4 = 0x6ffe18`，因此输出选 C。

## 第7题

在 C++ 中,执行下面代码后，输出的是（ ）。
```cpp
int point(int *p)
{
    return *p**p;
}
int main()
{
    int a=20;
    int *p=&a;
    *p=point(p);
    cout<<*p<<endl;
}
```

A. 400
B. 200
C. 20
D. 100

答案：A

解析：`p` 指向变量 `a`，初始时 `*p` 的值是 `20`。函数 `point(p)` 接收这个地址，表达式 `*p**p` 解析为 `(*p) * (*p)`，返回 `20*20=400`。主函数中的 `*p=point(p);` 又通过指针把 `a` 的值改成 `400`，最后 `cout<<*p` 输出的就是 `400`。

## 第8题

下列 C++ 语句执行以后结果是 true 的是（ ）。

A. 3&&false
B. 5&&2
C. 101&&000
D. 4&true

答案：B

解析：逻辑与 `&&` 中，非零整数按真处理，零按假处理。A 中 `false` 为假，所以 `3&&false` 为假；B 中 `5` 和 `2` 都非零，所以 `5&&2` 为真；C 中 `000` 是整数 0，所以 `101&&000` 为假。D 使用的是按位与 `&`，`true` 会转成 `1`，`4 & 1` 的结果是 `0`，也不是 true。

## 第9题

在如下的 C++ 代码中实现了对字符串中出现的 26 个字母的个数统计，横线处应填入是（ ）。

```cpp
string str="HELLO CHEN A DAI";
int strlen=str.length();
char alpha[26]={65};
int cnt[26]={0};
for(int i=1;i<26;i++)
{
    __________;
}
for(int i=0;i<26;i++)
{
    cout<<alpha[i]<<" ";
}
cout<<endl;
for(int i=0;i<26;i++)
{
    for(int j=0;j<strlen;j++)
    {
        if(alpha[i]==str[j])
        {
            cnt[i]++;
        }
    }
}
for(int i=0;i<26;i++)
{
    cout<<cnt[i]<<" ";
}
```

A. `alpha[i]=alpha[i-1]+1;`
B. `alpha[i]=alpha[i]+1;`
C. `alpha[i+1]=alpha[i]+1;`
D. `alpha[i-1]=alpha[i]+1;`

答案：A

解析：`char alpha[26]={65};` 会让 `alpha[0]` 等于 ASCII 码 65，也就是 `'A'`，其余位置初始为 `0`。循环从 `i=1` 开始，需要让当前位置等于前一个字母加 1，才能依次得到 `B,C,...,Z`，所以应填 `alpha[i]=alpha[i-1]+1;`。B 使用当前位置原值，无法承接前一个字母；C 会跳过 `alpha[1]` 且在最后写到 `alpha[26]` 越界；D 会反向覆盖前一个位置，都会破坏字母表。

## 第10题

下面 C++ 代码执行后生成的文件其字节数为（ ）。
```cpp
ofstream fout;
fout.open("1.txt");
for(int i=1;i<=10;i++)
{
    if(i%5==0)
    {
        int x=6;
        fout<<x;
    }
    else
    {
        char ch='A';
        fout<<ch;
    }
}
```

A. 10
B. 16
C. 40
D. 24

答案：A

解析：循环从 `1` 到 `10` 共执行 10 次，每次都向文本文件写入一个可见字符。`i` 为 `5` 和 `10` 时，`fout<<x` 把整数 `6` 按文本形式写成字符 `'6'`，各占 1 字节；其余 8 次写入字符 `'A'`，也各占 1 字节。代码没有写空格和换行，所以总字节数是 `10`。

## 第11题

下列 C++ 代码输入 `1,2,3,4`，执行后，将输出的是（ ）。
```cpp
string str="";
cin>>str;
int strlen=str.length();
for(int i=0;i<strlen;i++)
{
    if(str[i]<='9'&&str[i]>='0')
    {
        cout<<str[i];
    }
    else
    {
        cout<<"#";
    }
}
```

A. 1#4#
B. 1#3#
C. 1#2#3#4#
D. 1#2#3#4

答案：D

解析：`cin>>str` 以空白字符为分隔，输入 `1,2,3,4` 中没有空格，所以整个串都会读入。循环逐字符判断：`1`、`2`、`3`、`4` 都在 `'0'` 到 `'9'` 之间，原样输出；三个逗号不是数字，分别输出 `#`。按原顺序拼起来就是 `1#2#3#4`。

## 第12题

以下 C++ 代码用于实现每个整数对应的因数，如输入 12 ，则输出 1 2 3 4 6 12 ；如输入 18 ，则输出 1 2 3 6 9 18 。横线处应填入代码是（ ）。
```cpp
int n;
cin>>n;
for(int i=1;i<=n;i++)
{
    ________
    {
        cout<<i<<" ";
    }
}
```

A. `if(n%i==0)`
B. `if(n/i==0)`
C. `if(n%i!=0)`
D. `if(n/i!=0)`

答案：A

解析：要输出 `n` 的因数，就要判断当前枚举值 `i` 能否整除 `n`，也就是余数是否为 0，条件应为 `n%i==0`。B 的 `n/i==0` 是整数商为 0 的判断，对 `i<=n` 的循环几乎不会成立；C 会输出不能整除 `n` 的数；D 判断商不为 0，对 `1` 到 `n` 的所有 `i` 都成立，不能筛出因数。

## 第14题

现代计算机是指电子计算机，它所基于的是（ ）体系结构。

A. 艾伦·图灵
B. 冯·诺依曼
C. 阿塔纳索夫

## 第15题

输入一个正整数 $N$（$N \geq 7$），想找出它所有相邻的因数对，比如，输入 $12$，因数对有 $(1,2)$、$(2,3)$、$(3,4)$。下面哪段代码找不到所有的因数对?()

A. `for(i=1;i<N;i++) if(!(N%i) && !(N%(i+1))) printf("%d,%d\n", i, i+1);`
B. `for(i=2;i<N;i++) if(!(N%i) && !(N%(i+1))) printf("%d,%d\n", i, i+1);`
C. `for(i=2;i<N/2;i++) if(!(N%(i-1)) && !(N%i)) printf("%d,%d\n", i-1, i);`
D. `for(i=1;i<N/2;i++) if(!(N%i) && !(N%(i+1))) printf("%d,%d\n", i, i+1);`

答案：B

解析：相邻因数对形如 `(i,i+1)`，必须同时满足 `N%i==0` 和 `N%(i+1)==0`。A 从 `i=1` 枚举到 `N-1`，能覆盖所有可能的首项；D 枚举到 `i<N/2`，也足够，因为除 `1` 和 `N` 外，成对相邻因数的较大者不会达到 `N/2` 以上。C 用 `i` 表示较大的那个数，检查 `(i-1,i)`，从 `i=2` 开始同样覆盖了 `(1,2)`。B 从 `i=2` 开始检查 `(i,i+1)`，会漏掉 `(1,2)`；例如 `N=12` 时，`(1,2)` 是合法相邻因数对，但 B 不会检查它，所以找不到所有因数对。

## 第1题

以下 C++ 不可以作为变量的名称的是（）。

A. CCF GESP
B. ccfGESP
C. CCFgesp
D. CCF_GESP

答案：A

解析：C++ 变量名只能由字母、数字和下划线组成，且不能以数字开头。`CCF GESP` 中间含有空格，会被拆成两个标识符，不能作为一个变量名；`ccfGESP`、`CCFgesp` 和 `CCF_GESP` 都只包含合法字符。

## 第2题

C++ 表达式 `10 - 3 * (2 + 1) % 10` 的值是（）。

A. 0
B. 1
C. 2
D. 3

答案：B

解析：先算括号内 `2 + 1 = 3`，再算乘法 `3 * 3 = 9`。取模与乘法同级，从左到右计算，`9 % 10 = 9`，最后 `10 - 9 = 1`，所以结果是 `1`。

## 第3题

假设现在是上午十点，求出 $N$ 小时（正整数）后是第几天几时，如输入 $20$ 小时则为第 $2$ 天 $6$ 点，如 $N$ 输入 $4$ 则为今天 $14$ 点。为实现相应功能，应在横线处填写代码是（）。

```cpp
int N, dayX, hourX;
cin >> N;
dayX = _____, hourX = _____;
if (dayX == 0)
    cout << "今天" << hourX << "点";
else
    cout << "第" << (dayX + 1) << "天" << hourX << "点";
```

A. ` (10 + N) % 24 `, ` (10 + N) / 24 `
B. ` (10 + N) / 24 `, ` (10 + N) % 24 `
C. ` N % 24 `, ` N / 24 `
D. ` 10 / 24 `, ` 10 % 24 `

答案：B

解析：从上午 10 点开始，经过 `N` 小时后的总小时数是 `10 + N`。经过了多少个完整的 24 小时应放入 `dayX`，即 `(10 + N) / 24`；当天的小时数应放入 `hourX`，即 `(10 + N) % 24`。例如 `N = 20` 时得到 `dayX = 1`、`hourX = 6`，输出第 2 天 6 点；`N = 4` 时得到今天 14 点。

## 第4题

下面的程序用于判断 $N$ 是否为偶数，横线处应填写代码是（ ）。

```cpp
cin >> N;
if (______)
    cout << "偶数";
else
    cout << "奇数";
```

A. ` N % 2 == 0 `
B. ` N % 2 = 0 `
C. ` N % 2 `
D. ` N % 2 != 0 `

答案：A

解析：偶数的判断条件是能被 `2` 整除，也就是余数为 `0`，应写成 `N % 2 == 0`。`N % 2 = 0` 把赋值号用在表达式结果上，语法不合法；`N % 2` 在奇数时为真，判断方向相反；`N % 2 != 0` 判断的是奇数。

## 第5题

下面对 C++ 代码执行后输出的描述，正确的是（ ）。

```cpp
cin >> N;
cnt = 0;
for(int i = 1; i < N; i++)
    cnt += 1;
cout << cnt;
```

A. 如果输入的 $N$ 是小于等于 $2$ 整数，第 $5$ 行将输出 $0$。
B. 如果输入的 $N$ 是大于等于 $2$ 整数，第 $5$ 行将输出 $N-1$。
C. 如果输入的 $N$ 是大于等于 $2$ 整数，第 $5$ 行将输出 $N$。
D. 以上说法均不正确。

答案：B

解析：循环从 `i = 1` 开始，每次 `i++`，只要 `i < N` 就执行一次 `cnt += 1`。当 `N >= 2` 时，`i` 依次取 `1` 到 `N - 1`，一共执行 `N - 1` 次，所以输出 `N - 1`。选项 A 错在 `N = 2` 时会执行一次，输出 `1`，不是 `0`。

## 第6题

下面 C++ 代码执行后的输出是（ ）。

```cpp
cnt = 0;
for(int i = 1; i < 10; i++){
    cnt += 1;
    i += 2;
}
cout << cnt;
```

A. 10
B. 9
C. 3
D. 1

答案：C

解析：第一次循环 `i = 1`，`cnt` 变为 `1`，循环体内 `i += 2` 后为 `3`，再执行 `for` 的 `i++` 变为 `4`。之后 `i` 依次以 `4`、`7` 进入循环，`cnt` 分别变为 `2`、`3`；第三次循环后 `i` 变为 `10`，不再满足 `i < 10`，因此输出 `3`。

## 第7题

下面 C++ 代码执行后的输出是（ ）。

```cpp
cnt = 0;
for(int i = 1; i < 20; i++){
    if(i%2)
        continue;
    else if(i%3==0 && i%5==0)
        break;
    cnt += i;
}
cout << cnt;
```

A. 90
B. 44
C. 20
D. 10

答案：A

解析：`if(i%2)` 在奇数时为真，奇数会直接 `continue`，不会累加，也不会检查后面的 `else if`。偶数进入后续判断，`1` 到 `19` 之间没有既是偶数又同时被 `3` 和 `5` 整除的数，所以不会 `break`。累加的是 `2+4+6+8+10+12+14+16+18 = 90`。

## 第8题

下面 C++ 代码执行后的输出是（ ）。

```cpp
N = 10;
cnt = 0;
while(1){
    if(N == 0) break;
    cnt += 1;
    N -= 2;
}
cout << cnt;
```

A. 11
B. 10
C. 5
D. 4

答案：C

解析：初始 `N = 10`。每次循环先判断 `N == 0`，不为 `0` 时 `cnt` 加 `1`，再让 `N -= 2`。`N` 依次为 `10`、`8`、`6`、`4`、`2` 时各计数一次，减到 `0` 后下一轮立即 `break`，所以 `cnt = 5`。

## 第9题

下面 C++ 代码用于判断一个数是否为质数（素数），在横线处应填入代码是（ ）。

```cpp
cin >> N;
cnt = 0;
for (int i = 1; i < N + 1; i++)
    if (N % i == 0)
        _____;
if (cnt == 2)
    cout << N << "是质数。";
else
    cout << N << "不是质数。";
```

A. ` cnt = 1 `
B. ` cnt = 2 `
C. ` cnt =+ 1 `
D. ` cnt += 1 `

答案：D

解析：程序遍历 `1` 到 `N`，遇到能整除 `N` 的 `i` 就要把约数个数加一。只有 `cnt += 1` 表示在原值基础上增加 `1`。`cnt = 1` 和 `cnt = 2` 只是反复赋固定值，`cnt =+ 1` 等价于把正数 `1` 赋给 `cnt`，都不能统计约数个数。

## 第10题

对下面的代码，描述正确的是（ ）。

```cpp
#include <stdlib.h>
using namespace std;

int main(){
    int arr[] = {2,6,3,5,4,8,1,0,9,10};
    for(int i = 0; i < 10; i++)
        cout << arr[i] << " ";
    cout << i << endl;
    cout << endl;
    return 0;
}
```

A. 输出 2 6 3 5 4 8 1 0 9 10 10
B. 输出 2 6 3 5 4 8 1 0 9 9
C. 输出 2 6 3 5 4 8 1 0 9 10
D. 提示有编译错误

答案：D

解析：`for(int i = 0; i < 10; i++)` 中的 `i` 只在这个 `for` 语句的作用域内有效。循环结束后执行 `cout << i << endl;` 时，`i` 已经不可见，因此程序会编译错误，不能产生 A、B、C 中的任何输出。

## 第11题

执行下面 C++ 代码后输出是（ ）。

```cpp
cnt = 0;
for (i = 10; i > 3; i -= 3)
    cnt = cnt + i;
cout << cnt;
```

A. 3
B. 21
C. 27
D. 49

答案：B

解析：循环开始时 `i = 10`，满足 `i > 3`，把 `10` 加入 `cnt`；之后 `i -= 3` 变为 `7`，再加 `7`；再变为 `4`，再加 `4`。下一次 `i` 变为 `1`，不满足条件，循环结束，所以输出 `10 + 7 + 4 = 21`。

## 第12题

定义变量 `char c`，下面对 c 赋值的语句，不符合语法的是（ ）。

A. ` c = (char)66; `
B. ` c = (char)(66); `
C. ` c = char(66); `
D. ` c = char 66; `

答案：D

解析：`(char)66` 和 `(char)(66)` 是 C 风格强制类型转换，`char(66)` 是 C++ 的函数式类型转换，它们都能把整数 `66` 转为字符后赋给 `c`。`char 66` 不是合法的表达式写法，缺少必要的括号或变量声明形式，因此不能放在赋值号右侧。

## 第13题

下图所示 C++ 代码对大写字母 'A' 到 'Z' 分组，对每个字母输出所属的组号，那么输入 'C' 时将输出的组号是（ ）。

```cpp
char c;
while(1){
    cin >> c;
    if(c=='q') break;
    switch(c){
        case 'A': cout << "1 "; break;
        case 'B': cout << "3 ";
        case 'C': cout << "3 ";
        case 'D': cout << "5 "; break;
        case 'E': cout << "5 "; break;
        default: cout << "9 ";
    }
    cout << endl;
}
```

A. 3
B. 3 5
C. 3 5 9
D. 以上都不对

答案：B

解析：输入 `'C'` 时会直接进入 `case 'C'`，先输出 `3 `。该分支后面没有 `break`，所以会继续向下执行 `case 'D'`，再输出 `5 `，然后遇到 `break` 退出 `switch`。因此结果是 `3 5`，不会继续执行到 `default`。

## 第14题

某公司新出了一款无人驾驶的小汽车，通过声控智能驾驶系统，乘客只要告诉汽车目的地，车子就能自动选择一条优化路线，告诉乘客后驶达那里。请问下面哪项不是驾驶系统完成选路所必须的。（ ）

A. 麦克风
B. 扬声器
C. 油量表
D. 传感器

答案：C

解析：题干强调的是声控智能驾驶系统完成选路：麦克风用于接收乘客说出的目的地，传感器用于感知车辆和环境信息，扬声器可用于把选择的路线告知乘客。油量表反映剩余燃油，不是完成“听懂目的地并选择路线”这个过程所必须的部件。

## 第15题

现代计算机是指电子计算机，它所基于的是（ ）体系结构。

A. 艾伦·图灵
B. 冯·诺依曼
C. 阿塔纳索夫
D. 埃克特-莫克利

答案：B

解析：现代电子计算机通常基于冯·诺依曼体系结构，核心思想包括存储程序、程序和数据存放在存储器中，并由运算器、控制器、存储器、输入设备和输出设备协同工作。图灵、阿塔纳索夫、埃克特-莫克利都与计算机发展有关，但不是这道题所问的体系结构名称。

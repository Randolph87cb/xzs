## 第12题

下面程序的输出为（ ）。

```cpp
#include <iostream>
#define N 10
using namespace std;
int path[N][N];
int main() {
    for (int i = 1; i < N; i++)
        path[i][0] = i;
    for (int j = 1; j < N; j++)
        path[0][j] = j;
    for (int i = 1; i < N; i++)
        for (int j = 1; j < N; j++)
            path[i][j] = path[i - 1][j] + path[i][j - 1];
    cout << path[8][4] << endl;
    return 0;
}
```

A. 84
B. 495
C. 1012
D. 结果是随机的。

答案：C

## 第14题

下面 `fib` 函数的时间复杂度为（ ）。

```cpp
int fib_rcd[MAX_N];
int fib(int n) {
    if (n <= 1)
        return 1;
    if (fib_rcd[n] > 0)
        return fib_rcd[n];
    return fib(n - 1) + fib(n - 2);
}
```

A. $O(n)$
B. $O(\phi^n)$，$\phi = \frac{\sqrt{5}-1}{2}$
C. $O(2^n)$
D. 无法正常结束。

答案：C

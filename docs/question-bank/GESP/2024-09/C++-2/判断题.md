## 第1题

小杨最近开始学习 C++ 编程，老师说 C++ 是一门面向对象的编程语言，也是一门高级语言。（ ）

答案：正确

解析：该说法正确。C++ 支持类、对象、继承等面向对象特性，同时高级语言是相对机器语言和汇编语言而言，更接近人类书写和理解。题干两部分都成立，所以判断为正确。

## 第2题

在 C++ 中，`cout << (3, 4, 5)` 可以输出 $3$ $4$ $5$，且每个输出项之间用空格分开。（ ）

答案：错误

解析：`(3, 4, 5)` 是逗号表达式，会从左到右计算并取最后一个表达式的值，所以整体值是 $5$。`cout << (3, 4, 5)` 只输出 `5`，不会依次输出 `3 4 5`，也不会自动加空格。

## 第3题

C++ 表达式 `12%10%10` 的值为 $2$。（ ）

答案：正确

解析：该说法正确。`%` 与 `*`、`/` 同级，连续出现时按从左到右计算，因此 `12%10%10` 等价于 `(12%10)%10`。先算得 `2`，再算 `2%10` 仍是 `2`，所以题干正确。

## 第4题

C++ 语句 `cout << rand() << ' ' << rand();` 的第二个输出值较大。（ ）

答案：错误

解析：该说法错误。`rand()` 每调用一次都会返回一个伪随机数，连续两次调用没有固定的大小顺序。第二个结果可能更大、更小，或者恰好相等，因此不能说第二个输出值较大。

## 第5题

定义 C++ 的 `int` 类型的变量 `ch`，而且值为 `'1'`，则语句 `cout << int(ch);` 的输出为 $1$。（ ）

答案：错误

解析：`'1'` 是字符常量，赋给 `int ch` 时保存的是字符编码值。在常见 ASCII 编码中，字符 `'1'` 的编码是 $49$，所以 `cout << int(ch);` 输出的是 `49`，不是数字 `1`。

## 第6题

下面 C++ 代码执行后将输出 $10$。（ ）

```cpp
int i;
for (i = 0; i < 10; i++)
    continue;
if(i == 10)
    cout << i;
```

答案：正确

解析：`for` 循环中 `i` 依次为 $0$ 到 $9$，循环体只有 `continue`，它会结束本轮并继续执行循环后的 `i++`。循环结束时 `i` 已变为 $10$，满足 `if (i == 10)`，所以输出 `10`。

## 第7题

下面 C++ 代码能求整数 $N$ 和 $M$ 之间所有整数之和，包含 $N$ 和 $M$。（ ）

```cpp
int N, M, Sum;

cin >> N >> M;

if (N > M){
    int tmp = N;
    N = M, M = tmp;
}

for (int i = N; i < M+1; i++)
    Sum += i;

cout << Sum;
```

答案：错误

解析：代码中 `Sum` 只声明为 `int N, M, Sum;`，没有初始化为 $0$，之后直接执行 `Sum += i` 会在不确定初值上累加，结果不可靠。虽然交换 `N`、`M` 和循环边界能覆盖两端点，但未初始化变量使程序不能保证求出正确和。

## 第8题

将下面 C++ 代码中的 L3 标记的代码行调整为 `for (int i = 0; i < 5; i++)` 后输出结果相同。（ ）

```cpp
int loopCount = 0;
for (int i = 1; i < 5; i++) // L3
    for (int j = 0; j < i; j++)
        loopCount += 1;
cout << loopCount;
```

答案：正确

解析：原代码中外层 `i=1,2,3,4`，内层分别执行 $1,2,3,4$ 次，总次数为 $10$。改成 `i=0` 到 `4` 后，`i=0` 时内层执行 $0$ 次，`i=1,2,3,4` 仍分别执行 $1,2,3,4$ 次，总次数还是 $10$，输出相同。

## 第9题

某一系列数据的规律是从第 3 个数值开始是前两个数之和。下面的代码求第 $N$ 个数的值，$N$ 限定为大于 $2$。（ ）

```cpp
int start1; // 第1个数
int start2; // 第2个数
int N; //求N个数的值
int tmp;
cin >> start1 >> start2 >> N;

for (int i = 2; i < N; i++){
    tmp = start1 + start2;
    start1 = start2;
    start2 = tmp;
}
cout << start2;
```

答案：正确

解析：循环前 `start1`、`start2` 分别保存第 $1$、第 $2$ 个数。每轮先计算后一个数 `tmp = start1 + start2`，再把窗口向后移动一位。`N > 2` 时循环执行 $N-2$ 次，结束后 `start2` 正好是第 $N$ 个数。

## 第10题

下面 C++ 代码执行时如果输入 $2024$，则输出是 $4202$。（ ）

```cpp
int N, flag=0;
cin >> N;
while (N){
    if(!flag) cout << N % 10;
    N /= 10;
    flag = (flag + 1)%2;
}
```

答案：错误

解析：输入 $2024$ 后，循环从个位开始处理数字。`flag=0` 时输出，`flag=1` 时不输出，所以依次处理 $4,2,0,2$ 时只输出第 $1$ 个和第 $3$ 个数字，即输出 `40`，不是 `4202`。

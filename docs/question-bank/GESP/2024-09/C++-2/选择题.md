## 第1题

据有关资料，山东大学于 1972 年研制成功 DJL-1 计算机，并于 1973 年投入运行，其综合性能居当时全国第三位。DJL-1 计算机运算控制部分所使用的磁心存储元件由磁心颗粒组成，设计存贮周期为 $2\mu s$（微秒）。那么该磁心存储元件相当于现代计算机的（ ）。

A. 内存
B. 磁盘
C. CPU
D. 显示器

答案：A

解析：磁心存储元件用于保存计算机运行时要用到的数据和指令，属于早期主存储器的实现形式，作用相当于现代计算机的内存。磁盘主要用于长期存储，CPU 负责运算控制，显示器负责输出画面。

## 第2题

IPv4 版本的因特网总共有（ ）个 A 类地址网络。

A. 65000
B. 200 万
C. 126
D. 128

答案：C

解析：IPv4 的 A 类地址首位固定为 0，理论上首字节范围是 0 到 127。但 0 和 127 有特殊用途，不能作为普通 A 类网络号使用，因此可用的 A 类地址网络数是 $128-2=126$。

## 第3题

在 C++ 中，下列不可做变量的是( )。

A. ccf-gesp
B. ccf_gesp
C. ccfGesp
D. _ccfGesp

答案：A

解析：C++ 变量名只能由字母、数字和下划线组成，且不能以数字开头。`ccf-gesp` 中的 `-` 会被当作减号，不是变量名合法字符；其余三个名字只含字母或下划线，可以作为变量名。

## 第4题

在 C++ 中，与 `for (int i = 1; i < 10; i++)` 效果相同的是( )。

A. for (int i = 0; i < 10; i++)
B. for (int i = 0; i < 11; i++)
C. for (int i = 1; i < 10; ++i)
D. for (int i = 0; i < 11; ++i)

答案：C

解析：原循环中 `i` 从 $1$ 开始，到 $9$ 结束，每轮加 $1$。在这里 `i++` 和 `++i` 都是让 `i` 增加 $1$，不会改变循环取值，所以 C 的执行效果相同；A、B、D 的初值或终止范围不同。

## 第5题

在 C++ 中，`cout << (5 / 2 + 5 % 3)` 的输出是( )。

A. 1
B. 2
C. 4
D. 5

答案：C

解析：`5 / 2` 是整数除法，结果为 $2$；`5 % 3` 是 $5$ 除以 $3$ 的余数，结果为 $2$。两部分相加得到 $2+2=4$，因此输出 $4$。

## 第6题

假定变量 a 和 b 可能是整型、字符型或浮点型，则下面 C++ 代码执行时先后输入 -2 和 3.14 后，其输出不可能是( )。[已知字符 '+'、'-'、'=' 的 ASCII 码值分别是 43、45 和 61]

```cpp
cin >> a;
cin >> b;
cout << (a + b);
```

A. 1
B. 1.14
C. 47
D. 将触发异常

答案：D

解析：若 `a`、`b` 都是整型，输入会得到 `-2` 和 `3`，相加为 $1$；若含浮点型，可以得到 `-2+3.14=1.14`；若 `a` 是字符型，先读到字符 `'-'`，其 ASCII 码为 $45$，再读数值 $2$ 时可得到 $47$。`cin` 默认不会因为这种输入直接触发异常，所以“不可能”的是 D。

## 第7题

在 C++ 代码中假设 $N$ 为正整数，则下面代码能获得个位数的是（ ）。

A. N % 10
B. N / 10
C. N && 10
D. 以上选项均不正确

答案：A

解析：一个正整数除以 $10$ 的余数就是它的个位数字，例如 $123 \% 10 = 3$。`N / 10` 会去掉个位，`N && 10` 是逻辑运算，结果只表示真假，不能取得个位数。

## 第8题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int i;
for (i = 0; i < 10; i++) {
    if (i % 2)
        break;
    cout << "0#";
}
if (i == 10) cout << "1#";
```

A. 0#
B. 1#
C. 0#0#1
D. 没有输出

答案：A

解析：第一次循环 `i=0`，`i % 2` 为 $0$，不执行 `break`，输出 `0#`。第二次循环 `i=1`，`i % 2` 为 $1$，执行 `break` 跳出循环，此时 `i` 不是 $10$，最后的 `if` 不输出内容，所以总输出只有 `0#`。

## 第9题

执行下面 C++ 代码并输入 1 和 0，有关说法正确的是（ ）。

```cpp
int a, b;
cin >> a >> b;
if (a && b)
    cout << ("1");
else if (!(a | b))
    cout << ("2");
else if (a || b)
    cout << ("3");
else
    cout << ("4");
```

A. 1
B. 2
C. 3
D. 4

答案：C

解析：输入后 `a=1`、`b=0`。`a && b` 为假，不能输出 `1`；`a | b` 的结果为 $1$，所以 `!(a | b)` 为假，不能输出 `2`；`a || b` 为真，因此执行第三个分支输出 `3`。

## 第10题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int loopCount = 0;
for (int i = 1; i < 5; i += 2)
    loopCount += 1;
cout << (loopCount);
```

A. 1
B. 2
C. 3
D. 5

答案：B

解析：循环变量 `i` 从 $1$ 开始，每次增加 $2$，满足 `i < 5` 时进入循环。实际取值为 $1$ 和 $3$，共执行 $2$ 次，每次 `loopCount` 加 $1$，最后输出 $2$。

## 第11题

下图是 C++ 程序执行后的输出。为实现其功能，横线处应填入代码是（ ）。

```
7
1
2 3
3 4 5
4 5 6 7
5 6 7 8 9
6 7 8 9 10 11
7 8 9 10 11 12 13
```

```cpp
int lineNum;
cin >> lineNum;
for (int i = 1; i < lineNum + 1; i++) {
    for (int ____________)
        cout << j << " ";
    cout << endl;
}
```

A. `j = i; j < i; j++`
B. `j = 1; j < i; j++`
C. `j = i; j < i * 2; j++`
D. `j = i + 1; j < i + i; j++`

答案：C

解析：第 `i` 行要从 `i` 输出到 `2*i-1`，一共输出 `i` 个数。例如第 $4$ 行是 `4 5 6 7`。`j = i; j < i * 2; j++` 正好让 `j` 依次取这些值；A 不会进入循环，B 从 1 开始，D 少输出开头的 `i`。

## 第12题

下面 C++ 代码执行后输出逆序数。如输入 123 则输出 321。如输入 120 则输出 21。横线处先后应填入的代码是（ ）。

```cpp
int N;
cin >> N;
int rst = 0;
while (N) {
    _________________;
    _________________;
}
cout << (rst);
```

A. rst = rst * 10 + N % 10  N = N / 10
B. rst += N % 10  N = N / 10
C. rst = rst * 10 + N / 10  N = N % 10
D. rst += N / 10  N = N % 10

答案：A

解析：逆序数要每次取出当前个位 `N % 10`，接到结果末尾，即 `rst = rst * 10 + N % 10`；然后用 `N = N / 10` 去掉已经处理的个位。以 $120$ 为例，`rst` 依次为 $0,2,21$，最终输出 $21$，前导的 $0$ 自然不会显示。

## 第13题

下面的 C++ 代码用于输入学生成绩，并根据人数计算出平均成绩，有关说法错误的是（ ）。

```cpp
float Sum = 0; // 保存总成绩
int cnt = 0; // 保存学生人数
while (1) {
    int score;
    cin >> score;
    if (score < 0)
        break;
    cnt += 1;
    Sum += score;
}
cout << "总学生数: " << cnt << "平均分: " << Sum / cnt;
```

A. 代码 while (1) 写法错误
B. 如果输入负数，将结束输入，并正确输出
C. 如果输入的学生成绩含有小数，程序将无法正常执行
D. 变量 int score 初始值不确定，但不影响程序执行

答案：A

解析：`while (1)` 表示一直循环，程序内部用 `score < 0` 和 `break` 作为结束条件，这种写法本身没有语法错误。`score` 虽然声明时未初始化，但马上由 `cin >> score` 读入后才参与判断和累加；若成绩含小数，由于 `score` 是 `int`，读取流程会出现问题。因此错误说法是 A。

## 第14题

以下 C++ 代码判断输入的正整数是否为质数，如果该数字是质数，则输出 YES，否则输出 NO。质数是指仅能被 1 和它本身整除的正整数。请在横线上填写代码。（ ）

```cpp
int num, i;
cin >> num;
for (i = 2; i < num; i++)
    if (__________) {
        cout << ("NO");
        break;
    }
if (i == num)
    cout << ("YES");
```

A. num % i
B. num % i == 0
C. num / i
D. num / i == 0

答案：B

解析：判断质数时，要检查 $2$ 到 `num-1` 之间是否存在能整除 `num` 的数。若 `num % i == 0`，说明 `i` 是 `num` 的因子，`num` 不是质数，应输出 `NO` 并结束循环，所以横线处填 B。

## 第15题

一个数如果能被某个数（比如 7）整除，或者含有该数，则说该数是某个数的相关数。下面 C++ 代码用于判定输入的数与 7 是否有关。下列说法错误的是（ ）。

```cpp
int N, M;
bool Flag = false;
cin >> N;
M = N;

if (M % 7 == 0)
    Flag = true;

while (!Flag && M) {
    if (M % 10 == 7) {
        Flag = true;
        break;
    }
    M /= 10;
}

if (Flag)
    cout << N << "与7有关";
else
    cout << N << "与7无关";
```

A. 删除 break 语句不会导致死循环，但有时会导致结果错误
B. 删除 M /= 10 将可能导致死循环
C. 删除 M = N 并将其后代码中的 M 变量改为 N，并调整输出同样能完成相关功能
D. 删除 break 语句不会导致死循环，但有时会影响效率

答案：A

解析：找到某一位是 $7$ 时，`Flag` 已经被设为 `true`，即使删除这里的 `break`，本轮最多再执行一次 `M /= 10`，下一次判断 `!Flag && M` 也会退出循环，结果不会因此变错，只是多做一点无用操作。删除 `M /= 10` 则可能在没有找到 $7$ 时一直检查同一个数位，导致死循环；用 `N` 直接处理也可以，但要相应调整输出。

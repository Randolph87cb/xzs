## 第1题

据有关资料，山东大学于 1972 年研制成功 DJL-1 计算机，并于 1973 年投入运行，其综合性能居当时全国第三位。DJL-1 计算机运算控制部分所使用的磁心存储元件由磁心颗粒组成，设计存贮周期为 $2\mu s$（微秒）。那么该磁心存储元件相当于现代计算机的（ ）。

A. 内存
B. 磁盘
C. CPU
D. 显示器

答案：A

解析：磁心存储器是早期计算机中用来临时保存正在运行的数据和指令的主存储器，作用相当于现代计算机的内存。磁盘主要用于长期保存数据，CPU 负责运算和控制，显示器负责输出画面。

## 第2题

C++ 程序执行出现错误，不太常见的调试手段是（ ）。

A. 阅读源代码
B. 单步调试
C. 输出执行中间结果
D. 跟踪汇编码

答案：D

解析：阅读源代码、单步调试、输出中间结果都是学习和排查 C++ 程序错误时常用的方法。汇编码是更底层的机器相关代码，初学阶段很少直接跟踪它来调试普通 C++ 程序。

## 第3题

在 C++ 中，下列表达式错误的是（ ）。

A. `cout << "Hello,GESP!" << endl;`
B. `cout << 'Hello,GESP!' << endl;`
C. `cout << """Hello,GESP!""" << endl;`
D. `cout << "Hello,GESP!' << endl;`

答案：D

解析：字符串常量要用一对双引号括起来。D 中前面是双引号，结尾却用了单引号，双引号没有正确配对，所以表达式错误。A 是普通字符串输出；C 可看作相邻字符串字面量连接；B 虽不是字符串写法，但不是本题中引号不配对的错误。

## 第4题

C++ 表达式 `10 - 3 * 2` 的值是（ ）。

A. 14
B. 4
C. 1
D. 0

答案：B

解析：乘法优先级高于减法，应先算 `3 * 2 = 6`，再算 `10 - 6 = 4`。因此表达式的值是 $4$。

## 第5题

C++ 中，假设 $N$ 为正整数 $10$，则 `cout << (N / 3 + N % 3)` 将输出（ ）。

A. 6
B. 4.3
C. 4
D. 2

答案：C

解析：`N` 是正整数 $10$，所以 `N / 3` 是整数除法，结果为 $3$；`N % 3` 是除以 $3$ 的余数，结果为 $1$。两者相加得到 $4$。

## 第6题

C++ 语句 `printf("6%2={%d}", 6%2)` 执行后的输出是（ ）。

A. "6%2={6%2}"
B. 6%2={6%2}
C. 0=0
D. 6%2={0}

答案：D

解析：本题按题目选项的命题意图判断，前面的 `6%2={` 和后面的 `}` 按原样输出，`%d` 输出后面表达式 `6%2` 的值。`6` 能被 `2` 整除，余数是 $0$，所以输出 `6%2={0}`。严格 C/C++ 中如果要输出字面 `%`，格式串应写成 `%%`，即可写作 `printf("6%%2={%d}", 6%2);`。

## 第7题

成功执行下面的 C++ 代码，先后从键盘上输入 5 回车和 2 回车，输出是（ ）。

```cpp
cin >> a;
cin >> b;
cout << a + b;
```

A. 将输出整数 7
B. 将输出 52，5 和 2 之间没有空格
C. 将输出 5 和 2，5 和 2 之间有空格
D. 执行结果不确定，因为代码段没有显示 a 和 b 的数据类型

答案：D

解析：`cin >> a` 和 `cin >> b` 会把输入分别读入两个变量，但题目没有说明 `a`、`b` 的类型。若它们是整数，`a + b` 是 $7$；若是字符或其他类型，结果可能不同。因此只凭这段代码不能确定输出。

## 第8题

下面 C++ 代码执行后输出是（ ）。

```cpp
int Sum = 0;
for (int i = 0; i < 10; i++)
    Sum += i;
cout << Sum;
```

A. 55
B. 45
C. 10
D. 9

答案：B

解析：`i` 从 $0$ 开始，每次加 $1$，当 `i < 10` 时执行循环，所以依次把 $0,1,2,\ldots,9$ 加到 `Sum` 中。和为 $0+1+\cdots+9=45$，输出 $45$。

## 第9题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int N = 0;
for (int i = 0; i < 10; i++)
    N += 1;
cout << N;
```

A. 55
B. 45
C. 10
D. 9

答案：C

解析：循环变量 `i` 依次取 $0$ 到 $9$，共执行 $10$ 次。每次循环都执行 `N += 1`，所以 `N` 从 $0$ 增加到 $10$，最后输出 $10$。

## 第10题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int N = 0;
for (int i = 1; i < 10; i += 2){
    if (i % 2 == 1)
        continue;
    N += 1;
}
cout << N;
```

A. 5
B. 4
C. 2
D. 0

答案：D

解析：`i` 从 $1$ 开始，每次加 $2$，会依次取 $1,3,5,7,9$，这些数都是奇数。每次进入循环后 `i % 2 == 1` 都成立，于是执行 `continue` 跳过 `N += 1`，所以 `N` 一直是 $0$。

## 第11题

下面 C++ 代码执行时输入 14+7 后，正确的输出是（ ）。

```cpp
int P;
printf("请输入正整数P：");
scanf("%d", &P);
if (P % 3 || P % 7)
    printf("第5行代码%d, %d", P % 3, P % 7);
else
    printf("第7行代码%2d", P % 3 && P % 7);
```

A. 第5行代码2, 0
B. 第5行代码1, 0
C. 第7行代码 1
D. 第7行代码 0

答案：A

解析：`scanf("%d", &P)` 读整数时，输入 `14+7` 会先读入整数 $14$，遇到 `+` 停止。此时 `P % 3 = 2`，`P % 7 = 0`，条件 `P % 3 || P % 7` 为真，因此执行第 5 行输出 `第5行代码2, 0`。

## 第12题

执行下面 C++ 代码后得到的输出是（ ）。

```cpp
int count = 0, i, s;
for (i = 0, s = 0; i < 20; i++, count++)
    s += i++;
cout << s << " " << count;
```

A. 190 20
B. 95 10
C. 90 19
D. 90 10

答案：D

解析：循环体里的 `s += i++` 会先把当前 `i` 加到 `s`，再让 `i` 加 $1$；循环后的 `i++, count++` 又让 `i` 再加 $1$，并让 `count` 加 $1$。因此 `i` 实际按 $0,2,4,\ldots,18$ 进入循环，`s=0+2+\cdots+18=90`，循环 $10$ 次，`count=10`。

## 第13题

下面 C++ 代码拟用于计算整数 $N$ 的位数，比如对 123 则输出 `123是3位整数`，但代码中可能存在 bug。下面有关描述正确的是（ ）。

```cpp
int N, N0, rc = 0;
cout << "请输入整数：";
cin >> N;
N0 = N;
while (N) {
    rc++;
    N /= 10;
}
printf("%d是%d位整数\n", N, rc);  // L11
```

A. 变量 N0 占用额外空间，可以去掉
B. 代码对所有整数都能计算出正确位数
C. L11 标记的代码行简单修改后可以对正整数给出正确输出
D. L11 标记的代码行的输出格式有误

答案：C

解析：循环中不断执行 `N /= 10`，会把 `N` 改成 $0$，所以 L11 直接输出 `N` 时不能输出原来的整数。变量 `N0` 已保存原始输入，把 L11 中第一个输出值改成 `N0`，就能对正整数输出类似 `123是3位整数` 的结果。

## 第14题

下面的 C++ 代码用于求连续输入的若干正五位数的百位数之和。例如输入 32488 25731 41232 0，则输出 3 个正五位数的百位数之和为 13。有关描述错误的是（ ）。

```cpp
int M, Sum = 0, rc = 0;
cout << "请输入正整数：";
cin >> M;

while (M) {
    M = (M / 100 % 10);  // L6
    Sum += M;
    rc++;
    cin >> M;
}

cout << rc << "个正五位数的百位数之和为" << Sum;
```

A. 执行代码时如果输入 23221 23453 12345 11111 0，则最后一行 Sum 的值是 10
B. 执行代码时如果输入 2322 2345 1234 1111 0，程序也能运行
C. 将代码标记为 L6 那行改为 `M = (M % 1000 / 100);`，同样能实现题目要求
D. 将代码标记为 L6 那行改为 `M = (M % 100 / 10);`，同样能实现题目要求

答案：D

解析：五位数的百位可以用 `M / 100 % 10` 取得，也可以先取后三位再除以 $100$，即 `M % 1000 / 100`。但 `M % 100 / 10` 取得的是十位数字，不是百位数字，所以不能实现“求百位数之和”的要求。

## 第15题

如果一个正整数 $N$ 能够表示为 $X \times (X+1)$ 的形式，这里称它是一个“兄弟数”。例如，输入 6，则输出“6 是一个兄弟数”。下面 C++ 代码用来判断 $N$ 是否为一个“兄弟数”，在横线处应填入的代码可从 i)-iv) 中选择，则有几个能完成功能？（ ）。

```cpp
int N;
cin >> N;
for (int i = 0; i <= N; i++)
    if (__________________)
        cout << N << "是一个兄弟数\n";
```

i) `N == i * (i + 1)`  
ii) `N == i * (i - 1)`  
iii) `N / (i + 1) == i`  
iv) `N / (i - 1) == i`

A. 1
B. 2
C. 3
D. 4

答案：B

解析：i) 直接判断 `N == i * (i + 1)`，符合兄弟数定义；ii) 当 `i = X + 1` 时，`i * (i - 1)` 也等于 $X \times (X+1)$，也能找到兄弟数。iii) 使用整数除法，可能把不是兄弟数的数误判，例如 `8 / 3 == 2`；iv) 当 `i=1` 时会出现除以 $0$，不能安全完成判断。因此可用的有 $2$ 个。

## 第1题

小杨最近开始学习 C++ 编程，老师说 C++ 是一门面向对象的编程语言，也是一门高级语言。（ ）

答案：正确

解析：C++ 支持类和对象等面向对象编程特性，也比机器语言、汇编语言更接近人的表达方式，属于高级程序设计语言。因此题干说法正确。

## 第2题

在 C++ 中，表达式 `10/4` 和 `10%4` 的值相同，都是整数 $2$，说明 `/` 和 `%` 可以互相替换。（ ）

答案：错误

解析：`10/4` 是整数除法，结果是商 $2$；`10%4` 是取余数，结果也是 $2$。这只是这一组数碰巧相同，`/` 和 `%` 的含义不同，例如 `10/3=3`，`10%3=1`，不能互相替换。

## 第3题

$N$ 是 C++ 程序中的整型变量，则语句 `scanf("%d", &N)` 能接收形如正整数、负整数和 $0$ 输入，但如果输入含字母或带小数点数，将导致无法执行。（ ）

答案：错误

解析：`scanf("%d", &N)` 可以读取十进制整数，包括正整数、负整数和 $0$。如果输入含字母或小数点，通常是读取到不能继续组成整数的位置就停止，或读取失败，并不是“导致无法执行”。

## 第4题

下面 C++ 代码能够执行，则将输出 $45$。（ ）

```cpp
for (int i = 0; i < 10; i++)
    Sum += i;
cout << Sum;
```

答案：错误

解析：循环会把 $0$ 到 $9$ 加到 `Sum` 中，若 `Sum` 初值为 $0$，结果才是 $45$。但题目给出的代码片段没有定义和初始化 `Sum`，只说“能够执行”不能保证它一开始就是 $0$，所以不一定输出 $45$。

## 第5题

在 C++ 代码中整型变量 $X$ 被赋值为 $20.24$，则 `cout << (X++, X+1) / 10` 执行后输出的是 $2.124$。（ ）

答案：错误

解析：`X` 是整型变量，被赋值为 `20.24` 时会保存为整数 $20$。表达式 `(X++, X+1)` 使用逗号运算符，先让 `X` 变成 $21$，再取 `X+1` 得 $22$；`22 / 10` 是整数除法，输出 $2$，不是 $2.124$。

## 第6题

下面 C++ 代码执行后，最后一次输出是 $10$。（ ）

```cpp
for (int i = 1; i < 10; i+=3)
    cout << i << endl;
```

答案：错误

解析：`i` 从 $1$ 开始，每次增加 $3$，且要求 `i < 10`。实际输出的数依次是 $1,4,7$，再变成 $10$ 时条件不成立，不会输出 $10$。

## 第7题

在 C++，`break` 语句通常与 `if` 语句配合使用。（ ）

答案：错误

解析：`break` 的作用是跳出当前循环或 `switch` 语句，它和 `for`、`while`、`switch` 的关系更直接。虽然程序中可以写成 `if (...) break;` 来按条件退出，但 `break` 并不是通常与 `if` 本身配套使用的语句。

## 第8题

在 C++ 代码中，不可以将变量命名为 `five-star`，因为变量名中不可以出现 `-`（减号）符号。（ ）

答案：正确

解析：C++ 变量名只能由字母、数字和下划线组成，并且不能以数字开头。`five-star` 中的 `-` 会被当作减号，不能出现在变量名中，所以题干说法正确。

## 第9题

在 C++ 语言中，整型、实数型、字符型、布尔型是不同数据类型，但这四种类型的变量间都可以比较大小。（ ）

答案：正确

解析：整型、实数型、字符型、布尔型在 C++ 中属于不同的数据类型。进行大小比较时，字符型和布尔型也可以转换成对应的数值参与比较，例如字符按编码值比较，布尔值可看作 $0$ 或 $1$，所以题干说法正确。

## 第10题

C++ 中，定义变量 `int a=5,b=4,c=3`，则表达式 `(a<b<c)` 的值为逻辑假。（ ）

答案：错误

解析：表达式 `a<b<c` 会按从左到右计算成 `(a<b)<c`。当 `a=5,b=4,c=3` 时，`a<b` 为假，数值上相当于 $0$；再比较 `0<c`，即 `0<3` 为真，所以整个表达式不是逻辑假。

## 第2题

## 第3题

$N$ 是 C++ 程序中的整型变量，则语句 `scanf("%d", &N)` 能接收形如正整数、负整数和 $0$ 输入，但如果输入含字母或带小数点数，将导致无法执行。（ ）

答案：错误

## 第4题

下面 C++ 代码能够执行，则将输出 $45$。（ ）

```cpp
for (int i = 0; i < 10; i++)
    Sum += i;
cout << Sum;
```

答案：错误

## 第5题

答案：错误

## 第6题


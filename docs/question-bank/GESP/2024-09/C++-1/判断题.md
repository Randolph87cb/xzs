## 第1题

小杨最近开始学习 C++ 编程，老师说 C++ 是一门面向对象的编程语言，也是一门高级语言。（ ）

答案：正确

解析：C++ 支持类、对象、封装、继承等面向对象编程特性，因此可以说它是一门面向对象的编程语言。同时，C++ 比机器语言、汇编语言更接近人的表达方式，能用变量、语句和函数描述程序逻辑，属于高级程序设计语言。所以题干说法正确。

## 第2题

在 C++ 中，表达式 `10/4` 和 `10%4` 的值相同，都是整数 $2$，说明 `/` 和 `%` 可以互相替换。（ ）

答案：错误

解析：在两个操作数都是整数时，`10 / 4` 表示整数除法，结果是商 $2$；`10 % 4` 表示取余数，结果也恰好是 $2$。但这只是这一组数碰巧相同，并不说明两个运算符等价。例如 `10 / 3 = 3`，而 `10 % 3 = 1`，含义和结果都可能不同，所以不能互相替换。

## 第3题

$N$ 是 C++ 程序中的整型变量，则语句 `scanf("%d", &N)` 能接收形如正整数、负整数和 $0$ 输入，但如果输入含字母或带小数点数，将导致无法执行。（ ）

答案：错误

解析：`scanf("%d", &N)` 可以读取十进制整数，包括正整数、负整数和 $0$。如果输入是 `12abc` 或 `12.5`，它通常会先读入整数部分 $12$，在字母或小数点处停止；如果一开始就是字母，则本次读取失败，`scanf` 返回值可用于判断是否读入成功。这些情况并不是“程序无法执行”，所以题干说法错误。

## 第4题

下面 C++ 代码能够执行，则将输出 $45$。（ ）

```cpp
for (int i = 0; i < 10; i++)
    Sum += i;
cout << Sum;
```

答案：错误

解析：这段循环本身会把 $0$ 到 $9$ 依次加到 `Sum` 中；只有在 `Sum` 初值已经是 $0$ 时，最后才会输出 $45$。但题目给出的代码片段没有展示 `Sum` 的定义和初始化，只说“代码能够执行”并不能推出 `Sum` 一开始就是 $0$。如果 `Sum` 初值不是 $0$，输出就会不同，因此“将输出 $45$”不一定成立。

## 第5题

在 C++ 代码中整型变量 $X$ 被赋值为 $20.24$，则 `cout << (X++, X+1) / 10` 执行后输出的是 $2.124$。（ ）

答案：错误

解析：`X` 是整型变量，被赋值为 `20.24` 时，小数部分会被舍去，实际保存为 $20$。表达式 `(X++, X+1)` 使用逗号运算符，先计算 `X++`，使 `X` 从 $20$ 变为 $21$，再计算并取右侧表达式 `X+1`，得到 $22$。随后 `22 / 10` 是整数除法，结果为 $2$，不会输出 $2.124$。

## 第6题

下面 C++ 代码执行后，最后一次输出是 $10$。（ ）

```cpp
for (int i = 1; i < 10; i+=3)
    cout << i << endl;
```

答案：错误

解析：循环开始时 `i = 1`，每次循环结束后执行 `i += 3`，并且只有 `i < 10` 时才会进入循环体。实际输出顺序是 $1,4,7$；当 `i` 再增加到 $10$ 时，条件 `i < 10` 为假，循环结束，不会输出 $10$。所以“最后一次输出是 $10$”错误。

## 第7题

在 C++，`break` 语句通常与 `if` 语句配合使用。（ ）

答案：错误

解析：`break` 的作用是跳出当前所在的循环语句或 `switch` 语句，因此它和 `for`、`while`、`do while`、`switch` 的关系更直接。程序中常见写法确实可能是 `if (...) break;`，表示满足某个条件时退出循环，但 `if` 只是提供条件判断，`break` 并不是与 `if` 本身配套使用的语句。所以题干说法错误。

## 第8题

在 C++ 代码中，不可以将变量命名为 `five-star`，因为变量名中不可以出现 `-`（减号）符号。（ ）

答案：正确

解析：C++ 标识符通常只能由字母、数字和下划线组成，并且不能以数字开头。`five-star` 中间的 `-` 会被编译器当作减号运算符，而不是变量名的一部分。因此 `five-star` 不能作为一个合法变量名，题干说法正确。

## 第9题

在 C++ 语言中，整型、实数型、字符型、布尔型是不同数据类型，但这四种类型的变量间都可以比较大小。（ ）

答案：正确

解析：整型、实数型、字符型、布尔型确实是不同的数据类型。进行大小比较时，C++ 可以把字符型、布尔型等转换成对应的数值参与比较：字符通常按字符编码值比较，`false` 可看作 $0$，`true` 可看作 $1$。因此这几类变量之间可以进行大小比较，题干说法正确。

## 第10题

C++ 中，定义变量 `int a=5,b=4,c=3`，则表达式 `(a<b<c)` 的值为逻辑假。（ ）

答案：错误

解析：C++ 不会把 `a<b<c` 理解成数学中的“$a<b$ 且 $b<c$”，而是按关系运算符的结合方向计算为 `(a<b)<c`。当 `a=5,b=4,c=3` 时，`a<b` 为假，布尔值在后续比较中可转换为 $0$；再计算 `0<c`，即 `0<3`，结果为真。因此整个表达式的值不是逻辑假，题干说法错误。

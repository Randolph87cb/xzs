## 第1题

判断一个三角形是否成立的条件只有：任意两边长度之和大于第三边的长度

【知识点：枚举与基础算法】

答案：错误

解析：题干把条件说成“只有”这一条，表述不严谨。判断三条线段能否构成三角形时，边长应为正，并且要对三组边都满足“两边之和大于第三边”，即 $a + b > c$、$a + c > b$、$b + c > a$。如果只检查某一组“两边之和大于第三边”，例如 $1, 2, 3$ 满足 $1 + 3 > 2$，但 $1 + 2 = 3$，不能构成三角形。因此题干表述不成立，判断为错误。



## 第2题

这段程序进行的是判断一个从键盘输入的字符的 ASCII 是否是奇数，若是，输出 YES，否则，输出 NO

```cpp
int main()
{
    char x;
    scanf("%c", &x);
    int ASCII = (int)x;
    cout << (x & 1 ? "YES" : "NO") << '\n';
    return 0;
}
```

【知识点：位运算】

答案：正确

解析：`scanf("%c", &x)` 读入一个字符，字符在表达式中参与位运算时会按其 ASCII 编码值提升为整数。`x & 1` 检查最低二进制位：最低位为 `1` 表示该 ASCII 值是奇数，最低位为 `0` 表示偶数。三目表达式 `x & 1 ? "YES" : "NO"` 因外层括号包住了整个条件表达式，会在奇数时输出 `YES`，否则输出 `NO`，所以题干正确。



## 第3题

闰年的定义：
- 普通闰年：公历年份是 4 的倍数，且不是 100 的倍数的，为闰年（如 2004 年、2020 年等就是闰年）。
- 世纪闰年：公历年份是整百数的，必须是 400 的倍数才是闰年（如 1900 年不是闰年，2000 年是闰年）。

下面程序是判断是否是闰年的正确程序

```cpp
cin >> n;
cout << ((n % 4 == 0 && n % 100 != 0) || (n % 400 == 0)) ? 1 : 0;
return 0;
```

【知识点：运算符、表达式与类型转换】

答案：正确

解析：闰年的逻辑条件是“能被 $4$ 整除且不能被 $100$ 整除，或者能被 $400$ 整除”，代码中的 `((n % 4 == 0 && n % 100 != 0) || (n % 400 == 0))` 正好对应这个定义。虽然 `cout << 条件 ? 1 : 0` 会按优先级先执行 `cout << 条件`，再形成条件表达式，但 `cout` 输出布尔值时默认输出 `1` 或 `0`，结果仍与判断闰年一致。因此题干说它能正确判断，结论为正确。



## 第4题

C++语句 `cout << (n % 15 == 0 ? "YES" : "NO");` 能够判断一个整数能否被 3 和 5 同时整除

【知识点：运算符、表达式与类型转换】

答案：正确

解析：一个整数同时能被 $3$ 和 $5$ 整除，等价于能被它们的最小公倍数 $15$ 整除。表达式 `n % 15 == 0` 成立时三目运算输出 `YES`，否则输出 `NO`。因此这条语句可以判断整数是否同时被 $3$ 和 $5$ 整除，题干正确。



## 第5题

有 $ n $ 个同学，从中抽取任意个人数来参加学校组织的大合唱，共有 $ 2^n $ 种方法

【知识点：位运算】

答案：正确

解析：每个同学都有两种独立选择：参加或不参加。$n$ 个同学的选择数相乘为 $2 \times 2 \times \cdots \times 2 = 2^n$。这里“抽取任意个人数”包含抽取 $0$ 人的情况，也就是空集合，因此总方法数为 $2^n$，题干正确。



## 第6题

若将一个正整数化为二进制数，在此二进制数中，我们将数字 1 的个数是偶数的这类二进制数称为 A 类数，否则就称其为 B 类数。

例如：
- $ (13)_{10} = (1101)_2 $，其中 1 的个数为 3，则称此数为 B 类数；
- $ (10)_{10} = (1010)_2 $，其中 1 的个数为 2，称此数为 A 类数；

判断 $ (2025)_{10} $ 化为二进制后，1 的个数为偶数个，因此 2025 为 A 类数。

【知识点：进制转换与进制字面量】

答案：正确

解析：把 $2025$ 按二进制位权展开：$2025 = 1024 + 512 + 256 + 128 + 64 + 32 + 8 + 1$，所以 $(2025)_{10} = (11111101001)_2$。这个二进制数中 `1` 的个数是 $8$ 个，为偶数，因此按题目定义属于 A 类数，题干正确。



## 第7题

该段程序将 $ n $ 不停地除以 2，并输出此时的商和余数，直到 $ n = 0 $ 为止。

```cpp
long long n;
cin >> n;
while(n != 0) {
    cout << n/2 << ' ' << n%2 << '\n';
    n /= 2;
}
```

【知识点：控制结构与程序阅读】

答案：正确

解析：循环条件是 `while(n != 0)`，每次循环先输出当前 `n / 2` 和 `n % 2`，分别是当前数除以 $2$ 的商和余数；随后执行 `n /= 2`，把 $n$ 更新为刚才的商。这个过程会不断把 $n$ 整除 $2$，直到 $n$ 变成 $0$ 后停止。因此题干对程序行为的描述正确。



## 第8题

两个 13 进制的数 A 和 B，在 10 进制下分别表示 10 和 11。$ (A + B)_{13} = (18)_{13} $，也就是说 13 进制数 A 加上 13 进制数 B，和是 13 进制数 18。

【知识点：进制转换与进制字面量】

答案：正确

解析：在 $13$ 进制中，字母 `A` 表示十进制 $10$，`B` 表示十进制 $11$。两者相加得 $10 + 11 = 21$；再把 $21$ 转回 $13$ 进制，$21 = 1 \times 13 + 8$，所以结果是 $(18)_{13}$。题干中的等式成立，判断为正确。



## 第9题

$ k $ 进制，逢 $ k $ 进第二位，$ k^2 $ 进百位，$ k^3 $ 进千位。

【知识点：位运算】

答案：错误

解析：$k$ 进制的规则应理解为“逢 $k$ 进一”，从低位向高一位进位；各位的位权依次是 $k^0$、$k^1$、$k^2$、$k^3$。第二位的位权是 $k$，第三位的位权是 $k^2$，第四位的位权是 $k^3$，但不能说“逢 $k^2$ 进百位、逢 $k^3$ 进千位”：进位基数始终是 $k$，不是不同位使用不同的“逢 $k^2$”“逢 $k^3$”规则。因此题干表述错误。



## 第10题

$ \text{CCF}$（十九进制）$= 21\text{AC}$（十三进制）（不区分大小写）

【知识点：计算机基础常识】

答案：错误

解析：在十九进制中，`C = 12`，`F = 15`，所以 $(\text{CCF})_{19} = 12 \times 19^2 + 12 \times 19 + 15 = 12 \times 361 + 228 + 15 = 4575$。在十三进制中，$(21\text{AC})_{13} = 2 \times 13^3 + 1 \times 13^2 + 10 \times 13 + 12 = 4394 + 169 + 130 + 12 = 4705$。两个十进制值不相等，因此题干等式错误。

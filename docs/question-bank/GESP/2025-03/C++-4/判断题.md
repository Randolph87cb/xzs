## 第4题

答案：正确

## 第7题

考虑最坏情况下冒泡排序算法的时间复杂度，$ T(n) $ 为待排序数字的数目为 $ n $ 的复杂度，则其递推关系式为 $ T(n) = T(n - 1) + n $，$ T(0) = 1 $。

答案：正确

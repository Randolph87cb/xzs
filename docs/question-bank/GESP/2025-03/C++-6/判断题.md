## 第4题

以下代码实现的是二叉树的中序遍历：

```cpp
void traverse(TreeNode* root) {
    if (root == nullptr) return;
    traverse(root->left);
    cout << root->val << " ";
    traverse(root->right);
}
```

答案：正确

## 第7题

## 第8题

答案：正确

## 第9题

以下代码创建的树是一棵完全二叉树：

```cpp
TreeNode* root = new TreeNode{1};
root->left = new TreeNode{2};
root->right = new TreeNode{3};
root->left->left = new TreeNode{4};
```

答案：正确

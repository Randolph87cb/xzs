# 判断题

## 第1题

哈夫曼树在构造过程中，每次合并权值最小的两个节点，最终生成的树带权路径长度最小。

答案：正确

解析：题干表述成立。哈夫曼树的贪心构造每轮选取当前权值最小的两个结点合并，使小权值结点逐步被放到较深层，大权值结点保留较短路径。这样得到的前缀编码树能使带权路径长度 $\sum w_i l_i$ 最小，其中 $w_i$ 是字符权值，$l_i$ 是编码长度。因此判断为正确。



## 第2题

格雷编码的相邻两个编码之间必须有多位不同，以避免数据传输错误。

答案：错误

解析：题干表述不成立。格雷编码的关键性质恰好是相邻两个编码只相差 1 位，例如 2 位格雷码可写成 `00, 01, 11, 10`，相邻项之间每次只翻转一个二进制位。这样可以减少状态切换时多位同时变化带来的误判风险。题干说“必须有多位不同”与格雷码性质相反，因此判断为错误。



## 第3题

在树的深度优先搜索（DFS）中，使用队列作为辅助数据结构以实现“先进后出”的访问顺序。

答案：错误

解析：题干表述不成立。DFS 要沿着一条分支尽量向深处访问，非递归实现通常用栈保存待回溯结点，栈的访问顺序是后进先出。队列的顺序是先进先出，更适合实现 BFS 的按层访问；题干还把队列说成“先进后出”，这也是错误的。因此判断为错误。



## 第4题

以下代码实现的是二叉树的中序遍历：

```cpp
void traverse(TreeNode* root) {
    if (root == nullptr) return;
    traverse(root->left);
    cout << root->val << " ";
    traverse(root->right);
}
```

答案：正确

解析：题干表述成立。代码先递归执行 `traverse(root->left)`，访问左子树；再输出当前结点 `root->val`；最后递归执行 `traverse(root->right)`，访问右子树。访问顺序是“左、根、右”，正是二叉树中序遍历。因此判断为正确。



## 第5题

C++ 支持构造函数重载，但默认无参数的构造函数只能有一个。

答案：正确

解析：题干表述成立。C++ 的构造函数可以重载，例如同时定义带参构造函数和无参构造函数，但函数重载要求参数列表不同。无参数构造函数的参数列表都是空列表，不能再定义另一个同样无参数的构造函数，否则会产生重复定义。因此默认无参数的构造函数只能有一个，判断为正确。



## 第6题

二叉排序树（BST）中，若某节点的左子树为空，则该节点一定是树中的最小值节点。

答案：错误

解析：题干表述不成立。BST 的规则是任一结点左子树中的值小于该结点，右子树中的值大于该结点；整棵树的最小值要从根开始不断向左走，直到不能再向左为止。某个结点左子树为空，只能说明它在自己的子树中没有更小的左孩子，不能说明祖先方向不存在更小值。例如根为 $10$、右孩子为 $20$，结点 $20$ 左子树为空，但它显然不是整棵树最小值。因此判断为错误。



## 第7题

在动态规划解决一维硬币找零问题时，若硬币面额为 $[1, 3, 4]$，目标金额为 $6$，则最少需要 2 枚硬币（$3+3$）。

答案：正确

解析：题干表述成立。设 `dp[x]` 表示凑出金额 $x$ 所需的最少硬币数，转移为 `dp[x] = min(dp[x - coin] + 1)`。对面额 $[1,3,4]$ 和目标 $6$，可以用 $3+3$ 得到 2 枚；不可能用 1 枚得到 $6$，因为没有面额为 $6$ 的硬币。因此最少硬币数确实是 2，判断为正确。



## 第8题

面向对象编程中，封装是指将数据和行为绑定在一起，并对外隐藏实现细节。

答案：正确

解析：题干表述成立。封装把对象的数据和操作这些数据的方法放在同一个类中，并通过 `private`、`protected`、`public` 等访问控制限制外部直接访问内部实现。外部通常只通过公开接口使用对象，从而降低对内部细节的依赖。因此判断为正确。



## 第9题

以下代码创建的树是一棵完全二叉树：

```cpp
TreeNode* root = new TreeNode{1};
root->left = new TreeNode{2};
root->right = new TreeNode{3};
root->left->left = new TreeNode{4};
```

答案：正确

解析：题干表述成立。这棵树按层从左到右为：第 1 层结点 $1$，第 2 层结点 $2,3$，第 3 层只有结点 $4$，且它位于最左侧的 `root->left->left`。完全二叉树要求除最后一层外各层都满，最后一层结点从左向右连续排列；这里不存在中间空位后又出现结点的情况。因此判断为正确。



## 第10题

栈和队列均可以用双向链表实现，插入和删除操作的时间复杂度为 $O(1)$。

答案：正确

解析：题干表述成立。双向链表的结点同时保存前驱和后继指针，若维护好表头、表尾指针，栈可以在同一端完成入栈和出栈，队列可以在一端入队、另一端出队。每次操作只需要改动常数个指针，不需要遍历整条链表，所以插入和删除的时间复杂度为 $O(1)$。因此判断为正确。

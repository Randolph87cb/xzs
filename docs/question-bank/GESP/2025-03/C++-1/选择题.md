## 第1题

2025 年春节有两件轰动全球的事件，一个是 DeepSeek 横空出世，另一个是贺岁片《哪吒2》票房惊人，入了全球票房榜。下面关于 DeepSeek 与《哪吒2》的描述成立的是（ ）。

A. 《哪吒2》是一款新型操作系统
B. DeepSeek 是深海钻探软件
C. 《哪吒2》可以生成新的软件
D. DeepSeek 可以根据《哪吒2》的场景生成剧情脚本

答案：D

```cpp
int main() {

```
A.
```cpp
do {
} while (i != 0);
```

B.
```cpp
for (; i < 10; i++) {
}
```

C.
```cpp

}
```

D.
```cpp

}
```

B. B

## 第10题

## 第12题

下面 C++ 代码执行后，将输出能被 2 整除且除以 7 余数为 2 的数。下列选项不能实现的是（ ）。

```cpp
for (int i = 0; i < 100; i++)
    if ______________________
        cout << i << " ";
```

A. `((i % 2 == 0) && (i % 7 == 2))`
B. `((!(i % 2)) && (i % 7 == 2))`
C. `((!(i % 2)) && (!(i % 7)))`
D. `((i % 2 != 1) && (i % 7 == 2))`

答案：C

## 第15题

```cpp
    if (i % 2 == 0)

        break;

```

B. 450
C. 10

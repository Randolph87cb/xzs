## 第1题

2025 年春节有两件轰动全球的事件，一个是 DeepSeek 横空出世，另一个是贺岁片《哪吒2》票房惊人，入了全球票房榜。下面关于 DeepSeek 与《哪吒2》的描述成立的是（ ）。

A. 《哪吒2》是一款新型操作系统
B. DeepSeek 是深海钻探软件
C. 《哪吒2》可以生成新的软件
D. DeepSeek 可以根据《哪吒2》的场景生成剧情脚本

答案：D

解析：DeepSeek 属于人工智能工具，可以根据给定场景、人物和情节要求生成文本内容，所以 D 的“生成剧情脚本”成立。《哪吒2》是电影，不是操作系统，也不能作为软件开发工具直接生成新的软件，因此 A、C 错；DeepSeek 的名称中虽然有“Deep”，但它不是深海钻探软件，B 错。



## 第2题

在某集成开发环境中编辑一个源代码文件时不可以执行下面（ ）操作。

A. 修改变量定义
B. 保存代码修改
C. 撤销代码修改
D. 插入执行截图

答案：D

解析：源代码文件本质上是文本文件，集成开发环境的代码编辑器可以修改变量定义、保存改动，也可以撤销刚才的编辑操作，所以 A、B、C 都是常见编辑功能。执行截图是图片内容，不能像代码语句一样直接插入源代码文件并作为源程序的一部分执行，因此 D 不属于编辑源代码时应执行的操作。



## 第3题

在 C++ 中，下列可以做变量的是（ ）。

A. Var-1
B. $1
C. %%1
D. _Var_1

答案：D

解析：C++ 变量名只能由字母、数字和下划线组成，并且不能以数字开头。`_Var_1` 只含下划线、字母和数字，形式合法；`Var-1` 中的 `-` 会被识别为减号，`$1` 含有 `$`，`%%1` 含有 `%`，都不能作为变量名。



## 第4题

以下哪个是 C++ 语言的关键字？（ ）

A. abs
B. cin
C. do
D. endl

答案：C

解析：`do` 是 C++ 的关键字，用于组成 `do ... while` 循环语句。`abs` 是常见函数名，`cin` 是标准输入流对象，`endl` 是输出换行并刷新缓冲区的操纵符，它们都不是 C++ 语言关键字。



## 第5题

下面的框架在 L1 和 L2 标记的两行处分别填写选项中的代码，哪组不能通过编译（ ）。

```cpp
int main() {
    int i = 0;
    ____________ // L1
    break;
    ____________ // L2
    return 0;
}
```
A.
```cpp
do {
} while (i != 0);
```

B.
```cpp
for (; i < 10; i++) {
}
```

C.
```cpp
if (i == 0) {
}
```

D.
```cpp
switch (i) {
}
```

答案：C

解析：题目中的 `break;` 必须位于循环语句或 `switch` 语句内部。A 填入后形成 `do { break; } while (i != 0);`，B 填入后形成 `for (...) { break; }`，D 填入后形成 `switch (i) { break; }`，这些都能编译。C 填入后只是 `if (i == 0) { break; }`，`if` 本身不是循环或 `switch`，其中的 `break` 没有合法的跳出对象，所以不能通过编译。



## 第6题

下面 C++ 语句（ ）执行后的输出是 __ 0322$$ 。

A. printf("__ %2d%02d$$", 3, 22)
B. printf("__ %02d%2d$$", 3, 22)
C. printf("__ %02d%02d$$$$", 3, 22)
D. printf("____ %02d%02d$$$$", 3, 22)

答案：B

解析：目标输出是两个下划线、一个空格、`03`、`22`、两个美元符号。B 中 `__ ` 是固定输出，`%02d` 把整数 $3$ 按宽度 $2$、不足补 `0` 输出为 `03`，`%2d` 输出 $22$ 时宽度刚好为 $2$，仍为 `22`，末尾 `$$` 原样输出，所以得到 `__ 0322$$`。A 的 `%2d` 会把 $3$ 输出成前面带空格的 ` 3`；C 和 D 都会输出四个美元符号，D 还多了两个下划线。



## 第7题

有关下列 C++ 代码的说法，错误的是（ ）。

```cpp
printf("我爱码代码！");
```

A. 配对双引号内的汉字改为英文 Hello ,C++ 代码能正确执行
B. 配对双引号内的汉字改为 Hello 代码！ ,C++ 代码能正确执行
C. 代码中的每个双引号，都可以改为两个单引号
D. 代码中的每个双引号，都可以改为三个双引号

答案：C

解析：`printf` 的参数需要字符串字面量，原代码中双引号括起来的内容就是一个字符串。把字符串内容改为英文 `Hello` 或中英文混合内容，仍然是合法字符串，所以 A、B 正确。两个单引号不能替代字符串的双引号，`''...''` 不是合法的 C++ 字符串写法，因此 C 错。三个连续双引号可理解为空字符串、原字符串、空字符串相邻拼接，仍可形成合法字符串，所以 D 不作为错误项。



## 第8题

C++ 表达式 `16 / 4 % 2` 的值是（ ）。

A. 8
B. 4
C. 2
D. 0

答案：D

解析：`/` 和 `%` 的优先级相同，按从左到右结合。先算 `16 / 4 = 4`，再算 `4 % 2 = 0`，因此表达式的值是 $0$。



## 第9题

$N$ 是 C++ 的正整数，值为 12，则 `cout << (N % 3 + N / 5)` 的输出是（ ）。

A. 6.4
B. 2.4
C. 6
D. 2

答案：D

解析：$N = 12$。`N % 3` 表示 $12$ 除以 $3$ 的余数，结果为 $0$；`N / 5` 是整数除法，$12 / 5$ 的商为 $2$，小数部分舍去。因此 `N % 3 + N / 5 = 0 + 2 = 2`，输出为 `2`。



## 第10题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int N = 10;
printf("{N}*{N}={%d*%d}", N, N, N * N);
```

A. 10*10={10*10}
B. 100=10
C. N*N=100
D. {N}*{N}={10*10}

答案：D

解析：`printf` 中只有 `%d` 是格式占位符，普通的 `{N}`、`*`、`=`、花括号都会原样输出。格式串里有两个 `%d`，分别由前两个参数 `N`、`N` 替换成 `10`，第三个参数 `N * N` 没有对应占位符，不会输出。因此结果是 `{N}*{N}={10*10}`。



## 第11题

执行下面的 C++ 代码，在键盘上先后输入 100 和 200，输出是（ ）。

```cpp
int first, second;
cout << "请输入第1个正整数：";
cin >> first;
cout << "请输入第2个正整数：";
cin >> second;
cout << (first / second * second) << endl;
```

A. 200
B. 100
C. 1
D. 0

答案：D

解析：输入后 `first = 100`，`second = 200`。表达式 `first / second * second` 从左到右计算，先做整数除法：`100 / 200` 的商是 $0$；再算 `0 * 200 = 0`。所以输出为 `0`，不是把 $100$ “还原”出来。



## 第12题

下面 C++ 代码执行后，将输出能被 2 整除且除以 7 余数为 2 的数。下列选项不能实现的是（ ）。

```cpp
for (int i = 0; i < 100; i++)
    if ______________________
        cout << i << " ";
```

A. `((i % 2 == 0) && (i % 7 == 2))`
B. `((!(i % 2)) && (i % 7 == 2))`
C. `((!(i % 2)) && (!(i % 7)))`
D. `((i % 2 != 1) && (i % 7 == 2))`

答案：C

解析：题目要求同时满足两个条件：能被 $2$ 整除，即 `i % 2 == 0`；除以 $7$ 余 $2$，即 `i % 7 == 2`。A 直接写出了这两个条件；B 中 `!(i % 2)` 等价于 `i % 2 == 0`；D 中 `i % 2 != 1` 对循环里的非负整数也等价于偶数判断。C 的 `!(i % 7)` 表示 `i % 7 == 0`，筛出的是能被 $7$ 整除的数，不是除以 $7$ 余 $2$ 的数，所以不能实现要求。



## 第13题

下面 C++ 代码执行后输出是（ ）。

```cpp
int tnt = 0;
for (int i = -1000; i < 1000; i++)
    tnt += i;
cout << tnt << endl;
```

A. -1000
B. 0
C. 999
D. 1000

答案：A

解析：循环中 `i` 的取值是从 $-1000$ 到 $999$，不包含 $1000$。其中 $-999$ 与 $999$、$-998$ 与 $998$，一直到 $-1$ 与 $1$ 都能两两抵消，$0$ 不影响和，剩下没有被抵消的是 $-1000$。因此 `tnt` 最终为 $-1000$。



## 第14题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int i;
for (i = 1; i < 100; i += 5)
    continue;
cout << i << endl;
```

A. 101
B. 100
C. 99
D. 96

答案：A

解析：`i` 从 $1$ 开始，每轮循环结束后执行 `i += 5`，取值依次为 $1, 6, 11, \ldots, 96$。当 `i = 96` 时满足 `i < 100`，执行 `continue` 后仍会进行本轮的增量操作，`i` 变为 $101$；再判断 `101 < 100` 为假，循环结束，输出 $101$。



## 第15题

下面 C++ 代码执行后输出的是（ ）。

```cpp
int tnt = 0;
for (int i = 5; i < 100; i += 5){
    if (i % 2 == 0)
        continue;
    tnt += 1;
    if (i % 3 == 0 && i % 7 == 0)
        break;
}
cout << tnt << endl;
```

A. 500
B. 450
C. 10
D. 1

答案：C

解析：循环中的 `i` 是小于 $100$ 的正的 $5$ 的倍数。若 `i` 为偶数，就执行 `continue`，不会让 `tnt` 增加；所以只统计奇数倍的 $5$：$5, 15, 25, 35, 45, 55, 65, 75, 85, 95$，共 $10$ 个。`break` 条件要求同时被 $3$ 和 $7$ 整除，也就是被 $21$ 整除；这些被统计的数中没有满足该条件后提前结束循环的数，因此最终输出 $10$。

答案：错误

## 第3题

执行如下 C++ 代码如果在键盘上输入 10，执行后将输出 20。（ ）

```cpp
char N;
printf("请输入正整数：");
cin >> N;
printf("%d\n", N * 2);
```

答案：错误

## 第4题

删除下面 C++ 代码中的 continue 不影响程序的执行效果。（ ）

```cpp
for (int i = 0; i < 100; i++) {
    if (i % 2 == 0) {
        printf("偶数");
        continue;
    }
    else
        printf("奇数");
}
```

答案：正确

## 第5题

下面 C++ 代码执行时将报错，因为所在位置应该是变量名，而不可以做变量名。（ ）

```cpp
for (int _ = 0; _ < 100; _++)
    cout << "*" << endl;
```

答案：错误

## 第6题

下面 C++ 代码被执行后，将先后输出 3 和 5。（ ）

```cpp
    printf("%d ", i);
```

答案：错误

## 第7题

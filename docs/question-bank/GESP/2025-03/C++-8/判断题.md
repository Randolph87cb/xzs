# 判断题

- 洛谷有题页：https://ti.luogu.com.cn/problemset/1180
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1669257026863136.pdf

## 第1题

表达式 `'5' - 3.0` 的结果为 $2.0$，类型为 double。

答案：错误

## 第2题

在 C++ 语言中，如果想要在一个函数内调用一个类的私有方法，可以在该类中将该函数声明为友元函数。

答案：正确

## 第3题

插入排序一般是稳定的。

答案：正确

## 第4题

5 个相同的红球和 4 个相同的蓝球排成一排，要求蓝球不能相邻，则一共有 15 种排列方案。

答案：正确

## 第5题

使用 math.h 或 cmath 头文件中的函数，表达式 `pow(2, 5)` 的结果类型为 int、值为 32。

答案：错误

## 第6题

C++ 是一种面向对象编程语言，C 则不是。多态是面向对象三大特性之一，虚函数是动态多态的代表特性。因此，使用 C 语言无法实现虚函数。

答案：错误

## 第7题

在 $N$ 个节点的平衡二叉树中查找指定元素的最差时间复杂度为 $O(N)$。

答案：错误

## 第8题

定义 int 类型的变量 $a$ 和 $b$，求二次函数 $y = x^2 + ax + b$ 取最小值时 $x$ 的值，可以通过表达式 $-a / 2.0$ 求得。

答案：正确

## 第9题

判断无向图中是否有环，可以通过广度优先搜索实现。

答案：正确

## 第10题

从 32 名学生中选出 4 人分别担任班长、副班长、学习委员和组织委员，共有 $C(32, 4)$ 种不同的选法。

答案：错误

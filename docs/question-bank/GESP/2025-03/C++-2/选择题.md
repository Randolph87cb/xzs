```cpp
// 实现1
int i, N;
cin >> N;
int tnt = 0, last = 1;
for (i = 1; i < N + 1; i++) {
    last *= i;
    tnt += last;
}
cout << tnt << endl;
```

```cpp
// 实现2
int i, N;
cin >> N;
int tnt = 0, tmp;
for (i = 1; i < N + 1; i++) {
    tmp = 1;
    for (int j = 1; j < i + 1; j++)
        tmp *= j;
    tnt += tmp;
}
cout << tnt << endl;
```

B. 实现 2 的代码效率更高，且更易于理解
C. 实现 1 因为应用了前项计算结果，计算量更小，因此效率高
D. 两种实现，效率几乎一致

答案：C

解析：C 项「实现 1 因为应用了前项计算结果，计算量更小，因此效率高」符合题干要求。表达式和控制结构题要按运算符优先级、整数除法、取余规则以及循环控制语句逐步执行。 其余选项要么改变了关键条件，要么与 C++ 语法、算法性质或计算结果不一致，因此排除。

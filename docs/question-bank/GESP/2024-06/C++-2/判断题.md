## 第1题

执行 C++ 代码 `cout << '9' + '1';` 的输出为 $10$。（ ）

答案：错误

解析：`'9'` 和 `'1'` 都是字符常量，不是整数 9 和 1。它们参与 `+` 运算时会先转换成对应的字符编码整数；常见 ASCII 编码中 `'9'` 为 57，`'1'` 为 49，相加得到 106。因此输出不是 $10$，题干错误。

## 第2题

C++ 表达式 `-12 % 10` 的值为 $2$。（ ）

答案：错误

解析：C++ 整数除法会向 0 截断，余数满足“被除数 = 除数 × 商 + 余数”。这里 `-12 / 10` 的商是 -1，所以余数为 `-12 - 10 * (-1) = -2`。也就是说，`-12 % 10` 的结果是 -2，不是 $2$，题干错误。

## 第3题

C++ 表达式 `int(12.56)` 的值为 $13$。（ ）

答案：错误

解析：`int(12.56)` 表示把浮点数强制转换成 `int`。对正数来说，小数部分会被直接舍去，相当于只保留整数部分 12；它不会按照四舍五入得到 13。因此题干错误。

## 第4题

C++ 的整型变量 $N$ 被赋值为 $10$，则语句 `cout << N / 3 << "-" << N % 3` 执行后输出是 $3-1$。（ ）

答案：正确

解析：`N` 是整型变量，值为 10，因此 `N / 3` 执行整数除法，商为 3；`N % 3` 求余数，余数为 1。输出语句依次输出商、字符串 `"-"`、余数，所以结果正是 `3-1`，题干正确。

## 第5题

在 C++ 代码中，不可以将变量命名为 `scanf`，因为 `scanf` 是 C++ 语言的关键字。（ ）

答案：错误

解析：`scanf` 是 C 标准库函数名，不是 C++ 语言关键字。关键字如 `int`、`while`、`return` 等不能作为变量名，但普通库函数名不属于这一类。实际写代码时不建议把变量命名为 `scanf`，因为容易造成混淆或遮蔽函数名；但题干说“因为它是关键字”是错误的。

## 第6题

下面 C++ 代码执行后将导致死循环。（ ）

```cpp
for (int i = 0; i < 10; i++)
    continue;
```

答案：错误

解析：在 `for` 循环中执行 `continue`，只是结束当前这一轮循环体，之后仍会执行更新语句 `i++`，再判断 `i < 10`。所以 `i` 会从 0 依次增加到 10，条件变为假后循环正常结束，不会停在同一个值上反复执行，题干错误。

## 第7题

下面 C++ 代码执行后将输出 $10$。（ ）

```cpp
int cnt = 0;
for (int i = 0; i < 10; i++)
    for (int j = 0; j < i; j++){
        cnt += 1;
        break;
    }
cout << cnt;
```

答案：错误

解析：外层 `i=0` 时，内层初始 `j=0` 不满足 `j < i`，所以一次也不执行。外层 `i=1` 到 9 时，内层至少能进入一次；进入后先执行 `cnt += 1`，紧接着 `break` 跳出内层循环，因此每个这样的 `i` 只计数 1 次。共有 9 个这样的外层取值，最终输出 9，不是 $10$，题干错误。

## 第8题

下面 C++ 代码执行后，将输出 $5$。（ ）

```cpp
int cnt = 0;
for (int i = 1; i < 5; i++)
    for (int j = i; j < 5; j += i)
        if (i * j % 2 == 0)
            cnt += 1;
cout << cnt;
```

答案：正确

解析：按外层 `i` 分类统计更清楚：`i=1` 时，`j=1,2,3,4`，其中 `j=2,4` 使乘积为偶数，计 2 次；`i=2` 时，`j=2,4`，乘积都为偶数，计 2 次；`i=3` 时，只有 `j=3`，乘积为奇数，计 0 次；`i=4` 时，只有 `j=4`，计 1 次。总数为 $2+2+0+1=5$，题干正确。

## 第9题

下面 C++ 代码能实现正整数各位数字之和。（ ）

```cpp
int N, Sum = 0;
cin >> N;
while (N){
    Sum += N % 10;
    N /= 10;
}
cout << Sum;
```

答案：正确

解析：对正整数 `N`，表达式 `N % 10` 可以取出当前个位数字，把它累加到 `Sum`；随后 `N /= 10` 会去掉个位，让下一轮处理原来的十位。循环一直进行到 `N` 变为 0，说明所有数位都已处理完，因此这段代码能求出正整数各位数字之和，题干正确。

## 第10题

GESP 测试是对认证者的编程能力进行等级认证，同一级别的能力基本上与编程语言无关。（ ）

答案：正确

解析：GESP 的等级划分关注的是认证者在相应级别应具备的编程思维、程序结构理解和问题解决能力。不同编程语言在语法和库函数上会有差异，但同一级别对应的能力要求基本一致，语言主要是表达这些能力的工具。因此题干说法正确。

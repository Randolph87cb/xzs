## 第1题

小杨父母带他到某培训机构给他报名参加 CCF 组织的 GESP 认证考试的第 1 级，那他可以选择的认证语言有几种？（ ）

A. 1
B. 2
C. 3
D. 4

答案：C

解析：GESP 第 1 级支持图形化编程、C++、Python 三种认证语言，因此可选择的语言数量是 3 种，选 C。

## 第2题

下面流程图在 yr 输入 2024 时，可以判定 yr 代表闰年，并输出 2 月是 29 天，则图中菱形框中应该填入（ ）。

![](https://cdn.luogu.com.cn/upload/image_hosting/shumi9qb.png)

A. `(yr%400==0) || (yr%4==0)`
B. `(yr%400==0) || (yr%4==0 && yr%100!=0)`
C. `(yr%400==0) && (yr%4==0)`
D. `(yr%400==0) && (yr%4==0 && yr%100!=0)`

答案：B

解析：闰年的判断规则是“能被 400 整除，或者能被 4 整除但不能被 100 整除”。2024 满足 `yr%4==0 && yr%100!=0`，所以应填写 `||` 连接的完整条件，选 B。

## 第3题

在 C++ 中，下列不可做变量的是（ ）。

A. `five-Star`
B. `five_star`
C. `fiveStar`
D. `_fiveStar`

答案：A

解析：C++ 变量名只能由字母、数字和下划线组成，且不能以数字开头。`five-Star` 中的 `-` 会被当作减号，不属于变量名合法字符，因此不能作为变量名，选 A。

## 第4题

在 C++ 中，与 `for(int i=0; i<10; i++)` 效果相同的是（ ）。

A. `for(int i=0; i<10; i+=1)`
B. `for(int i=1; i<=10; i++)`
C. `for(int i=10; i>0; i--)`
D. `for(int i=10; i<1; i++)`

答案：A

解析：原循环从 `i=0` 开始，每次加 1，执行到 `i=9` 为止。`i++` 与 `i+=1` 都表示每轮增加 1，A 的初值、条件和变化完全相同；B、C 虽也循环 10 次但 `i` 的取值顺序不同，D 一开始条件不成立。

## 第5题

在 C++ 中，`cout << (5 % 2 && 5 % 3)` 的输出是（ ）。

A. `1`
B. `2`
C. `true`
D. `false`

答案：A

解析：`5 % 2` 的结果是 1，`5 % 3` 的结果是 2，二者在逻辑运算中都表示真；`真 && 真` 的结果为 `true`，用 `cout` 默认输出布尔值时显示为 `1`，选 A。

## 第6题

执行下面的 C++ 代码时输入 1，则输出是（ ）。

```cpp
int month;

cin >> month;

switch(month){
    case 1:
        cout << "Jan ";
    case 3:
        cout << "Mar ";
        break;
    default:
        ;
}
```

A. Jan
B. Mar
C. Jan Mar
D. 以上均不对

答案：C

解析：输入 1 时进入 `case 1`，先输出 `Jan `。由于 `case 1` 后没有 `break`，程序会继续执行 `case 3`，再输出 `Mar `，直到遇到 `break` 停止，所以输出为 `Jan Mar`，选 C。

## 第7题

执行下面 C++ 代码后，有关说法错误的是（ ）。

```cpp
int a, b;
cin >> a >> b;
if (a && b)
    cout << "1";
else if (!(a || b))
    cout << "2";
else if (a || b)
    cout << "3";
else
    cout << "4";
```

A. 如果先后输入 1 和 1，则将输出 1
B. 如果先后输入 0 和 1 或者 1 和 0，则将输出 3
C. 如果先后输入 0 和 0，则将输出 2
D. 如果先后输入 0 和 0，则将输出 4

答案：D

解析：当输入 `0 0` 时，`a && b` 为假，`a || b` 也为假，因此 `!(a || b)` 为真，会输出 `2`。所以 A、B、C 都符合对应输入下的结果，D 说 `0 0` 会输出 `4` 是错误说法，选 D。

## 第8题

某货币由 5 元、2 元和 1 元组成。输入金额（假设为正整数），计算出最少数量。为实现其功能，横线处应填入代码是（ ）。

```cpp
int N;
cin >> N;

int M5, M2, M1;
M5 = N / 5;
M2 = _______________;
M1 = _______________;
printf("5*%d+2*%d+1*%d", M5, M2, M1);
```

A. 第 1 横线处应填入：`N / 2`  第 2 横线处应填入：`N - M5 - M2`
B. 第 1 横线处应填入：`(N - M5 * 5) / 2`  第 2 横线处应填入：`N - M5 * 5 - M2 * 2`
C. 第 1 横线处应填入：`N - M5 * 5 / 2`  第 2 横线处应填入：`N - M5 * 5 - M2 * 2`
D. 第 1 横线处应填入：`(N - M5 * 5) / 2`  第 2 横线处应填入：`N - M5 - M2`

答案：B

解析：要让张数最少，应先用最多的 5 元：`M5 = N / 5`。剩余金额是 `N - M5 * 5`，再用 2 元，所以 `M2 = (N - M5 * 5) / 2`；最后的 1 元数量等于扣掉 5 元和 2 元后的剩余金额，即 `N - M5 * 5 - M2 * 2`，选 B。

## 第9题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int loopCount = 0;
for (int i=0; i < 10; i++)
    for (int j=1; j < i; j++)
        loopCount += 1;
cout << loopCount;
```

A. 55
B. 45
C. 36
D. 28

答案：C

解析：外层 `i` 从 0 到 9。内层 `j` 从 1 到 `i-1`，因此当 `i=0,1` 时执行 0 次，`i=2` 到 9 时分别执行 `1,2,...,8` 次，总次数为 `1+2+...+8=36`，所以输出 36，选 C。

## 第10题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int loopCount = 0;
for (int i=0; i < 10; i++){
    for (int j=0; j < i; j++)
        if (i * j % 2)
            break;
    loopCount += 1;
}
cout << loopCount;
```

A. 25
B. 16
C. 10
D. 9

答案：C

解析：`loopCount += 1` 位于外层循环体内、内层循环之后，所以外层每执行一轮都会加 1。内层的 `break` 只会提前结束内层循环，不会跳出外层循环；外层 `i=0` 到 9 共 10 轮，因此输出 10，选 C。

## 第11题

假设下面 C++ 代码执行过程中仅输入正负整数或 0，有关说法错误的是（ ）。

```cpp
int N, Sum = 0;
cin >> N;
while (N){
    Sum += N;
    cin >> N;
}
cout << Sum;
```

A. 执行上面代码如果输入 0，将终止循环
B. 执行上面代码能实现所有非 0 整数的求和
C. 执行上面代码第一次输入 0，最后将输出 0
D. 执行上面代码将陷入死循环，可将 `while (N)` 改为 `while (N==0)`

答案：D

解析：`while (N)` 表示只要当前输入不是 0 就继续循环，每轮都会重新读入下一个 `N`，所以输入 0 时能结束，并不会必然死循环；第一次就输入 0 时 `Sum` 仍为 0。把条件改成 `while (N==0)` 反而会在输入 0 时进入循环，因此 D 是错误说法，选 D。

## 第12题

执行下面的 C++ 代码，有关说法正确的是（ ）【质数是指仅能被 1 和它本身整除的正整数】。

```cpp
int N;
cin >> N;
bool Flag = true;
for (int i = 2; i < N; i++){
    if (i * i > N)
        break;
    if (N % i == 0){
        Flag = false;
        break;
    }
}
if (Flag)
    cout << N << "是质数" << endl;
else
    cout << N << "不是质数" << endl;
```

A. 如果输入正整数，上面代码能正确判断 $N$ 是否为质数
B. 如果输入整数，上面代码能正确判断 $N$ 是否为质数
C. 如果输入大于等于 0 的整数，上面代码能正确判断 $N$ 是否为质数
D. 如将 `Flag = true` 修改为 `Flag = N>=2? true:false` 则能判断所有整数包括负整数、0、正整数是否为质数

答案：D

解析：原程序一开始把 `Flag` 设为 `true`，所以输入 1、0 或负数时，循环不会找到因子，最后会被误判为质数。若改成 `Flag = N>=2? true:false`，则先把小于 2 的整数判为非质数，再检查大于等于 2 的数是否有因子，能覆盖负数、0 和正整数，选 D。

## 第13题

下面 C++ 代码用于实现如下图所示的效果，其有关说法正确的是（ ）。

```
1
2 4
3 6 9
4 8 12 16
5 10 15 20 25
```

```cpp
for (int i = 1; i < 6; i++) { // L1
    for (int j = 1; j < i+1; j++) // L2
        cout << i*j << " ";
    cout << endl;
}
```

A. 当前代码能实现预期效果，无需调整代码
B. 如果 `cout << endl;` 移到循环 L2 内部，则可实现预期效果
C. 如果 `cout << endl;` 移到循环 L1 外部，则可实现预期效果
D. 删除 `cout << endl;` 行，则可实现预期效果

答案：A

解析：外层 `i` 控制行数为 1 到 5，内层 `j` 从 1 到 `i`，每行依次输出 `i*1, i*2, ..., i*i`。内层循环结束后执行一次 `cout << endl;`，正好换到下一行，当前代码已经得到题目所示图形，选 A。

## 第14题

下面 C++ 代码执行后，输出是（ ）。

```cpp
int cnt1 = 0, cnt2 = 0;
for (int i = 0; i < 10; i++){
    if (i % 2 == 0)
        continue;
    if (i % 2)
        cnt1 += 1;
    else if (i % 3 == 0)
        cnt2 += 1;
}
cout << cnt1 << " " << cnt2;
```

A. 5 2
B. 5 0
C. 0 2
D. 0 0

答案：B

解析：`i` 为偶数时先执行 `continue`，后面的判断都跳过，所以偶数不会影响 `cnt2`。剩下的奇数是 1、3、5、7、9，它们都满足 `i % 2`，使 `cnt1` 加 1，共加 5 次；`else if` 分支没有机会执行，`cnt2` 仍为 0，输出 `5 0`，选 B。

## 第15题

在下面的 C++ 代码中，$N$ 必须是小于 $10$ 大于 $1$ 的整数，$M$ 为正整数（大于 $0$）。如果 $M$ 被 $N$ 整除则 $M$ 为幸运数，如果 $M$ 中含有 $N$ 且能被 $N$ 整除，则为超级幸运数，否则不是幸运数。程序用于判断 $M$ 是否为幸运数或超级幸运数或非幸运数。阅读下面代码，有关说法正确的是（ ）。

```cpp
int N, M;
cout << "请输入幸运数字：";
cin >> N;
cout << "请输入正整数：";
cin >> M;

bool Lucky;
if (M % N == 0)
    Lucky = true;
else
    Lucky = false;
while (M){
    if (M % 10 == N && Lucky){
        printf("%d是%d的超级幸运数!", M, N);
        break;
    }
    M /= 10;
}
if (M == 0)
    if (Lucky)
        printf("%d是%d的幸运数!", M, N);
    else
        printf("%d非%d的幸运数!", M, N);
```

A. 如果 $N$ 输入 $3$，$M$ 输入 $36$ 则将输出：`36是3的超级幸运数!`
B. 如果 $N$ 输入 $7$，$M$ 输入 $21$ 则将输出：`21是7的幸运数!`
C. 如果 $N$ 输入 $8$，$M$ 输入 $36$ 则将输出：`36非8的超级幸运数!`
D. 如果 $N$ 输入 $3$，$M$ 输入 $63$ 则将输出：`63是3的超级幸运数!`

答案：D

解析：程序在检查各位数字时会不断执行 `M /= 10`，所以若不是一开始就遇到幸运数字，输出中的 `M` 可能已不是原来的数。`N=3, M=63` 时，`63%3==0` 且个位就是 3，循环第一次就输出 `63是3的超级幸运数!`；A 会输出 `3是3...`，B 会输出 `0是7...`，C 会输出 `0非8的幸运数!`，只有 D 正确。

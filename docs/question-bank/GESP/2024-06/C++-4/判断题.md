## 第1题

`int& a` 和 `&a` 是一样的，都是取 `a` 的地址。

答案：错误

解析：`int& a` 出现在声明中，表示 `a` 是一个 `int` 类型变量的引用；`&a` 出现在表达式中，才是取变量 `a` 的地址。两者位置和含义都不同，不能说都是取地址。

## 第2题

以下代码不能够正确执行。

```cpp
#include<iostream>
using namespace std;
int main()
{
    int a=20;
    int& ra;
    ra=&a;
    cout<<ra<<endl;
}
```

答案：正确

解析：引用定义时必须立即绑定到一个已有对象，`int& ra;` 没有初始化，已经是语法错误；后面的 `ra=&a;` 也不对，因为 `ra` 是 `int` 的引用，不能用 `int*` 类型的地址给它赋值，所以这段代码不能正确执行。

## 第3题

引用是一个指针常量。

答案：正确

解析：从使用效果看，引用一旦初始化就始终绑定同一个对象，不能再改为引用别的对象，这一点类似“指针常量”的指向不可变。需要注意引用不是普通指针变量，题干是在说明它绑定对象不可改变这一特性。

## 第4题

下面程序两个输出结果是一样的。

```cpp
#include<iostream>
using namespace std;
int main()
{
    int a[2][3]={0};
    cout<<a<<endl;
    cout<<&a[0][0]<<endl;
}
```

答案：正确

解析：二维数组 `a` 的首地址就是第一个元素 `a[0][0]` 所在的地址。`cout<<a` 输出的是指向第 0 行的地址值，`cout<<&a[0][0]` 输出的是第一个 `int` 的地址值，指针类型不同，但数值地址相同。

## 第5题

函数不可以调用自己。

答案：错误

解析：函数可以调用自己，这种写法叫递归，例如求阶乘或遍历树形结构时常见。递归函数必须设置能停止调用的边界条件，否则会一直调用下去直到栈空间耗尽。

## 第6题

函数参数传递过程中，如果传常量值、常量引用和常量指针都是不能被修改的，它们可以防止函数对实参的值或地址进行修改。

答案：正确

解析：`const` 修饰形参时，会限制函数体内对该形参或其所代表对象的修改：常量值形参不能在函数内被重新赋值，常量引用不能通过引用改实参，带 `const` 限定的指针形参也能限制对应的值或指向被改写。因此它们常用于避免函数意外修改调用者传入的数据。

## 第7题

下面代码输出的值等于 $0$。

```cpp
#include<iostream>
using namespace std;
int main()
{
    int *p=NULL;
    cout<<p<<endl;
}
```

答案：正确

解析：`NULL` 表示空指针常量，`int *p=NULL;` 后 `p` 不指向任何有效的 `int` 对象。直接输出指针变量 `p` 会输出空指针的地址表示，按本题表述可看作值为 `0`；这里没有解引用 `*p`，所以不是访问非法内存。

## 第8题

在下面这个程序里，`a[i][j]` 和一个普通的整型变量一样使用。

```cpp
#include<iostream>
using namespace std;
int main()
{
    int a[10][10]={0};
    for(int i=0;i<10;i++)
    {
        for(int j=0;j<10;j++)
        {
            if(i==j)
            {
                a[i][j]=1;
            }
        }
    }
}
```

答案：正确

解析：`a[i][j]` 表示二维数组中第 `i` 行第 `j` 列的一个 `int` 元素。代码里当 `i==j` 时执行 `a[i][j]=1;`，就是给这个整型元素赋值，因此它可以像普通整型变量一样读写。

## 第9题

一个一维数组，至少含有一个自然数 $N$，是一个合法的数列。可以在一维数组末尾加入一个自然数 $M$，$M$ 不能超过一维数组末尾元素的一半，形成一个新的合法的一维数组，如果 $N=6$，那么可以有 $6$ 个不同的合法数组。

答案：正确

解析：从初始数组 `[6]` 开始，可以选择停止，也可以在末尾接不超过前一项一半的自然数。所有情况为 `[6]`、`[6,1]`、`[6,2]`、`[6,2,1]`、`[6,3]`、`[6,3,1]`，共 6 个，因此题干判断正确。

## 第10题

插入排序算法中，平均时间复杂度是 $O(n^2)$，最坏的情况逆序情况下，达到最大时间复杂度。

答案：正确

解析：插入排序平均情况下，每个元素需要在前面已排序部分中查找插入位置并移动若干元素，总体时间复杂度为 $O(n^2)$。当原数组逆序时，每次插入都要把前面元素几乎全部右移，比较和移动次数达到最多，也是 $O(n^2)$ 的最坏情况。

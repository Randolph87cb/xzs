## 第1题

小杨父母带他到某培训机构给他报名参加 CCF 组织的 GESP 认证考试的第 $1$ 级，那他可以选择的认证语言有（ ）种。

【知识点：计算机基础常识】

A. $1$
B. $2$
C. $3$
D. $4$

答案：C

解析：GESP 第 1 级的认证语言可分为图形化编程、Python 编程和 C++ 编程三类。题目要求填写“有几种”，因此应按可选语言类别计数，而不是只看本卷是 C++ 方向。三类合计为 $3$ 种，所以选 C。A、B 少算了实际可报名的语言类别；D 多算了一类题目中并不存在的认证语言。

## 第3题

一般默认 $64$ 位计算机系统中整型变量（int）还是 $32$ 位，则整数能够表示的数据范围是（ ）。

【知识点：原码反码补码与整数表示】

A. $0 \sim 2^{32}$
B. $0 \sim 2^{64}$
C. $-2^{31} \sim (2^{31}) - 1$
D. $-2^{63} \sim (2^{63}) - 1$

答案：C

解析：题目说明 `int` 仍按 $32$ 位有符号整数理解，补码表示中最高位用于表示符号，因此范围是 $-2^{31}$ 到 $2^{31}-1$，选 C。A、B 是从 $0$ 开始的无符号范围，D 是 $64$ 位有符号整数范围。

## 第5题

二进制数 $101.11$ 对应的十进制数是（ ）。

【知识点：进制转换与进制字面量】

A. $6.5$
B. $5.5$
C. $5.75$
D. $5.25$

答案：C

解析：二进制整数部分 $101_2=1\times 2^2+0\times 2^1+1\times 2^0=5$；小数部分 $.11_2=1\times 2^{-1}+1\times 2^{-2}=0.5+0.25=0.75$。合起来是 $5.75$，选 C。

## 第7题

下列代码的输出结果是（ ）。
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 12;
    int result = a >> 2;
    cout << result << endl;
    return 0;
}
```

【知识点：位运算】

A. $12$
B. $6$
C. $3$
D. $1$

答案：C

解析：`12` 的二进制是 `1100`，右移 $2$ 位变成 `0011`，十进制就是 $3$。对正整数来说，右移 $2$ 位也相当于整除 $4$，所以输出 $12/4=3$，选 C。

## 第8题

下列代码的输出结果是（ ）。
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    int b = 10;

    a = a ^ b;
    b = a ^ b;
    a = a ^ b;

    cout << "a = " << a << ", b = " << b << endl;
    return 0;
}
```

【知识点：位运算】

A. a = 5, b = 10
B. a = 5, b = 5
C. a = 10, b = 5
D. a = 10, b = 10

答案：C

解析：异或满足 `x ^ x = 0`、`x ^ 0 = x`。第一次后 `a` 变为原来的 `5 ^ 10`；第二句 `b = (5 ^ 10) ^ 10` 得到 $5$；第三句 `a = (5 ^ 10) ^ 5` 得到 $10$。因此两数被交换，输出 `a = 10, b = 5`，选 C。

#include <iostream>

## 第11题

小杨在做数学题，题目要求找出从 $1$ 到 $35$ 中能被 $7$ 整除的数字，即 $[7, 14, 21, 28, 35]$，则横线处应填入哪个代码？（ ）
```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[35];

    int count = 0;
    for (int i = 1; i <= 35; i++) {
        if (i % 7 == 0)
            ________________ // 在此处填入代码
    }

    for (int i = 0; i < count; i++)
        cout << arr[i] << endl;

    return 0;
}
```

【知识点：数组与字符数组】

A. `arr[count++] = i;`
B. `arr[i] = count++;`
C. `arr[i] = count;`
D. `arr[count] = count++;`

答案：A

解析：`count` 表示已经找到的倍数个数，也正好是下一个存放位置。`arr[count++] = i;` 会把当前满足条件的 $i$ 依次放到 `arr[0]`、`arr[1]` 等位置，再把 `count` 加 $1$，最后循环输出 `arr[0]` 到 `arr[count-1]` 就是 $7,14,21,28,35$。

## 第12题

已知字符 `'0'` 的ASCII编码的十进制表示为 $48$，则执行下面 C++ 代码后，输出是（ ）。

```cpp
#include <iostream>
using namespace std;

int main() {
    string s = "0629";
    
    int n = s.length();
    int x = 0;
    for(int i = 0; i < n; i++)
        x += s[i];
    
    cout << x << endl;
    return 0;
}
```

【知识点：ASCII/字符编码】

A. $17$
B. $158$
C. $209$
D. $316$

答案：C

解析：循环中 `x += s[i]` 加的是字符的 ASCII 编码，不是数字本身。已知 `'0'` 是 $48$，则 `'6'` 是 $54$，`'2'` 是 $50$，`'9'` 是 $57$，总和 $48+54+50+57=209$，选 C。

## 第14题

下面可以正确输出 `They're planning a party for their friend's birthday.` 的 C++ 语句是？（ ）

【知识点：控制结构与程序阅读】

A. `cout << 'They\'re planning a party for their friend'\s birthday." << endl;`
B. `cout << "They\'re planning a party for their friend's birthday.'<< endl;`
C. `cout << 'They're planning a party for their friend's birthday.'<< endl;`
D. `cout << "They\'re planning a party for their friend\'s birthday." << endl;`

答案：D

解析：要输出一整句话，应使用双引号表示字符串字面量；单引号只能表示字符常量，不能包住整句。D 用双引号包住整句，并把两个英文撇号写成转义形式，能够输出目标句子，因此按官方答案选 D。

## 第15题

如果执行下面 C++ 代码后，输出的结果是 `gesp ccf org cn `（即 `cn` 后还有一个空格），则横线上应填入哪个代码？（ ）
```cpp
#include <iostream>
using namespace std;

int main() {
    string str = "gesp.ccf.org.cn";

    string delimiter = ".";
    string result = "";
    string token;
    size_t found = str.find(delimiter);
    while (found != string::npos) {
        token = str.substr(0, found);
        result += token;
        result += " ";
        ________________ // 在此处填入代码
        found = str.find(delimiter);
    }

    //最后一部分
    result += str;
    result += " ";

    cout << result << endl;
    return 0;
}
```

【知识点：字符串处理】

A. `str = str.substr(found + delimiter.length(), str.length() - 1);`
B. `str = str.substr(found, str.length());`
C. `str = str.substr(found, str.length() -1);`
D. 以上都不对

答案：A

解析：每轮已经把第一个点号前的 `token` 加入结果，接下来必须把 `str` 更新为点号后面的剩余部分。A 从 `found + delimiter.length()` 开始截取，能依次得到 `ccf.org.cn`、`org.cn`、`cn`；第二个参数长度偏大也会被 `substr` 自动截到字符串末尾。B、C 都从点号本身开始截取，下一次 `find` 仍会在开头找到点号，不能正确推进。

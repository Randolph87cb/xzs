## 第9题

执行下面 C++ 代码后，输出的结果不可能是 $89781$。

```cpp
#include <iostream>
#include <cstdlib>  // 为了使用 rand() 和 srand()
#include <ctime>    // 为了使用 time()

using namespace std;

int main() {
    // 设置随机种子
    srand(time(NULL));

    int i = 1;
    int s[5];
    while(i <= 5)
    {
        int a = rand() % 10;

        if(a % 3 == (i + 1) % 3)
            s[i++] = a;
    }
    for(int i = 1; i <= 5; i++)
        cout << s[i];
    cout << endl;
    return 0;
}
```

【知识点：控制结构与程序阅读】

答案：正确

## 第1题

如果将双向链表的最后一个结点的下一项指针指向第一个结点，第一个结点的前一项指针指向最后一个结点，则该双向链表构成循环链表。

答案：正确

解析：双向链表中每个结点既能通过 `next` 向后走，也能通过 `prev` 向前走。若尾结点的 `next` 指回头结点、头结点的 `prev` 指回尾结点，链表首尾相接，从任意方向都能循环访问，因此构成循环双向链表。

## 第2题

数组和链表都是线性表，链表的优点是插入删除不需要移动元素，并且能随机查找。

答案：错误

解析：数组和链表都属于线性表，但“能随机查找”是数组的典型优点，不是链表的优点。链表插入、删除时可以通过改指针避免大量移动元素，但查找第 $k$ 个元素通常要从头沿指针走，时间复杂度为 $O(k)$ 或 $O(n)$。

## 第3题

链表的存储空间物理上可以连续，也可以不连续。

答案：正确

解析：链表结点之间靠指针连接，逻辑上的相邻结点不要求物理地址相邻。程序可以连续申请到一批结点，也可能在不同时间申请到分散位置的结点，所以链表的物理存储可以连续，也可以不连续。

## 第4题

找出自然数 $n$ 以内的所有质数，常用算法有埃拉托斯特尼（埃氏）筛法和线性筛法，其中埃氏筛法效率更高。

答案：错误

解析：埃氏筛会从每个质数出发标记它的倍数，复杂度通常记为 $O(n \log \log n)$；线性筛通过“每个合数只被最小质因子筛一次”把复杂度降到 $O(n)$。因此不能说埃氏筛效率更高。

## 第5题

唯一分解定理表明任何一个大于 $1$ 的整数都可以唯一地表示为一系列质数的乘积，即质因数分解是唯一的。

答案：正确

解析：唯一分解定理说明每个大于 $1$ 的整数都能分解为若干质数的乘积，并且除去质因子排列顺序后分解结果唯一。例如 $12=2^2 \times 3$，不能分解成另一组不同的质因子组合。

## 第6题

贪心算法通过每一步选择局部最优解来获得全局最优解，但并不一定能找到最优解。

答案：正确

解析：贪心算法每一步只根据当前状态做看起来最优的选择。只有当题目满足贪心选择性质和最优子结构时，局部最优才能保证导向全局最优；否则局部选择可能堵住后续更优方案，所以贪心不一定总能找到最优解。

## 第7题

归并排序和快速排序都采用递归实现，也都是不稳定排序。

答案：错误

解析：快速排序通常是不稳定排序，因为划分和交换可能改变相等元素的相对顺序；但归并排序在合并时若相等元素优先取左侧元素，就可以保持稳定。题干把归并排序也说成不稳定排序，因此说法错误。

## 第8题

插入排序有时比快速排序时间复杂度更低。

答案：正确

解析：插入排序最好情况下数组已经有序，只需线性扫描，时间复杂度为 $O(n)$；快速排序平均为 $O(n \log n)$，最坏还可能退化到 $O(n^2)$。所以在某些数据状态或规模较小时，插入排序的时间复杂度可以更低。

## 第9题

在进行全国人口普查时，将其分解为对每个省市县乡来进行普查和统计。这是典型的分治策略。

答案：正确

解析：全国人口普查可以先按省、市、县、乡等层级拆成较小统计任务，分别完成后再逐级汇总得到全国结果。这符合分治“分解子问题、解决子问题、合并结果”的思想。

## 第10题

在下面 C++ 代码中，由于删除了变量 `ptr`，因此 `ptr` 所对应的数据也随之删除。故执行下述代码时，将报错。

```cpp
int* ptr = new int(10);
cout << *ptr << endl;
delete ptr;
cout << ptr << endl;
```

答案：错误

解析：`delete ptr;` 释放的是 `ptr` 指向的动态内存，并不会把指针变量 `ptr` 本身删除。之后再解引用 `*ptr` 才是危险的悬空指针访问；代码只是输出 `ptr` 保存的地址值，一般不会因此编译或运行时报错，所以题干说法错误。

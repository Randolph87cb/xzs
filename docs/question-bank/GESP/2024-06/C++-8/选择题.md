## 第1题

GESP 活动期间，举办方从获胜者 ABCDE 五个人中选出三个人排成一队升国旗，其中 A 不能排在队首，请问有多少种排法？

A. $24$
B. $48$
C. $32$
D. $12$

答案：B

解析：先从 $5$ 人中选出 $3$ 人并排队，共有 $5 \times 4 \times 3 = 60$ 种。A 排在队首时，后两位从其余 $4$ 人中排列，有 $4 \times 3 = 12$ 种，需要排除，所以共有 $60 - 12 = 48$ 种。

## 第2题

$7$ 进制数 $235$ 转换成 $3$ 进制数是（ ）。

A. $11121$
B. $11122$
C. $11211$
D. $11112$

答案：A

解析：先转成十进制：$(235)_7 = 2 \times 7^2 + 3 \times 7 + 5 = 124$。再把 $124$ 按 $3$ 的幂展开，$124 = 1 \times 81 + 1 \times 27 + 1 \times 9 + 2 \times 3 + 1$，所以三进制为 $11121$。

## 第3题

$0,1,2,3,4,5$ 这些数字组成一个三位数，请问没有重复数字的情况下，有多少种组法（ ）。

A. $180$
B. $120$
C. $80$
D. $100$

答案：D

解析：百位不能为 $0$，有 $1,2,3,4,5$ 共 $5$ 种选法；十位可从剩下 $5$ 个数字中选；个位再从剩下 $4$ 个数字中选。总数为 $5 \times 5 \times 4 = 100$。

## 第4题

有 $V$ 个顶点、$E$ 条边的图的深度优先搜索遍历时间复杂度为（ ）。

A. $O(V)$
B. $O(E)$
C. $O(V + E)$
D. $O(\log(V + E))$

答案：C

解析：用邻接表进行 DFS 时，每个顶点最多被访问一次，每条边也只会在遍历邻接表时被检查常数次，因此总时间为 $O(V + E)$。只写 $O(V)$ 或 $O(E)$ 都漏掉了另一部分输入规模。

## 第5题

一对夫妻生男生女的概率相同。已知这对夫妻有两个孩子，其中一个是女孩，另一个是男孩的概率是多少？

A. $\frac{2}{3}$
B. $\frac{1}{4}$
C. $\frac{1}{2}$
D. $\frac{1}{3}$

答案：C

解析：题目按“两个孩子一男一女”的无条件概率计算。两个孩子按出生顺序共有男男、男女、女男、女女 $4$ 种等可能情况，其中一男一女有男女、女男 $2$ 种，所以概率为 $\frac{2}{4} = \frac{1}{2}$。这不是“已知至少一个是男孩”的条件概率问题。

## 第6题

从 $1$ 到 $2024$ 这 $2024$ 个数中，共有（ ）个包含数字 $6$ 的数。

A. $544$
B. $546$
C. $564$
D. $602$

答案：A

解析：先数 $1$ 到 $1999$ 中含 $6$ 的数。把 $0000$ 到 $1999$ 看成两千个四位串，千位为 $0$ 或 $1$，后三位不含 $6$ 的有 $9^3$ 种，所以不含 $6$ 的共有 $2 \times 9^3 = 1458$ 个，含 $6$ 的为 $2000 - 1458 = 542$ 个。$2000$ 到 $2024$ 中只有 $2006,2016$ 含 $6$，总数为 $542 + 2 = 544$。

## 第7题

二进制数 $100.001$ 转换成十进制数是（ ）。

A. $4.25$
B. $4.125$
C. $4.5$
D. $4.75$

答案：B

解析：整数部分 $100_2 = 1 \times 2^2 = 4$；小数部分 $.001_2 = 1 \times 2^{-3} = \frac{1}{8} = 0.125$。合起来是 $4.125$。

## 第8题

以下函数声明，哪个是符合 C++ 语法的？（ ）。

A. `void BubbleSort(char a[][], int n);`
B. `void BubbleSort(char a[][20], int n);`
C. `void BubbleSort(char a[10][], int n);`
D. `void BubbleSort(char[,] a, int n);`

答案：B

解析：C++ 函数参数中的二维数组可以省略第一维，但必须给出第二维大小，编译器需要用它计算元素地址。因此 `char a[][20]` 合法；`a[][]` 和 `a[10][]` 都缺少第二维，`char[,] a` 也不是 C++ 数组参数写法。

## 第9题

下面有关 C++ 重载的说法，错误的是（ ）。

A. 两个参数个数不同的函数可以重名。
B. 两个参数类型不同的函数可以重名。
C. 两个类的方法可以重名。
D. 所有 C++ 运算符均可以重载。

答案：D

解析：函数重载可以依靠参数个数或参数类型区分，不同类中的成员函数同名也不会互相冲突。错误的是 D，因为 C++ 中并非所有运算符都能重载，例如 `.`、`::`、`sizeof`、`?:` 等不能被重载。

## 第10题

小于或等于给定正整数 $n$ 的数中，与 $n$ 互质的数的个数，我们称为欧拉函数，记作 $\phi(n)$。下面说法错误的是（ ）。

A. 如果 $n$ 是质数，那么 $\phi(n) = n - 1$。
B. 两个质数一定是互质数。
C. 两个相邻的数一定是互质数。
D. 相邻的两个质数不一定是互质数。

答案：D

解析：质数 $n$ 只和 $1$、自身有公共因子，所以 $\phi(n)=n-1$；两个不同质数一定互质；相邻整数若有大于 $1$ 的公因子，也会整除它们的差 $1$，不可能。相邻的两个质数仍然是两个不同质数，一定互质，所以 D 的说法错误。

## 第11题

已知一棵二叉树有 $10$ 个节点，则其中至多有（ ）个节点有 $2$ 个子节点。

A. $4$
B. $5$
C. $6$
D. $3$

答案：A

解析：设有 $x$ 个节点有 $2$ 个子节点，二叉树中叶子节点数为 $x + 1$。若要让 $x$ 尽量大，应尽量不出现只有 $1$ 个子节点的节点，此时总结点数为 $x + (x + 1) = 2x + 1$。由 $2x + 1 \le 10$ 得 $x \le 4$，所以最多为 $4$ 个。

## 第12题

二项展开式 $(x + y)^n = x^n + n x^{n-1} y + \frac{n(n-1)}{2} x^{n-2} y^2 + \ldots + y^n$ 的系数，正好满足杨辉三角的规律。当 $n = 10$ 时，二项式展开式中 $x y^9$ 项的系数是（ ）。

A. $5$
B. $9$
C. $10$
D. $8$

答案：C

解析：$(x+y)^{10}$ 中 $xy^9$ 项对应从 $10$ 个因子里选 $1$ 个提供 $x$，其余 $9$ 个提供 $y$，系数为组合数 $\binom{10}{1}=10$。在杨辉三角第 $10$ 行中，它也对应靠边的第二个系数 $10$。

## 第13题

下面程序的时间复杂度为（ ）。
```cpp
bool notPrime[N] = {false};
void sieve() {
    for (int n = 2; n * n < N; n++)
        if (!notPrime[n])
            for (int i = n * n; i < N; i += n)
                notPrime[i] = true;
}
```

A. $O(N)$
B. $O(N \times \log N)$
C. $O(N \times \log \log N)$
D. $O(N^2)$

答案：C

解析：这段代码是埃氏筛。只有当前 $n$ 仍是质数时，才从 $n^2$ 开始按步长 $n$ 标记倍数，总标记次数约为 $N \left(\frac{1}{2}+\frac{1}{3}+\frac{1}{5}+\cdots\right)$，其中分母只取质数，规模为 $O(N \log\log N)$。

## 第14题

下面程序的最差时间复杂度为（ ）。
```cpp
int gcd(int m, int n) {
    if (m == 0)
        return n;
    return gcd(n % m, m);
}
```

A. $O(\sqrt{n})$
B. $O(\log(n))$
C. $O(n)$
D. $O(1)$

答案：B

解析：这是欧几里得算法，每次递归把问题变成 `(n % m, m)`，较大的数会快速变小；最坏情况出现在相邻斐波那契数附近，递归次数与数字位数成正比，因此为 $O(\log n)$，不是线性或常数级。

## 第15题

下面程序的输出为（ ）。
```cpp
#include <iostream>
using namespace std;
int main() {
    int cnt = 0;
    for (int x = 0; x <= 10; x++)
        for (int y = 0; y <= 10; y++)
            for (int z = 0; z <= 10; z++)
                if (x + y + z <= 15)
                    cnt++;
    cout << cnt << endl;
    return 0;
}
```

A. $90$
B. $91$
C. $710$
D. $711$

答案：D

解析：若只要求非负整数 $x+y+z \le 15$，解数为 $\binom{18}{3}=816$。但程序还限制 $x,y,z \le 10$，需要减去某个变量至少为 $11$ 的情况：令该变量减去 $11$，剩余和不超过 $4$，每个变量有 $\binom{7}{3}=35$ 种，三种变量共减 $105$ 种。两个变量同时超过 $10$ 时和已至少 $22$，不可能，所以输出 $816-105=711$。

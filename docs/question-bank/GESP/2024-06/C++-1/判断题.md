## 第1题

在 C++ 代码中变量 X 被赋值为 16.44，则 `cout << X / 10` 执行后输出的一定是 1。（ ）

答案：错误

解析：题干没有说明 `X` 是整型还是浮点型。如果 `X` 是 `float` 或 `double`，`X / 10` 的结果约为 $1.644$，输出不一定是 $1$；即使整型变量也不能保存 `16.44` 这个小数值。因此“一定是 1”的说法过于绝对。

## 第2题

C++ 的整型变量 N 被赋值为 10，则语句 `cout << N / 4 << "->" << N % 4 << "->" << N / 4.0` 执行后输出是 `2->2->2.0`。（ ）

答案：错误

解析：`N / 4` 是两个整数相除，结果为 $2$；`N % 4` 是余数，结果为 $2$；但 `N / 4.0` 中有浮点数 `4.0`，结果是 $2.5$。因此输出不会是 `2->2->2.0`。

## 第3题

定义 C++ 的 `float` 型变量 N，则语句 `cin >> N; cout << int(float(N))` 可以输入正负整数和浮点数，并将其转换为整数后输出。（ ）

答案：正确

解析：`float` 变量可以通过 `cin` 读入整数形式或小数形式的数，也可以读入带负号的数。`float(N)` 仍表示把 `N` 当作浮点数，外层 `int(...)` 会把它强制转换为整数，转换时舍去小数部分后输出。

## 第4题

C++ 的整型 $N$ 被赋值为 5，语句 `printf("%d*2",N)` 执行后将输出 `10`。（ ）

答案：错误

解析：格式字符串 `"%d*2"` 中，`%d` 会输出变量 `N` 的值，后面的 `*2` 只是普通字符。`N=5` 时输出的是 `5*2`，不会自动把它计算成 $10$。

## 第5题

在 C++ 中，`break` 语句用于终止当前层次的循环，循环可以是 `for` 循环，也可以是 `while` 循环。（ ）

答案：正确

解析：`break` 的作用是立即跳出它所在的当前一层循环或 `switch`。在 `for` 循环和 `while` 循环中都可以使用 `break` 来提前结束当前循环，所以题干说法正确。

## 第6题

在 C++，`continue` 语句通常与 `if` 语句配合使用。（ ）

答案：正确

解析：`continue` 的作用是跳过本次循环剩余语句，直接进入下一次循环判断。实际写程序时，常常用 `if` 判断某种情况是否需要跳过本次循环，因此说它通常与 `if` 配合使用是合理的。

## 第7题

在 C++ 代码中，不可以将变量命名为 `printf`，因为 `printf` 是 C++ 语言的关键字。（ ）

答案：错误

解析：`printf` 是 C/C++ 标准库中的输出函数名，不是 C++ 关键字。关键字如 `int`、`if`、`while` 等才不能作为变量名；题干把“函数名”说成“关键字”，判断依据错误。

## 第8题

在 C++ 中有整型变量 $N$，则表达式 `N += 8/4/2` 相当于 `N += 8/(4/2)`。（ ）

答案：错误

解析：除法运算符 `/` 的结合方向是从左到右，所以 `8/4/2` 等价于 `(8/4)/2`，结果为 $1$。而 `8/(4/2)` 先算括号，结果是 $8/2=4$。两边结果不同，因此不相当。

## 第9题

C++ 中定义变量 `int N`，则表达式 `(!!N)` 的值也是 $N$ 的值。（ ）

答案：错误

解析：`!!N` 是先对 `N` 取逻辑非，再对结果取逻辑非，结果只可能是 `true` 或 `false`，在输出或整数语境中通常表现为 $1$ 或 $0$。例如 `N=5` 时，`!!N` 的值为 $1$，并不是 $5$，所以它不是 `N` 的值。

## 第10题

GESP 测试是对认证者的编程能力进行等级认证，同一级别的能力基本上与编程语言无关。（ ）

答案：正确

解析：GESP 是编程能力等级认证，同一级别考查的是相近的编程能力目标，例如基本输入输出、变量、表达式和控制结构等。不同语言的语法写法会不同，但同一级别对应的能力要求基本一致。

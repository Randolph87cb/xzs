## 第1题

在 C++ 中，下列不可做变量的是（ ）。

A. `five-Star`
B. `five_star`
C. `fiveStar`
D. `_fiveStar`

答案：A

解析：C++ 标识符只能由字母、数字和下划线组成，并且不能以数字开头。`five-Star` 中的 `-` 会被看作减号，不是变量名的一部分，所以不能作为变量名；`five_star`、`fiveStar`、`_fiveStar` 都符合命名规则。

## 第2题

C++ 表达式 `3 - 3 * 3 / 5` 的值是（ ）。

A. $-1.2$
B. $1$
C. $0$
D. $2$

答案：D

解析：乘法和除法优先于减法，且整数相除会取整数部分。先算 `3 * 3 / 5`，得到 `9 / 5 = 1`，再算 `3 - 1 = 2`，所以结果是 $2$。

## 第3题

在 C++ 中，假设 $N$ 为正整数，则表达式 `cout << (N % 3 + N % 7)` 可能输出的最大值是（ ）。

A. $6$
B. $8$
C. $9$
D. $10$

答案：B

解析：`N % 3` 的最大可能值是 $2$，`N % 7` 的最大可能值是 $6$。这两个最大值可以同时出现，例如 $N=20$ 时，`20 % 3 = 2`、`20 % 7 = 6`，和为 $8$，所以最大输出是 $8$。

## 第4题

C++ 语句 `printf("5%%2={%d}\n",5 % 2)` 执行后的输出是（ ）。

A. `1={1}`
B. `5%2={5%2}`
C. `5%2={1}`
D. `5 ={1}`

答案：C

解析：在 `printf` 的格式字符串中，`%%` 表示输出一个普通的 `%`，`%d` 用来输出后面整数表达式的值。`5 % 2` 的结果是 $1$，所以整句输出 `5%2={1}`。

## 第5题

对整型变量 `i`，执行 C++ 语句 `cin >> i, cout << i` 时如果输入 `5+2`，下述说法正确的是（ ）。

A. 将输出整数 7
B. 将输出 5
C. 语句执行将报错，输入表达式不能作为输出的参数
D. 语句能执行，但输出内容不确定

答案：B

解析：`cin >> i` 读取整型变量时，会先读到前面的数字 `5`，遇到不能作为整数继续读取的 `+` 就停止。因此 `i` 的值是 $5$，`cout << i` 输出 $5$，不会把输入的 `5+2` 当表达式计算成 $7$。

## 第6题

下面 C++ 代码执行后的输出是（ ）。

```cpp
float a;
a = 101.101;
a = 101;
printf("a+1={%.0f}",a+1);
```

A. `102={102}`
B. `a+1={a+1}`
C. `a+1={102}`
D. `a` 先被赋值为浮点数，后被赋值为整数，执行将报错

答案：C

解析：变量 `a` 先被赋值为 `101.101`，随后又被 `a = 101;` 覆盖成 $101$，赋值给 `float` 不会报错。`a+1` 的值是 $102$，格式 `%.0f` 表示按浮点数输出且保留 $0$ 位小数，前面的 `a+1={` 是普通字符，所以输出 `a+1={102}`。

## 第7题

表达式 `9/4 - 6 % (6 - 2) * 10` 的值是（ ）。

A. $-17.75$
B. $-18$
C. $-14$
D. $-12.75$

答案：B

解析：先算括号 `6 - 2 = 4`。整数除法 `9/4` 得 $2$，取模 `6 % 4` 得 $2$，再算 `2 * 10 = 20`，最后 `2 - 20 = -18`。

## 第8题

下面 C++ 代码执行时输入 10 后，正确的输出是（ ）。

```cpp
int N;
cout << "请输入正整数：";
cin >> N;
if (N % 3)
    printf("第5行代码%2d", N % 3);
else
    printf("第6行代码%2d", N % 3);
```

A. 第5行代码1
B. 第6行代码1
C. 第5行代码 1
D. 第6行代码 1

答案：C

解析：输入 $10$ 后，`N % 3 = 1`。在 `if (N % 3)` 中，非 $0$ 表示条件为真，所以执行第 5 行的 `printf`。格式 `%2d` 表示宽度至少为 $2$，数字 $1$ 前会补一个空格，因此输出 `第5行代码 1`。

## 第9题

下面 C++ 代码执行后输出是（ ）。

```cpp
int Sum = 0, i = 0;
for ( ; i < 10; )
    Sum += i++;
cout << i << " " << Sum;
```

A. 9 45
B. 10 55
C. 10 45
D. 11 55

答案：C

解析：循环条件是 `i < 10`，循环中执行 `Sum += i++`，先把当前 `i` 加到 `Sum`，再让 `i` 增加 $1$。因此累加的是 $0+1+\cdots+9=45$，循环结束时 `i` 已变成 $10$，输出 `10 45`。

## 第10题

下面 C++ 代码用于判断 $N$ 是否为质数（只能被 $1$ 和它本身整除的正整数）。程序执行后，下面有关描述正确的是（ ）。

```cpp
int N;
cout << "请输入整数：";
cin >> N;

bool Flag = false;

if (N >= 2){
    Flag = true;
    for (int i=2; i < N; i++)
        if (N % i == 0){
            Flag = false;
            break;
        }
}

if(Flag)
    cout << "是质数";
else
    cout << "不是质数";
```

A. 如果输入负整数，可能输出“是质数”
B. 如果输入 2，将输出“不是质数”，因为此时循环不起作用
C. 如果输入 2，将输出“是质数”，即便此时循环体没有被执行
D. 如果将 `if (N >= 2)` 改为 `if (N > 2)` 将能正确判断 $N$ 是否质数

答案：C

解析：输入 $2$ 时满足 `N >= 2`，程序先把 `Flag` 设为 `true`。接着 `for (int i=2; i < N; i++)` 的条件一开始就是 `2 < 2`，循环体不执行，`Flag` 仍为 `true`，最后输出“是质数”。负整数不会进入外层 `if`，仍为 `false`；若改成 `N > 2`，反而会把 $2$ 判成“不是质数”。

## 第11题

下面的 C++ 代码用于求 $1\sim N$ 之间所有奇数之和，其中 $N$ 为正整数，如果 $N$ 为奇数，则求和时包括 $N$。有关描述错误的是（ ）。

```cpp
int N;
cout << "请输入正整数：";
cin >> N;

int i = 1, Sum = 0;

while (i <= N){
    if (i % 2 == 1)
        Sum += i;
    i += 1;
}

cout << i << " " << Sum;
```

A. 执行代码时如果输入 10，则最后一行输出将是 `11 25`
B. 执行代码时如果输入 5，则最后一行输出将是 `6 9`
C. 将 `i += 1` 移到 `if (i % 2 == 1)` 前一行，同样能实现题目要求
D. 删除 `if (i % 2 == 1)`，并将 `i += 1` 改为 `i += 2`，同样可以实现题目要求

答案：C

解析：原程序先判断当前 `i` 是否为奇数，再把它累加，最后 `i += 1`。如果把 `i += 1` 移到判断前，循环第一次会从 `i=1` 变成 `2` 后再判断，导致漏掉 $1$，并且可能把原本不该计入的下一个数用于判断，不能实现题目要求。A、B 可按原循环算出，D 从 $1$ 开始每次加 $2$，正好遍历奇数。

## 第12题

如果一个整数 $N$ 能够表示为 $X\times X$ 的形式，那么它就是一个完全平方数，下面 C++ 代码用于完成判断 $N$ 是否为一个完全平方数，在横线处应填入的代码是（ ）。

```cpp
int N;

cin >> N;
for(int i = 0; i <= N; i++)
    if(________________)
        cout << N << "是一个完全平方数\n";
```

A. `i == N*N`
B. `i*10 == N`
C. `i+i == N`
D. `i*i == N`

答案：D

解析：完全平方数的定义是存在整数 $X$，使得 $N = X \times X$。循环变量 `i` 正是在尝试每一个可能的整数，所以判断条件应写成 `i*i == N`；其他选项判断的不是“某个整数的平方是否等于 `N`”。

## 第13题

执行下面 C++ 代码后输出的 `cnt` 的值是（ ）。

```cpp
int cnt=0;

for(int i = 0; i*i < 64; i+=2)
    cnt++;
cout << cnt;
```

A. 8
B. 7
C. 4
D. 1

答案：C

解析：`i` 从 $0$ 开始，每次增加 $2$，只要 `i*i < 64` 就执行一次 `cnt++`。满足条件的 `i` 依次为 $0,2,4,6$，到 $8$ 时 `8*8 = 64` 不再小于 $64$，所以循环执行 $4$ 次，`cnt` 输出 $4$。

## 第15题

ENIAC 于 1946 年投入运行，是世界上第一台真正意义上的计算机，它的主要部件都是（ ）组成的。

A. 感应线圈
B. 电子管
C. 晶体管
D. 集成电路

答案：B

解析：ENIAC 是早期电子计算机，主要使用电子管作为基本电子元件。晶体管和集成电路属于后来计算机发展的主要器件，感应线圈不是 ENIAC 的主要组成部件。

## 第1题

又到期末考试周，小明发现这次许多闭卷考试不仅禁止携带手机、平板电脑，还有最近比较时髦的各类 AI 眼镜（也有叫智能眼镜）也同样不允许带入考场。这些 AI 眼镜应该也是内置了操作系统并可能支持 Wi-Fi 或蓝牙连接。

答案：A

解析：本题判断为正确。AI 眼镜属于智能设备，通常需要系统软件来管理摄像头、显示、通信、电源等硬件，因此可以说它内置了操作系统。许多智能眼镜还会通过 Wi-Fi 或蓝牙进行联网、同步数据或连接其他设备，所以题干中的说法合理。

## 第2题

C++ 代码 `cout << (not ('5' % 2 == 0) == ((not '5' % 2) == 0))` 执行后的输出是 `1`。

答案：A

解析：本题判断为正确。字符 `'5'` 参与运算时会使用它的字符编码，常见编码中 `'5'` 的值是 53，所以 `'5' % 2 == 0` 为假，前面加 `not` 后为真。右边的 `not '5' % 2` 按 C++ 运算优先级可理解为先对非零字符 `'5'` 取逻辑非，得到 `0`，再对 `2` 取余，结果仍为 `0`，所以 `((not '5' % 2) == 0)` 也为真。两边都为真，比较结果为真，`cout` 输出 `1`。

## 第3题

执行 C++ 语句 `cout << (int(3.5) * 2)` 将输出 `6`。

答案：A

解析：本题判断为正确。`int(3.5)` 会把小数 `3.5` 转换成整数，直接舍去小数部分得到 `3`。随后计算 `3 * 2`，结果是 `6`，所以输出为 `6`。

## 第4题

下面 C++ 代码执行后将输出 `1-4-7-`。

```cpp
for (int i = 1; i < 10; i += 3) {
    if (not i % 3)
        break;
    cout << i << "-";
}
```

答案：A

解析：本题判断为正确。循环中 `i` 的值依次为 `1, 4, 7`，之后变为 `10` 时循环结束。对于这些非零的 `i`，`not i` 为假，条件不会成立，因此不会执行 `break`，每次都会输出当前的 `i` 和 `-`。最终输出就是 `1-4-7-`。

## 第5题

执行如下 C++ 代码，将从小到大依次输出 `abs(N)` 个整数，并在最后输出 `1`。

```cpp
int N;
cin >> N;
int start_num = 1, end_num = N + 1, i;

if (N < 0)
    start_num = N, end_num = 0;

for (i = start_num; i < end_num; i++)
    cout << i << " ";

cout << endl << ((i - 1) == abs(N)) << endl;
```

答案：B

解析：本题判断为错误。如果 `N` 为正数，程序会输出 `1` 到 `N`，最后 `i - 1` 等于 `N`，确实会输出 `1`。但如果 `N` 为负数，例如 `N = -3`，程序会输出 `-3 -2 -1`，最后循环结束时 `i = 0`，于是 `(i - 1) == abs(N)` 变成 `-1 == 3`，结果为假，最后输出 `0`。所以题干说“在最后输出 1”并不总是成立。

## 第6题

如下 C++ 代码执行后，输出值为 `9`。

```cpp
int cnt = 0, i;
for (i = 0; i < 10; i++) {
    for (int j = 0; j < i; j++)
        cnt += 1;
    break;
}
cout << i;
```

答案：B

解析：本题判断为错误。外层循环第一次执行时 `i = 0`，内层循环条件 `j < i` 不成立，所以内层循环一次也不执行。随后执行外层循环体中的 `break`，直接跳出外层循环，此时 `i` 仍然是 `0`。因此程序输出的是 `0`，不是 `9`。

## 第7题

如下 C++ 代码执行时如输入 `10`，输出将是 `100`。

```cpp
int cnt = 0, N;
cout << "请输入正整数：";
cin >> N;
for (int i = 0; i < N; i++)
    for (int j = -i; j < i; j++)
        cnt += 1;
cout << cnt;
```

答案：B

解析：本题判断为错误。输入 `10` 时，外层循环中 `i` 从 `0` 到 `9`。固定一个 `i`，内层变量 `j` 从 `-i` 到 `i - 1`，一共执行 `2 * i` 次。因此总次数为 `0 + 2 + 4 + ... + 18 = 90`，最后输出 `90`，不是 `100`。

## 第8题

如下 C++ 代码执行其输出是 `3`。

```cpp
int count = 0;
int i = 0;
while (i < 3) {
    int j = 0;
    while (j < 3) {
        if (i + j >= 3)
            count += 1;
        j += 1;
    }
    i += 1;
}
cout << count;
```

答案：A

解析：本题判断为正确。`i` 和 `j` 都会取到 `0, 1, 2`。满足 `i + j >= 3` 的组合有 `(1, 2)`、`(2, 1)`、`(2, 2)`，一共 3 组，所以 `count` 最终为 `3`，程序输出 `3`。

## 第9题

如下 C++ 代码执行时如果输入正整数，其输出将是输入的正整数。

```cpp
int N;
cin >> N;
int i = 0, Nbase = 1;
int rst = 0;
while (N != 0) {
    rst = rst + N % 10 * Nbase;
    N /= 10, Nbase *= 10;
    i += 1;
}
cout << rst;
```

答案：A

解析：本题判断为正确。循环每次用 `N % 10` 取出当前个位数字，再乘以当前位权 `Nbase` 加到 `rst` 中；随后 `N /= 10` 去掉个位，`Nbase *= 10` 移到下一位。因为取出的数字又被放回原来的位权上，所以 `rst` 会重新得到原来的正整数。例如输入 `123`，依次累加 `3 * 1`、`2 * 10`、`1 * 100`，最终输出 `123`。

## 第10题

如下 C++ 代码执行时如输入 `5`，将输出代码后的字符图形。

```cpp
int n;
cin >> n;
for (int i = 1; i < n + 1; i++) {
    for (int j = 1; j < n - i + 1; j++)
        cout << 0;
    for (int k = 1; k < 2 * i; k++) {
        if (k <= i)
            cout << k;
        else
            cout << 2 * i - k;
    }
    for (int j = 1; j < n - i + 1; j++)
        cout << 0;
    cout << endl;
}
```

```text
000010000
000121000
001232100
012343210
123454321
```

答案：A

解析：本题判断为正确。输入 `5` 时，外层循环共输出 5 行。第 `i` 行先输出 `5 - i` 个 `0`，再输出从 `1` 递增到 `i`、再递减到 `1` 的数字，最后再补 `5 - i` 个 `0`。例如第 3 行是两个 `0`、`12321`、两个 `0`，即 `001232100`。逐行计算得到的图形与题干给出的字符图形一致。

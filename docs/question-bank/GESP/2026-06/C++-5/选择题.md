## 第1题

假设 `head != nullptr`，下面是实现单向循环链表在头节点后插入新节点的代码，横线处应填入（ ）。

```cpp
struct Node {
    int val;
    Node* next;
};

void insertAfterHead(Node* head, int x) {
    Node* newNode = new Node;
    newNode->val = x;
    ______________________ // 在此处填入代码
}
```

A.

```cpp
newNode->next = head;
head->next = newNode;
```

B.

```cpp
newNode->next = head->next;
head->next = newNode;
```

C.

```cpp
head->next = newNode;
newNode->next = head->next;
```

D.

```cpp
newNode->next = head->next;
head = newNode;
```

答案：B

解析：本题选 B。在头节点 `head` 后插入新节点时，应先让 `newNode->next` 指向原来的后继 `head->next`，再把 `head->next` 改成 `newNode`，这样新节点就被接到 `head` 和原后继之间。A 把新节点指回 `head`，不是插到 `head` 后面。C 先改了 `head->next`，此时 `head->next` 已经是 `newNode`，再执行 `newNode->next = head->next` 会让新节点指向自己。D 只是改变形参 `head` 的值，没有把新节点接入链表。

## 第2题

下面代码遍历并输出一个循环单链表，其中 `head` 指向链表的第一个节点，横线处应填入的是（ ）。

```cpp
struct Node {
    int val;
    Node* next;
};

void printList(Node* head) {
    if (head == nullptr) return;
    Node* p = head;
    _______________________ // 在此处填入代码
    cout << endl;
}
```

A.

```cpp
while (p != nullptr) {
    cout << p->val << " ";
    p = p->next;
}
```

B.

```cpp
while (p->next != nullptr) {
    cout << p->val << " ";
    p = p->next;
}
```

C.

```cpp
do {
    cout << p->val << " ";
    p = p->next;
} while (p != head);
```

D.

```cpp
for (; p; p = p->next) {
    cout << p->val << " ";
}
```

答案：C

解析：本题选 C。循环单链表的最后一个节点不会指向 `nullptr`，而是重新指向第一个节点 `head`。由于一开始 `p` 就等于 `head`，要先输出当前节点，再移动到下一个节点，并在 `p` 再次回到 `head` 时停止，所以应使用 `do...while (p != head)`。A、B、D 都依赖 `nullptr` 作为结束标志，适用于普通单链表，不适用于这里的循环单链表；其中 B 还会漏掉最后一个节点。

## 第3题

双链表结点定义如下，若要删除双链表中的中间结点（非首尾节点）`p`，下面写法正确的是（ ）。

```cpp
struct Node {
    int val;
    Node* prev;
    Node* next;
};
```

A.

```cpp
p->prev->next = p->next;
p->next->prev = p->prev;
delete p;
```

B.

```cpp
p->next->prev = p->next;
p->prev->next = p->prev;
delete p;
```

C.

```cpp
p->prev = p->next;
p->next = p->prev;
delete p;
```

D.

```cpp
p->next->next = p->prev;
p->prev->prev = p->next;
delete p;
```

答案：A

解析：本题选 A。删除双链表中的中间结点 `p` 时，要让 `p` 的前驱结点的 `next` 指向 `p` 的后继，即 `p->prev->next = p->next`；同时让 `p` 的后继结点的 `prev` 指向 `p` 的前驱，即 `p->next->prev = p->prev`，最后再 `delete p`。B 把前驱、后继指针都接错了。C 只修改了 `p` 自己的指针，链表中其他节点仍然指向 `p`。D 修改的是后继的后继和前驱的前驱，也没有把 `p` 从链表中正确摘除。

## 第4题

使用如下欧几里得算法求 `gcd(105, 45)` 时，函数 `gcd(a, b)` 的递归调用序列正确的是（ ）。

```cpp
int gcd(int a, int b) {
    return b == 0 ? a : gcd(b, a % b);
}
```

A. `gcd(105, 45) -> gcd(45, 60) -> gcd(60, 15) -> gcd(15, 0)`

B. `gcd(105, 45) -> gcd(45, 15) -> gcd(15, 0)`

C. `gcd(105, 45) -> gcd(60, 45) -> gcd(15, 45)`

D. `gcd(105, 45) -> gcd(15, 45) -> gcd(15, 0)`

答案：B

解析：本题选 B。代码在 `b != 0` 时递归调用 `gcd(b, a % b)`。因此 `gcd(105, 45)` 的下一步是 `gcd(45, 105 % 45)`，也就是 `gcd(45, 15)`；再下一步是 `gcd(15, 45 % 15)`，也就是 `gcd(15, 0)`，此时返回。A 把取余结果写成了 `60`，但 `105 % 45 = 15`。C、D 都没有按 `gcd(b, a % b)` 的参数顺序和取余结果递推。

## 第5题

下面代码实现线性筛（欧拉筛），以筛选出 `n` 以内的所有素数。横线处的代码应为（ ）。

```cpp
vector<int> sieve(int n) {
    vector<bool> is_prime(n + 1, true);
    vector<int> primes;

    if (n >= 0) is_prime[0] = false;
    if (n >= 1) is_prime[1] = false;

    for (int i = 2; i <= n; ++i) {
        if (is_prime[i]) {
            primes.push_back(i);
        }
        for (int j = 0; j < primes.size() && i * primes[j] <= n; j++) {
            is_prime[i * primes[j]] = false;
            if (________________) break; // 在此处填入代码
        }
    }
    return primes;
}
```

A. `i % primes[j] == 0`

B. `primes[j] % i == 0`

C. `i % primes[j] != 0`

D. `i == primes[j]`

答案：A

解析：本题选 A。线性筛的关键是每个合数只由它的最小质因子筛掉。当 `i % primes[j] == 0` 时，说明 `primes[j]` 已经是 `i` 的一个质因子，此时继续用更大的质数去乘 `i` 会破坏“由最小质因子筛掉”的规则，所以要 `break`。B 把取模方向写反了，通常没有意义。C 会在不整除时停止，反而漏筛。D 只在 `i` 等于当前质数时停止，不符合线性筛的停止条件。

## 第6题

下面关于埃氏筛法的说法正确的是（ ）。

A. 每个合数只会被筛掉一次

B. 从每个素数出发，把它的倍数标记为合数

C. 只能判断一个数是不是偶数

D. 不能求出素数表

答案：B

解析：本题选 B。埃氏筛法的基本思想是：从小到大找到还没有被标记的素数，然后把这个素数的倍数标记为合数，最后未被标记的数就是素数。A 是线性筛更强调的性质，埃氏筛中一个合数可能被多个质因子重复标记，例如 `12` 可能被 `2` 和 `3` 标记。C 错在埃氏筛能判断和筛出所有范围内的素数，不只是偶数。D 与算法用途相反，埃氏筛正是常用来求素数表的。

## 第7题

下面代码实现了计算 $x^n$ 的快速幂算法，该算法体现的编程思想是（ ）。

```cpp
long long power(long long x, int n) {
    if (n == 0) return 1;
    long long res = power(x, n / 2);
    if (n % 2 == 0) return res * res;
    else return res * res * x;
}
```

A. 枚举

B. 贪心

C. 分治

D. 模拟

答案：C

解析：本题选 C。快速幂把计算 $x^n$ 的问题转化为先计算 $x^{\lfloor n/2 \rfloor}$，再根据 `n` 的奇偶性合并结果：偶数次幂为 `res * res`，奇数次幂为 `res * res * x`。这是把大问题拆成较小的同类子问题，再合并子问题答案，体现的是分治思想。A 枚举通常是逐个尝试所有情况；B 贪心是每步选当前最优；D 模拟是按题意过程一步步执行，都不符合这段代码的核心思想。

## 第8题

下面代码用于统计 `n` 中因子 `2` 出现了多少次。若 `n = 40`，输出是（ ）。

```cpp
int n = 40;
int cnt = 0;
while (n % 2 == 0) {
    cnt++;
    n /= 2;
}
cout << cnt;
```

A. `1`

B. `2`

C. `3`

D. `4`

答案：C

解析：本题选 C。程序每发现当前 `n` 能被 `2` 整除，就把 `cnt` 加 `1`，并令 `n /= 2`。从 `40` 开始：`40 -> 20 -> 10 -> 5`，一共连续除以 `2` 三次，所以输出 `3`。也可以理解为 `40 = 2^3 × 5`，因子 `2` 出现了 3 次。A、B 都少数了除以 `2` 的次数，D 则多算了一次，因为变成 `5` 后已经不能再被 `2` 整除。

## 第11题

下面代码段实现了快速排序的划分操作（以首元素为基准），横线处代码应填入（ ）。

```cpp
int partition(vector<int>& arr, int low, int high) {
    int pivot = arr[low];

        while (i < j && arr[j] >= pivot) j--;
        while (i < j && arr[i] <= pivot) i++;
        if (i < j) swap(arr[i], arr[j]);
    }
    ________________; // 在此处填入代码
    return i;
}
```

A. `swap(arr[low], arr[high])`

B. `swap(arr[low], arr[i])`

C. `swap(arr[i], arr[high])`

## 第13题

在对长度为 $n$（$n \ge 1$）的数组进行归并排序的过程中，`mergeArray` 函数（合并两个有序子数组的操作）被调用的次数是（ ）。

```cpp
const int MAXN = 100005;

int a[MAXN];
int tempArr[MAXN];

void mergeArray(int left, int mid, int right) {
    int i = left;       // 左半部分起点
    int j = mid + 1;    // 右半部分起点
    int k = left;       // 临时数组下标

    while (i <= mid && j <= right) {
        if (a[i] <= a[j]) {
            tempArr[k++] = a[i++];
        } else {
            tempArr[k++] = a[j++];
        }
    }

    while (i <= mid) {
        tempArr[k++] = a[i++];
    }

    while (j <= right) {
        tempArr[k++] = a[j++];
    }

    for (int p = left; p <= right; p++) {
        a[p] = tempArr[p];
    }
}

void mergeSort(int left, int right) {
    if (left >= right) {
        return;
    }

    int mid = left + (right - left) / 2;

    mergeSort(left, mid);
    mergeSort(mid + 1, right);

    mergeArray(left, mid, right);
}
```

A. $n - 1$

B. $\log n$

C. $n \log n$

D. $2n$

答案：A

解析：本题选 A。归并排序把每个单独元素看作一个已经有序的小数组，然后不断把两个有序部分合并。若开始有 `n` 个单元素部分，最后要合成 1 个完整有序数组，每调用一次 `mergeArray` 都会把两个部分合成一个部分，部分数量减少 1。因此总共需要减少 `n - 1` 次，`mergeArray` 被调用 `n - 1` 次。B 是归并排序递归层数的量级。C 是归并排序总时间复杂度的量级。D 与合并次数不对应。

## 第15题

高精度减法中，假设两个高精度数按低位在前存储，且已经保证被减数不小于减数。下面处理借位逻辑代码中横线处应填入（ ）。

```cpp
if (a[i] < b[i]) {
    a[i + 1]--;
    ________________;
}
t = a[i] - b[i];
```

A. `a[i] += 10`

B. `a[i] -= 10`

C. `b[i] += 10`

D. `a[i] = a[i+1] + 10`

答案：A

解析：本题选 A。按低位在前做高精度减法时，如果当前位 `a[i] < b[i]`，就需要向更高一位借 1。借位后，高一位 `a[i + 1]--`，当前位相当于增加 10，所以应执行 `a[i] += 10`，再计算 `t = a[i] - b[i]`。B 会让当前位更小，无法完成减法。C 修改的是减数当前位，方向错误。D 把当前位直接改成高一位加 10，会丢掉当前位原来的数值，不是借位规则。

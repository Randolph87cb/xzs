## 第1题

又到期末考试周，小明发现这次许多闭卷考试不仅禁止携带手机、平板电脑，还有最近比较时髦的各类 AI 眼镜（也有叫智能眼镜）也同样不允许带入考场。这些 AI 眼镜应该也是内置了操作系统并可能支持 Wi-Fi 或蓝牙连接。

答案：A

解析：本题判断为正确。AI 眼镜属于智能设备，通常需要系统软件来管理摄像头、显示、通信、电源等硬件，也可能通过 Wi-Fi 或蓝牙和其他设备连接。因此说它们内置操作系统并可能支持无线连接是合理的。

## 第2题

如果 `n` 为大于 100 的整数，则 C++ 表达式 `(n / 10) % 10` 与 `(n % 100) / 10` 的结果相同。

答案：A

解析：本题判断为正确。在整数运算中，`n / 10` 会去掉个位，再对 10 取余，得到原数的十位数字；`n % 100` 会留下后两位，再除以 10，也得到十位数字。例如 `n = 123` 时，两式都得到 `2`，所以结果相同。

## 第3题

下面 C++ 代码执行后将输出 `19`。

```cpp
int i;
for (i = 0; i < 20; i += 3)
    if (i % 3 != 0)
        break;
cout << i;
```

答案：B

解析：本题判断为错误。循环中 `i` 的值依次为 `0, 3, 6, 9, 12, 15, 18`，这些数都能被 3 整除，所以 `i % 3 != 0` 一直不成立，不会执行 `break`。当 `i` 从 18 再加 3 变成 21 时，循环条件 `i < 20` 不成立，循环结束，最后输出的是 `21`，不是 `19`。

## 第4题

在数学中，`N!` 称之为 `N` 的阶乘，其含义是 1 到 `N` 之积，包括 `N`。如 `3! = 1 × 2 × 3 = 6`。如下 C++ 代码能输出 `N!` 的结果。

```cpp
int N;
cout << "请输入正整数：";
cin >> N;
int rst = 0;
for (int i = 1; i < N + 1; i++)
    rst *= i;
cout << rst;
```

答案：B

解析：本题判断为错误。求阶乘时需要从 `1` 开始不断相乘，但代码中 `rst` 一开始被赋值为 `0`。无论后面乘以多少个数，`0 * i` 的结果仍然是 `0`，所以程序会输出 `0`，不能正确输出 `N!`。正确做法应把 `rst` 初始化为 `1`。

## 第5题

如下 C++ 代码能实现输出正整数 `N` 的各位数字。

```cpp
int N;
cin >> N;
while (N > 0) {
    cout << (N % 10) << endl;
    N = N / 10;
}
```

答案：A

解析：本题判断为正确。`N % 10` 可以取出当前数字的个位，`N = N / 10` 会去掉当前个位。循环不断重复这个过程，就能把正整数 `N` 的每一位数字输出出来。需要注意的是，这段代码的输出顺序是从个位到最高位。

## 第6题

将如下 C++ 代码中的 `printf("\n")` 更换为 `printf("%c", '\n')`，输出效果相同。

```cpp
for (int i = 0; i < 100; i++) {
    if (i % 5 == 0)
        printf("\n");
    printf("%2d ", i);
}
```

答案：A

解析：本题判断为正确。`printf("\n")` 会直接输出一个换行符，`printf("%c", '\n')` 是按照字符格式输出字符 `'\n'`，输出的也是换行符。因此两种写法在这里都会产生换行，输出效果相同。

## 第7题

C++ 代码 `cout << (2.5 % 2)` 能够生成可执行代码，但执行时将报错，因为 `%` 只能适用于正整数。

答案：B

解析：本题判断为错误。`%` 是求余运算符，在 C++ 中要求两边的操作数是整数类型；`2.5` 是小数，不能和 `%` 这样搭配使用。因此这段代码通常在编译阶段就会报错，不能生成可执行代码。题干还说“`%` 只能适用于正整数”也不准确，`%` 的关键限制是整数类型。

## 第8题

下面的 C++ 代码执行后将输出 `24`。

```cpp
int total;
for (int n = 1; n <= 4; n++) {
    total *= n;
}
cout << total;
```

答案：B

解析：本题判断为错误。局部整型变量 `total` 只声明了，但没有初始化，它开始时的值是不确定的。后面执行 `total *= n` 时，是在一个不确定的初始值上继续相乘，结果不能保证是 `24`。如果想计算 `1 × 2 × 3 × 4`，应先写 `int total = 1;`。

## 第9题

如果 `a` 是 C++ 的整型变量，值为 `4`，则表达式 `(a % 2 == 0) + (a / 2)` 的值为 `4`。

答案：B

解析：本题判断为错误。当 `a = 4` 时，`a % 2 == 0` 成立，布尔值 `true` 参与加法时会当作整数 `1`；`a / 2` 是整数除法，结果为 `2`。所以整个表达式的值是 `1 + 2 = 3`，不是 `4`。

## 第10题

删除下面 C++ 代码中的 `continue` 后，其输出与原来相同。

```cpp
int x = 0;
while (x < 4) {
    x++;
    if (x == 2) continue;
    cout << x << ",";
}
```

答案：B

解析：本题判断为错误。原代码中，当 `x` 变成 `2` 时会执行 `continue`，直接进入下一轮循环，所以不会输出 `2,`，原输出为 `1,3,4,`。如果删除 `continue`，`x == 2` 时也会继续执行下面的输出语句，输出会变成 `1,2,3,4,`。两者输出不同。

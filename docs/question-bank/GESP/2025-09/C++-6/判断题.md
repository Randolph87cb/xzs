## 第7题

以下代码实现了二叉树的中序遍历。输入以下二叉树，中序遍历结果是 4 2 5 1 3 6 。

```cpp
//       1
//      / \
//     2   3
//    / \   \
//   4   5   6

struct TreeNode {
    int val;
    TreeNode* left;
    TreeNode* right;
    TreeNode(int x) : val(x), left(nullptr), right(nullptr) {}
};

void inorderIterative(TreeNode* root) {
    stack<TreeNode*> st;
    TreeNode* curr = root;

    while (curr || !st.empty()) {
        while (curr) {
            st.push(curr);
            curr = curr->left;
        }
        curr = st.top(); st.pop();
        cout << curr->val << " ";
        curr = curr->right;
    }
}
```

答案：正确

解析：题干表述成立。输入输出题要注意格式控制符、转义字符、整数除法和输出语句中字面量与变量的区别。 将题干条件代入相关规则后不会产生矛盾，因此判断为正确。



## 第8题

下面代码实现的二叉排序树的查找操作时间复杂度为 $O(h)$，其中 $h$ 为树高。

```cpp
TreeNode* searchBST(TreeNode* root, int val) {
    while (root && root->val != val) {
        root = (val < root->val) ? root->left : root->right;
    }
    return root;
}
```

答案：正确

解析：题干表述成立。排序题要区分算法性质：快速排序平均 $O(n\log n)$ 但通常不稳定，归并排序稳定且合并有序子区间，冒泡和插入排序通常稳定，选择排序通常不稳定。 将题干条件代入相关规则后不会产生矛盾，因此判断为正确。

## 第10题

有一排香蕉，每个香蕉有不同的甜度值。小猴子想吃香蕉，但不能吃相邻的香蕉。以下代码能找到小猴子吃到最甜的香蕉组合。

```cpp
// bananas: 香蕉的甜度
void findSelectedBananas(vector<int>& bananas, vector<int>& dp) {
    vector<int> selected;
    int i = bananas.size() - 1;

    while (i >= 0) {
        if (i == 0) {
            selected.push_back(0);
            break;
        }

        if (dp[i] == dp[i-1]) {
            i--;
        } else {
            selected.push_back(i);
            i -= 2;
        }
    }

    reverse(selected.begin(), selected.end());
    cout << "小猴子吃了第: ";
    for (int idx : selected)
        cout << idx+1 << " ";
    cout << "个香蕉" << endl;
}

int main() {
    vector<int> bananas = {1, 2, 3, 1}; // 每个香蕉的甜
    vector<int> dp(bananas.size());
    dp[0] = bananas[0];
    dp[1] = max(bananas[0], bananas[1]);
    for (int i = 2; i < bananas.size(); i++) {
        dp[i] = max(bananas[i] + dp[i-2], dp[i-1]);
    }
    findSelectedBananas(bananas, dp);
    return 0;
}
```

答案：正确

解析：题干表述成立。计数题要先区分是否有顺序、是否可重复、是否互斥，再用乘法原理、加法原理或组合数计算。 将题干条件代入相关规则后不会产生矛盾，因此判断为正确。

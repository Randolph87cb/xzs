## 第1题

人工智能现在非常火，小杨就想多了解一下，其中就经常听人提到“大模型”。那么请问这里说的“大模型”最贴切是指（ ）。

A. 大电脑模型
B. 大规模智能
C. 智能的单位
D. 大语言模型

答案：D

解析：“大模型”在当前人工智能语境中通常指参数规模很大、能处理自然语言等任务的模型，本题给出的最贴切选项是“大语言模型”。“大电脑模型”把重点误解成硬件大小，“大规模智能”和“智能的单位”都不是常用的技术概念。



## 第2题

在 TCP 协议中，完成连接建立需要通过（ ）握手。

A. 一次
B. 二次
C. 三次
D. 四次

答案：C

解析：TCP 建立连接采用“三次握手”：客户端发起连接请求，服务器确认并回应，客户端再确认一次。这样双方才能确认收发能力和初始序号，一次或二次都不足以完成可靠确认，四次不是建立 TCP 连接的标准过程。



## 第3题

下面的 C++ 代码用于输入姓名，然后输出姓名，正确的说法是（ ）。

```cpp
string XingMing;
cout << "请输入您的姓名：";
cin >> XingMing;
cout << XingMing;
```

A. `XingMing` 是汉语拼音，不能作为变量名称
B. 可以将 `XingMing` 改为 `Xing Ming`
C. 可以将 `XingMing` 改为 `xingming`
D. 可以将 `XingMing` 改为 `Xing-Ming`

答案：C

解析：C++ 变量名可以由字母、数字和下划线组成，且不能以数字开头；`XingMing` 和 `xingming` 都是合法变量名，只要声明和后续使用同步修改即可。拼音字母可以作为变量名，空格会把标识符拆开，连字符 `-` 会被当作减号运算符，因此 B、D 不是合法变量名写法。



## 第4题

下面 C++ 代码用于获得正整数 $N$ 的第 $M$ 位数，如 $N$ 等于 $1234$，$M$ 等于 $2$，则输出 $3$。此题假设 $M$ 的值大于等于 $1$ 且小于等于 $N$ 的位数。横线处应填入的代码是（ ）。

```cpp
int N, M, div=1;
cout << "请输入一个正整数：";
cin >> N;
cout << "请输入从右到左取第几位数：";
cin >> M;
for (int i = 0; i < (M - 1); i++) div *= 10;
cout << (_____________);
```

A. `N % div / 10`
B. `N / div / 10`
C. `N % div % 10`
D. `N / div % 10`

答案：D

解析：循环结束后 `div = 10^{M-1}`，先用 `N / div` 去掉目标位右边的 $M-1$ 位，再对 $10$ 取余就得到这一位。例如 $N=1234$、$M=2$ 时，`div=10`，`1234 / 10 % 10 = 123 % 10 = 3`。使用 `N % div` 会只保留目标位右边的数字，无法取到目标位。



## 第5题

某种编号的规则是“XX-Y”，其中 XX 从 00 到 11，Y 从 0 到 9。第 1 个编号是 00-0，第 2 个编号是 01-1，…，第 12 个编号 11-1，第 13 个编号 00-2，即其编码规则是 XX 和 Y 同时增 1，到 XX 到 11 时下一个变为 00，Y 到 9 时，下一个变为 0。下面的 C++ 代码用于生成第 N 个编号，横线处应填上的代码是（ ）。

```cpp
cout << "请输入编号位置：";
cin >> N;
part1 = N % _____;
part2 = N % _____;
if (part1 < 10)
    printf("0%d-%d\n", part1, part2);
else
    printf("%d-%d\n", part1, part2);
```

A. 12 10
B. 10 10
C. 11 9
D. 9 9

答案：A

解析：`XX` 的取值从 `00` 到 `11`，一共 $12$ 种，超过后回到 `00`，所以第一空要按 $12$ 取余；`Y` 的取值从 `0` 到 `9`，一共 $10$ 种，超过后回到 `0`，所以第二空要按 $10$ 取余。其他选项把循环长度写成 $10$、$11$ 或 $9$，都会提前或延后回绕，无法覆盖完整编号范围。



## 第6题

下面的 C++ 代码执行后其输出是（ ）。

```cpp
cnt = 0;
for (int i = -10; i < 10; i++)
    for (int j = 0; j < i; j++)
        cnt += 1;
cout << cnt;
```

A. 145
B. 125
C. 55
D. 45

答案：D

解析：外层 `i` 从 $-10$ 到 $9$。当 `i <= 0` 时，内层循环初始 `j = 0`，条件 `j < i` 为假，不会计数；当 `i` 为 $1$ 到 $9$ 时，内层分别执行 $i$ 次。因此 `cnt = 1 + 2 + ... + 9 = 45`。



## 第7题

下面 C++ 代码执行后其输出是（ ）。

```cpp
for (i = 1; i < 12; i++) {
    if (i % 2 == 0)
        continue;
    for (j = 0; j < i; j++)
        if (i * j % 2)
            break;
}
if (i >= 12)
    cout << (i * j);
```

A. 110
B. 12
C. 不确定
D. 无输出

答案：B

解析：外层循环结束时，`i` 会从 $11$ 自增到 $12$，所以 `if (i >= 12)` 会执行。对奇数 `i`，内层从 `j = 0` 开始，`i * 0 % 2` 为 $0$，到 `j = 1` 时乘积为奇数，于是 `break`；最后一次有效的奇数是 `i = 11`，留下 `j = 1`。因此输出 `i * j = 12 * 1 = 12`。



## 第8题

阅读下面的 C++ 代码，其中变量都是整型，则说法正确的是（ ）。

```cpp
cin >> a >> b;
while (b != 0) {
    remainder = a % b;
    a = b;
    b = remainder;
}
cout << a;
```

A. `b` 不能为 $0$，因为 `a % b` 将导致错误
B. `a` 必须小于 `b`，否则 `a % b` 将导致错误
C. `a` 和 `b` 都必须为正整数，否则 `a % b` 将导致错误
D. 如果 `a` 输入为 $0$，则不管 `b` 的输入值是什么，输出值的绝对值都是 `abs(b)`

答案：D

解析：这段代码是辗转相除法的结构。若一开始 `a = 0` 且 `b != 0`，第一次循环得到 `remainder = 0 % b = 0`，随后 `a` 变成原来的 `b`，`b` 变成 $0$，输出的就是原来的 `b`，其绝对值为 `abs(b)`；若一开始 `b = 0`，循环不进入，输出 $0$，也满足。`b` 初始为 $0$ 不会触发取模，`a` 大于 `b` 也可以取模，负整数只要除数不为 $0$ 也不是语法错误。



## 第9题

下面 C++ 代码执行后输出是（ ）。

```cpp
num = 0;
while (num <= 5) {
    num += 1;
    if (num == 3)
        continue;
    printf("%d#", num);
}
```

A. 1#2#4#5#6#
B. 1#2#4#5#6
C. 1#2#3#4#5#6#
D. 1#2#3#4#5#6

答案：A

解析：每轮循环先执行 `num += 1`，所以实际检查和输出的是 $1$ 到 $6$。当 `num == 3` 时执行 `continue`，跳过本轮的 `printf`，其余数字都会按 `"%d#"` 输出并带上井号。因此输出为 `1#2#4#5#6#`，最后也有一个 `#`。



## 第10题

下面 C++ 代码用于记录多个输入数中的最大数和最小数（输入 $-999$ 则输入结束），相关说法错误的是（ ）。

```cpp
cin >> now_num;
min_num = max_num = now_num;

while (now_num != -999){
    if (max_num < now_num)
        max_num = now_num;

    if (min_num > now_num)
        min_num = now_num;

    cin >> now_num;
}
cout << min_num << ' ' << max_num;
```

A. 程序运行时如果第一个数输入 $-999$，则输出将是 $-999$ $-999$
B. 程序输入过程中，如果输入的第一个数不是 $-999$，则如果待输入的数据中没有 $-999$，则程序能求出已输入整数中的最大数和最小数
C. 如果用于输入考试成绩，即成绩中不可能有 $-999$，则程序能求出已输入成绩中的最高成绩和最低成绩
D. 可以将 `cin >> now_num;` 移动到 `while (now_num != -999) {` 下面，结果不变

答案：D

解析：原程序先读入第一个数，用它初始化最大值和最小值，然后在每次循环末尾读取下一个数，这样 `while` 条件检查到 `-999` 时不会把它参与统计。若把循环末尾的 `cin >> now_num;` 移到 `while` 下面，程序会在进入循环体后先读新数，读到 `-999` 时仍可能继续执行比较，从而把结束标记当成普通数据，结果会改变。因此 D 的说法是错误的。



## 第11题

下面 C++ 代码执行后输出与 5 有关数的数量。“与 5 有关的数”定义为含有 5 或者能被 5 整除的数。相关说法正确的是（ ）。

```cpp
cnt = 0;
for (i = 1; i < 1000; i++){
    if (i % 5 == 0){
        cnt += 1;
        continue;
    }

    j = i;
    while (j > 0)
        if (j % 10 == 5){
            cnt += 1;
            break;
        }
        else
            j /= 10;
}
cout << cnt;
```

A. 删除代码中 `continue` 不影响程序执行结果
B. 删除 `j = i` 并将 `while` 循环内的 `j` 修改为 `i`，不影响程序执行结果
C. 代码中 `break` 修改为 `j = 0`，不影响程序执行结果
D. 将 `while (j > 0)` 修正为 `while (j >= 0)` 不影响程序执行的结果

答案：C

解析：当某一位数字为 $5$ 时，程序已经把 `cnt` 加 $1$，随后只需要退出检查这个数的 `while` 循环。把 `break` 改成 `j = 0` 后，下一次判断 `while (j > 0)` 为假，也会结束当前数字检查，所以计数结果不变。删除 `continue` 会让能被 $5$ 整除且含有数字 $5$ 的数被重复统计；直接修改外层变量 `i` 会破坏 `for` 循环；把条件改成 `j >= 0` 会在 `j` 变成 $0$ 后无法结束。



## 第12题

下面 C++ 代码实现输出如下图形，应该在横线处填入的代码是（ ）。

```
请输入层数：10
         1
        23
       456
      7891
     23456
    789123
   4567891
  23456789
 123456789
1234567891
```

```cpp
cout << "请输入层数：";
cin >> N;
K = 1;
for (i = 1; i < N + 1; i++){
    for (int _ = 1; _ < __________; _++)
        cout << " ";
    for (int _ = 1; _ < __________; _++){
        cout << K;
        K += 1;
        if (K == 10)
            K = 1;
    }
    cout << '\n';
}
```

A.

```cpp
N - i + 1
i + 1
```

B.

```cpp
N - i
i
```

C.

```cpp
N
i
```

D.

```
N - i
i + 1
```

答案：A

解析：第 `i` 行需要先输出 `N - i` 个空格，再输出 `i` 个数字。循环写法是 `for (int _ = 1; _ < 上界; _++)`，实际执行次数比上界少 $1$，所以空格循环的上界应为 `N - i + 1`，数字循环的上界应为 `i + 1`。选项 A 正好对应这两个次数，且 `K` 到 $10$ 后重置为 $1$，可以得到图中的连续数字效果。



## 第13题

下面 C++ 代码执行，其输出是（ ）。

```cpp
int a=9, b=27;
a = 'a' + 'b';
b = 'a' - 'b';
a = a - b;
cout << a << ' ' << b << endl;
```

A. 196 -1
B. 27 9
C. 98 97
D. 不确定

答案：A

解析：字符参与算术运算时会按字符编码转成整数。在常用 ASCII 编码中，`'a'` 是 $97$，`'b'` 是 $98$，所以先得到 `a = 97 + 98 = 195`，`b = 97 - 98 = -1`。接着 `a = a - b = 195 - (-1) = 196`，最终输出 `196 -1`。



## 第14题

有个无限长的链，由 3 种外形相同但材质不同的环链成。3 种环的重量分别是 $3$、$4$、$6$ 克，相同材质的多个环每 $12$ 克一组，分别记为 G3、G4、G6。链依次 G3、G4、G6、G3、G4、G6、...。同时对链上所有环从头依次编号 $1$、$2$、$3$、$4$……。输入正整数代表环编号，求该编号前所有环（不含该环本身）的重量。下面是 C++ 代码是实现，正确说法是（ ）。


```cpp
int N, G, R;
int wc = 0;

cin >> N;  // 输入正整数

G = (N - 1) / 9; // L1
R = (N - 1) % 9; // L2：保存余数
wc += 36*G;

if((1 <= R) && (R <= 4))
    wc += 3*R; // L3
else if ((5 <= R) && (R <= 7))
    wc += 4*R; // L4
else if(R == 8)
    wc += 6*(R - 1); // L5
cout << wc << endl;
```

A. 必须同时修改 L1 和 L2 代码行才能实现功能
B. 必须同时修改 L3 和 L4 代码行才能实现功能
C. 必须同时修改 L3 和 L5 代码行才能实现功能
D. 其他说法都不对

答案：D

解析：每个 `G3` 有 $4$ 个 $3$ 克环，每个 `G4` 有 $3$ 个 $4$ 克环，每个 `G6` 有 $2$ 个 $6$ 克环，因此一轮共有 $9$ 个环、重量 $36$ 克，`L1` 和 `L2` 用 `(N - 1) / 9`、`(N - 1) % 9` 划分完整轮数和剩余环数是合理的。剩余部分中，`R = 1..4` 时重量是 `3 * R`；`R = 5..7` 时应为 $12 + 4(R - 4)$，不是 `4 * R`；`R = 8` 时应为 $12 + 12 + 6 = 30$，不是 `6 * (R - 1)`。真正需要改的是 `L4` 和 `L5`，选项 A、B、C 都没有准确描述，所以选 D。

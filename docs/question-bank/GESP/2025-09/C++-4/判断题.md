## 第1题

以下代码能正确初始化指针。

```cpp
int a = 5;
int *p = a;
```

答案：错误

## 第2题

执行下面 C++ 代码将输出 $11$。

```cpp
int x = 10;
void f() {
    int x = x + 1;
    cout << x << endl;
}

int main() {
    f();
}
```

答案：错误

## 第3题

以下 C++ 代码合法。

```cpp
struct Student {
    string name;
    int age;
    float score;
};
Student* students = new Student[20];
```

答案：正确

## 第4题

执行下面 C++ 代码将输出 $10$。

```cpp
void func(int* p) {
    *p = 10;
}

int main() {
    int a = 5;
    func(&a);
    cout << a << endl;
    return 0;
}
```

答案：正确

## 第5题

下面代码将二维数组 `arr` 传递给函数 `f`，函数内部用 `arr[i][j]` 访问元素，函数参数声明为 `int arr[][4]` 是错误的。

```cpp
void f(int arr[][4], int rows) {
    // 访问 arr[i][j]
}

int main() {
    int arr[3][4] = { /* 初始化 */ };
    f(arr, 3);
}
```

答案：错误

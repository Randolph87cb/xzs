# 判断题

来源：
- 洛谷有题页：https://ti.luogu.com.cn/problemset/1196
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1703973115396128.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

C++ 语言中，表达式 `'9' ^ 3` 的结果值为 `'999'`。

答案：错误

## 第2题

下列 C++ 语言代码，能够安全地输出 `arr[5]` 的值。

```cpp
int n = 5;
int arr[n] = {1, 2, 3};
std::cout << arr[5];
```

答案：错误

## 第3题

对 $n$ 个元素的数组进行排序，最差情况的时间复杂度为 $O(n^2)$。

答案：错误

## 第4题

有 4 个红球、3 个蓝球和 2 个绿球排成一排（相同色球视为完全相同），则不同的排列方案数为 1260 种。

答案：正确

## 第5题

使用 `math.h` 或 `cmath` 头文件中的函数，对于 `int` 类型的变量 $x$，表达式 `fabs(x)` 和 `sqrt(x * x)` 的结果总是近似相等的。

答案：错误

## 第6题

运算符重载是 C++ 语言静态多态的一种典型体现，而使用 C 语言则无法实现运算符重载。

答案：正确

## 第7题

存在一个简单无向图满足：顶点数为 6，边数为 8，6 个顶点的度数分别为 3、3、3、3、2、2。

答案：正确

## 第8题

已知两个 `double` 类型的变量 $r$ 和 $theta$ 分别表示一个扇形的圆半径及圆心角（弧度），则扇形的周长可以通过表达式 `(2 + theta) * r` 求得。

答案：正确

## 第9题

Dijkstra 算法的时间复杂度为 $O(V^2)$，其中 $V$ 为图中顶点的数量。

答案：错误

## 第10题

从 32 名学生中选出 2 人分别担任男生班长和女生班长（男生班长必须是男生，女生班长必须是女生），则共有 $C(32, 2) / 2$ 种不同的选法。

答案：错误

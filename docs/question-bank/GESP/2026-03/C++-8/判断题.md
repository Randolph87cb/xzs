# 判断题

来源：
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1734124729532448.pdf

说明：本文件依据官网 PDF 逐题视觉核对整理。判断题答案中，`A` 表示正确，`B` 表示错误。

## 第1题

题干：在 C++ 中，若结构体中包含一个 `static` 成员变量，则该变量的存储空间属于结构体对象的一部分。

答案：B

解析：本题判断为错误。`static` 成员变量属于结构体或类本身，而不是某个具体对象的一部分。每个对象中不会各自保存一份该静态成员，所有对象共享同一个静态成员变量。正确说法是：非静态数据成员通常占用对象自身的存储空间，而静态成员变量需要在类外定义并拥有独立的静态存储空间。

## 第2题

题干：对于任意正整数 $n$，二项式 $(a+b)^n$ 展开式中各项的二项式系数之和等于 $2^n$。

答案：A

解析：本题判断为正确。二项式 $(a+b)^n$ 的展开式为 $\sum_{k=0}^{n} \binom{n}{k}a^{n-k}b^k$，其中各项的二项式系数是 $\binom{n}{0},\binom{n}{1},\ldots,\binom{n}{n}$。令 $a=1,b=1$，左边变为 $(1+1)^n=2^n$，右边正好是所有二项式系数之和，因此结论成立。

## 第3题

题干：在 C++ 中，若函数参数类型为 `const int &`，则该参数既可以绑定左值，也可以绑定右值。

答案：A

解析：本题判断为正确。普通非常量左值引用 `int &` 只能绑定可修改的左值，而常量左值引用 `const int &` 既可以绑定左值，也可以绑定临时对象、字面量或表达式结果等右值。绑定右值时，临时对象的生命周期会延长到该引用的作用域结束，因此函数参数写成 `const int &` 可以接收如变量 `x`、字面量 `3` 或表达式 `x + 1` 等实参。

## 第4题

题干：若一个无向图的最小生成树唯一，则图中所有边权必定各不相同。

答案：B

解析：本题判断为错误。边权全部不同可以保证最小生成树唯一，但最小生成树唯一并不反推出所有边权都不同。图中可以存在相同边权，只要这些相同权值的边不会在某个割或环上造成可替换的最小选择，最小生成树仍可能唯一。正确说法是：所有边权互不相同是最小生成树唯一的充分条件，不是必要条件。

## 第5题

题干：使用快速排序对 $n$ 个元素进行排序时，无论最好、最坏还是平均情况，时间复杂度均为 $O(n\log n)$。

答案：B

解析：本题判断为错误。快速排序在划分较均衡时，每一层处理 $O(n)$ 个元素，递归深度约为 $O(\log n)$，最好和平均时间复杂度通常是 $O(n\log n)$。但如果每次枢轴都把序列划分得极不均衡，例如一边为空、另一边只有少一个元素，递归深度会退化为 $O(n)$，总时间复杂度变为 $O(n^2)$。正确说法是：快速排序平均复杂度为 $O(n\log n)$，最坏情况可能为 $O(n^2)$。

## 第6题

题干：若一个图中所有顶点的度数为偶数，则一定存在欧拉回路。

答案：B

解析：本题判断为错误。对于无向图，所有顶点度数为偶数只是存在欧拉回路的重要条件之一，还需要所有有边相连的顶点位于同一个连通分量中。若图不连通，即使每个顶点的度数都是偶数，也无法通过一条闭合路径经过所有边一次。正确说法是：无向图存在欧拉回路通常要求所有非零度顶点连通，并且每个顶点的度数都是偶数。

## 第7题

题干：使用倍增法预处理区间最值问题时，预处理的时间复杂度为 $O(n\log n)$，查询的时间复杂度为 $O(1)$。

答案：A

解析：本题判断为正确。区间最值问题常用 ST 表进行倍增预处理，设 `st[i][j]` 表示从位置 `i` 开始、长度为 $2^j$ 的区间最值。预处理需要枚举每个起点和每个长度层级，层数为 $O(\log n)$，因此时间复杂度为 $O(n\log n)$。查询区间 $[l,r]$ 时，只需取 $k=\lfloor\log_2(r-l+1)\rfloor$，合并两个长度为 $2^k$ 的覆盖区间即可，时间复杂度为 $O(1)$。

## 第8题

题干：如果将一个连通无向图 $G_1$ 中所有边的权值都统一增加同一个正整数常数 $C$，形成图 $G_2$。则 $G_1$ 的最小生成树中每条边在 $G_2$ 中对应的边组成的树，一定是 $G_2$ 的最小生成树。

答案：A

解析：本题判断为正确。连通无向图中任意生成树都恰好包含 $n-1$ 条边。把所有边权统一增加同一个常数 $C$ 后，每棵生成树的总权值都会增加同样的 $(n-1)C$，不同生成树之间的总权值大小关系不会改变。因此，原来在 $G_1$ 中总权值最小的生成树，在 $G_2$ 中仍然总权值最小。

## 第9题

题干：在图论算法中，Kruskal 算法和 Prim 算法都可以用来求解最小生成树，且这两者的贪心策略无论在任何连通无向图上求得的最小生成树总边权和必定相同。

答案：A

解析：本题判断为正确。Kruskal 算法按边权从小到大选择不会形成环的边，Prim 算法从一个点集出发不断选择连接当前点集与外部点的最小边，它们采用的贪心过程不同，但都基于最小生成树的割性质。在同一个连通无向带权图中，最小生成树可能不唯一，具体选出的边集也可能不同；不过所有最小生成树的总边权都等于该图的最小生成树权值。因此两种算法求得的最小生成树总边权和必定相同。

## 第10题

题干：在动态规划问题中，“状态转移方程+递推”和“递归+记忆化搜索”通常是解决同一问题的两种不同实现方式，它们的时间复杂度总是相同的。

答案：B

解析：本题判断为错误。递推和记忆化搜索确实常常是在同一状态定义和同一转移关系下的两种实现方式；如果它们计算的状态集合相同，且每个状态的转移代价也相同，时间复杂度通常相同。但题干说“总是相同”过于绝对：记忆化搜索可能只访问从初始状态可达的部分状态，而递推可能按表格顺序计算全部状态；同时递归调用、边界处理和剪枝方式也可能影响实际计算规模。正确说法是：二者在状态空间和转移代价相同、且访问状态数量相同的情况下，渐进时间复杂度通常相同，但不能一概而论为总是相同。

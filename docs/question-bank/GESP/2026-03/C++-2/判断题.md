## 第1题

题干：小明的妈妈最近刚刚给他买了一块电话手表，除了可以看时间，小明也可以用它和妈妈打电话、收发信息，那么可以推测这块手表中装有一款特定操作系统。（）

答案：正确

解析：电话手表要完成通话、短信、界面显示和硬件控制，背后一定需要系统软件统一管理。这个系统可能是专门的嵌入式系统，也可能是简化版智能设备系统。题目说“装有一款特定操作系统”是合理推测，所以该说法正确。

## 第2题

题干：C++ 语句执行后的输出是 1。（）

```cpp
cout << ('4' % '2' == '2' * '2' % 2);
```

答案：错误

解析：字符 `'4'` 和 `'2'` 在运算时会按字符编码参与计算，常见编码下它们分别是 52 和 50。于是左边 `52 % 50` 的结果是 2，右边 `50 * 50 % 2` 的结果是 0，比较结果为假。`cout` 输出假时显示 `0`，不是 `1`。

## 第3题

题干：下面的 C++ 代码段执行时将报错，因为 10 是整数类型，`a` 是布尔类型。由于类型不同，不能进行加法运算。（）

```cpp
bool a = true;
cout << (10 + a);
```

答案：错误

解析：在 C++ 中，`bool` 可以参与算术运算。`true` 会先转换成整数 `1`，所以表达式 `10 + a` 实际上等于 `11`。这段代码能正常执行并输出 `11`，不会因为类型不同而报错。

## 第4题

题干：下面 C++ 代码段执行后将输出 `0-3-6-9-`。（）

```cpp
for (int i = 0; i < 10; i++){
    if (i % 3)
        continue;
    cout << i << "-";
}
```

答案：正确

解析：当 `i % 3` 不为 0 时，执行 `continue`，本轮不会输出。只有 `i` 是 3 的倍数时才会执行 `cout`，也就是 `0、3、6、9`。因此最终输出正是 `0-3-6-9-`。

## 第5题

题干：执行下面的 C++ 代码段，如果 `N` 是基本数据类型，则语句 `cout << (N);` 将被执行 0 次或无数次（即死循环）。（）

```cpp
cin >> N;
while (N)
    cout << (N);
```

答案：正确

解析：循环条件只检查 `N` 当前是否为 0，但循环体里并没有修改 `N`。如果输入后 `N` 为 0，循环一次也不执行；如果输入后 `N` 非 0，条件会一直成立，程序就会不停输出同一个值。于是执行次数只可能是 0 次或者无限次。

## 第6题

题干：下面的 C++ 代码段可正常执行，删除 `continue` 不影响执行效果。（）

```cpp
for (i = 0; i < 10; i++){
    i += 1;
    continue;
}
cout << (i);
```

答案：正确

解析：`continue` 后面在这个循环体内已经没有别的语句了，所以删掉它不会改变本轮循环的实际行为。每轮先执行 `i += 1`，再执行 `for` 语句自带的 `i++`，所以 `i` 每次增加 2。循环结束后 `i` 变为 10，程序可正常运行。

## 第7题

题干：下面的 C++ 代码段用于计算 1 到 `N` 之间且包含 `N` 的所有数字中含有的 3 的个数，如 1 到 40 含有 14 个 3，而 1 到 20 则只含有 2 个 3。如果将 `while (i != 0)` 改为 `while (abs(i))`，则执行结果相同。（）

```cpp
cout << "请输入正整数N：";
cin >> N;
cnt = 0; // 保存3的个数
for (k = 1; k < N + 1; k++){
    i = k;
    while (i != 0){
        if (i % 10 == 3)
            cnt += 1;
        i /= 10;
    }
}
cout << cnt;
```

答案：正确

解析：这里的 `i` 来自 `k`，而 `k` 从 1 开始递增，所以 `i` 在整个过程中始终是非负整数。对非负整数来说，`i != 0` 与 `abs(i)` 作为循环条件的真假完全一致，都是“非 0 继续、等于 0 停止”。因此把条件改成 `while (abs(i))` 后，统计结果不会变。

## 第8题

题干：下面的 C++ 代码段执行将不会有输出，因为内层循环 `j` 总是 0 开始，`i * j % 10 == 0` 将会被满足，执行 `break`，故而 `i` 小于 10，不会满足 `if` 判断条件。（）

```cpp
for (i = 1; i < 10; i++)
    for (j = 0; j < i; j++)
        if (i * j % 10 == 0)
            break;
if (i >= 10)
    cout << (i * j);
```

答案：错误

解析：`break` 只会跳出内层 `for (j = 0; j < i; j++)`，不会跳出外层 `for (i = 1; i < 10; i++)`。所以外层循环仍会继续执行到结束，最终 `i` 会变成 10，满足 `if (i >= 10)`。此时 `j` 在最后一次内层循环中为 0，因此程序会输出 `0`，不是“不会有输出”。

## 第9题

题干：下列 C++ 代码执行后将输出 `1#4#9#16#16`。（）

```cpp
cnt = 0;
for (i = 1; i < 5; i++){
    for (j = 1; j < i + 1; j++)
        if (i * j % 10 == 0)
            break;
    if (j >= i + 1)
        cout << (i * j) << "#";
}
if (i >= 5)
    cout << (i * j);
```

答案：错误

解析：当 `i = 1, 2, 3, 4` 时，内层循环都不会提前 `break`，循环结束后 `j` 分别变成 `2, 3, 4, 5`。因此循环内输出的是 `2#6#12#20#`，不是平方数序列。外层循环结束后 `i = 5, j = 5`，最后还会再输出 `25`，所以完整输出是 `2#6#12#20#25`。

## 第10题

题干：下面 C++ 代码执行后输出如左图所示，将 `" %d"` 修改为 `"%3d"` 即可实现右图输出。（）

左图：

```text
 1 2 3 4 5 6 7 8 9
 2 4 6 8 10 12 14 16 18
 3 6 9 12 15 18 21 24 27
 4 8 12 16 20 24 28 32 36
 5 10 15 20 25 30 35 40 45
 6 12 18 24 30 36 42 48 54
 7 14 21 28 35 42 49 56 63
 8 16 24 32 40 48 56 64 72
 9 18 27 36 45 54 63 72 81
```

右图：

```text
  1  2  3  4  5  6  7  8  9
  2  4  6  8 10 12 14 16 18
  3  6  9 12 15 18 21 24 27
  4  8 12 16 20 24 28 32 36
  5 10 15 20 25 30 35 40 45
  6 12 18 24 30 36 42 48 54
  7 14 21 28 35 42 49 56 63
  8 16 24 32 40 48 56 64 72
  9 18 27 36 45 54 63 72 81
```

```cpp
for (i = 1; i < 10; i++){
    for (j = 1; j < 10; j++)
        printf(" %d", i * j);
    printf("\n");
}
```

答案：正确

解析：`" %d"` 中前面的空格是固定输出的，所以每个数前面都会先打印 1 个空格；一位数总宽度变成 2，两位数总宽度变成 3，于是左图里各列会逐渐错开。`"%3d"` 则表示把整数按宽度 3 右对齐输出：一位数前面补 2 个空格，两位数前面补 1 个空格。这样每个乘积都占满 3 列，整张乘法表的列宽一致，因此就能得到右图那样对齐的效果。

## 第1题

如果字符变量 `_1` 的值是字符 $1$,那么 `(int) _1` 的值是? （ ）。

【知识点：ASCII/字符编码】

A. $1$
B. -$1$
C. $49$
D. $+1$ 或者 -$1$

答案：C

## 第2题

a,b 是整型变量，各自有互不相同的初始值。下列程序实现了什么效果（ ）。

```cpp
a = a ^ b;
b = a ^ b;
a = a ^ b;
```

【知识点：位运算】

A. a,b 的值从始至终都没有改变。
B. a,b 的值实现了互换。
C. a,b 的值互换了以后，又还回去了，相当于没有变化。
D. a,b 的值最后和原值不一样，没有任何意义。

答案：B

## 第14题

用枚举法解决“鸡兔同笼问题：头共 $35$ 个，脚共 $94$ 只，求鸡和兔的数量”，以下枚举逻辑最合理的是：（ ）

【知识点：枚举与基础算法】

A. 枚举鸡的数量 `x` （$0 \sim 35$），兔的数量 `y=35-x` ，判断 `2*x + 4*y == 94`
B. 枚举兔的数量 `y` （$0 \sim 94$），鸡的数量 `x=35-y` ，判断 `2*x + 4*y == 94`
C. 枚举所有整数 `x` （$0 \sim 100$）和 `y` （$0 \sim 100$），判断 `x+y==35 && 2*x+4*y==94`
D. 枚举脚的总数 `sum` （$0 \sim 94$），判断 `sum == 94`

答案：A

## 第15题

模拟“字符串加密”：规则为“每个字符 ASCII 码 +3 ，若超过 z（$122$）则从 a 重新开始”，以下代码中正确的条件判断是：（ ）

```cpp
for (int i = 0; i < str.length(); i++) {
    // 需补充条件:
    ________________
    else {
        str[i] += 3;
    }
}
```

【知识点：ASCII/字符编码】

A. `if (str[i]> 122) str[i] = str[i] + 3 - 26;`
B. `if (str[i] > 122) str[i] = str[i] - 26;`
C. `if (str[i] + 3 > 122) str[i] = str[i] - 26;`
D. `if (str[i] + 3 > 'z') str[i] = 'a' + (str[i] + 3 - 'z') - 1;`

答案：D

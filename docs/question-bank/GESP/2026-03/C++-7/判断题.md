# 判断题

来源：
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1734312623865888.pdf

说明：本文件依据官网 PDF 第 5 页渲染截图视觉核对整理。判断题中 `A` 表示正确，`B` 表示错误。

## 第1题

C++ 语言中，表达式 `4 ^ 2` 的结果类型为 `int`，值为 `6`。

答案：A

解析：本题判断为正确。在 C++ 中，`^` 是按位异或运算符，不是乘方运算符。整数 `4` 的二进制可看作 `100`，整数 `2` 的二进制可看作 `010`，按位异或后得到 `110`，也就是十进制的 `6`。两个操作数都是 `int` 类型的整数字面量，因此表达式结果类型也是 `int`。

## 第2题

C++ 中引用可以重新绑定。

答案：B

解析：本题判断为错误。C++ 引用在定义时必须初始化，并且初始化后就一直作为同一个对象的别名，不能再改成引用另一个对象。对引用变量赋值，改变的是它所绑定对象的值，而不是把引用重新绑定到新的对象上。

## 第3题

在 C++ 中，若函数形参为引用类型，则在函数内部对该形参的修改会影响对应的实参。

答案：A

解析：本题判断为正确。引用形参不是实参的一份独立拷贝，而是实参对象的别名。函数内部通过这个引用形参进行赋值或修改成员时，实际修改的就是调用处传入的对象。因此，非 `const` 引用形参常用于让函数直接修改调用者提供的变量。

## 第4题

如果一个最值问题可以用动态规划在多项式时间内求解，那么也一定存在一种贪心策略，可以在多项式时间内求得最优解。

答案：B

解析：本题判断为错误。动态规划和贪心都常用于最优化问题，但适用条件不同。动态规划通常依赖最优子结构和状态转移，把多个可能选择都纳入比较；贪心还需要满足“当前局部最优选择一定能导向全局最优”这样的性质。一个问题能用动态规划在多项式时间内求解，只说明可以通过状态设计和转移求出最优解，不能推出一定存在正确的贪心策略。

## 第5题

使用归并排序对 $n$ 个元素进行排序时，无论最好、最坏还是平均情况，时间复杂度均为 $O(n log n)$。

答案：A

解析：本题判断为正确。标准归并排序每一层都会把当前区间分成两部分递归排序，再用线性时间完成合并。每一层合并总工作量是 $O(n)$，递归层数是 $O(log n)$，所以总时间复杂度为 $O(n log n)$。即使原数组已经有序，标准归并排序通常仍然要完成分治和合并过程，因此最好、最坏和平均情况都是这个量级。

## 第6题

在使用 Dijkstra 算法求单源最短路径时，如果发现某条边被选入从源点出发的最短路径生成树中，那么这条边也一定属于该图的某棵最小生成树。

答案：B

解析：本题判断为错误。最短路径树关心的是从源点到每个顶点的路径长度最短，最小生成树关心的是连通所有顶点的边权总和最小，这两个目标不同。举例来说，三角形图中有边 `s-a=2`、`a-b=2`、`s-b=3`，从源点 `s` 到 `b` 的最短路会直接使用边 `s-b`，因为长度 `3` 小于经过 `a` 的长度 `4`；但最小生成树会选两条权值为 `2` 的边，边 `s-b` 是环中权值最大的边，不会出现在最小生成树中。

## 第7题

在一个带权无向图中，若所有边的权值都不相同，则该图的最小生成树是唯一的。

答案：A

解析：本题判断为正确。在存在生成树的带权无向连通图中，如果所有边权都不相同，就不会出现两条边权完全相同导致的选择并列。根据最小生成树的割性质，每个割上权值最小的跨割边必须被选入最小生成树；也可以用环性质理解，任意环中权值最大的边不可能属于最小生成树。边权互不相同时，这些选择不会产生平局，因此最小生成树唯一。

## 第8题

若所有字符出现频率相同，则哈夫曼编码一定会得到完全二叉树。

答案：B

解析：本题判断为错误。哈夫曼编码构造出的哈夫曼树是一棵每个内部结点通常都有两个孩子的二叉树，但这不等于一定是完全二叉树。只有在字符个数和合并过程刚好使各叶子深度分布非常均匀时，形状才可能接近完全二叉树；当字符个数不是 $2$ 的幂时，等频字符也可能得到不同深度的编码。频率相同只能说明每次合并时有许多并列选择，不能保证最终树形一定满足完全二叉树的结构要求。

## 第9题

使用 `math.h` 或 `cmath` 头文件中的函数，表达式 `sin(90)` 的结果为 `1`。

答案：B

解析：本题判断为错误。C/C++ 数学库中的三角函数使用弧度制参数，而不是角度制参数。`sin(90)` 计算的是 $90$ 弧度的正弦值，并不是 $90°$ 的正弦值，所以结果不会等于 `1`。如果要得到 $90°$ 对应的结果，应传入 $π/2$ 弧度附近的数值，例如 `sin(acos(-1) / 2)`。

## 第10题

在一个无向连通图中，从任意顶点开始进行深度优先遍历，最终得到的 DFS 生成树一定包含图中的所有顶点。

答案：A

解析：本题判断为正确。无向图连通表示任意两个顶点之间都存在路径，因此从任意一个起点出发，总能沿着边逐步到达图中的其他顶点。深度优先遍历虽然访问顺序会受邻接表顺序影响，但只要按规则继续访问未访问的相邻顶点，最终会访问所有顶点。每个非起点顶点第一次被访问时对应的树边组成 DFS 生成树，所以这棵生成树包含图中的所有顶点。

# 判断题

来源：
- 洛谷有题页：https://ti.luogu.com.cn/problemset/1132
- 官网原卷 PDF：https://gesp.ccf.org.cn/101/attach/1580976400498720.pdf

说明：本文件优先采用洛谷结构化数据整理，并以官网 PDF 做交叉校对。

## 第1题

TCP/IP 的传输层的两个不同的协议分别是 UDP 和 TCP。

答案：正确

解析：TCP/IP 协议体系中，传输层最常见、最核心的两个协议就是 TCP 和 UDP。TCP 提供面向连接、可靠传输等机制，UDP 提供无连接、开销较小的报文传输；题干只是在区分这两个传输层协议，表述正确。

## 第2题

5G 网络中，5G 中的 G 表示 Gigabytes/s，其中 1 GB = 1024 MB。

答案：错误

解析：5G 中的 `G` 是 `Generation`，表示“第五代移动通信技术”，不是 `Gigabytes/s`。后半句 `1 GB = 1024 MB` 在常见二进制换算语境下可以成立，但题干把 5G 的含义解释成数据传输单位，整体说法仍然错误。

## 第3题

在面向对象中，类是对象的实例。

答案：错误

解析：面向对象中，类描述对象共有的数据和行为，是创建对象的抽象模板；对象才是类的实例。题干把“类”和“对象”的实例关系说反了，因此错误。

## 第4题

在 C++ 类的定义中，使用 `static` 修饰符定义的静态成员被该类的所有对象共享。

答案：正确

解析：`static` 成员属于类本身，而不是某一个具体对象各自独有的一份。多个对象访问同一个静态数据成员时，访问的是同一份存储；静态成员函数也不依赖某个对象的 `this` 指针。题干说静态成员被所有对象共享，表述正确。

## 第5题

在 C++ 类的定义中，可以定义初始化函数或运算符函数等。

答案：正确

解析：C++ 类中可以定义用于对象初始化的构造函数，也可以定义重载运算符的函数，例如 `operator<`、`operator+`、`operator<<` 等。它们都属于类接口设计的一部分，所以题干说法正确。

## 第6题

DFS 是深度优先算法的英文简写。

答案：正确

解析：DFS 是 `Depth-First Search` 的缩写，中文通常译为“深度优先搜索”或“深度优先算法”。它的特点是沿着一条路径尽量向深处搜索，再回溯处理其他分支，因此题干表述正确。

## 第7题

哈夫曼编码是一种有损压缩算法。

答案：错误

解析：哈夫曼编码是一种基于字符出现频率构造前缀码的无损压缩方法。编码后仍能唯一还原原始符号序列，不会像有损图像、音频压缩那样丢弃信息；把它说成有损压缩算法是错误的。

## 第8题

有些算法或数据结构在 C/C++ 语言中使用指针实现，一个典型的例子就是链表。因此，链表这一数据结构在 C/C++ 语言中只能使用指针来实现。

答案：错误

解析：链表常用指针保存“下一个节点”的地址，但这不是唯一实现方式。也可以用数组存储节点数据，并用下标或整数编号表示 `next`，这种“静态链表”在 C/C++ 中很常见。因此“只能使用指针实现”过于绝对，判断为错误。

## 第9题

如果节点数为 $N$，广度搜索算法的最差时间复杂度为 $O(N)$。

答案：正确

解析：按题干只以节点数 $N$ 衡量搜索空间时，BFS 最坏情况下可能需要把所有 $N$ 个节点都入队、出队并检查一遍，时间复杂度为 $O(N)$。若在一般图中把边数也计入输入规模，更严谨的写法常是 $O(V+E)$；本题按节点规模判断为正确。

## 第10题

二叉搜索树的左右子树也是二叉搜索树。

答案：正确

解析：二叉搜索树的定义本身具有递归性：任一节点的左子树中所有节点值小于该节点，右子树中所有节点值大于该节点，并且左右子树也都要满足二叉搜索树性质。因此二叉搜索树的左右子树仍然是二叉搜索树，题干正确。

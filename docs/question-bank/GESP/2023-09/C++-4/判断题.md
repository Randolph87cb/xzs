## 第1题

在 C++ 语言中，指针变量在逻辑上指向另一个变量在内存中的位置，指针变量本身不占用内存。

答案：错误

解析：指针变量的值表示某个对象的地址，因此可以说它“指向”另一个变量的内存位置；但指针变量本身也是变量，同样需要占用存储空间来保存这个地址。题干后半句说指针变量本身不占用内存，过于绝对，因此错误。

## 第2题

对 $N$ 个元素的数组执行插入排序算法，通常的时间复杂度是 $O(N^2)$。

答案：正确

解析：插入排序通常通过逐个插入元素来维护前缀有序。第 $i$ 个元素最坏可能要和前面约 $i$ 个元素比较、移动，总次数为 $1+2+\cdots+(N-1)$，主导项是 $N^2$，所以通常时间复杂度为 $O(N^2)$。

## 第3题

在 C++ 语言中，每个变量都有其作用域。

答案：正确

解析：作用域指一个名字在程序中可以被访问的代码范围。C++ 中变量可能具有块作用域、函数作用域、命名空间作用域或类作用域等，不同位置定义的变量可见范围不同，因此“每个变量都有其作用域”是正确的。

## 第4题

在 C++ 语言中，在函数调用时，通过引用传递的参数不会复制实际参数，因此不会额外占用内存。

答案：错误

解析：引用传参不会像值传参那样复制一份实参对象，这是前半句的核心意思；但“不会额外占用内存”不能作为保证。实现函数调用时通常仍需要传递地址或建立形参引用绑定，形参和调用过程也可能产生额外存储开销，所以题干结论过于绝对。

## 第5题

在 C++ 语言中，可以通过定义结构体，定义一个新的数据类型。

答案：正确

解析：`struct` 可以把若干成员组合成一个自定义类型，例如 `struct Student { int id; };` 定义后就可以声明 `Student s;`。它不只是声明一个变量，而是在程序中引入了一个新的结构体类型。

## 第6题

在 C++ 语言中，可以定义结构体类型的数组变量，定义结构体时也可以包含数组成员。

答案：正确

解析：结构体类型和内置类型一样，可以作为数组元素类型，例如 `Student a[10];`。结构体内部也可以包含数组成员，例如 `char name[20];`，用来保存一组同类型数据，因此题干两个说法都成立。

## 第7题

如果希望记录 10 个最长为 99 字节的字符串，可以将字符串数组定义为 `char s[10][100];`。

答案：正确

解析：C 风格字符串需要以空字符 `'\0'` 结尾。`char s[10][100];` 表示 10 行、每行 100 个 `char`，每行最多可存放 99 个有效字符再加 1 个结尾空字符，因此可以记录 10 个最长为 99 字节的字符串。

## 第8题

一个可能抛出异常的函数，调用它的位置没有在 try 子句中，会引起编译错误。

答案：错误

解析：C++ 不要求所有可能抛出异常的调用点都必须写在 `try` 子句中。没有匹配的局部 `try-catch` 时，异常会继续向外层调用栈传播；如果最终仍无人处理，程序通常会终止，但这不是编译错误。

## 第9题

`==` 和 `:=` 都是 C++ 语言的运算符。

答案：错误

解析：`==` 是 C++ 中的相等比较运算符，常用于判断两个表达式的值是否相等；`:=` 不是 C++ 运算符。C++ 的赋值运算符是 `=`，不能把其他语言或伪代码中的 `:=` 当作 C++ 语法。

## 第10题

通过使用文件重定向操作，可以将程序中输出到 `cout` 的内容输出到文件中，这是常用的记录程序运行日志的方法之一。

答案：正确

解析：文件重定向可以在运行程序时把标准输出改到文件，例如命令行中使用 `>` 将原本写到 `cout` 的内容保存到指定文件。程序内部仍然写 `cout`，但操作系统把标准输出连接到文件，这确实是记录运行信息的一种常见方式。

## 第14题

```cpp

using namespace std;

int main() {
    int fib[10];
    fib[0] = 0;
    fib[1] = 1;
    for (int i = 2; i < 10; i++)
        fib[i] = fib[i - 1] + fib[i - 2];
    cout << fib[10] << endl;
    return 0;
}
```

B. 5
C. 55
D. 无法确定。

答案：D

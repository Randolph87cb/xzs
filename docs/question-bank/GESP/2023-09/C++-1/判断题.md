## 第1题

早期计算机内存不够大，可以将字库固化在一个包含只读存储器的扩展卡中插入计算机主板帮助处理汉字。

答案：正确

解析：本题判断为正确。早期计算机主机内存容量有限，处理汉字时需要保存大量汉字字形数据。把字库存放在带有只读存储器（ROM）的扩展卡中，可以让计算机读取固定的汉字字形信息，用来支持汉字显示或打印，因此这种做法符合当时的硬件条件。

## 第2题

神威·太湖之光超级计算机是中国自主研制的超级计算机，在全球超级计算机 TOP500 排行榜中多次荣膺榜首。

答案：正确

解析：本题判断为正确。神威·太湖之光是我国研制的超级计算机，曾在全球超级计算机 TOP500 榜单中位居第一，并且不是只出现过一次榜首记录。题干说“自主研制”和“多次荣膺榜首”都符合这一事实。

## 第3题

C++ 表达式 `int(3.14)` 的值为 3。

答案：正确

解析：本题判断为正确。`int(3.14)` 表示把浮点数 `3.14` 转换成 `int` 类型。转换为整数时会去掉小数部分，保留整数部分 `3`，不是四舍五入成 `3`，也不会得到 `4`。

## 第4题

C++ 语句 `cout << (2, 3, "23")` 的输出为 2, 3, 23。

答案：错误

解析：本题判断为错误。括号中的 `(2, 3, "23")` 使用的是逗号运算符，会从左到右计算各个表达式，但整个表达式的结果取最后一个 `"23"`。因此 `cout` 实际只输出 `23`，不会把前面的 `2`、`3` 和逗号一起输出。

## 第5题

C++ 的循环语句 `for (int i = 0; i < 10; i += 2)` 表示 $ i $ 从 0 开始到 10 结束但不包含 10，间隔为 2。

答案：正确

解析：本题判断为正确。这个 `for` 循环先令 `i = 0`，每次循环后执行 `i += 2`，所以 `i` 依次为 `0、2、4、6、8`。条件是 `i < 10`，当 `i` 变成 `10` 时循环结束，因此包含 0，不包含 10，间隔为 2。

## 第6题

C++ 表达式 `('1' + '1')` 的值为 '2'。

答案：错误

解析：本题判断为错误。`'1'` 是字符常量，参与加法时会按字符编码转换成整数计算。在常见编码中 `'1'` 的编码值是 49，所以 `'1' + '1'` 得到的是整数 `98`，而字符 `'2'` 的编码值是 50，二者不是同一个值。

## 第7题

在 C++ 语言中，do-while 循环不可能导致死循环，但 while 有可能。

答案：错误

解析：本题判断为错误。`do-while` 的特点是先执行一次循环体，再判断条件；但如果循环条件一直为真，它同样会一直执行下去。例如 `do { } while (true);` 就是死循环。因此不能说 `do-while` 不可能导致死循环。

## 第8题

在下面的 C++ 代码中，由于循环中的 continue 是无条件被执行，因此将导致死循环。

```cpp
for (int i = 1; i < 10; i++) continue;
```

答案：错误

解析：本题判断为错误。`continue` 在 `for` 循环中会跳过本次循环体后面的语句，然后执行更新表达式 `i++`，再判断 `i < 10`。这里 `i` 会从 1 增加到 10，条件最终变为假，所以循环会正常结束，不会死循环。

## 第9题

在 C++ 代码中，不可以将变量命名为 cout，因为 cout 是 C++ 的关键字。

答案：错误

解析：本题判断为错误。C++ 的关键字是语言规定的保留词，例如 `int`、`for`、`while` 等；`cout` 不是关键字，而是标准库中用于输出的对象名。实际写程序时应避免随意遮蔽标准库名字，但题干把 `cout` 说成关键字是不正确的。

## 第10题

C++ 是一种高级程序设计语言。

答案：正确

解析：本题判断为正确。C++ 使用接近自然语言和数学表达式的语法编写程序，程序通常需要经过编译器转换后才能由计算机执行。相对于机器语言或汇编语言，C++ 属于高级程序设计语言。

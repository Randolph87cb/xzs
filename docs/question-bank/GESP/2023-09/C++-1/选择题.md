## 第1题

我们通常说的“内存”属于计算机中的（ ）。

A. 输出设备
B. 输入设备
C. 存储设备
D. 打印设备

答案：C

解析：本题选 C。内存用来临时保存正在运行的程序和数据，属于计算机的存储设备。输出设备负责把结果呈现给人，如显示器；输入设备负责把信息送入计算机，如键盘；打印设备只是输出设备的一种，都不是“内存”的分类。

## 第2题

以下 C++ 不可以作为变量的名称的是（ ）。

A. redStar
B. RedStar
C. red_star
D. red star

答案：D

解析：本题选 D。C++ 变量名可以由字母、数字和下划线组成，且不能包含空格。`redStar`、`RedStar` 和 `red_star` 都符合命名规则，其中大小写还会被看成不同字符；`red star` 中间有空格，会被拆成两个单词，不能作为一个变量名。

## 第3题

C++ 表达式 `2 - 1 && 2 % 10` 的值是（ ）。

A. 0
B. 1
C. 2
D. 3

答案：B

解析：本题选 B。先算算术运算：`2 - 1` 得到 `1`，`2 % 10` 得到 `2`。在逻辑运算 `&&` 中，非 0 值都表示真，所以表达式相当于“真 && 真”，结果为真；在本题选项中用数值表示就是 `1`。

## 第4题

下面 C++ 代码段执行后的输出是（ ）。

```cpp
int a = 3, b = 4;
cout << "a+b=" << a+b;
```

A. 3+4= 7
B. 3+4=7
C. a+b=7
D. a+b=a+b

答案：C

解析：本题选 C。第一段输出的是字符串常量 `"a+b="`，所以屏幕上会原样出现字母 `a+b=`。后面的 `a+b` 没有引号，是变量 `a` 和 `b` 的加法，`3 + 4 = 7`，因此完整输出为 `a+b=7`。A、B 把变量名当成了数字表达式的一部分，D 则把后面的加法也当成了普通文字。

## 第5题

在 C++ 语言中，int 类型的变量 x、y、z 的值分别为 2、4、6，以下表达式的值为真的是（ ）。

A. x > y || x > z
B. x != z - y
C. z > y + x
D. x < y || !x < z

答案：D

解析：本题选 D。代入 `x = 2`、`y = 4`、`z = 6`：A 中 `x > y` 和 `x > z` 都是假；B 中 `z - y = 2`，所以 `x != z - y` 是假；C 中 `y + x = 6`，`z > 6` 是假。D 中 `x < y` 已经为真，`||` 两边只要有一边为真，整个表达式就为真，所以 D 正确。

## 第6题

对 int 类型的变量 a、b、c，下列语句不符合 C++ 语法的是（ ）。

A. `c += 5;`
B. `b = c % 2.5;`
C. `a = (b = 3, c = 4, b + c);`
D. `a -= a = (b = 6) / (c = 2);`

答案：B

解析：本题选 B。取模运算符 `%` 的左右两边都必须是整数类型，而 `2.5` 是小数，不能写成 `c % 2.5`。A 是合法的复合赋值；C 使用逗号表达式，依次给 `b`、`c` 赋值后再计算 `b + c`；D 的写法虽然不适合作为初学者范例，但从语法形式上可以构成赋值表达式。

## 第7题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int m = 14;
int n = 12;
if (m % 2 == 0 && n % 2 == 0)
    cout << "都是偶数";
else if (m % 2 == 1 && n % 2 == 1)
    cout << "都是奇数";
else
    cout << "不都是偶数或奇数";
```

A. 都是偶数
B. 都是奇数
C. 不都是偶数或奇数
D. 以上说法都不正确

答案：A

解析：本题选 A。`14 % 2` 和 `12 % 2` 都等于 `0`，所以条件 `m % 2 == 0 && n % 2 == 0` 的两部分都为真，程序进入第一个 `if` 分支，输出“都是偶数”。进入第一个分支后，后面的 `else if` 和 `else` 都不会再执行。

## 第8题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int m = 14;
int n = 12;
if (m % 2 && n % 2)
    cout << "都是偶数";
else if (m % 2 == 1 && n % 2 == 1)
    cout << "都是奇数";
else
    cout << "不都是偶数或奇数";
```

A. 都是偶数
B. 都是奇数
C. 不都是偶数或奇数
D. 以上说法都不正确

答案：C

解析：本题选 C。这里第一句条件是 `m % 2 && n % 2`，不是判断“等于 0”。当 `m = 14`、`n = 12` 时，两个余数都是 `0`，在逻辑判断中表示假，所以第一个分支不会执行；第二个条件要求两个余数都等于 `1`，也不成立，最后执行 `else`，输出“不都是偶数或奇数”。

## 第9题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int m = 7;
if (m / 5 || m / 3)
    cout << 0;
else if (m / 3)
    cout << 1;
else if (m / 5)
    cout << 2;
else
    cout << 3;
```

A. 0
B. 1
C. 2
D. 3

答案：A

解析：本题选 A。整数除法中 `7 / 5` 得到 `1`，`7 / 3` 得到 `2`，这两个值都是非 0，在条件判断中都表示真。因此 `m / 5 || m / 3` 为真，程序直接执行第一个分支输出 `0`，后面的分支不会再判断。

## 第10题

下面 C++ 代码段执行后的输出是（ ）。

```cpp
int cnt = 0;
for (int i = 1; i <= 5; i++)
    cnt = cnt + 1;
cout << cnt;
```

A. 1
B. 4
C. 5
D. 10

答案：C

解析：本题选 C。循环变量 `i` 从 1 开始，每次加 1，只要 `i <= 5` 就执行循环体，所以 `i = 1、2、3、4、5` 时共执行 5 次。`cnt` 初始为 0，每执行一次加 1，最终为 `5`。

## 第11题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int tnt = 0;
for (int i = 1; i < 5; i += 2)
    tnt = tnt + i;
cout << tnt;
```

A. 2
B. 4
C. 9
D. 10

答案：B

解析：本题选 B。`i` 从 1 开始，每次增加 2，并且要求 `i < 5`，所以循环中 `i` 只会取 `1` 和 `3`。`tnt` 初始为 0，依次加上 1 和 3，最终输出 `4`。`i = 5` 时不满足 `i < 5`，不会再加 5。

## 第12题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int n = 5;
int cnt = 1;
while (n >= 0) {
    cnt += 1;
    n -= 2;
}
cout << cnt;
```

A. 3
B. 4
C. 6
D. 7

答案：B

解析：本题选 B。开始时 `n = 5`、`cnt = 1`。循环条件 `n >= 0` 在 `n = 5、3、1` 时成立，共执行 3 次，每次 `cnt` 加 1、`n` 减 2。执行后 `cnt` 从 1 变成 4，`n` 变成 -1，循环结束，所以输出 `4`。

## 第13题

下面 C++ 代码用于求正整数的所有因数，即输出所有能整除一个正整数的数。如，输入 10，则输出为 1、2、5、10；输入 12，则输出为 1、2、3、4、6、12；输入 17，则输出为 1、17。在横线处应填入代码是（ ）。

```cpp
int n = 0;
cout << "请输入一个正整数：";
cin >> n;

for (__________________) // 此处填写代码
    if (n % i == 0)
        cout << i << endl;
```

A. `int i = 1; i < n; i + 1`
B. `int i = 1; i < n + 1; i + 1`
C. `int i = 1; i < n; i++`
D. `int i = 1; i < n + 1; i++`

答案：D

解析：本题选 D。要输出正整数 `n` 的所有因数，候选数 `i` 应从 1 一直检查到 `n` 本身，因为 `n` 也一定能整除自己。D 的 `i < n + 1` 等价于 `i <= n`，并且 `i++` 会让 `i` 每轮加 1。A、B 的第三部分 `i + 1` 只是计算表达式，没有修改 `i`；C 只检查到 `n - 1`，会漏掉因数 `n`。

## 第14题

在下列代码的横线处填写（ ），可以使得输出是正整数 1234 各位数字的平方和。

```cpp
int n = 1234, s = 0;
for (; n; n /= 10)
    s += ____________; // 此处填写代码
cout << s << endl;
```

A. `n / 10`
B. `(n / 10) * (n / 10)`
C. `n % 10`
D. `(n % 10) * (n % 10)`

答案：D

解析：本题选 D。循环每轮末尾执行 `n /= 10`，会去掉当前个位数字；因此循环体中应先用 `n % 10` 取出当前个位。题目要求“各位数字的平方和”，所以要把这个个位数字乘以自己，即 `(n % 10) * (n % 10)`。C 只求数字和，A、B 使用 `n / 10`，取到的不是当前个位数字。

## 第15题

执行以下 C++ 语言程序后，输出结果是（ ）。

```cpp
int n = 5, s = 1;
for (; n = 0; n--)
    s *= n;
cout << s << endl;
```

A. 1
B. 0
C. 120
D. 无法确定

答案：A

解析：本题选 A。`for` 的中间部分写的是 `n = 0`，这是赋值，不是判断相等。程序先把 `n` 赋值为 0，然后把这个表达式的值 0 当作循环条件；0 表示假，所以循环体一次也不会执行，`s` 保持初值 `1`，最终输出 `1`。选项 C 的 `120` 是把这段代码误看成求 `5!` 才会得到的结果。

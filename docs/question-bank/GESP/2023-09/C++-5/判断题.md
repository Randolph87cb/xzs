## 第10题

下面的 C++ 代码执行后将输出 0 5 1 6 2 3 4。

```cpp
#include <iostream>
#include <algorithm>
using namespace std;

bool compareModulo5(int a, int b) {
    return a % 5 < b % 5;
}

int main() {
    int lst[7];
    for (int i = 0; i < 7; i++)
        lst[i] = i;

    // 对序列所有元素按compareModulo5结果排序
    sort(lst, lst + 7, compareModulo5);

    for (int i = 0; i < 7; i++)
        cout << lst[i] << " ";
    cout << endl;
    return 0;
}
```

答案：正确

解析：本题判断为正确。按控制语句语义判断：if 可以没有 else，while/for 先判断条件，do while 先执行循环体一次，嵌套语句可用大括号明确范围。

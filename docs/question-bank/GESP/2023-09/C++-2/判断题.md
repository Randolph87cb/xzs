## 第1题

我们常说的互联网（Internet）是一个覆盖全球的广域网络，它不属于任何一个国家。

答案：正确

解析：互联网是把世界各地许多计算机网络连接起来的广域网络，使用统一的通信协议进行互联。它由不同国家、地区、机构和个人共同接入与维护，不归某一个国家单独所有，所以题干说法正确。

## 第2题

神威·太湖之光超级计算机是中国自主研制的超级计算机，在全球超级计算机TOP500排行榜中多次荣膺榜首。

答案：正确

解析：神威·太湖之光是我国自主研制的超级计算机，曾在 TOP500 超级计算机排行榜中连续多期位列第一。题干同时说到“中国自主研制”和“多次荣膺榜首”，这两点都符合事实，所以判断为正确。

## 第3题

C++ 表达式 `7.8 / 2` 的值为 $3.9$，类型为 float。

答案：错误

解析：`7.8` 在 C++ 中默认是 `double` 类型浮点常量，`2` 会在运算时转换为 `double` 后再相除，数值结果是 `3.9`。但表达式结果类型是 `double`，不是 `float`，所以题干整体为错误。

## 第4题

C++ 表达式 `(2 * 3) || (2 + 5)` 的值为 $67$。

答案：错误

解析：`2 * 3` 的值是 `6`，`2 + 5` 的值是 `7`，二者都是非零值。在逻辑或 `||` 中，非零会被当作真，`真 || 真` 的结果为真，作为数值通常是 `1`，不是把 `6` 和 `7` 拼成 `67`。

## 第5题

如果 m 和 n 为 int 类型变量，则执行 `for (m = 0, n = 1; n < 9; ) n = ((m = 3 * n, m + 1), m - 1);` 之后 n 的值为偶数。

答案：正确

解析：循环开始时 `n = 1`。循环体中先执行 `m = 3 * n`，再通过逗号表达式取外层的 `m - 1` 赋给 `n`，所以每轮相当于把 `n` 变成 `3 * n - 1`。`n` 的变化为 `1 -> 2 -> 5 -> 14`，此时 `14 < 9` 不成立，循环结束，最终 `n = 14`，是偶数。

## 第6题

如果 a 为 int 类型的变量，则表达式 `(a >= 5 && a <= 10)` 与 `(5 <= a <= 10)` 的值总是相同的。

答案：错误

解析：`(a >= 5 && a <= 10)` 表示 `a` 同时不小于 `5` 且不大于 `10`。但 `(5 <= a <= 10)` 在 C++ 中会先算 `5 <= a`，得到 `0` 或 `1`，再判断这个结果是否 `<= 10`，因此总为真。两者并不总是相同，所以题干错误。

## 第7题

下面 C++ 代码执行后的输出为 10。

```cpp
int cnt = 0;
for (int i = 1; i < 10; i++) {
    cnt += 1;
    i += 1;
}
cout << cnt;
```

答案：错误

解析：每次循环体内先 `cnt += 1`，再执行 `i += 1`，循环末尾还会执行 `i++`，所以 `i` 每轮实际增加 `2`。`i` 依次为 `1, 3, 5, 7, 9` 时进入循环，共执行 `5` 次，输出 `5`，不是 `10`。

## 第8题

执行以下 C++ 代码后的输出为 0。

```cpp
int rst = 0;
for (int i = -100; i < 100; i += 2)
    rst += i;
cout << rst;

```

答案：错误

解析：循环变量依次取 `-100, -98, ..., 98`，共有 `100` 个偶数。它们组成等差数列，和为 `(-100 + 98) * 100 / 2 = -100`，因此最终输出 `-100`，不是 `0`。

## 第9题

执行以下 C++ 代码后的输出为 30。

```cpp
int rst = 0;
for (int i = 0; i < 10; i += 2)
    rst += i;
cout << rst;
```

答案：错误

解析：循环中 `i` 的取值是 `0, 2, 4, 6, 8`，累加和为 `0 + 2 + 4 + 6 + 8 = 20`。程序输出的是 `20`，不是 `30`，所以题干错误。

## 第10题

C++ 是一种高级程序设计语言。

答案：正确

解析：高级程序设计语言相对于机器语言、汇编语言更接近人的表达方式，能用变量、语句、函数等结构描述程序逻辑。C++ 正是这类语言，虽然它需要编译后运行，但仍属于高级程序设计语言，所以题干正确。

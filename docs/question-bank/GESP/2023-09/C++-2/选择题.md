## 第15题

某班级人数不知，连续输入成绩直到输入负数停止，输入结束后求出平均成绩。在以下 C++ 代码横线处应填入是（ ）。

```cpp
double totalScore = 0; // 总分
int studCount = 0; // 总人数
while (________) { // 此处填写代码
    cin >> score;
    if (score < 0)
        break;
    totalScore += score;
    studCount += 1;
}
cout << "平均分=" << totalScore / studCount;
```

A. true
B. false
C. True
D. False

答案：A

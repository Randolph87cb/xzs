## 第1题

我国第一台大型通用电子计算机使用的逻辑部件是（ ）。

A. 集成电路
B. 大规模集成电路
C. 晶体管
D. 电子管

答案：D

解析：我国第一台大型通用电子计算机属于早期电子计算机，主要逻辑部件是电子管。晶体管、集成电路和大规模集成电路都属于后来更先进的硬件发展阶段，所以应选 D。

## 第2题

下列流程图的输出结果是（ ）？

B. 12 5

## 第3题

如果要找出整数 a、b 中较大一个，通常要用下面哪种程序结构？（ ）。

A. 顺序结构
B. 循环结构
C. 分支结构
D. 跳转结构

答案：C

解析：要在 `a` 和 `b` 中找较大值，程序必须先判断 `a > b` 是否成立，再根据判断结果选择不同赋值或输出，这正是分支结构。顺序结构不能做条件选择，循环结构用于重复执行，跳转结构不是解决这类比较问题的通常写法。

## 第4题

以下不是 C++ 关键字的是（ ）。

A. continue
B. cout
C. break
D. goto

答案：B

解析：`continue`、`break`、`goto` 都是 C++ 语言保留的关键字，分别用于控制循环或跳转。`cout` 是标准库中的输出流对象名称，需要配合头文件和命名空间使用，不是 C++ 关键字，所以选 B。

## 第5题

C++ 表达式 `int(-123.123 / 10)` 的值是（ ）。

A. -124
B. -123
C. -13
D. -12

答案：D

解析：先计算 `-123.123 / 10 = -12.3123`。再用 `int(...)` 转为整数时，小数部分会被直接舍去，向 `0` 截断，结果是 `-12`。它不是向下取整，所以不是 `-13`，也不是只保留被除数的前几位。

## 第6题

以下 C++ 代码实现从大到小的顺序输出 $N$ 的所有因子。例如，输入 N = 18 时输出 18 9 6 3 2 1，横线处应填入（ ）。

```cpp
int N = 0;
cin >> N;
for (________) // 此处填写代码
    if (!(N % i))
        cout << i << ' ';
```

A. `; ;`
B. `int i = 1; i < N; i++`
C. `int i = N; i > 0; i--`
D. `int i = N; i > 1; i--`

答案：C

解析：要从大到小输出所有因子，循环变量应从 `N` 开始递减到 `1`，每次用 `N % i == 0` 判断 `i` 是否为因子。C 选项 `int i = N; i > 0; i--` 正好覆盖 `N, N-1, ..., 1`。B 是从小到大且漏掉 `N`，D 漏掉因子 `1`，A 不能提供合适的 `i` 变化。

## 第7题

如下图所示，输出 $N$ 行 $N$ 列的矩阵，对角线为 $1$，横线处应填入（ ）

![](https://cdn.luogu.com.cn/upload/image_hosting/oe36n8kg.png)

```cpp
int N = 0;
cout << "请输入行列数量:";
cin >> N;
for (int i = 1; i < N + 1; i++) {
    for (int j = 1; j < N + 1; j++)
        if (________) // 此处填写代码
            cout << 1 << " ";
        else
            cout << 0 << " ";
    cout << endl;
}
```

A. `i = j`
B. `j != j`
C. `i >= j`
D. `i == j`

答案：D

解析：矩阵主对角线上的位置满足“行号等于列号”，也就是 `i == j` 时输出 `1`。A 的 `i = j` 是赋值，不是比较；B 的 `j != j` 永远为假；C 的 `i >= j` 会把主对角线及其一侧都输出为 `1`，不是只有对角线。

## 第8题

下面 C++ 代码用于判断 $N$ 是否为质数（素数），约定输入 $N$ 为大于等于 $2$ 的正整数，请在横线处填入合适的代码（ ）。
```cpp
int N = 0, i = 0;
cout << "请输入一个大于等于2的正整数:";
cin >> N;

for (i = 2; i < N; i++)
    if (N % i == 0) {
        cout << "非质数";
        __________; // 此处填写代码
    }
if (i == N)
    cout << "是质数";
````

A. break
B. continue
C. exit
D. return

答案：A

解析：一旦找到某个 `i` 能整除 `N`，就已经可以确定 `N` 不是质数，应立即跳出循环。使用 `break` 后，循环结束时 `i` 仍小于 `N`，后面的 `if (i == N)` 不会输出“是质数”。`continue` 会继续检查后面的数，可能最后仍输出“是质数”；`exit` 和 `return` 都不符合这里用来结束当前循环的目的。

## 第9题

下面 C++ 代码执行后的输出是（ ）。
```cpp
int N = 9;
for (int i = 2; i < N; i++)
    if (N % i)
        cout << "1#";
cout << "0" << endl;
```

A. 1#0
B. 1#
C. 1#1#1#1#1#1
D. 1#1#1#1#1#1#0

答案：D

解析：循环中 `i` 从 `2` 到 `8`。当 `N % i` 非零时输出 `1#`，对 `N = 9` 来说，`i = 2, 4, 5, 6, 7, 8` 都不能整除 `9`，共输出 6 次 `1#`；`i = 3` 能整除，不输出。循环结束后还会输出 `0`，所以结果是 `1#1#1#1#1#1#0`。

## 第10题

下面 C++ 代码执行后的输出是（ ）。

```cpp
int cnt = 0;
for (int i = 1; i < 9; i++)
    for (int j = 1; j < i; j += 2)
        cnt += 1;
cout << cnt;
```

A. 16
B. 28
C. 35
D. 36

答案：A

解析：外层 `i` 取 `1` 到 `8`。内层 `j` 从 `1` 开始，每次加 `2`，且要求 `j < i`，各轮执行次数分别为 `0, 1, 1, 2, 2, 3, 3, 4`。这些次数相加得到 `16`，所以 `cnt` 最终为 `16`。

## 第11题

下面 C++ 代码执行后的输出是（ ）。
```cpp
int cnt = 0;
for (int i = 1; i < 13; i += 3)
    for (int j = 1; j < i; j += 2)
        if (i * j % 2 == 0)
            break;
        else
            cnt += 1;
cout << cnt;
```

A. 1
B. 3
C. 15
D. 没有输出

答案：B

解析：外层 `i` 依次为 `1, 4, 7, 10`。`i = 1` 时内层不执行；`i = 4` 和 `i = 10` 时，第一次 `j = 1` 就有 `i * j` 为偶数，直接 `break`，不增加 `cnt`；只有 `i = 7` 时，`j = 1, 3, 5` 都让乘积为奇数，`cnt` 增加 3 次，最终输出 `3`。

## 第12题

下面 C++ 代码执行后的输出是（ ）。
```cpp
int x = 1;
while (x < 100) {
    if (!(x % 3))
        cout << x << ",";
    else if (x / 10)
        break;
    x += 2;
}
cout << x;
```

A. 1
B. 3,9,11
C. 3,6,9,10
D. 1,5,7,11,13,15

答案：B

解析：`x` 初值为 `1`，每轮末尾加 `2`，所以会依次检查奇数。`x = 3` 和 `x = 9` 能被 `3` 整除，分别输出 `3,` 和 `9,`；当 `x = 11` 时不能被 `3` 整除，但 `x / 10` 为 `1`，触发 `break`，随后循环外输出当前的 `11`。因此完整输出为 `3,9,11`。

## 第13题

下面图形每一行从字母 A 开始，以 ABC 方式重复。行数为输入的整数。请在 C++ 代码段横线处填入合适代码（ ）

![](https://cdn.luogu.com.cn/upload/image_hosting/8qyfxxc6.png)

```cpp
int N = 0;
cout << "请输入行列数量:";
cin >> N;
for (int i = 1; i < N + 1; i++) {
    for (int j = 0; j < i; j++)
        cout << _________; // 此处填写代码
    cout << endl;
}
```

A. `'A' + j / 3`
B. `(char)('A' + j / 3)`
C. `'A' + j % 3`
D. `(char)('A' + j % 3)`

答案：D

解析：每行都要按 `A, B, C` 循环输出，关键是让 `j` 对 `3` 取余，得到 `0, 1, 2, 0, 1, 2...`，再加到字符 `'A'` 上。表达式还需要转成 `char`，否则 `cout` 可能输出字符编码数值。`j / 3` 会连续多个相同字母，不能形成 `ABC` 重复。

## 第14题

输入行数，约定 $1\leq \text{lineCount} \leq 9$，输出以下图形。应在 C++ 代码横线处填入（ ）。

![](https://cdn.luogu.com.cn/upload/image_hosting/getl70oj.png)

```cpp
int lineCount = 0;
cout << "请输入行数量:";
cin >> lineCount;
for (int i = 0; i < lineCount; i++) {
    for (int j = 0; j < _________; j++) // 此处填写代码
        cout << ' ';
    for (int j = 1; j < i + 1; j++)
        cout << j << " ";
    for (int j = i + 1; j > 0; j--)
        cout << j << " ";
    cout << endl;
}
```

A. `(lineCount - i - 1) * 2`
B. `(lineCount - i) * 2`
C. `lineCount - i - 1`
D. `lineCount - i`

答案：A

解析：第 `i` 行左侧需要留出的空格会随着 `i` 增大而减少，最后一行 `i = lineCount - 1` 时不需要前导空格，所以基本数量是 `lineCount - i - 1`。由于后面每个数字后还输出一个空格，图形按两个字符宽度对齐，因此前导空格数要乘以 `2`，应填 `(lineCount - i - 1) * 2`。

## 第15题

某班级人数不知，连续输入成绩直到输入负数停止，输入结束后求出平均成绩。在以下 C++ 代码横线处应填入是（ ）。

```cpp
double totalScore = 0; // 总分
int studCount = 0; // 总人数
double score = 0; // 当前输入成绩
while (________) { // 此处填写代码
    cin >> score;
    if (score < 0)
        break;
    totalScore += score;
    studCount += 1;
}
cout << "平均分=" << totalScore / studCount;
```

A. true
B. false
C. True
D. False

答案：A

解析：题目要求“不知道人数，连续输入，直到遇到负数停止”，所以循环本身应一直执行，停止条件放在循环体里的 `if (score < 0) break;`。`while (true)` 能形成这种持续读取的循环；`false` 会让循环一次也不执行，`True`、`False` 不是 C++ 的布尔字面量。

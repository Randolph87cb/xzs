## 第7题

C++ 表达式 `z = a > b ? x : y` 等同于：

```cpp
if (a > b) {
    z = x;
} else {
    z = y;
}
```

以下代码执行后，输出将会是 2。

```cpp
int main() {
    int arr[5] = {1, 2, 3, 4, 5};
    cout << arr[3 > 2 ? 1 : 2] << endl;
    return 0;
}
```

【知识点：数组与字符数组】

答案：正确

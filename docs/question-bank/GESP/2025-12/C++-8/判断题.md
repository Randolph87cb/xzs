## 第1题

若一项任务可用两种互斥方案完成：方案 A 有 $ m $ 种做法，方案 B 有 $ n $ 种做法，则总做法数为 $ m + n $。

答案：正确

解析：这是加法原理的典型场景。任务可以由方案 A 或方案 B 完成，并且两类方案互斥，说明同一种完成方式不会同时被计入 A 和 B。于是总做法数就是两类方案数相加，为 $m+n$。如果两类方案不互斥，则还要减去重复计数部分；题干已经给出“互斥”，所以判断正确。



## 第2题

在 C++ 语言中，引用一旦被初始化，就不能再改为引用另一个变量。

答案：正确

解析：C++ 引用是被引用对象的别名，必须在定义时完成初始化，例如 `int &r = a;`。初始化以后，`r` 一直绑定到 `a`，不能再改为绑定另一个变量。之后写 `r = b;` 的含义也不是“让 `r` 引用 `b`”，而是把 `b` 的值赋给 `a`，所以题干说法正确。



## 第3题

快速排序和归并排序的平均时间复杂度都是 $ O(n \log n) $，但快速排序是不稳定的排序算法，归并排序是稳定的排序算法。

答案：正确

解析：快速排序在基准值划分较均衡时，递归层数约为 $\log n$，每层总划分代价为 $O(n)$，平均时间复杂度为 $O(n\log n)$；但划分交换可能改变相等元素的相对顺序，通常不是稳定排序。归并排序每层合并总代价为 $O(n)$，层数为 $\log n$，时间复杂度也是 $O(n\log n)$；合并两个有序段时若相等元素按原相对顺序取出，可以保持稳定。因此题干判断正确。



## 第4题

使用 `math.h` 或 `cmath` 头文件中的函数，表达式 `sqrt(4)` 的结果类型为 `double`。

答案：正确

解析：`sqrt` 用来计算平方根，C/C++ 数学库中的常用返回类型是浮点类型。对整型实参 `4` 调用 `sqrt(4)` 时，实参会匹配到返回 `double` 的重载或转换后参与计算，表达式结果为 `double` 类型的 $2.0$，不是整型 `int` 的 $2$。所以题干说法正确。



## 第5题

在杨辉三角形中，第 $ n $ 行（从 0 开始计数，即第 $ n $ 行有 $ n + 1 $ 个数）的所有数字之和等于 $ 2^n $。

答案：正确

解析：杨辉三角第 $n$ 行的各项正是二项式系数 $\binom{n}{0},\binom{n}{1},\ldots,\binom{n}{n}$。把二项式定理中的 $(1+x)^n$ 令 $x=1$，得到 $\sum_{k=0}^{n}\binom{n}{k}=(1+1)^n=2^n$。题干明确从 $0$ 开始计数，因此第 $n$ 行的和为 $2^n$，判断正确。



## 第6题

使用二叉堆优化的 Dijkstra 最短路算法，在某些特殊情况下时间复杂度不如朴素实现的 $ O(V^2) $。

答案：正确

解析：二叉堆优化 Dijkstra 的常见复杂度为 $O((V+E)\log V)$，适合稀疏图；朴素邻接矩阵实现每轮线性找最近点，复杂度为 $O(V^2)$。在稠密图中 $E$ 可能接近 $V^2$，此时堆优化版本会达到 $O(V^2\log V)$，渐近上反而不如 $O(V^2)$ 的朴素实现。因此“某些特殊情况下不如朴素实现”是正确的。



## 第7题

$ n $ 个不同元素依次入栈的出栈序列数与将 $ n $ 个不同元素划分成若干非空子集的方案数相等。

答案：错误

解析：固定入栈顺序的 $n$ 个不同元素，其合法出栈序列数是第 $n$ 个 Catalan 数 $C_n=\frac{1}{n+1}\binom{2n}{n}$。把 $n$ 个不同元素划分成若干非空子集的方案数是 Bell 数 $B_n$。这两个数列含义不同，也不恒相等，例如 $n=4$ 时，出栈序列数为 $14$，集合划分数为 $15$。所以题干把两类计数混为一谈，判断错误。



## 第8题

快速排序在最坏情况下的时间复杂度为 $ O(n \log n) $，可以通过随机化选择基准值（pivot）的方法完全避免退化。

答案：错误

解析：快速排序最坏情况不是 $O(n\log n)$，而是 $O(n^2)$。例如每次选到当前区间的最小值或最大值作为基准，划分会退化成规模 $n-1$ 和 $0$ 的子问题，比较次数累加为 $1+2+\cdots+(n-1)$ 量级。随机化选基准可以显著降低持续退化的概率，但不能从理论上“完全避免”所有坏情况，因此题干判断错误。



## 第9题

在 C++ 语言中，一个类可以拥有多个构造函数，也可以拥有多个析构函数。

答案：错误

解析：一个类可以通过函数重载拥有多个构造函数，例如默认构造、带参数构造、拷贝构造等，它们的参数列表可以不同。析构函数则不同：析构函数名固定为 `~类名`，没有返回类型，也不能带参数，因此不能重载，一个类至多有一个析构函数。题干把构造函数和析构函数都说成可以有多个，所以判断错误。



## 第10题

求两个序列的最长公共子序列（LCS）时，使用滚动数组优化空间后，仍然可以还原出具体的 LCS 序列。

答案：错误

解析：LCS 的二维 DP 表不仅给出长度，还保留了从哪个状态转移而来的信息，才能从 `dp[n][m]` 往回走还原具体序列。滚动数组优化后通常只保留当前行和上一行，旧行信息会被覆盖，可以求出长度，但缺少完整回溯路径。若要还原具体 LCS，需要额外保存决策、保留完整 DP 表，或用更复杂的分治还原方法；不能仅靠普通滚动数组直接还原。因此题干判断错误。

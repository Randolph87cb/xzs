## 第1题

以下代码中，构造函数被调用的次数是 1 次。

```cpp
class Test {
public:
    Test() { cout << "T "; }
};

int main() {
    Test a;
    Test b = a;
}
```

答案：错误

解析：`Test a;` 会调用一次默认构造函数，输出一次 `T`。`Test b = a;` 是用已有对象初始化新对象，调用的是拷贝构造函数；即使类中没有显式写出拷贝构造函数，编译器也会生成一个默认拷贝构造函数，它同样属于构造函数调用。实际构造了两个对象，构造函数调用次数不是 $1$ 次，所以判断为错误。



## 第2题

面向对象编程中，封装是指将数据和操作数据的方法绑定在一起，并对外隐藏实现细节。

答案：正确

解析：封装强调把对象的数据和操作这些数据的成员函数组织在同一个类中，并通过访问控制隐藏不希望外部直接依赖的实现细节。例如把成员变量设为 `private`，只提供必要的 `public` 接口，外部就不需要知道内部如何存储和维护数据。因此题干对封装的描述是正确的。



## 第3题

以下代码能够正确统计二叉树中叶子结点的数量。

```cpp
int countLeaf(TreeNode* root) {
    if (!root) return 0;
    if (!root->left && !root->right) return 1;
    return countLeaf(root->left) + countLeaf(root->right);
}
```

答案：正确

解析：函数先处理空树，空指针没有叶子，返回 $0$。如果当前结点左右孩子都为空，它本身就是叶子，返回 $1$。否则叶子只可能出现在左子树或右子树中，递归统计两边再相加。这个递归划分覆盖了所有结点，且每个叶子只会在自身位置被计数一次，所以能正确统计叶子结点数量。



## 第4题

广度优先遍历二叉树可用栈来实现。

答案：错误

解析：二叉树的广度优先遍历要求按层从上到下、同层按先后顺序访问，典型数据结构是队列：先发现的结点先访问。栈是后进先出，若把孩子压入栈，后发现的结点会先被弹出，访问顺序会转向深度优先。普通栈不能直接保证 BFS 所需的先进先出顺序，所以题干说“可用栈来实现”是错误的。



## 第5题

函数调用管理可用栈来管理。

答案：正确

解析：函数调用具有后调用先返回的特点：调用函数时要保存返回地址、参数、局部变量等信息，函数结束时再恢复到上一层调用现场。递归调用中这种关系最直观，最后进入的递归层最先结束。这个管理方式正符合栈的后进先出规则，因此函数调用管理可以用栈来管理。



## 第6题

在二叉排序树（BST）中，若某结点的左子树为空，则该结点一定是整棵树中的最小值结点。

答案：错误

解析：BST 中整棵树的最小值一定在“从根开始不断向左走”能到达的最左下结点。某个结点左子树为空，只能说明在它自己的子树中没有比它更小的结点，不能说明整棵树中没有更小的祖先或祖先的左侧结点。反例：根为 $5$，右孩子为 $8$，结点 $8$ 的左子树为空，但整棵树最小值是 $5$，不是 $8$。所以判断为错误。



## 第7题

下面的函数能正确判断一棵树是不是二叉排序树（左边的数字要比当前数字小，右边的数字要比当前数字大）。

```cpp
bool isBST(TreeNode* root, int minVal, int maxVal) {
    if (!root) return true;
    if (root->val <= minVal || root->val >= maxVal)
        return false;
    return isBST(root->left, minVal, root->val) &&
           isBST(root->right, root->val, maxVal);
}
```

答案：正确

解析：判断 BST 不能只比较一个结点和它的直接孩子，还要保证整棵左子树都小于当前结点、整棵右子树都大于当前结点。这个函数用 `minVal` 和 `maxVal` 记录当前结点允许取值的开区间，当前值不在范围内就返回 `false`；递归左子树时把上界收紧为 `root->val`，递归右子树时把下界提高为 `root->val`。在初始边界足够覆盖所有可能结点值的前提下，它能正确判断严格 BST。



## 第8题

格雷编码相邻两个编码之间必须有多位不同，以避免数据传输错误。

答案：错误

解析：格雷编码的核心性质恰好是相邻两个编码只有一位不同。这样从一个状态切换到相邻状态时，只需要改变一个二进制位，可以降低多位同时变化导致的中间状态误读风险。题干说“必须有多位不同”与格雷编码定义相反，所以判断为错误。



## 第9题

小杨在玩一个闯关游戏，从第 $1$ 关走到第 $4$ 关。每一关的体力消耗如下（下标表示关卡编号）：`cost = [0, 3, 5, 2, 4]`。其中 `cost[i]` 表示到达第 `i` 关需要消耗的体力，`cost[0]=0` 表示在开始状态，体力消耗为 $0$。小杨每次可以从当前关卡前进 $1$ 步或 $2$ 步。按照上述规则，从第 $1$ 关到第 $4$ 关所需消耗的最小体力为 $7$。

答案：错误

解析：按题干“从第 $1$ 关走到第 $4$ 关”理解，起点已经在第 $1$ 关，之后每次前进 $1$ 或 $2$ 步并支付到达目标关卡的消耗。可选路径中，$1\to3\to4$ 的消耗为 `cost[3] + cost[4] = 2 + 4 = 6`，比 $1\to2\to4$ 的 $5 + 4 = 9$ 更小。如果把第 $1$ 关的到达消耗也算入，最小值也会变为 $3 + 2 + 4 = 9$，仍不是 $7$。因此题干给出的 $7$ 不成立。



## 第10题

假定只有一个根节点的树的深度为 $1$，则一棵有 $n$ 个节点的完全二叉树，则树的深度为 $\lfloor \log_2(n) \rfloor + 1$。

答案：正确

解析：完全二叉树除最后一层外都被填满，最后一层从左到右连续排列。若树的深度为 $d$，它的结点数满足 $2^{d-1} \le n \le 2^d-1$。对这个不等式取以 $2$ 为底的对数，可得 $d-1 \le \log_2 n < d$，因此 $d=\lfloor \log_2 n \rfloor+1$。根节点单独成树时 $n=1$，公式也给出深度 $1$。

## 第1题

近日，空中客车公司表示，约 $6000$ 架空客 A320 系列飞机需要紧急更换一种易受太阳辐射影响的飞行控制软件。空客表示，在对一起飞行事故分析后的结果显示，强烈的太阳辐射可能会损坏飞行控制系统所需的关键数据，导致判断失误，进而引发飞行异常。那这里的飞行控制系统执行判断的部件最可能是下面的（ ）。

A. 辐射传感器
B. 处理器
C. 内存单元
D. 输出设备

答案：B

解析：飞行控制系统需要根据传感器数据和软件逻辑进行“判断”，真正执行指令、运算和条件判断的是处理器。辐射传感器主要负责感知环境，内存单元主要负责存储数据，输出设备负责把结果输出或执行显示，都不是执行判断的核心部件。因此选 B。



## 第2题

小明最近为了备考 GESP，开始看 B 站上关于网络知识的视频。其中提到计算机网络系统有不同的划分标准，那他平时上学所在的教学楼内的网络是一个（ ）。

A. PAN
B. LAN
C. MAN
D. WAN

答案：B

解析：教学楼内的网络覆盖范围通常只在一栋楼或一个校园局部区域内，属于局域网，即 LAN。PAN 是个人区域网，范围更小；MAN 是城域网，覆盖城市范围；WAN 是广域网，覆盖范围更大。因此选 B。



## 第3题

下面有关 C++ 变量的说法，正确的是（ ）。

A. 不可以用 for 作为变量名，因为 for 是 C++ 的关键字（保留字）。
B. `_tnt` 不可以是变量名，因为变量名的第一个字符必须是英文字母。
C. `_tnt_` 不可以是变量名，因为最后一个字符容易与减号混淆。
D. 可以用 printf 作为变量名，因为 printf 是关键字，但这不是好习惯，因为 printf 有约定的功能与含义。

答案：A

解析：C++ 的变量名不能使用关键字，`for` 是循环语句的关键字，所以不能作为变量名，A 正确。变量名可以由字母、数字和下划线组成，但不能以数字开头，因此 `_tnt` 和 `_tnt_` 都可以作为变量名，B、C 错误。`printf` 不是 C++ 关键字，它是常见库函数名；即使某些作用域下技术上可能被重新声明，也不应把它说成关键字，D 错误。因此选 A。



## 第4题

一个小数是 $0.123123123\ldots$ 无限循环，其小数点后 $1$ 位是 $1$，后 $2$ 位是 $2$，依此类推，求第 $N$ 位的值。横线处应填入的代码是（ ）。

```cpp
int N;
cin >> N;
remainder = ___________;

if (remainder == 0)
    cout << 1;
else if (remainder == 1)
    cout << 2;
else
    cout << 3;
```

A. `N % 3`
B. `(N - 1) % 3`
C. `N / 3`
D. `(N - 1) / 3`

答案：B

解析：小数部分按 `1, 2, 3` 每 3 位循环一次。第 1、4、7 位都是 1，第 2、5、8 位都是 2，第 3、6、9 位都是 3。代码中 `remainder == 0` 时输出 1，`remainder == 1` 时输出 2，否则输出 3，所以要让第 1 位对应余数 0，应使用 `(N - 1) % 3`。因此选 B。



## 第5题

某同学执行 C++ 代码时 `printf("%g\n", (3 + 3.1415926535));` 输出 $6.14159$，其原因最可能是（ ）。

A. C++ 中整数和浮点数相加时，整数会被转换为浮点数，而某些十进制小数无法精确表示为二进制小数，从而产生某些舍入误差。
B. C++ 的 `printf` 函数在输出浮点数时根据格式有默认小数点位数，因此输出了较少的位数。
C. $3.1415926535$ 是一个无限循环小数，在计算机中无法精确表示。
D. 由于计算机 CPU 的运算错误导致。

答案：B

解析：表达式 `3 + 3.1415926535` 的数学结果约为 `6.1415926535`，而 `%g` 在没有指定精度时默认使用 6 位有效数字输出，所以显示为 `6.14159`。这里主要原因是输出格式控制了显示位数，不是 CPU 运算错误；`3.1415926535` 也不是无限循环小数。A 提到的二进制浮点误差确实可能存在，但不能解释“只输出 6 位有效数字”这个现象的主要原因。因此选 B。



## 第6题

某单位工号的编码规则：编码总长 5 位，均为数字。前 4 位依次整除以 3 其值累加之和除以 10 的余数为第 5 位数字。如某工号为 76587，前 4 位分别整除以 3 后，商分别为 2、2、1、2，其累加之和为 7，除以 10 的余数为 7，故第 5 位为 7。下面代码依次输入前 4 位后，两个横线处分别应填的是（ ）。

```cpp
int rst = 0, N;
for (int i = 0; i < 4; i++) {
    cin >> N;
    rst += __________; // L1
}
cout << __________; // L2
```

A.

```cpp
N % 3
rst / 10
```

B.

```cpp
N % 3
rst % 10
```

C.

```cpp
N / 3
rst / 10
```

D.

```cpp
N / 3
rst % 10
```

答案：D

解析：规则要求前 4 位数字分别“整除以 3”，也就是取整数商，应写 `N / 3`，不是取余数 `N % 3`。4 个商相加后，再“除以 10 的余数”为第 5 位，所以输出应写 `rst % 10`。两个空分别是 `N / 3` 和 `rst % 10`，因此选 D。



## 第7题

下面的 C++ 代码执行后的输出是（ ）。

```cpp
for (int i = -2; i < 2; i++)
    if (i % 2)
        printf("%d#", i);
```

A. -1#1#
B. -1#0#1#
C. -2#-1#1#
D. -2#-1#1#2#

答案：A

解析：循环变量 `i` 依次取 `-2, -1, 0, 1`，因为条件是 `i < 2`，不包含 2。`if (i % 2)` 判断的是取余结果是否非 0：`-2 % 2` 和 `0 % 2` 都是 0，不输出；`-1 % 2` 和 `1 % 2` 都非 0，会输出。因此输出为 `-1#1#`，选 A。



## 第8题

下面的 C++ 代码执行后其输出是（ ）。

```cpp
int cnt = 0, N;
for (int i = 1; i < 10; i += 2)
    for (int j = 0; j < i; j++)
        cnt += 1;
cout << cnt;
```

A. $100$
B. $55$
C. $45$
D. $25$

答案：D

解析：外层循环 `i` 从 1 开始，每次加 2，且小于 10，所以 `i` 依次为 `1, 3, 5, 7, 9`。内层循环 `j` 从 0 到 `i - 1`，每次执行 `i` 次，因此 `cnt` 增加的总次数是 `1 + 3 + 5 + 7 + 9 = 25`。输出为 25，选 D。



## 第9题

下面 C++ 代码执行后其输出是（ ）。

```cpp
int i, j;
for (i = 1; i < 12; i++) {
    if (i % 2 == 0)
        continue;
    for (j = 0; j < i; j++)
        if (i * j % 2 == 0)
            break;
    if (j >= i)
        cout << i * j << " ";
}
if (i >= 12)
    cout << (i * j);
```

A. $0\ 0$
B. $11$
C. $0$
D. $0\ 11$

答案：C

解析：外层循环中偶数 `i` 会被 `continue` 跳过，只处理奇数 `1, 3, 5, 7, 9, 11`。对任意这些奇数，内层循环从 `j = 0` 开始检查，`i * 0 % 2 == 0` 成立，立即 `break`，所以内层循环结束时 `j` 仍为 0，`j >= i` 不成立，不会在循环内输出。外层循环结束时 `i` 已变为 12，`j` 保持最后一次内层循环的值 0，因此最后输出 `i * j = 12 * 0 = 0`。选 C。



## 第10题

与下面 C++ 输出效果不一致的代码是（ ）。

```cpp
int i;
for (i = 0; i < 10; i++)
    cout << i;
```

A.

```cpp
int i = 0;
while (i < 10) {
    cout << i;
    i += 1;
}
```

B.

```cpp
int i = 0;
while (i < 10) {
    i += 1;
    cout << i;
}
```

C.

```cpp
int i = 0;
while (true) {
    cout << i;
    i += 1;
    if (i >= 10)
        break;
}
```

D.

```cpp
int i = 0;
while (true) {
    if (i >= 10)
        break;
    cout << i;
    i += 1;
}
```

答案：B

解析：原代码输出 `0` 到 `9`，结果是 `0123456789`。A 先输出再加 1，输出相同；C 也是先输出当前 `i`，再加 1，达到 10 后退出，输出相同；D 先判断是否到 10，再输出当前 `i`，也输出 `0` 到 `9`。B 在循环体内先 `i += 1` 再输出，所以输出的是 `1` 到 `10`，与原代码不一致。因此选 B。



## 第11题

下面 C++ 代码执行后输出是（ ）。

```cpp
int num = 0;
while (num <= 5) {
    num += 1;
    if (num % 3)
        continue;
    printf("%d#", num);
}
if (num > 5)
    printf("%d", num);
```

A. 3#6#
B. 3#6#6
C. 1#2#3#4#5#6#
D. 1#2#3#4#5#6#6

答案：B

解析：`num` 初始为 0，进入循环后先加 1。只有当 `num % 3 == 0` 时，`if (num % 3)` 为假，不执行 `continue`，才会打印。循环中 `num` 依次变为 1、2、3、4、5、6，其中 3 和 6 会打印为 `3#6#`；当 `num` 变为 6 后下一次判断 `num <= 5` 不成立，退出循环。退出后 `num > 5` 成立，又打印 `6`，所以总输出是 `3#6#6`。选 B。



## 第12题

下面 C++ 代码执行后，其输出是（ ）。

```cpp
int cnt = 0;
for (int i = 0; i < 5; i++)
    for (int j = i; j < 4; j++)
        cnt += 1;
cout << cnt;
```

A. $9$
B. $10$
C. $14$
D. $20$

答案：B

解析：外层 `i` 取 `0, 1, 2, 3, 4`。内层循环条件是 `j < 4`，且 `j` 从 `i` 开始，所以各轮次数分别为：`i=0` 时 4 次，`i=1` 时 3 次，`i=2` 时 2 次，`i=3` 时 1 次，`i=4` 时 0 次。总次数为 `4 + 3 + 2 + 1 + 0 = 10`，因此输出 10，选 B。



## 第13题

漂亮数的定义是：如果 $N$ 能被 $M$ 整除，或者某位是 $M$，或者 $N$ 的每位数之和能被 $M$ 整除，则说 $N$ 是 $M$ 的漂亮数。如果三个条件都满足，则是完整漂亮数。123 是 3 的完整漂亮数，因为 123 能被 3 整除，也含有 3，其每位数之和为 6 也能被 3 整除。下面的代码用于判断 $N$ 是否为 $M$ 的完整漂亮数并输出。相关说法正确的是（ ）。

```cpp
int N, M, Flag, Sum, num;
cout << "请输入N, 不等于0的正整数: ";
cin >> N;
cout << "请输入M: M必须大于1小于9: ";
cin >> M;

Sum = 0; // 记录各位数之和
Flag = 0; // 假设记录N不含有M

while (N != 0) {
    num = N % 10;
    Sum += num;
    if (num == M)
        Flag = 1;
    N /= 10;
}

if ((N % M == 0) && (Flag == 1) && (Sum % M == 0)) // L1
    printf("%d 是 %d 的完整漂亮数", N, M);
else
    printf("%d 不是 %d 的完整漂亮数", N, M);
```

A. 代码能完成题目设定目标。
B. 在 while 循环中 $N$ 最终将变成 0，因此 L1 行代码中 `N % M` 将总是满足条件，可以在 while 前增加一行代码 `int old_num = N;`，并将 L1 开始这 4 行代码中的 $N$ 都改为 old_num 就可以做出正确的判定。
C. while 循环中 if 判断语句可以增加 else 句，其内容为 `Flag = 0`。
D. 如果先后输入 0 和 3，则肯定会输出 0 是 3 的完整漂亮数。

答案：B

解析：这段代码在 `while (N != 0)` 中不断执行 `N /= 10`，循环结束后 `N` 一定变成 0，原来的数已经丢失。因此 L1 中的 `N % M == 0` 实际检查的是 `0 % M == 0`，总是成立，输出时也只能输出 0，不能正确表示原数。若在循环前保存 `int old_num = N;`，并在 L1 的判断和后续输出中使用 `old_num`，就能同时保留原数、检查原数能否被 `M` 整除并正确输出。C 错在一旦后面某位不等于 `M` 就会把已经找到的标记清掉；D 中输入 0 和 3 时 `Flag` 仍为 0，不会输出完整漂亮数。因此选 B。



## 第14题

阅读下面的 C++ 代码。执行后如输入 $5$，其输出的字符图形是（ ）。

```cpp
int n, i, j, k;
cin >> n;
for (i = 0; i < n; i++) {
    for (j = 0; j < n - i - 1; j++)
        cout << " ";
    for (k = 0; k < 2 * i + 1; k++)
        cout << "*";
    cout << endl;
}
```

A.

```plain
 *****
  ****
   ***
    **
     *
```

B.

```plain
     *
    ***
   *****
  *******
 *********
```

C.

```plain
 *
 **
 ***
 ****
 *****
```

D.

```plain
     *
    **
   ***
  ****
 *****
```

答案：B

解析：输入 `n = 5` 时，外层 `i` 取 `0` 到 `4`。每一行先输出 `n - i - 1` 个空格，再输出 `2 * i + 1` 个星号，所以星号个数依次是 `1, 3, 5, 7, 9`，前导空格依次是 `4, 3, 2, 1, 0`。这正是居中向下增宽的三角形，对应选项 B。



## 第15题

某学校举办“十佳歌手大奖赛”，经过选拔最终参赛选手有 25 人，评委 10 人，最终计分规则为去掉一个最高分去掉一个最低分作为该参赛选手的最终得分，并输出该得分。如果单个评委可给满分为 10 分，则相关说法正确的是（ ）。

```cpp
float total_score, max_score, min_score, now_score;
for (int i = 0; i < 25; i++) {
    max_score = 0; // 记录最高分
    min_score = 10; // 记录最低分
    total_score = 0; // 记录总分

    for (int j = 0; j < 10; j++) {
        cin >> now_score; // 录入评委打分
        max_score = max(max_score, now_score); // L1
        min_score = min(min_score, now_score); // L2
        total_score += now_score;
    }

    cout << (total_score - max_score - min_score);
}
```

A. 程序总体逻辑错误。因为要去掉最高分和最低分，需要排序，而程序没有相关代码。
B. 内层循环和外层循环之间的三行代码也就是 `max_score = 0` 开始的三行代码应该移动到外层循环外。
C. L1 和 L2 标记的两行代码可以分别改为简单的 if 语句或 ? : 语句。
D. `total_score += now_score` 不可以更改为 `total_score = total_score + now_score`。

答案：C

解析：程序每名选手读入 10 个分数，同时维护当前最高分、最低分和总分，最后用总分减去最高分和最低分即可，不一定需要排序，所以 A 错。`max_score`、`min_score`、`total_score` 必须对每名选手重新初始化，不能移到外层循环外，B 错。`max(max_score, now_score)` 可以等价写成 `if (now_score > max_score) max_score = now_score;`，`min` 也可以类似改写，或用条件表达式实现，C 正确。`total_score += now_score` 与 `total_score = total_score + now_score` 等价，D 错。因此选 C。

## 第6题

下面的 C++ 代码能用于判断输入的正整数是否为对称数。所谓对称数是指从左到右和从右到左读该数，其值相同，$121$ 或 $414$ 等是对称数，而 $123$ 不是对称数。（ ）

```cpp
int n, new_number;
cout << "请输入正整数:";
cin >> n;

new_number = 0;
while (n != 0){
    new_number = new_number * 10 + n % 10;
    n /= 10;
}

if (n == new_number)
    cout << "对称数";
else
    cout << "非对称数";
```

答案：错误
